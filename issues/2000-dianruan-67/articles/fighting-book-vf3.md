# 格斗天书·霸王塾——VF3(三)

> 格斗天书·霸王塾 / PDF p.67 / 刊页41

●天津钢管公司天车 高读者，你对野读者的意见十分中肯，确实让你给说着了。而 + r1 + 你是头一份。

## VF3(三) 广州 野

### 攻防十分严密 系统非常之多

天语：VF进化至TB版后，愈发感觉其博大精深。广州 + r1 + 读者通过自己的街机实践总结出一些连续技，虽然尚缺杰夫大里、贤布、沃尔夫等人，但精神的确可嘉。南京航空学院的王勤亦从美国网站上找了一些 + r1 + 连技，并结合自己的DC实践整了过来。限定型连技主要是壁限定和高低差限定，配合着角色体重的级别，必须做到见机心必办，否则就连不上，可见其要求之高。不过还是要介绍的。这里先补充一点小技巧和小诀窍。建议手柄设置，

L：闪避，R：防御，X：拳，Y：视点，A：胸，B：脚

除隐云去斗双虎拿极难发出外，其他人物的招式用此种设置皆可顺利发出。如陈洛的↓KK且KK＋G。DC版的手柄只能设定最基础的G、P、K、E，而无法对它们进行组合，如K＋G、P＋G等不能设定在一个键上。故提一点。

---

### ■层岚

相扑手虽形体巨大，但绝对不容忽视，他的突张手（PPPPPP）速度极快，易令对手连续挨打。而中段投般当（→P），而时当后立刻前冲接各种大威力投技往往也令对手突笑不得。因此层岚虽拿投技较少，但却是最有希望15秒内KO对手的强者。上级者向，重量级。

连技：自己处于高地而对手位于低处时，双手突进（←→P）＞稍停＞诸手突张（PPPPP）＞追加。威40%，难70%，中25%。

注：此招难在双手突进打中手位于低处时，双手突进（←→P）＞稍停＞诸手突张（PPPPP）＞追加。威40%，难70%，中25%。

注：此招难在双手突进在空中不中，若按键慢易打不中，若太快，则易形成诸手突张（←→P）＞稍停＞诸手突张（PPPPPP）这招。应多练习。

### ■梅小路 葵

实在是VR3中最难修炼的角色。此人因打击技攻击力偏小，因此实战中主力武器是投技与当身技。本人的当身投经过近两个月 + r1 + 的修炼后，约已可准确判断出对手40%的主要攻击技法。但我的一位朋友发现已强到对手几乎不敢轻易向他 + r1 + 只可任他宰割的肉的地步。这也许真是天份的吧！

1.横打（←P＋K）＞各种快速打击技或冲前投技。稳20%，难中50%。命50%。

注：横打这一招是一种较为赖皮的技法，它有50%的概率可破坏对方防御。因此各种快速打击技如二连突（PP）即可命中，而大月车（↓K＋G）这一招因有准备时间，故难命中。

2.绞手取（↓起立途中P＋G）＞按碎（对手倒地时追加投↘P＋G）＞追打。

3.雷刃破（→K）对手浮空＞PP＞大月车（↓K＋G）＞按碎（↘P＋G）＞追加。威45%，难65%，中30%。

### ■影丸

速度型角色，近身战极为有利。时打（→P）及贴奸（↓K）连缩交叉使用足以令对手心烦。旋蹴（↓K＋G）堪称是一切中段足技的克星。弧延落（←P＋G）后追打手段十分丰富，有些虽很难，但还是应该多练习使用。不折不扣的初级角色。

1.侧阳（↘K）＞对手浮空＞影刃（→↑↘P）＞追加。威25%，难50%，中50%。

2.弧延落（←P＋G）＞影刃＞追加。威30%，难13%，中80%。

3.龙尾闪（←←K＋G）＞后贴奸（↓K）＞追加。威25%，难40%，中100%。

注：勿将追加按误出成浮身膝蹴（↓→K），解决方法是后贴奸后松开摇杆，再（↓K）。

4.背后是壁壘，弧延落（←P＋G）＞月丹蹴（↗K＋G）＞散弹奸脚（PPPK）＞追加。

注：笔者认为此招就是VF3中威力最大的一招了，若还有玩家可击出更大的威力，请告之。这一招还有一分歧，即月丹蹴后接↑P＞影刃＞追加。不过威力较小，约50%。

本人对VR3的研究还只是管窥蠡测，望全国VR高手指正。

广州热切盼望VR4的人

### ■陈洛

自VF3大会得到亚军后，洛的饭馆突然陷入了经营危机，这次再度参赛竟是为赚钱救饭馆。首先指出，↘（长按）P←P→P。另一人的 + r1 + 亦算好用，出招之后转过身的技巧有←←P，←←K＋G等，背向对手的即可P←P→P或↓Ko。陈洛用←↑P＋G拍过对手后，可及时↘PPP↑K，他最高难度连续技是，斜上掌（↘P）COUNTER对手＞P＋K＞PPP↑Ko。

### ■碧甫

技巧搞笑的角色，威力虽小但酒6以上各种COMBO实现。饮酒技当然首选→P＋K，命中则5拍，倒对手可以特定技可击酵醒双拳点击。老汉最大的特征是应对打边PPE↑KKG（酒6），→↙↗↘→↙↗↓↑↑（侧翻）KP↑KK，等等。

注：要求敢我逆足，即我在左（右）脚前面敢（左）脚蹴后面。否则就是顺足定，须需↑↘→P→↑↘→P→↑P的追加。

1.↓↘→P＞↓↘→/↑→←P→↑P的追加。

2.顺足正常态时，只需↑↘→P→P＞↑↘→P↑P的追加。

3.↘P＋G＞E.P＋K。

注：敢在我右前。第一段翻过对手后右单按闪避键，然后立即按下P＋G投之。即嗯……若敢在我左边，P＋G投之，则须将E配合方向↓使出。

4.↘P＋G翻至对手身后（敢在我右边）＞↓↘→P→P→←↘P。

注：第二和第三段摇杆的方向应与上述表记相反才对。
