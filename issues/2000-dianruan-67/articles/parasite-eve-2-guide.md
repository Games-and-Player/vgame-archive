## Day 1

2000年9月4日晚8:18，洛山矶市的MIST中心地下一层。

由于一时疏忽，Aya在靶场击中了平民靶，她走出屋子希望冷静一下。此时，她的同事Pierce接到了Baldwin的电话，得知在Akropolois Tower发生了异常情况，需要人员前往调查。于是Aya先到（2）处的武器库，向Jodie购买物品。然后到停车场（MIST PARKING）与Pierce交谈（这里的电话可以存进度）。前往Akropolois Tower。

2000年9月4日晚8:56，洛山矶市Akropolois Tower。

Aya先是向楼下的警员了解情况，而后乘电梯到达事件发生场所（3）处。经过广场（Square）时只见SWAT的队员像被屠杀一样尸横遍地，电话机旁又有个古怪的影子一闪即逝。这时，Balelwin把电话打到了这边了解情况（可以不接电话）。Aya作出简单汇报，又进入东侧电梯间（4）处，从幸存的SWAT Officer手中得到一把 + r1 + 。

从广场西南角下楼梯，Aya遇到了第一批怪物（用魔法可轻松胜出）。打倒它们之后，她用刚到手的钥匙进入自助餐厅（Cafeteria），见到一名女子独自坐在桌边，一言不发。Aya走上前去想帮她离开这里，不料那人竟发生急剧变异，这情景令Aya不由得回想起数年前的卡耐基大剧院。掀翻这只怪物后，Aya从它的头部取出一个零件 + r1 + ，但不知有何用途。这时，那怪物正悄悄地竖直身体，准备全力一击，幸好Rupert及时赶到，一枪将其送进地狱。

Aya接受了Rupert的建议，用广场上的电话向Balowin请求支援，准知竟遭拒绝。她返回自助餐厅，在餐桌上找到一本杂志，而Rupert已经离开。穿过东南角的门，Aya从（5）处墙上的金属盒内取得一把 + r1 + ，并用它打开（6）处左边的锁（先用前头关锁眼，再选物品），使得（7）处通向瞭台（observatory）的电梯起动。经瞭台的电梯再到圣堂（Sanctuary），正巧见Rupert与一名怪人的博弈。那个怪人一见到Aya就跳窗而逃，而屋子中央的地板上落有一把 + r1 + 。

救起Rupert，Aya为了追查那个怪人进入屋顶花园（Roof Garden）。偷听到那人与其幕后组织的连络信息并发现了他安装的炸弹，而炸弹旁边还有一张识别卡 + r1 + 。

如果希望在BOSS战中打得轻松些，可以返回（6）处，用 + r1 + 打开石边的锁，使（8）处的池水排光。那里有一具SWAT队员的尸体，可以获得榴弹发射器 + r1 + 和4发子弹 + r1 + 。

到达屋顶的机场，铁桥（bridge）是必经之路，但因为水中有怪物，所以强渡必然会受伤。Aya在桥头（9）处的机器上输入 + r1 + ，铁桥便升出水面，这样也很容易解决那些怪物。

登上楼顶的Aya调查了一圈却毫无收获，她正准备回去，突然遇到了那个神出鬼没的怪人，狭路相逢唯勇者能胜。

---

这个BOSS虽然初级却也不好对付，他的火焰刀（莫非是吐着焰师……）杀伤力很强，如果被逼到窄路的尽头，更是一刀必杀。此外他还有飞弹连发，令Aya陷入麻痹状态。我方的优势是能利用墙上的几个配电箱，等BOSS靠近时射击铁箱，电流会给予他极大伤害。魔法主要是用水属性的，特别是状态回复，一中飞弹就赶紧用。临战时边打边后退，别怕被逼到尽头，只要前面的电线都电中了BOSS，而且一直在用枪打他，到最后再电他一下（这最后的电流伤害最大），拼着放个火魔法多半就赢了。

打跑了怪人，Aya乘升降机来到直升机平台，虽然夺下了引爆器，却被Rupert一通乱揿按中了起爆钮，情急之下，二人乘直升机脱离险境。

然而这些只是整个事件的序幕，Aya回到MIST总部以后不久便领到了一项新任务：前往内华达Mojave沙漠中的小镇（DRYFIELD）调查UFO事件。看过Baldwin提供的所有资料之后，Aya与停车场的Pierce谈话便可离开。
