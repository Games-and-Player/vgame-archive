## 援护攻击与COUNTER连续技

天津红桥 李续国

由于KOF99中启用了新系统——援护攻击，以及COUNTER模式中可由必杀技CANCEL超必杀技，虽使本作增色不少，但如果要成为KOF99的高手，就必须对每个角色的援护攻击与COUNTER连续技了如指掌，下面就笔者发现的有关援护攻击的追打和COUNTER态下大威力连续技作一个简要介绍。

### 一、援护攻击的追打

★援护角色：特瑞

追打方法（全过程的步骤以小写字母顺序表示）：

选用八神时，法①（自己身处版边，对手距2个身位左右）a.援助攻击，b.八稚女，c.（特瑞将对手打飞）用八稚女或祈爪柝（↓↘→↓↘→B/D）追打。

法②跳至对手背后百合折（←B）>a>b>c

注：上述连续击威力不低，普通状态下可打掉对手约1/2血，MAX状态下威力更为恐怖。

★援护角色：罗伯特

追打方法：

①任何角色都可用大威力的连续技追打

②选用包时，a.超杀→←↙↓↘→B/D命中，b.跃起C>↓B×2，c.援助攻击d.C>→A>↓↙←↓↙←A/C。

注：MAX态下，b中可再作适当追打。

③选用乐文，a.将对手逼至版边，呼出援助，b.C>↓↘→↘↓↙←A/C，c.低踢（↓↘→↓↘→+B/D）蓄住，待对手倒地放出。

★援护角色：MAXIMAM、二阶堂红丸、安迪、东文、不知火舞、尤利、拉尔夫、克拉克、包、MARY、藤堂香澄、金家藩、蔡宝健

追打方法：以1HIT大威力技追打为最佳，推荐。

K'↓↘→↓↘→A/C，拉尔夫↙←↙↓↘→B/D，矢吹↓↘→↓↘→A/C，李香绯↓↘→↓↘→B/D

★援护角色：矢吹真吾

追打方法：任何技

★援护角色：椎拳崇

使用方法：援护角色出来后，使用任何必杀技，均可加不同程度的血，HIT数越多越好。

★援护角色：KING

使用方法：选用K'，待对手距自己2～3个身位时，呼出援护角色，a.快速使出超杀↓↘→↓↘→A/C，b.待对手从空中落下时，再用上述超杀追打。

### 二、COUNTER态下部分角色大威力连续技：

草薙京：跳D>站C>轻铁1HIT时>百八拾二式

二阶堂：对手在角落，跳D>站C>轻雷初拳>幻影飓风

矢吹真吾：跳C>站C>重花研2HIT时>凤[?]

特瑞：跳D>站C>重能量波>高轨喷泉

安迪：跳D>站C>重背水掌2HIT时>新影流星拳

不知火舞：跳D>蹲C>↘B1HIT>小夜千鸟连龙炎舞>超杀忍蜂。此技打掉对手2/3血。

罗伯特：跳C>↓B>↓A>↓蓄↑A（2HIT时）>十段脚

KING：跳D>站D>←↙↓↘→C（3HIT时）>沉默闪光

MARY：跳D>站C>←A>←蓄→B>台风。打掉对手3/5血。

## 决～天草降临

霸王塾

天津 燕尾王

小天天主持的高打乐园

侍魂。侍，武士；魂，灵魂，精神。武士的灵魂，即刀。但在下认为它有另一个意思，那就是武士蕴藏于 + r1 + 字之上的精神。在下从《真·侍魂》到《天草降临》上千次的与人对战，有悟于这个 + r1 + 字。（注：在下不是绯村剑心）

玩家分为两类：为赢而决的人，为决而决的人。

前者有三，一是对战时只会使用无赖技巧，如 + r1 + 中覇王丸的旋风烈斩（轻斩）>↓↘→A，对手只有闪避或跳过，由于连发旋风，闪避根本不是办法，只得跳过，覇王丸收招极快，又会发弧月斩击落对手，如此反复，堪称无赖。或使用枯燥无味封杀对手的玩赖技巧，如塔木塔木的轻斩很长且连打，使对手根本无法还击。

其次是使用无限连杀，使对手即使满血中招也会一命呜呼。例子皆是，这里就不举了。

使用以上两种技巧的人用贵刊以前的一篇文章说许是 + r1 + 。他们取胜不靠经验，不靠反射神经，不靠对手的观察力；而只凭手按的快，按的熟，按得准。好像是学生专攻文科，背政治，背历史，背得滚瓜乱熟而根本不带脑筋思考一样，他们绝不会感受到凭多方面综合能力而决胜的那种快感。为赢而决的人简直是像在完成任务似的取胜，如此两人遇在一起对战，比起手快，在下看来不如去美国西部，当年仗枪铳战去算了。其实为赢而决真是愚夫的表现，他们的对战，根本谈不上是真正的格斗。那些败给他们的人都会不甘心失败而只能鄙视地说一句： + r1 + 

上述两种技巧有一个共同点，就是反复使用同样的招数。但有的人虽使也不过是偶尔小调节奏，不会连续使用，如牙神（罗利）击对手反身的前中斩是很好用的无限连杀，可有人在此技斩两下后，使不再重复使，而用三空杀，对手虽损血倒地，但也毫无怨言。这样使牙神的人不算是赢而决的人。

还有，SNK公司设计的武器击飞程序在下看来是一种考验或测试，它可以体现玩家的品质，确定他决的目的。当在下用超必杀技或在拼刀中打飞对手的武器后是不会再进攻的，因为打败一个赤手空拳的人比战死更颜面无光，叫对方拾起武器再战，让他输得心服口服。但第三种为赢而决的人，一见对方武器离手，便趁机大肆猛攻，大举进犯，以此大获全胜。其实这恰恰说明他怕了，是惧对手拾起武器反击获胜，也就是承认自己不比对方。空手失败实不甘心，在下感触最深的就是一声 + r1 + 了此局便告失败。前面提到不去斩一个空手的人当然也非绝对 + r1 + ，如对方协尔福特，喜欢拳击的无赖技，那便不要轻饶，拾于练之。但也不要留情于对自己武器脱手仍进攻的人，鲁迅也曾提过（论 + r1 + 应该缓行）（Fair Play，公正决斗的意思），以痢打落水狗，当然这只是以牙还牙，在下还是认为Fair Play应当执行了。

在下善使柳生十兵卫（罗利），二、四代皆同，且是为决而决的人。对战时使出柳生心眼刀架中对方的快感是那些为赢而决的人所不能感受到的。在下深感无赖，玩赖和无限杀技巧，如遇之，哪怕倾我所有，弹尽粮绝，损失惨重，也要死战到底，斩之而后快。在下觉得有义务，有责任去决杀这些 + r1 + 的人。只是在侍魂Ⅳ十七人中独头痛覇王丸，无赖技 + r1 + （编者注：这大概是天津话，波动加升龙的意思？↓↘→？→↓↘→？）实不知如何对付。

借此机会，谈了在下对战上千次的一点心得，也许有过激之处，请谅，是因不可用自己的观点强加于别人。不过在下真的相信这篇文章与 + r1 + 二字的本质含义是有联系的，且相信所有的格斗游戏皆如此。

真正的对战，真正的高手，已明白决不是为赢，而是为从中寻求激情，寻求惬意，寻求领先经验、反射神经、洞察力等多方面能力综合在一起并经努力发挥而取胜的欣喜。这是一门艺术，人生亦是。谁愿一生平庸，有此小悟空选择盲死喝下超神水，而不是去稳妥地修炼几年，以战胜短笛。又比悟空[?]在人造人出现的三年前便得知而不马上去破坏以追求战斗。侍魂对战，如能和一个真正可以称得上是对手的人决，那是一种莫大的幸福与享受，在按键挥刀的一刹那，已忘却了自己是谁。
