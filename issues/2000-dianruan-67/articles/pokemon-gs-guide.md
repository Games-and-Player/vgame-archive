### 饲养屋的新作用

把捕获的口袋妖怪放在饲育屋中寄养，而后玩家只要操纵着主角继续冒险，被寄养的口袋妖怪就会自动得到经验值！在 + r1 + 、 + r1 + 中，饲养屋的地位更加重要了。现在，只要把捕获性别不同的口袋妖怪一起寄养就会出现宝蛋！玩家可以利用某些特定的组合方式诞生出新品种的口袋妖怪。具体的妖怪配合方法和进化情况将在下期一并公开志中一并公开。

### 欢迎来到闪耀着 + r1 + 光辉的口袋妖怪世界！！

前作的口袋妖怪也会在 + r1 + 、 + r1 + 中登场。

使用通信连线与朋友交换妖怪也是本系列的一大魅力。当然， + r1 + 也能与前作交换口袋妖怪。也就是说，在前作中登场的151只妖怪全都可以在这次推出的 + r1 + 、 + r1 + 中使用！不过，这次新登场的口袋妖怪和交换后已学会新技巧的妖怪就不能再交换回去了。在交换时，要记得先把这些妖怪从名单上去掉才好。顺便提一下， + r1 + 也可以与 + r1 + 、 + r1 + 连线，交换道具。

### 用口袋妖怪球收服口袋妖怪

要收服口袋妖怪就要用口袋妖怪球。在战斗时，把对方妖怪HP削减得越多，捕捉的几率就越大。另外，如果令对方妖怪遭遇麻痹与睡眠等异常状态的话，捕捉妖怪的机会会更高。

### 如何编成最佳队伍？

主人公能够随身携带的口袋妖怪只有六只。大家一定希望只带着自己喜欢的口袋妖怪去冒险吧！不过这样一来在作战时可能就会很辛苦了。因为妖怪有属性之力，若在战斗时遇到相克属性的妖怪就会变得非常不利，所以在编成时要特别注意属性的平衡。

### 可以砍新树？

在游戏中会有一些与众不同的树散布在各处。如果让玩家将这些树砍倒的话，就能够到达更多的地区了。因此要特别留意这些树的位置。

### 用ID号码来抽奖

所有的口袋妖怪都有设定有ID号码。在某一个地方进行抽奖，只要口袋妖怪的ID号码与幸运号码一样，玩家就可以得到各式各样的奖品！由于抽奖是以所有的口袋妖怪为对象，所以玩家必须尽可能地与朋友们相互交换，以便收集更多种类的口袋妖怪，提高中奖的几率！

### 培育不参战的口袋妖怪

要想培育幼小的口袋妖怪是件很辛苦的事。不过，在本作中只要编入在队中，就算不作战也能得到经验值。方法是把想培育的口袋妖怪放在先头部队中，一进入战斗时，就马上会很辛苦了。因为妖怪有属性之力，若在战斗中时遇到相克属性的妖怪就会变得非常不利，所以在编成时要特别注意属性的平衡。

### 上记想看到的口袋妖怪与其它的口袋妖怪做交换

上记想若有的口袋妖怪与其它的口袋妖怪做交换。这是一个既简单又实用的小技巧。

### 健全的赌场

前作中出现的老虎机、猜纸牌等迷你游戏，在 + r1 + 、 + r1 + 中也可以玩到！与前作一样，若进行迷你游戏，就必须先兑换代币。而忙于存一定数量的代币后，就可以换到珍贵的奖品。

### 无法捕获的口袋妖怪

在 + r1 + 、 + r1 + 中，也存在着用普通玩法无法收集的特殊的口袋妖怪。事实上，要得到这些稀有的口袋妖怪只能靠和朋友交换或利用通信交换来使用特定的道具。本刊将在下期杂志中为读者介绍各种稀有妖怪的获取方法。

让口袋妖怪进化的时机是

在口袋妖怪中，大部分妖怪在等级提升到一定程度后就会进化为其它种类。虽然进化可令怪兽的基本能力值上升，而且也会学会攻击技巧的基本等级也会上升，而且也存在着无法学会某些特定技巧的缺点。因此，最好还是等待妖怪学习得所有技巧后，再让其进化比较好。

##  + r1 + 简明流程攻略

清晨，金得到ポケットギア（口袋工具），向母亲道别后，展开冒险。首先，前往上方ウツギ博士的研究室，博士会让金选择一种怪物（モンスターボール），不用说，是抓怪物用的重要道具。

从左侧出村，跟前一片绿地，先练上几级后，很快到了第二个村子（ヨシノシティ），与村口的老者对话，他会为主人公介绍村里的景物，并赠送了地图（今后不会再迷路了）。在北出村，在路口金被两个讨厌的家伙挡住了去路。没办法，还是先去右上方的房子看看吧，一代中的オーキド博士已在此久候，他赠送图�的鉴（ずかん）后就匆匆离开了。

在返回ワカバ镇（老家）的路上，金第一次碰到了的对手（ライバン），幸好他的怪物不强，只有5段，轻松搞定。回到研究室，博士已与警察一同回来了。金继续北上，原来是 + r1 + 两的战的二人早已休战。前进中，金碰到了第一个怪物教练，金边战边行，最终到达一国大城市。在守卫展示8枚徽章后就可以进入其中了。城市里有很多障碍和怪物，但金还是先穿过一个山洞，就在离开山洞的那即儿，ライバン又来战斗了。虽说强了一点，但还不是金所介意的对手。
