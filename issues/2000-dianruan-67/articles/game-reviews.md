## 永恒之戒

全新画面 全新系统

随着PS2的火爆发售，那些没有PS的玩家会不会直接选择这台 + r1 + 的主机呢？而PS被淘汰后，日本媒体会以新世代机的游戏为主，因此本栏的机种比例可能要随之发生变化。在2000年，你准备POW↑自己的主机吗？

### PS2考验制作公司的技术力

简单的说， + r1 + 这款游戏的特色就在于将多样化的戒指重复交错组合，即可创造与众不同的强力魔法，而决定魔法配制的强弱可就得看玩家如何地灵活运用了， + r1 + 始终是整个游戏系统的主干。左图即是游戏背景，制作者对虚拟世界的构筑是非常真实的，明显未用 + r1 + 技术。

### 层出不穷变化多端的游戏空间

游戏发生在 + r1 + ，此岛为龙的居住地，细分8个地区，受到龙的力量的影响，岛上内部地域皆呈现异常的形态，否认是天气、时间、地形等……玩家要在这岛上解谜，扮演国王派来侦察长老派的夺权情况，男主角接受了国王的密令而战。岛上的地形情况如下：

 + r1 + 在火山喷出岩浆而冷却凝结于海面上方，可连通对岸的洞窟； + r1 + 是万年冰雪覆盖下的山顶深藏着的百多年古代遗迹； + r1 + 是谁都弃之不用的神殿，至今半边沉入海中，内部已被海水堵塞不得进入，并且听说有水龙在此出没，更增其神秘感。此外还有不少惊险地区在等待着玩家的发掘，是一款提供全新娱乐理念的RPG。

### ■戒指的成长

戒指（指轮）要通过魔水晶这项ITEM在魔导士的作用下进行合成。指轮是在不断的变化、成长，外形越变越复杂，代表着主角能力值的相应提高。

### 相互牵制是戒指（指轮）的特性

戒指按属性分为火、水、土、风、光、暗、无等；按种类则分为 + r1 + 和 + r1 + 两种，前者在装备中可以决定使用某种魔法，而后者能够增加其魔法属性。在装备戒指时，一只手指即可戴上10个。

而制作戒指方面，打倒敌人后所获得的 + r1 + ，每一个都可经由魔导士混合制作而成，以强化戒指的利用价值，不但能够使用魔法，更能当作防具用。每种 + r1 + 都有各自的参数，在其相互影响之下，攻击值、耐久值和属性都会随着改变。魔法总数52种，其中包括 + r1 + 约10种左右的强力召唤魔法，戒指的排列组合越和变化莫测。
