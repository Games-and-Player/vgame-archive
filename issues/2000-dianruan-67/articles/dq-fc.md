### DQ III疑难解答

虽然DQ的FANS快要迎来它的第七作了，但是不少骨灰级玩家却始终被DQ III困在FC的世界中。为了解救这些战友，笔者特将DQ III中出现的疑点作一一解答。

**◇疑点之一：如何取得魔法钥匙？**

答案其实很简单：钥匙就在金字塔中。玩家只需上到金字塔第三层，就会发现在迷宫正中央有一块巨石将迷宫隔成两半，而钥匙就在巨石之后。移开巨石的方法是在这层迷宫的左下角和右下角各有两个机关（共四个），必须把左右两个正确机关都打开，才能使巨石降下。（正确触动机关是靠一定运气的，不成功就会掉到第二层，多试试吧）

**◇疑点之二：如何取得船？**

船是由葡萄牙国王提供的。取得魔法钥匙后通过一座桥便可到达葡萄牙。见到国王后，他会叫你去找黑胡椒，并给你一封 + r1 + 。拿去给パハラタ东边洞窟里那个士兵看吧，他会给你 + r1 + 出一条地道来。通过新开的地道到达一个新地域，在南走可以发现一个村庄，要取得黑胡椒就能完成这里的事件了。拿黑胡椒去给葡萄牙国王，他便会以大船相赠。

**◇疑点之三：如何取得黑胡椒？**

如果不是想象力怪异人士（早期RPG中毒症之一），这个疑点可算是极难解决，方法是先救出被关在黑胡椒树东边洞窟里的一对恋人。可奇怪的是，洞窟里似乎根本没什么人嘛。事实上答案在上面没有石墙的地方，对着这些地方用钥匙，咔嚓一声，竟然……有道门！剩下的就简单了！

**◇疑点四：如何取得 + r1 + ？**

最终钥匙在太平洋北部海域。这里有处 + r1 + 字型暗礁，对它使用 + r1 + ，就可使暗礁浮上来……

至于 + r1 + ，是在英国皇室内可取得的，但需要隐身术或隐身草才能进入。下到地下室完成一个推石头游戏（把石头全推到上部三块有花纹的地板上），就会打开秘道，得到千早之壶。

**◇疑点五：如何取得 + r1 + ？**

要取得火炎之剑，先要从幽灵船（出没在地中海一带）上得到 + r1 + ，这是破除西亚某处湖畔诅咒的东方的东西。在试图进入该湖时会响起一段音乐，然后船被强行驶回，在驶回途中使用 + r1 + ，诅咒便被破除。进入湖区后发现中间有个小岛，岛上某处角落头就有火炎之剑，调查即可。

**◇疑点六：如何得到六颗火种？**

六颗火种可以使不死鸟出世（不死鸟神殿在北极附近），从而使勇者们能够进入魔界。

①第一颗：在非洲中部（通过河流进入）有个小村子，村里的 + r1 + 晚上是人（幽灵吧），白天却是白骨，在此村中有座半房，里面关着一个 + r1 + 。晚上用最终钥匙打开半门跟他对话，就可以得到第一颗火种。

②第二颗：在日本。击败怪物所变成的天皇后，就可以打开皇座边的宝箱得到。

③第三颗：在主角出发地（大洋洲）西南方的一个大岛上。在此之前须取得回音之笛（就是使用后会 + r1 + 地一声的东西），利用它通过迷宫（只有主角一个人能进入），便可得到。

④第四颗：在南美地区的 + r1 + 里。这个小村子都是些海盗，火种就被他们藏在村边一块大石头下的地下宝里。

⑤第五颗：这颗也许是玩家最不容易得到的。首先在队伍中编入一名商人，然后坐船去北美。北美洲的东边树林中有一块空地，进去后发现有一位老人，交谈中老人希望主角们帮助他建设这里，把那个商人送出去吧！然后每次再进入这里都会发现有很大的development。不过越到后来，那个商人越发的贪得无厌，终于被人们反抗而进出城外。不用管他了，调查皇座旁边的宝箱就有第五颗火种和吧！

⑥第六颗：同样是很困难（不光是找地方难，打起来也难），具体位置大约在非洲中部吗？反正是从北亚（！）的一条河进去，达尻头后发现有个黑色方格，还以为是bug呢，不过对它使用 + r1 + 后，奇迹发生了——火山爆发？！这是火山？（笔者当时差点惊得千巴已磨跎平了河，出现了一条新路。而尽头处有个山洞，总体等级没37级就小心点吧！打过山洞后就可以从一个小面里的家伙手中获得火种。

**◇疑点七：如何到达大魔宫？**

到达魔界后剧情较易发展，不过到达大魔宫的方法倒是个疑点，方法是得到光之矛、勇者之盾、光明护符和一个什么杖。其中光明护符的取得稍难，需要在妖精之塔中对石像使用妖精之酒（使用后有一段很尖细的音乐），使石像变回妖精原形，她会送你光明护符（勇者装备后走一步加1点HP）。拿这四样东西去魔界西南方的一个小岛找一个老头，他会交给你 + r1 + ，到地图中去，找到最穷处使用彩虹桥就行了。

**◇疑点之八：如何找到大魔王？**

正确的方法是，在第一层魔王玉座的上方，有一个隐藏通道，调查出来后杀下去吧！
