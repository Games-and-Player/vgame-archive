### 技术第一。目标：成为STRIKER。建议：双打对练。

文：天语

#### ■一、基本操作法：

就三个键，能打到什么程度就看玩家的了。想提醒一下，当你防守时，控制住一个队员后，应配合着方向键按住长传键，让他努力地跑跑至对方控球队员那里抢球。抢球时不用按任何键，只要用方向键让抢球者碰触对手即可。对于一些队伍中的强力带球者，该放倒他就放倒，尽量不给他机会。长传或短传键也控制头球的远近，头球攻门时应视位置灵活施展。最难的是倒脚和过人，带球过人非常难，估计用摇杆时可能还好一点，手柄太难用了，所以你要特别了解上球员位置，打配合吧。

#### ■二、战术：

你的选择空间只是该队的 + r1 + 、 + r1 + 和 + r1 + 这三种方式而已。根据场上情况适时改变攻守程度即可。

#### ■三、高点数的射门并不容易：

这一系统街机上也有，要求打者对游戏有相当的上手度。绝对要主动进攻，绝对要积极跑动，要紧迫着对方传球甚至拌断，不这样，就形不成机会，只能被灌（此过程充满了紧张与刺激）。

通常的 + r1 + 战术即设法从边路下底，然后看看禁区有无人接应，传中后的混战中要么头球斗争，要么直接身体腾空或蹲倒勾射门。这是个典型，一般都在280点以上。如果只知 + r1 + 则最多150点。得分的要素如下：

①倒到DIRECT PASS（直传、传中）。

②形成射门的机会是一个过程，这个过程越周折、经手越多的人评价越高。二传一射280点附近。

③禁区内按传球后DIRECT SHOOT，距目炫，很难低于200点。

④让某位勇者独自运球至对方门前并射门成功。

#### ■四、PK战：

简单至极，颇能锻炼玩家的反射神经。

①罚方量表（射门时的力量）0%～30%：地滚球或低平球。守方因应为方向键的斜下或置之不理。

②罚方量表30%～50%：此时球的高度约为立柱的一半。守方应输入左或右键，或置之不理。

③罚方量表60%～80%：基本是吊角儿了。守方应以方向键的斜上来应付，若球的来路太正也可置之不理。

④罚方量表90%～100%：大力发泄，皮球恐怕要到横梁上面去了。吊角入网是极小概率事件。门将跳起来也够不着球。

#### ■五、隐藏球队：

本家用机版共有八支，分别是FC SEGA（最强世嘉队）/MVP-YUKI-CHAN（原创雪人冬季队）/MVP-ROYAL-GENKI（皇家元气队）、葡萄牙/瑞典/俄罗斯/乌拉圭/希腊。

前三者在街机模式的选择方法如下，●最强世嘉队：队选择画面下，在限定时间20秒内，分别在法国(F)、智利(C)、沙特(S)、英国(E)、德国(G)、奥地利(A)的标志上按一下开始键。然后在32支球队的上方出现世嘉队旗，迅速将游标移至那里，按A决定！●原创雪人冬季队：同理可知，分别在YUGOSLAVIA、USA、KOREA、IRAN的队标上按下START键，又出现一个旗帜，那便是了！●皇家播植元气队：在 + r1 + 的 + r1 + 上按一下开始键即可！其标志正是元气公司的LOGO。

后五者是系列前代复活队，在VARIATION中的RANKING模式下的出现法如下，● + r1 + 级取得后遇葡萄牙。● + r1 + 级取得后遇瑞典。● + r1 + 取得后遇俄罗斯。● + r1 + 取得后遇乌拉圭。● + r1 + 级遇希腊。

注意事项：①世嘉队遇队则强、遇弱队则弱，至于如何区分，请看场上有无一个109号，即一颗球就蹦倒的那位，有他在，说明你遇上弱队了，此时世嘉队只能发挥50%水平。也一队身高偏低，高球吃亏。阵型未明。②冬季队有雪人有动物，人员都穿着御寒服装，能力、阵型与日本队同。场地飘雪。③元气很世嘉队全是些大头外星人，皮球则变成了UFO。水平与冬季队同。④若分别用上述三队将街机模式统统打通，则其他模式可直选（SP区）。⑤对于五支复活队，只要战胜之即可使用了。五支队伍各有特色，如战国能防围墙，希腊长于头球……

#### ■六、隐藏体育场：

简单地说，你将特定模式都给打通，并有一次打出300点以上的射门，则能增加5个特色体育场（MATCH PLAY模式下可任选）。具体为：①VARIATION模式中的LEAGUE战打通。②世界杯模式CLEAR。③只要一次300点以上的射门。④街机模式清版。⑤VARIATION模式中的TOURNAMENT战打通。

完成其中的任一项，即会增加一个对应赛场，其有的观众席前带有金属网，有的建筑特色令人怀念，有的照明特别神奇，总之，能极大的提高比赛双方的投入感。因为这些体育场太酷了。你一定要试试。
