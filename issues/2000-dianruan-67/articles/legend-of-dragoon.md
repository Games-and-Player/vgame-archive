【图】左侧为游戏角色设定画：一男一女两位龙骑士，全副武装，背景为雷电闪烁的天空。

体贴的设计：在游戏中按住R1键，画面中会出现提示（第一次按下，过一会儿就会消失；第二次按住，提示会一直存在）。蓝色大三角代表玩家操作的人物；而绿色小三角代表一般居民，蓝色小三角代表武器屋、道具屋；黄色三角代表宿屋或治疗院。游戏过程中，玩家所操纵的角色头上的三角会依 + r1 + 转变，变红时则说明即将遇敌（小兵）。

### ～附加情节的BOSS攻略～

在进入 + r1 + 之前，可以去以下地点引发一些有趣的附加情节。这样不仅能够提升等级，又可以获得隐藏道具，何乐而不为？！（括号内为该BOSS属性）

#### ●フランベル的塔

BOSS：魔老ファウスト（无）

魔法攻击对其基本无效，取胜的关键是利用HP高、速度快以及魔防高的角色与ダート配合，进行强力的物理攻击。战前要为ダート装备 + r1 + 。此外，魔老ファウスト会在最终阶段使用全体的炎属性攻击，对没有变身的人类可达2500点的伤害，而在其HP剩余1/2至1/4时，分别有水、风属性的全体攻击，攻击力同样为2500点左右。所以己方平时最好不变身，或只让ダート变身，因为变身后无法使用道具。在三次强大攻击来临之前，再全体变身以减少伤害。其HP大约有32000左右。

参考装备为 + r1 + 、 + r1 + 、 + r1 + 、 + r1 + 。胜利后得到： + r1 + 与9000G！！！

#### ●マグラード岩

BOSS：在最深处先后与ヘルム、アーマー、ンード战斗。

ヘルム主要使用状态魔法；而アーマー以强力魔法攻击为主；ンード则善于物理攻击与即死攻击。参考装备为 + r1 + 。

#### ●帝都ベルウェブ

在ミヤーリー解救龙骑士们的灵魂。

BOSS：ミュベール（风）

使用与アルバート相同的魔法。其碧绿龙攻击力为1500左右。胜利后获得 + r1 + 。

BOSS：ダミア（水）

使用与メル相同的魔法，魔法攻防能力均很高。我方尽量使用炎属性魔法及物理攻击，其连发魔法对人类状态的攻击力为2000点左右。胜利后获得 + r1 + 。

BOSS：ヴァルザック（地）

使用与コンゴール相同的魔法，物理攻击和防御能力极高，攻击人类状态的角色可达1500点的伤害，但其速度缓慢，而且魔防极差。胜利后得到 + r1 + 。

BOSS：カンザス（雷）

使用与ハツシエル相同的魔法，其紫电龙攻击力在2000以上，但其所有魔法均为单体攻击。胜利后得到 + r1 + 。
