### 致富篇

钱在 + r1 + 作中有着重要的作用，主要用于各项技能的习得，行贿受贿，购买药品，医治疾病等。正所谓君子爱财，取之有道。 + r1 + 作中仅靠每月几贯的薪禄怎能用？且看赌场秀吉发家史。游戏之初，主人公金钱少于10贯。这样一些大赌场是不受理你的筹码的，但你可以从小赌到大赌的顺序开始。部分酒场赌码如下：1贯8枚，尾张之国�的�的海城。10贯8枚，纪大多数城池酒场。50贯8枚，伊势之国长岛＜推荐＞。

赌博规则：大概三个骰子中有同数如 + r1 + ，或者连数如 + r1 + 为胜，自己作庄赢不多，别人作庄时一口气全押上，嘻嘻——输光了吧！如果不存档可真怪了金赌神仙输。

赌中藏宝：在上文提及的伊势国国赌局中，如果玩家所赢得的筹码超过庄家所能负担的，那么庄家就是以抵押物出售！此时千万不要错过！抵押物大多为价值7的宝物（需作�的鉴定）。想想看，淀川一滋所亲爱的珠光小烫子也只有6而已！赚死了！赚死了！

### 技能篇

技能习得对武将很重要。 + r1 + 作中武将技共分12种，每种又分三级。一、二级一般易习得，三级则往往需要在特殊城、特殊武将处习得，现公布如下：

| 技能 | 习得途径 |
|---|---|
| 算术 | 1～2级：任意商家；3级：界之町商家、今井宗久 |
| 筑城 | 1级：前田利家（免费）；2级：其它武将；3级：明智光秀 |
| 开发 | 1～3级：界之町南蛮寺 |
| 弁舌 | 1级：前田利家（免费）；2级：其他武将；3级：明智光秀 |
| 礼法 | 1～3级：任意城的寺中习得，条件相应价值的书 |
| 茶道 | 1～3级：界之町南蛮寺、千利休、条件相应价值的茶具 |
| 武术 | 1～2级：任意城的道场；3级：伊贺上野城、柳生石舟斋 |
| 马术 | 1～3级：任意城的马屋 |
| 铁炮 | 1～2级：任意城的锻冶屋；3级：今浜町烟火屋、国友一贯�的 |
| 乱波 | 1～3级：伊贺上野城、遂之屋、百地丹波 |
| 计略 | 1～2级：任意武将；3级：竹中半兵卫 |
| 战术 | 1～2级：任意武将；3级：竹中半兵卫 |

注意：在自己下属武将处可以免费习得武将技。

### 升级与人际关系

想要在织田家错综复杂的斗争中步步高升、官运亨通，光靠一身本领和埋头苦干是不够的，玩家必须努力扩大自己的木下派。而前田利家是你的前哨阵地，从他处可以打听到各武将的爱好，投其所好，一般两三次合意的赠物就会使其转入你派。武将们所喜爱的宝物多分如下：长浜之町多有武具；岐阜城多有书；长�的多有艺术品；界之町的南蛮商行自然是多有南蛮货物。（哎呀！廉政公署别抓我……辛苦多准备了几个宝物）不过，有些人是礼物收入情不爽，现公布黑名单如下：森可成、河尻秀隆、可见、在左成政、XX家嘉。

此外，在家老级人物中，丹羽长秀、明智光秀可以笼络，而柴田胜家、�的久间信盛则刀切不见。至于其他家老吗？用钱把他们砸过来吧。当然，每次的评定是陈述方案也很重要，可以狂升功勋值200点，支持率正比于你的派阀数，特别是以老臣入为重，所以多多争夺吧。

附1：当主角官职提升后，可以找织田信长 + r1 + （要求增加下属武将）且以己身份和能力为重，比如前田庆次。

附2：评定结束后，多让手下武将去 + r1 + ，从而培养武将技。另外玩家还可对自己欣赏的武将进行 + r1 + ，让其从你处习得高级技。

附3：宝物薪用。凡价值在6或6以上的宝物均可增加人物的四项基本数值，对应关系如下：内政（史书及利器）、外交（茶具）、军事（武具）、魅力（南蛮物）。因此，多准备些宝物分给部下吧！

附4：有时回家居尼尼说： + r1 + ，不妨去拜访一下，顺便带上医宅的药，正所谓雪中送炭效果加倍。

附5：评定时内务拜访家老级人物，可见 + r1 + 项，做个民意测验吧。

### 难点释疑

★游戏初期，尼尼得痨发高烧，去界之町南蛮寺购药（青霉素即可）。

★织田主命，劝诱竹中半兵卫，在美浓�的杆仟山城，三请乃成。

★市浅井羽合联军后，秀长被派往横山作城代，任期中由浅一次浅井家的进攻，静待几日即可。

★伊贺上野城道场，柳生石舟斋会向你推荐浪人，弥补因时间差而损失的武将。本人的福岛就是花钱找来的，而迷之屋百地丹波 + r1 + 可以使你迅速了解自己想要攻略城池的情况。

★长筱之战以 + r1 + 击溃武田军后，回长浜找一贯需定制大筒（神武大炮），作战时只需带上1辆，外加足够的铁炮，组合即可。
