### 小人儿拉尔夫大冒险

（THE ADVENTURE OF LITTLE RALPH）

机种：PS　厂商：NEW　类型：ACT　媒体：CD-ROM

在这个3D横行的时代，似乎能够在平面动作迷心中留下些烙印的游戏已屈指可数了。但值得庆幸的是，这个叫做NEW的厂商又为我们带来了一个经典。

这是一款正宗的平面卷轴ACT，故事情节仍然是围绕着传统的英雄救美而展开的。既无敌人面前的对白，也没有精彩绝伦的CG。但却有的，却是那一种令资深玩家久违了的纯朴和清新，以及那极高的可玩性，与现在的某些一味追求3D效果而用POLYGON草草描绘但称做ACT GAME形成了鲜明的对比。

正如游戏包装上所描述的那样，在整个游戏过程中我都有一种爽快的感觉。而这种感觉不仅仅表现在操控感上。当然，此GAME专业级的操控感是绝对可以让那些提拉毛病的ACT老鸟们兴奋一阵子了。而且你完全能够用这套引擎来训练手指的灵活性和你的反应力，我身边的几个朋友就是用它来磨练此类游戏功力的，并且收到了不错的效果。

最值得称道的还算是游戏的美工部分，充满魔幻色彩的自然风格充斥着画面的每一个角落，丘远近的二重卷轴背景尤显真实、华丽，有一种如凡尼赤之感。记得某某尾尻之处主角小拉尔夫沿一颇显出暮色的地下隧道奔向地上，当那缓慢上升的画面将一幅大漠飞沙的景象展入眼前之瞬间，我真的被那感动得落下了热泪……

其次，在关卡的安排上，明显体现了制作者们独具匠心的设计思想，版面空间的自由度很高，一些机关设置也极具魄力和智慧性，虽然要一些技巧和思考才能顺利通过的。尽管游戏只有EASY和NORMAL两个难度档，但NORMAL的过程的确是个挑战！比较适合高手们的口味。游戏的音乐音效有很强的卡通风格，也是值得一听的。此外，更多的是那些新鲜而又叫人兴奋的东西等着你在关卡自由会，录自探索。你不会失望的！

厂商NEW的作品是我见过的叫声就那么一个，但却是个绝对的精品！若你是个真正的平面迷的话，就来领略一下这款使人爽快的杰作吧！

### 英国足球经理（LMA MANAGER）

机种PS　厂商CODEMASTERS　类型SLG　媒体CD-ROM

PS上的第一款足球经理游戏——LMA MANAGER（英国足球经理）是codemasters之手，在 + r1 + 吃尽PS的今天登陆PS。

既然名为英国足球经理，玩家要扮演英格兰四个级别中任一支球队的总经理兼主教练。通过球员市场的转会创造阵容；聘请资深教练进行系统训练；雇用 + r1 + 调查球员状况；确定下球员合同；决策体育场扩建及球票价格等。游戏方式分自由模式和挑战模式两种。自由模式不必多述。挑战模式要求玩家在特定时间完成冠军、保级成功（升班马守擂）等10个项目的任务。具有新意及挑战性游戏中的赛制和日程与现实中英超完全一致，可参加超级联赛、足总杯、联赛杯、欧洲冠军杯或联盟杯等四项赛事。球员资料库资料常详尽，有欧洲32个国家302个俱乐部近8000名球员的详细资料（当然有意志精和场情数，数值也当然不那么高）。

游戏的操作界面简单明了。比赛中战术采用图形辅以文字表述下底传中、中路渗透等基本术，使人一目了然，更易于临场操控。球员训练中的动画很精致。

该游戏的一大特色在于赛后的射门集锦，清晰的语音解说，真实的现场感，加之全3D制作的接球、过人、射门等精彩情节镜头的回放，多层次、多角度再现了英国足球快、准、狠的风格。当拿冠军、升华来年度最佳教练时所得到的成就感更让你融入其中。

由于是英国足球经理，不能操控AC米兰、巴塞罗那等队便成为一大遗憾。又因资料库太大，就中读盘速度稍慢，存档时更是蛙吞。要求14格记忆卡容量快，也算美中不足。

### 实况J联盟1999·Perfect striker

机种PS　厂商KONAMI　类型SPG　媒体CD-ROM

抛开一切技术上的进步，这款游戏很明显是在追求超越时代KONAMI足球的那种氛围，从比赛自由度极高的设定菜单，到开场而那绝无仅有的地硬而面面，再到比赛中可随时Replay压倒拆围[?]守门了贡献的性能设定，足以让一个超任老玩家大发感慨，忆往昔峥嵘岁月稠。

这一路线很明显和PS上的WE的发展趋势大相径庭，PS上已经推出的4款WE，其偏向相榜的关系非常紧密而且实操作性、流畅性和爽快感，恰恰反这款99即在这些[?]之外没给什么特殊的感觉。

其实这款 + r1 + 的出现并不是多么奇怪的事，KONAMI足球游戏的发展策略从来都是以世界范围对方为主，以J联盟为辅，这种辅助性从来就体现在它的试验性能。PS上最早的J被认为一无是处，除了真人扫描照片，已经没有什么保留下来了，其后的一代引入了传身足跃这一革命性的系统，至今已成为KONAMI足球的一大招牌，J98-99引入了覆盖动和声[?]的主赛，而看是可可，但是人现在正强烈期望到了所引[?]州的高度必要性。这些试验性的战术操纵——移用至世界杯足球，弹就了一批又一批经典游戏实况。

但这款99即使在J系列中也是额外特殊了。大概是因为其开发者是KCEO（KONAMI大阪），而不是KCET（KONAMI东京）吧！这种以现在的技术传达过去的情绪的尝试能否成功，让我们拭目以待。

### 007明日帝国（007 Tomorrow Never Dies）

机种PS　厂商EA　类型3D-STG　媒体CD-ROM

邦德肖恩·康纳利的出色演艺，好莱坞拍摄的007系列又开始令广大影迷如痴如醉。同样，以007电影改编的游戏也令游戏迷火冒不已。虽然没有玩过N64上的 + r1 + ，但它的名气实在太大了。当看到PS出了 + r1 + 后，就毫不犹豫地买下了。事实证明，这是款相当不错的游戏。

游戏的情节基本与同名电影相同。各个任务都是电影中的经典场面。因为不仅为了满足游戏的需要，在某些关中增加了很难打的BOSS。因为是三维游戏，所以场景的方向感很强，必须时刻注意周围是否有敌人，真实感非常强。游戏中所设立的任务难度并不大，一般来讲，只要按照原序去完成就可以了。最大的困难在于如何减少自己的死亡，同时尽量快地将敌人击毙。虽然制作者为游戏者设置了偏颇的键位，但所起的作用很有限。尤其是对BOSS时，美版游戏的难度增大，会让历经磨难的你有一种绝望的感觉。不过还好不会到那么自动瞄准的键让游戏者感受到了十分体贴，可以方便地瞄准敌人的要害。当你在瞄准镜中看到还没有来得及向你射击的敌兵头部中弹死掉时，那种快感是不言而喻的。

当然，游戏的快感不仅止于此。如果同时遇到四名敌人，扫射带来的除了快感还有实用，同时还会视得自己很勇敢，很猛。

### 龙骑士传说（LEGEND OF DRAGOON）

机种PS　厂商SCEI　类型RPG　媒体CD-ROM×4

如果对这部游戏的评价仅仅停留在画面做的如何美丽、CG的水平如何高、剧情引人入胜等层面上，却未免有些瞧不起SCEI，现在这样的大公司无论如何也要在这些方面表示自己的实力的，在华丽程度上是绝对不会输给任何公司的。

之所以说它是一款超级大作，是因为它很多地方模仿了 + r1 + 。模仿最明显的地方是剧情构成方式，场景中、战斗时的画面简直与 + r1 + 如出一辙。但是如果完全模仿的话，SCEI就要从此颜面扫地，去SQUARE面前抬不起头来。因此，SCEI在战斗系统上花了很大的精力，这也是唯一可能区别、甚至超越SQUARE的地方。本作的战斗模式最突出的一点就是连击系统。由于这个系统在游戏中的重要性，使得原本很容易令人厌恶的RPG战斗变得很吸引人。就连小数人战斗时也必须集中精力来使用连击技，而不是象一般RPG那样只按确定键键设就闭眼睛也过去了事。在RPG中如此关注战斗的情况，通常只有在A·RPG中才会出现，而这款正统RPG却达到了同样的效果。

本作的故事主线非常清晰，游戏玩起来不会有 + r1 + 的事情发生。不过，在背景音乐的造诣上，SCEI还是比SQUARE差了一点。不过音乐的气势不如 + r1 + 极致。人物性格的设定上，也还有些距离。

### 苍魔灯（苍魔灯）

机种PS　厂商TECMO　类型ACT　媒体CD-ROM

在《影牢》发售一年半之后，这部《苍魔灯》又与我们见面了。作为（刻命馆）这个陷阱杀人游戏系列的第三作，《苍魔灯》不仅继承了前作的优点，还对它们做了改进和完善。

做为游戏中心的陷阱部分，这次你不但可以通过实战积累 + r1 + 未习得新的陷阱，还可以用这些 + r1 + 对已有的陷阱加以改造，使它们改变双击方式或是加大威力，从而制造出具有玩家原创风格的陷阱。据说由此组合出来的陷阱不下2000多种呢！本人到通关也只制得60多种！更妙的是这些陷阱还可以被记录在游戏提供的 + r1 + 中，让玩家随时欣赏自己的杰作。这些要素的加入无疑是提高了玩家对游戏的投入度，也是本作最吸引人之处。

游戏处理方面，剧作中 + r1 + 的公共陷阱，现在也有一部分归玩家控制了，多利用它们也是取得高分的关键之一。新加入的侧翻动作，使得女主角的身法灵活了不少。敌人的身份、职业和攻击方式都丰富了许多，但游戏的难度终极人感觉反而有所下降。

游戏的不足之处也是相当明显的。画面还是一如既往的粗糙，过于简单的剧情也没有改变。还好，多分支、多结局的设定实仍然保持在，算是对单一的剧情做了弥补。

总得来说，《苍魔灯》的好处在于其良好的手感和与众不同的创意，但遗憾的是，它过于阴暗和所表现出的灰暗的世界观，只能成为不易被大众所接受的另类游戏。

——内蒙古呼市　桃太郎

### 寄生前夜2（PARASITE EVE 2）

机种PS　厂商SQUARE　类型AVG　媒体CD-ROM×2

世界好像变得越来越小了。请注意，我这里所指的是游戏世界。最近一段时间，类似的作品越来越多，雷同的比例正愈增加大，只是这几个人愿意点就，直到这《PE2》出现以后，大家不由�的声叹：真像（×× 危机）。

两部作品有些相似之处是不容置疑的，比如操纵方式，好像仅仅是把R1与□键摆放了一下。但这次不是主要的，我认为《PE2》之所以不像个RPG，自由度和世界观是两大因素。本作的自由度实在太小，根本没有 + r1 + 的感觉。若还能说它有世界观的话，这个世界真不算大。

当然，固作的RPG风格就不很 + r1 + ，这次虽然有魔法，有EXP，有BP，但是依然像个 + r1 + 。假如抛开雷同因素，本作在游戏性方面足以名列前茅，即使与《生化危机3》作比对，画面、音乐等方面也毫不逊色，而战斗时还要更有战略性。

《PE》系列的另一大卖点是CG。本作在这方面应该算是有所进步，一些细节（比如影子、表情）已经能被好地利用起来。也许你对AYA的形象改变有些意见，不过相信在看过所有CG以后，你的观点会有所转变。

### GT赛车2（GRAN TURISMO 2）

机种PS　厂商SCEI　类型RAC　媒体CD-ROM×2

让一个对游戏不甚了解的人来看的话， + r1 + 比 + r1 + 并没有什么太大的差别吧！确实，PS毕竟机能限，对于 + r1 + 这样一个在Ⅰ代时就竭尽全力制作的游戏来说，Ⅱ在画面上确实很难再有什么令人瞩目的提高了。

即使从玩家角度来说，相对于Ⅰ的改变也并不明显：交通标识是同样的桔红色图标，有着同样柞木索索的操作画面，教科书一样的车体性能数据。唯一明显的变化是在原有系统的基础上加入了车种和赛事，这种以不变应万变的策略应是SCE的高明之处。

GT在最初起步时就是一款以游戏迷和车迷的交集为目标消费层的游戏（虽然日后的成功远超过这一预想），它的味道暗示着除了纯粹之外的公众性就像一种典型的SLG要素一样吸引着广大玩友和非玩友！毕竟GT的销量与FF相当，不能不说其中相当多人手一款的感热情起了很大的作用！一款游戏能够成为车迷的象征和骄傲，便是GT的成功之处。

SCE看准这一点，推出了这款 + r1 + 的GT 2，而 + r1 + 正是其专业性更进一步的具体表现，简就这点来说，它能帮给那些GT的玩家的，一定是更大的享受。

——北京　LEXUS
