### 新系统

这次的 + r1 + 中，系统方面做了大幅的变更。它除了 + r1 + 系列游戏中著名的 + r1 + 系统，使战斗变成单一战线的战斗。除此之外，新作还收录了许多新的系统呢！在这期的报道中，将介绍众多新追加系统的其中2项。它们分别是可以任意设定角色的力量提升时机的 + r1 + ，以及战斗中可以回复体力的 + r1 + 。请大家仔细研究新追加系统的内容吧！

### T.O.P.系统

在本游戏中，角色可惜由此实时的残余体力的量来提升威力。而且玩家还可以任意选择力量提升的时机，以上所介绍的就是所谓的T.O.P.（为Tactocal Offensive Power的缩写，取每个字的英文字首，念做TOP系统。在比赛开始前移动生命槽上的白色部分，从3个阶段来选择提升威力的时机。然后，在比赛中生命槽达到白色部分的话，角色的身体就会一闪一闪地发光并进行 + r1 + 。

#### 选择威力提升的时机

**前**
比赛一开始就提升威力。所谓的先攻型。

**中**
当体力剩下2/3~1/3左右的话提升威力。

**后**
当残余体力少的话会提升威力的逆转型。

#### 启动T.O.P.!

一当生命槽到达指定的白色部分时，会启动T.O.P.！身体发光就表示正在启动。启动时，角色将会享有各种特征。

#### T.O.P.攻击

一启动T.O.P.的时候，攻击力会提升，体力会慢慢回复。另外，玩家也可以使用各个角色所特有的 + r1 + 哦！

### 及时防御

在快要遭到敌人攻击之前进行防御的话，角色身体就会发出蓝光，并可稍微回复自己的体力。这就是本游戏这次新追加系统之一的 + r1 + 。及时防御不论是在地上或者在空中都可以使用，在这次新作的比赛中，如何善加利用此系统，则对玩家的胜败将会有很大的影响。
