# EXO行动组

> 机种：MD　厂商：PLAYMATES　类型：ACT　容量：8M　文／刘鹏　杨柯来

## 穿越时空，维护和平！

### 一款美制低容量、高品质的游戏！

这是一款由美国厂商制作的游戏，其容量虽然不高，但其内容却十分丰富多彩，不单是游戏的各方面效果不俗，更精彩的是游戏中有多种作战模式，可令游戏者在闯关的过程中不会感觉重复，每过一关都会有新鲜的体验。

2118 年，人类战胜了自己制造的具有高度智慧的类人物体——新人类之后不久，又出现了新的危机。有人用时间机器回到了 73 年前，其目的及身份均不明，维和组织 EXO 便派出三名时空警察前往调查。究竟是谁在搞阴谋呢？故事就从这里开始了。

### 游戏共有三种不同的模式！

#### 2D 射击模式

这款游戏共有三种不同的进行方式，分别为 3D 射击、2D 射击及双人对战。游戏共分 21 小关。

2D 射击模式中的画面很精致，只是操纵方法不易很快掌握，具体的操作方法：用上下调整射击方向，左键为蹲下，可躲避上方的敌人。在后几关中要注意攻击关底的有效位置。

[游戏截图：2D射击模式画面，编号9]

▲瞄准时机用 A 键防御即可避免受到伤害。

[游戏截图：中央射击场景]

▲按住 B 键可看出威力很大的火炎，但也要灵活应用。

[游戏截图：关底场景]

▲一般情况下，关底用火炎打较好，不必理采它的攻击或利用防御即可。

#### 3D 射击模式

在 3D 射击模式中，如何能躲避迎面飞来的敌人，是关键所在。其中一些自杀性的飞行器一定要干掉，否则它会自动向你撞击，而陨石、飞雷不必去打。有些关底虽然枪很快，但子弹有显示，只要赶快避开即可，万一不幸被其跟上，用快速移动（按 C 再按方向键）可以闪开，另外要注意补充体力。

[游戏截图：3D射击模式画面，侧边注释：你着准点，的机器人，千方百计的瞄准对方，全瞄的机关到侧面]

#### 对战模式

这款游戏虽然是闯关型的，但其中有大量的对战关，更可以两个人进行对战，十分有趣。下面将战斗机器人的招式介绍给大家：

● 机器人着地时，按 B 键出现准星，可调整枪的方向，再按 B 键可开枪。

[游戏截图：对战画面，编号B和G]

▲即使在倒地时也可以攻击对方。

● 下 + B 键为俯身前冲。

[游戏截图：对战场景]

▲虽然这招不太好看，但却是十分有效的。

● 当机器人双脚离地时，按 B 键即可开枪，按住不放还可撂出三发火箭弹，但只能向正前方发射。

[游戏截图：城市背景对战，编号B和G]

▲一下发射三枚火箭弹，让机器人纵向移动可使攻击范围加大。

[游戏截图：外部场景对战，编号4和G]

▲抓紧时间连续进攻使对方措手不及。

● 空中常近敌人按 B 键可把敌人抓住，再按 B 键便可击打敌人。

[游戏截图：空中抓取画面]

▲把敌人抓到空中可以白打，还可以进行连续攻击。

● 空中下 + C 键为从上至下的俯冲攻击，但如果打不中反而会趴在地上。

[游戏截图：俯冲攻击场景]

▲前冲攻击速度很快，但很缓危很大，要小心使用。

● 如单按 B 键为轻拳，A 为重拳，当对方开枪时 A 为防御。
● 当机器人倒地时，按下键可以把敌人蹬出。

[游戏截图：摔技场景，编号F和B]

● 除了倒地时，按上 + C 键为空翻踏人。前 + C 键为前冲攻击。后 + C 键为背后闪击。下 + C 键为自动倒地。

[游戏截图：空翻场景]

▲当对方倒地时用这招可对其造成重创。

[游戏截图：室内对战场景，编号3和4]

▲背后闪击一定要跳好距离再发。

另外，在与电脑进行对战时，有一特效必杀技：如你想用枪将电脑干掉是非常难的，但只要平时蹲下，当对方飞起开枪时冲过去，看准时机用重拳将其打下，再飞过他头部所在的方向，用重拳打他，胜率会大大提高。

[游戏截图：城市废墟对战，编号5和B]

▲脑是最要害之处，从这个角度用重拳攻击电……

[游戏截图：关底攻击场景]

▲这就是 EXO 流秘奥义——兔子蹬鹰。
