## QUAKE

经常有这种事发生：某某游戏的开发组为了游戏推出后不让玩家失望，一再推延发行时间，玩家们给这些游戏冠以“传说中的游戏”之名，如《魔眼王座》（STONE KEEP）（已发行）、《三国演义 II》等。ID 小组的《QUAKE》也是“传说”之一。ID 声称《QUAKE》将给全球的 DOOM 迷带来巨大震撼，为了保持摆在他们的神秘感，ID 曾说过给大批 ID 小组的粉丝批版本的机会，也使许多人不再把注意力放在 QUADE 身上，ID 小组见形式不妙，在道不可已的情况下，在 INTERNET 上推出了一个“QUAKE 网络问题与硬件兼容性测试版”，本人粗粗玩了两遍，没有太新鲜的感觉，这并不说明《QUAKE》将会失败，因为 DESCENT 虽然是真 3D 的，但造形简单，贴图粗糙，而 QUAKE 中的任何物品均是贴图细腻的 3D 造形，就连粉碎的尸首都是 Polygon 的，加上 ID 小组高超的光线运用技术，QUAKE 的世界看起来是那么的真实立体。为了加深真实感，QUAKE 中还加入了许多特殊的视觉效果，如玩家在上坡或上楼梯时，会发现视角自动变成仰视，造成一种上行的感觉，（一般人上行时好象都低头看脚下，造成向下的弓腰纹身）；又如潜入水中，由于水的折射，眼前景物会扭曲变形（试版中只有 320×200 的模式十分平滑），移动速度也会减慢。现在，QUAKE 的发行已经进入倒计时阶段，希望它不要成为“传说中的失望”。据悉一下，QUAKE 要求 486—66，8 兆以上内存。

## STRIFE

又一个 DooM Too？怎么搞的，96 年刷过了三个月，一下子推出七八个 DooM Too 游戏（影武者、DESCENT II 等因资料不足无法向大家全介绍），不过，换了个别几款实在没跑以外，大多数还是有创新的。这个 STRIFE 也是这样。现在的 DooM Too 大致分为两派，一派是以 ID 为首的，使用的是 ID 开发的游戏引擎；另一派是以 AP-POGE 为首的，使用的是 3DREALM 的引擎。STRIFE 相对 ID 派的一员，游戏背景比较奇特，人们穿着的是欧洲汉式的服装，装上是“特警判打”与“ROBOCOP”式的武备，内容大概也是什么反军机武邦及势力这类的老套情节。图形方面也只能说 SO—SO，音乐更是一般般。但是，STRIFE 与众不同的是，它结合了 DooM Too 与 RPG，NPC、敌人都会原地立正、一动不动，你可以上前交谈以获取情报，也可以做然贸然武、充点武器、遇上官兵成要人，立即切换成全开对话游离画，对话采用选项方式，说不同的话，对故事的发展会有截然不同的影响；但只要在市区开一枪，就会引来大量敌人，在战区状态时，不用说，就是纯粹的打打杀杀。如果您是个不折不扣的 DOOM 迷，又对以前的游戏腻了，（STRIFE）还是个不错的选择。

两种游戏风格，战斗不再是仅仅为了找到出口，能不能完成任务不再是身手被捷就行了，还要用大脑思考。在游戏中分为市区与战区两种状态，在市区状态时，只要你不开枪，所有的圣殿骑士团的七骑士之一荣德迪，被召唤出十四世纪时的记忆不仅仅是要救出自己的女友，还被化解于圣殿骑士团的数千世纪的思想恩怨，游戏的画风来来了《鬼屋魔影》系列的一贯风格，不过 Polygon 的人物造型似乎有了少许进步，而且颠覆灌流的加入了一些 3D 的动画，背景、动作是《鬼屋魔影》系列所标榜的，游戏的困难度可以说，是挑战了难题，仍然需要玩家发挥自己的想象力才能解开谜关。该游戏对硬件要求并不太高，只要一台能玩《鬼屋魔影》的机器都能胜任，但是，游戏的剧情不会影响到中文玩家的冒险呢？说实话，台湾的“智超科技”已经受托把此游戏化，《鬼屋》还们现在更做的只有拒好，然后祈祷（KNIGHT CHASE）在大陆发行吧！

## RAYMAN

横版的动作游戏，只要做得好，永远都会受欢迎，象《阿拉丁》（狮子王）《漫画地带》（炫超战士）等。现在，又一款移植自次世代游戏机（在 PS 与 SS 上均有）的超平面游戏——《RAYMAN》登场了。《RAYMAN》是由法国 UBI 软件公司发行的，法国的卡通一向轻松、幽默、神奇风光，这款《RAYMAN》也不例外。主人公 RAY-MAN 是只有头无身双腿，瘦瘦身子，大嘴子，香蕉皮般的头发的家伙，为了拯救自己友爱的家园，RAYMAN 开始了自己飘荡和飞翔的旅程，但每过一大关，仙女都会赠给 RAYMAN 新的能力，如 RAYMAN 的主要攻击方式——飞拳，还有在地图上上，甩发当飞镖，还有乘着六爪飞，游戏共有六大关，每大

关有分成几个小关，玩家可在地图上选择自己的行进路线，为了不让玩家玩得太腻，（RAYMAN）的画面实在叫人着迷，大棒了！多层叠加，即时变形、扭曲多种图形效果都给出现，人物的动作如刺激的极为生动，比如让 RAYMIN 抱地不动，他也会不时地左右晃动脑袋或对着屏幕说“HEY！”正如果 RAYMAN 吃到水珠（类似超级马里奥中的金币），水珠会变成无数花晶晶的碎环四处飞溅，动人的音效也是《RAYMAN》的一大卖点了。《RAYMAN》的音效制做似乎用到了 Qsound 技术，使一般的立体声变成了有环绕感觉的环绕立体声，并且《RAYMAN》中的许多语音都是由真人配音，听起来十分清晰。《RAYMAN》是以 CD 形式发行的，CD 上有 90 兆左右的数据，以及用来作为 BGM 的音乐，一个模拟动作游戏能做到这么大（试玩版仅一久玩就用了 8 兆），一定是的不可不玩的好游戏。如果说《仙剑奇侠传》是中文 RPG 之王的话，《RAYMAN》就是 PC 横版动作游戏之王！
