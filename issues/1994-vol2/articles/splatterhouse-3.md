# 鬼屋 III

> 栏目：世嘉世家 ｜ 刊页 27 ｜ 无署名
> MD 版《鬼屋 3》（スプラッターハウス PART 3 / Splatterhouse 3）游戏介绍

---

## 正文（PDF p.29 / 刊页 27）

这是一个如恐怖片一样的动作游戏，前两集都受到玩家欢迎，这一集与前两集相比有很大变化，主角RICK的动作，除了过去的拳击、踢腿之外，增加了扭打、摔角等技术，还可以使用各种新式武器，进一步加强了格斗本色，画面也增加了深度，可以上下左右移动。各版的构成像迷宫一样。

**故事内容：**

自从二代中RICK救出杰妮芙已五年了，RICK和杰妮芙结婚了，并生下一子叫迪比特，一家人过着幸福的生活。但有一天，杰妮芙和迪比特都被怪物捉去，"地狱阎王"又出现了，战斗吧！RICK。

看似简单的拳击，若连续打出，破坏力大增，变成upper cut。

### 运用多姿多彩的攻击方法

由于攻击方法增加，与怪物的作战幅度亦扩展了。虽然仍以拳击为主，但连续攻击可有各种动作变化，打击力量也倍增，紧握对手后攻击也是多种多样的。

### RICK凭着面具的力量变身

这一集特地做了变身攻击，只要取得落在各版中的超能石，就可在画面下方的POW中储存灵力，有了灵力，可随时变身，攻击力和防御力都会提高，还有强力的变身攻击招法。灵力一旦消失，RICK会恢复原样。

<!-- 截图说明：使自己的内脏硬化，并将其突出部分用以攻击，此时完全无敌 -->

### 有用的武器仍健在

前集中，只能在特定版中使用武器，现分别在各版均设有两种武器，而且可以拿着走，使用多次。武器的威力很大，对过关大有帮助，掉在地上的武器，还会有拾宝猎人帮你捡回来。

### 清版的时间很重要

各版均设有清版时间，时间不同，故事的发展也会改变。例如在第一版，在预定时间内清版，可将杰妮芙平安救出，但假如超过限时，杰妮芙就会死掉……。另外，根据清版时间不同，还设有称为"X版"的BONUS版。

## 编辑备注

- **无署名**——本文未见作者署名
- **游戏信息**——《鬼屋 3》（スプラッターハウス PART 3 / Splatterhouse 3），Namco 1993年发行，MD 独占。系列第三作，也是最后一作登陆家用机的作品
- **“RICK”**——原文一处作“RLCK”（L与I混淆），应为“RICK”。主角全名 Rick Taylor [?]
- **“杰妮芙”**——即 Jennifer，Rick 的妻子。Splatterhouse 系列女主角
- **“迪比特”**——即 David，Rick 与 Jennifer 之子。日文原名デイビッド，此处从日文音译为“迪比特”而非标准英文译法“大卫”/“戴维” [?]
- **“地狱阎王”**——指游戏中的最终 Boss，即 Evil One / The Dark Stone 等邪恶存在
- **“清版”**——港台用语，等同于“通关”/“过关”。全文多处使用港台表达（“各版”“幅度亦扩展”“甚为”等），文风与苍狼与白鹿一文类似，疑同出自港台游戏杂志转译
- **“upper cut”**——英文原文保留，即上勾拳。连续出拳后自动变为上勾拳的连招系统是本作特色
- **变身系统**——收集超能石（Power Stone）充满 POW 槽后可变身为强化形态，这是 Splatterhouse 3 的核心新机制。变身后“使自己的内脏硬化，并将其突出部分用以攻击”描述的是变身后的特殊攻击动画
- **时间分歧系统**——本作最大创新：每关有时间限制，是否在限定时间内通关会影响剧情走向和结局。共有四种结局（Good / Bad / Worse / Worst）
