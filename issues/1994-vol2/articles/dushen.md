《赌神》——赌中战神。《赌神》是个难得的摆脱了"常规"游戏单面强调战斗动画束缚的新型游戏。从整体上看，是一个以完整故事为主体的冒险型的角色扮演游戏（简称RPG游戏，这类游戏在计算机游戏中较常见）

在这个游戏中，战斗并不象《街霸》那样血淋淋的格杀，但一样扣人心弦；进攻与冒险也不象《魂斗罗》那样横冲直撞，但这种步步为营的"天涯之旅"也一样艰苦，一样浪漫；也许比起《三国》来，"天算"较之"人算"相对多了一些，但综合的逻辑又远非"强手棋"这一类游戏所能比拟。

这并不是一个很长很长的故事，但是它发生在很远很远的地方……

那是在塞兰多，一个善良人的乐园，他们敢于战斗，但厌恶战争，他们懂长魔法，但那些魔法只用来保卫家园，用来惩罚那些为非作歹、不愿与和平共存的恶徒。

不杀生是他们行事的一贯准则，他们甚至善良地认为哪怕杀死罪大恶极屡教不改的恶魔也是一种罪过。他们没有想到，十足的坏蛋是不会考虑他们这一份善意的，这样做的结果只能是给顽冥不化的坏蛋一个卷土重来的机会。

幸而，天地间还有正义在。

<!-- 截图：游戏画面，角色头像对话框 -->

塞兰多拥有四个勇士：梦想成为最强者的斗士法依塔，把生命当作一场冒险的小矮人特瓦夫，聪明伶俐的小精灵艾尔夫和念念不忘名利的野姑娘哈夫，虽然他们都不是土生土长的本地人，但在和蔼可亲而又见多识广的塞兰多国王磨下，早已和这里有了不解之缘。

这时，世上不安宁的因素渐渐露出端倪，大魔王哈德斯，就是那个号称冥王的哈德斯，给太一世界留下了

第二，对手的力量也有明显加强，不过这倒不必过份担心，因为不久四位勇士也能变得更为强大。

想法：其一：充分利用每一眼泉水补充体力魔力。（我想你到现在已经知道怎样在泉水边逗留了吧）其二，战斗之前最好弄清楚怪狗们的弱点和强处（譬如某种魔法对某种怪物最有效，或是某种怪物力量骰子相当大，但魔法骰子却很差劲）其三：出征有两类目标：是想攻击敌人大本营还是想提高水平如是前者，那么一路上就应该尽量保存，增加体力，可不打的仗就不打（用逃跑命令），如是后者，那么就应该尽量多地战斗。其四，最好派有能力使用试骰或考尔般的勇士上阵。

## 编辑备注

- **作者"陈江"**——非编委成员，为外部撰稿人。署名位于刊页29右上角
- **游戏原名不详**——《赌神》为中文汉化版标题，原版 FC 游戏名称不详。根据卷首语"盼"中提及"一家公司将于94年初陆续推出全部翻译成中文的《赌神》"，应为早期汉化组（疑为外星科技）的中文化作品。"赌神"之名取自游戏核心的骰子赌博式战斗系统，也可能借用1989年周润发同名电影的知名度 [?]
- **角色译名源自英文**——四位主角名称为 D&D 风格种族/职业的中文音译：
  - 法依塔 = Fighter（战士）
  - 特瓦夫 = Dwarf（矮人）
  - 艾尔夫 = Elf（精灵）
  - 哈夫 = Half（半身人 / Halfling）
- **Boss 译名源自希腊/西方神话**——
  - 科普林 = Goblin（哥布林） [?]
  - 哈德斯 = Hades（冥王哈迪斯）
  - 麦迪尤 = Medusa（美杜莎）
  - 库拉肯 = Kraken（克拉肯/北海巨妖）
  - 雷巴洛德 = 疑为 Balrog（炎魔） [?]
  - 乌阿布/乌阿尔 = 不详 [?]

- **地名**——塞兰多（Cerandor）、普阿兰多（Puarando）、阿斯兰多、卡斯特兰（Castlelan）等，均为幻想世界地名
- **"磨下"**——应为"麾下"（旗帜之下，比喻部下）。"磨"疑为"麾"的形近误植 [?]

- **"疲劣"**——疑为"疲累"或"费劲"之误 [?]
- **"臊罗"**——疑为"喽啰"之误 [?]

- **记录所**——游戏中的存档点，仅在塞兰多有一处。作者多次强调一定要及时存档
- **考试森林**——STAGE 4 中的问答迷宫，回答问题正确与否决定路线。作者建议用纸记录问题答案
