## 世嘉观览车（SEGA TOURING CAR）

**机种** SS　**厂商** SEGA　**类型** RAC　**媒体** CD-ROM

对于 MODEL 2 基板的游戏来说，“完全移植”一直是士星可望而不可及的梦想。这次“观览车”是世嘉的又一次尝试，但仍然不太成功。

“观览车”力图完全保留下 MODEL 2 版原作的一些特色，因此为了保证画面的完整性，以及光源处理和后视镜等效果，“观览车”只能采用较低的解析度，画面粗劣是让许多玩家不能满意的。

速度感和流畅性方面也遇到了一些问题。也许真的是为了做出“士星史上最快的赛车游戏”，游戏中不得不采用降低画面帧数来培养加速度感的方法，流畅度便大为降低，玩家很容易就能够发现背景移动的生硬和不自然。

当然，技术上的成熟也使得赛车本身的操作感变得更加真实，这是士星版“观览车”比较突出的一个方面，此外赛程回放的处理比起“世嘉拉力”来也有了长足的进步，画面缺损的情况也很少出现。然而赛车互相超越和冲撞时的感觉又有些虚假了。

总体上说，尽管世嘉从技术上到宣传上都做了不小的改进和努力，但 SS 中后期的几个赛车作品，没有一个能够超过当初的“DAYTONA USA”，虽然那个节目看上去有很多不足。另一个让人感到不可理解的是，世嘉本部门在技术上的努力似乎是不够平均的。因为就在“TOURING CAR”让人失望的同时，士星也有不少制作严谨的优秀三维游戏推出，这不能不让人感到一些疑惑。

## 索尼克赛跑（SONIC R）

**机种** SS　**厂商** SEGA　**类型** ACT＋RAC　**媒体** CD-ROM

这不是 N64 的游戏？！至少也是高水平的 PS 节目！这就是笔者第一眼看到“SONIC R”时的第一反应。

士星在近年作品中有不少节目在三维方面有了突破性的进展。TECMO 的“DOA”是一个，“SONIC R”也是一个。而且从效果上来说，“SONIC R”更为出色。

SONIC TEAM 所制作的游戏之所以常常出人意料，主要原因就在于他们经常能够最大限度的发挥硬件的机能，或者采用一些特殊的处理方式来掩饰主机的短处、夸张主机的长处，“SONIC R”的标杆表现正是与这种努力分不开的。

在三维游戏中，使用二倍速光驱的次世代主机常常由于数据传送上的问题发生远景图像突然跳出的情况，这在 PS 和 SS 的赛车游戏中常常可以看到，同时也是发现次世代游戏如何比不过 N64 卡带游戏的一个方面。而在“SONIC R”中程式的设计者巧妙地使用了半透明机能，使远方的景物产生从淡雾中淡出的效果，相当完美地解决了这个问题，这不能不说是一个创举。

由于使用了高解析度，“SONIC R”的画面相当清晰，色彩搭配非常协调，不仅赏心悦目，更使人产生了光源处理后的实体感觉（实际上 SONIC R 并没有使用光源处理）。超高速画面高速移动的情况下，画面缺损出现地很少，这不能不说是土星软件的一大进步。游戏性和画面合二为一，音响效果也非常出色。

正是由于看到了这个“SONIC R”，笔者才会对“TOURING CAR”产生更为失望的感觉。玩家的推论是简单而直接的：如果 SS 的赛车游戏都照着“SONIC R”的路子走，何愁没有佳作出现？当然对程式设计者来说，也许其中有很多不能逾越的技术问题，也许也有本身的不少原因。这就不是我等能够理解的了。

## 魔法学园·露娜（魔法学园·LUNAR）

**机种** SS　**厂商** GAME ARTS　**类型** RPG　**媒体** CD-ROM

同“银色之星”一样，“魔法学园”也是以丰富的动画过场见长的 RPG。只不过竟是“外传”，如果以过瘾为目的，自然是不能得到完全满足的。

人物的设计上继承了原作的风格，仍然是非常可爱的。以独立的篇节为单位的情节设计则令 RPG 的感觉，使人仿佛回到了 AVG 的世界（事实上这和故事的设计方法非常类似当初 MEGA CD 的著名 RPG“魔法少女 UPS”有点接近的嫌疑！），而故事的节奏却把握的并不是很好。

客观地来说，“魔法学园”基本上以展示多姿多彩的魔法为主题的游戏，自然少不了展示多姿多彩的魔法，然而“魔法学园”在战斗方面却有点问题。过高的敌方遭遇设置和过低的逃脱率容易使玩家产生厌烦的感觉——“银色之星”显然就更加完善一些。

总体来说，“魔法学园”基本上只相当于一个小品，是一个介于“LUNAR”和“LUNAR 2”之间的调剂品。笔者至今难以忘怀完整版 MEGA CD 版“LUNAR 2”后的那种感动，也期盼着新一轮冲击到的到来。

## 盗墓者 2（TOMB RAIDER 2）

**机种** PS　**厂商** CORE　**类型** A·AVG　**媒体** CD-ROM×2

从最先的南美，到如今的中国，从当初的解谜，到今日的动作。这就是“TOMB RAIDER 2”给笔者带来的感受。

画面同原版作相比确实有了改进，但不是很大，动作成分却有很大的增加。开场的动作要求不亚于一代结尾的部分。而相比之下，几乎没有什么过难的机关需要玩家用脑子去开启的，笔者对此持保留意见。

在使用 3DFX 图形卡的电脑上看这个节目，效果要比 PS 版出色得多。次世代机引以为豪的图形处理现在也遇到了来自电脑的挑战，看来新主机的出现已是大势所趋。

使笔者感到不满和无法理解的是，在“TR2”中，许多地方的完成顺序是不可替换的，其完成的方法和道路也常常只有一条。这就使得整个游戏的可玩性受到很大程度的影响，相信是不会有多少人愿意再来第二遍的了。对于 AVG 来说，多线路和多情节应当是主要技巧之一，而“TOMB RAIDER 2”的处理方法显然更对应迫近流行的玩家，其经典 AVG 的地位已有所下降。

## 血腥咆哮（BLOODY ROAR）

**机种** PS　**厂商** HUDSON　**类型** 3D-FTG　**媒体** CD-ROM

HUDSON 的“BLOODY ROAR”，基本上是一款 PS 版的“格斗之蛇”加上一个兽化的噱头。画面方面不算出类拔萃，不过有了光源处理的烘托，人物的实体感官是比较明显的。游戏的最大魅力，还是要归功于它的爽快感。

在 PS 上制作动作游戏，由于受到手柄的限制，要想突出爽快感是比较困难的。南梦宫的几个作品算是比较成功的，在此之外，也只有这个“BLOODY ROAR”可以相比拟了。

在吸引其他公司长处、推出首张动作游戏的尝试方面，HUDSON 比较成功，这也许与公司的规模有关。风光正健的公司大概是常常不愿意向别人学习的，所以出品的效果就会多一些傲慢，少一些出色。而 HUDSON 的日子并不太好过，所以磨通道也也取长补短。

笔者最感兴趣的是游戏中的人物设计，不同人物与不同动物的对应非常有意思，造型也独特而酷。是另外一个可同“格斗之蛇”相比的方面，“BLOODY ROAR”在对付地面高和扫扫地面，加上扫下檐台方面，“BLOODY ROAR”做得比较粗糙，使人感到不够细腻。

## 第一猎手（EIN HÄNDER）

**机种** PS　**厂商** SQUARE　**类型** STG　**媒体** CD-ROM

史克威尔在 RPG 上誉了个褒贬参半，在 SLG 上又是很高不断（前线任务 2），这个“EIN HÄNDER”又是一个被很多玩家视为失败的作品。

游戏力图造成一种出色的段三维、真二维的效果，果真巧夺天工、把游戏的流畅性和游戏性搞得一团糟，史克威尔没有掌握到射击游戏的“快感点”便匆忙出手，效果不如人意也是在意料之中。

自从加入次世代行列中以后，史克威尔给人的感觉是 CG 极极超源亮，包装超超超出色，广告超超超诱人，游戏内容则是超超极规相人。笔者就在铺天盖地的“EIN HÄNDER”宣传上了当的，于是打起精神，告诫自己今后也要小心。

史克威尔的作品重视在大处的感觉，而细处常常为人们所忽视。比如“FF7”中背景美得绝世，人物加工却常常比单线条一般，甚至连连光源处理来支持。说这样的游戏“XX 绝对不可能移植”，有连相信？这个“EIN HÄNDER”就更进一步了，索性连以难的片头部没有做极。工作之粗糙可想而立。

平心而言，“EIN HÄNDER”中还是有一些新鲜的创意的。比如背景的巧妙转换、但是这些创意难为及在史克威尔部分的不小小节中，已经难以为人所见了。游戏还有着本质的东西，光靠名头和媒体的炒作是不行的。

## 恶魔召唤师 2·灵魂租借者（SOUL HACKERS）

**机种** SS　**厂商** ATLUS　**类型** RPG　**媒体** CD-ROM×2

自从真·女神系列转移到次世代主机以后，游戏整个的风格和系统都有了不少改进。而从主机的特性和已有几个节目的质量来看，该系列也更适合在士星上移植。可能这也是 ATLUS 为何一直乐于 SS 而不破的原因之一吧！

同“D 之食卓”、“生化危机”等游戏的那种强烈刺激不同，“真·女神”列所具有的是一种较低调的、阴暗森森的恐怖感。这种感觉就向一种气息弥漫在玩家的每一个行动之中，通过加强环境的渲染使得玩家从心底里产生恐惧（注意不是恐怖！）的感觉。这种做法在 SOUL HACKERS 中表现得更为突出，这也是笔者更为喜爱这个游戏的主要原因。

从整个系列来看，SOUL HACKERS 无疑是最愿为玩家考虑的。初学者比较容易上手，适当的地图配置使得玩家不至于在三维迷宫中迷到过于单调和无聊。魔神的合体又有了新的变化，与主角之间的对话也有了很好的写实取向，有意（对不懂日文的玩家来说这是一大遗憾！）。CG 演示画面的质量有了较大程度的提高，在数量上也有所增加，总体效果相当不错。

实际上，如果多推出一些象 SOUL HACKERS 那样的作品，士星还是大有溯源力的。SS 的关键就是在选择正确的游戏，而不是匹力不讨好。

## X 超人对街霸（X-MAN VS STREET FIGHTER）

**机种** SS　**厂商** CAPCOM　**类型** FTG　**媒体** CD-ROM

4M 就是 4M，加上了这块 RAM 卡就是不一样。

从这个基础上来比较士星和 PS 显然是不平等的。但士星也正是因为这个独一无二的优势，占据了二维动作游戏的大半江山，这也是 PS 不愿承认的。在新 4M RAM 卡的支持下，士星移植所有的二维对战游戏都将变成一件非常容易的事情：读盘时间快得简直让人望尘不到，所有动作的完整移植……这对于 PS 来说几乎是不可能的。

各种游戏中的主角进行混战，这是 CAPCOM 最近比较热衷的一项活动。前一阵子美国漫画英雄与 RYU、KEN 打得火热，现在又忙到美国队长这一批，看来 CAPCOM 已经可以以妙冷饭的大师了。

话虽如此，但是本着 CAPCOM 一贯的水准，这些“荒唐对战”的游戏在流畅性和动作游戏的感觉方面都还是比较出色的，画面也很细腻。冷饭应该说煮得如何，如果是看动作游戏角色来说，如果不是能够看到如此想望在电玩主角，CAPCOM 在沉寂了那么多时候之后，是否也会重新起动一个动作游戏的风潮呢？这是所有动作游戏玩家都非常关心的。

4M 卡的出现给大作游戏的玩家带来了新的希望，相信 SS 以后推出的 VAMPIRE SAVOR、SF3 和 KOF 97 都会有更为出色的表现。

## 七风岛故事（七ッ風の島物語）

**机种** SS　**厂商** ENIX　**类型** A·RPG　**媒体** CD-ROM

虽然加盟了 SS，但是游戏不好说这是“七风岛”给人留下的最大印象。

从实际情况和一贯的做法来看，PS 比 SS 更好的新产品试验基地。ENIX 这次选择 SS 来试验这个“七风岛”，不知究竟出于什么动机，反正是没有给人留下好印象。

因为只知道 ENIX 是个还可以看着的 RPG 制作商，所以才会买回这个游戏，但是很快就晕乎了上了光圈。相信不光是笔者，许多国内玩家也不会对这个节目有太多兴趣的。

故事不引人、画面不出色、角色不可爱、操作系统晦涩，是“七风岛”最大的失败之处。也许这个情节，和故事的作者以及游戏的监督对于日本玩家来说是很熟悉，而且认可的，但仅仅只是对于日本人而言。别人必也无福消受。

事实上以 ENIX 制作经典 RPG 的水平，和 SS 在三维机能上的贡献，双方完全可以合作推出一个效果非常出色的经典二维 RPG，但现在的游戏并没有这样做。也许是因为二维已过于落伍？也许是因为 PS 已经弃出品了？也许二者兼而有之。大势所趋，有些事也是很无奈的。

贵者唯一还有一点希望的，是 ENIX 明年春天推出的 SS 版足球 RPG 节目，ENIX 总要在士星上出一个“务正业”的游戏了，但愿它不让人失望。

## J 联盟·职业球会的创造！2（Jリーグ・プロサッカークラブをつくろう！2）

**机种** SS　**厂商** SEGA　**类型** SLG　**媒体** CD-ROM

足球经理类的游戏，从电脑到游戏机都推出了不少，但是最出色和成功的，当属这个“职业球会的创造 2”。

该游戏的系统是所有游戏中最简洁、最完善的。玩家非常容易上手，经过掌握后，很快就能使游戏产生成就感。在对各个队员不同性格的设置方面，也是非常逼真（比如刚加盟的外籍球员在往往难以得到本队队员的认可，等等）。比赛重点突出战略思想和战略意图，队员的个性显得极为突出。游戏竞争也还设计了观众的心态与场内队员之间的影响，不可调不不细致。至于竞赛时的解说，动作的细腻处理和重放等效果也是非常真实和出色，效果已经超越了“V GOAL”系列和街机版的“VR STRIKER”。

游戏玩家出生地，队员 10 时前后，会感到日本玩家很有亲切感。创造球员的设计更是能够激发玩家的感情与游戏中，简直是因某节目的一大创举（笔者就曾“创造”了一个非常逼真的“忍志敲”！）。通过游戏，球迷玩家也能够了解经经历日本职业赛事的种种苦辣，相信以往在看台上对着所谓教练大喊“下课”的大口也算声了！

同“V GOAL”系列一样，“职业球会的创造”把令人失望的一点想到了汉化版。如果游戏能够改为面向中国的中级联赛，该会有多少球迷为之热血沸腾！
