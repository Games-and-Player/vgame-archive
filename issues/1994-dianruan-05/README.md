## 本期特征

本期为 1994 年末刊（年度终刊），结构与前几期显著不同：
- **无闯关族宣言**：取消独立卷首语页
- **目录仅 1 页**：目录·版权页·编者的话合并于 PDF 4 单页（前几期为 2 页目录 + 独立宣言页）
- **内容从刊页 2 起**：前几期从刊页 4 起
- **后半全漫画**：刊页 33-64（32 页）全部为超级街霸Ⅱ漫画连载
- **无 GAME BAR / 新卡快讯 / 闯关族的家 / 电刑室手记**
- **二手货市场为别册附录**：单独装订，不在本 PDF 中
