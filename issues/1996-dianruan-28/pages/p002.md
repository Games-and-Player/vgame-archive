# 乱舞势力VOL.3——风云SUPER TAG BATTLE＆侍魂天草降临

> 乱舞势力 VOL.3 / PDF p.2 / 《电子游戏软件》GAME CAMP 96·11

这次“势力”登场的频率快了吧，还满意吗？相信看到这篇稿子，也许其中主要介绍的异族格斗游戏《风云SUPER TAG BATTLE》（又称《风云默示录2》）不出意外的话已经在日本上市了。

比起前作《风云默示录之格斗创世》来说她又会带给你一种新的格斗感觉。以“真狮子王复活”为号召的她此次撤消了独特的二重战线系统，不免有些可惜。再次召开的兽神武斗会依然是10名角色参战，但人物与前作不同，将嘉依和利克拉换成了一对帅男靓女，也就是新的主角。那位真狮子王还会出现的。

这回加入的新的战斗系统为同时可以4人对战，即一方中的一人体力不支时，另一人可上场，而退下的人可利用连打来回复体力，这样一场还不尽的宿命格斗就此展开了……

也许由于资料不太详尽，以后本栏目会有续报，另外SNK已决定97年推出“侍魂”的新作《侍魂之天草降临》，具体情况仍不太清楚，有了NEWS我一定会无私奉献的，GO！梵天闪光阵……
