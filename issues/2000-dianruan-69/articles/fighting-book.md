### 一、通常技及特殊技的CANCEL：

此类CANCEL技大致可分成三个方面：

①可CANCEL接驳必杀技及特殊技。

②通常技后使出才可CANCEL。

③可CANCEL接驳必杀技（含超杀）。

其余则不可CANCEL。

下面对以上三类进行分析。

**1. 属于第一类的通常技包括：**

●真.京：近立任意键，远立A，下蹲A/C/D。●庵：近立A/B/C，下蹲C，超重击。●京-1：近立任意键，远立A/B，下蹲A/C。●京-2：近A/C/D，远立A/C，下蹲A/C，小跳A。●K'：近立任意键，下蹲任意键。●MAXIMA：近立A/C/D，下蹲A/B/C。●MAXIMA：近立A/C/D，下蹲A/B/C。●红丸：近立或下蹲任意键。●真吾：近立或下蹲任意键，超重击，远立A。●特瑞：同真吾。●安迪：近立A/B/C，下蹲A/C，超重击。●东丈：近立任意键，下蹲A/D，远立B。●舞：近立任意键，下蹲C/D，空中超重击，跳D（限通常跳跃）。●良：近立任意键，远立A，下蹲A/C/D。●罗伯特：近立B/C，下蹲A/C/D。●尤莉：近立或下蹲任意键，远立A。●琢磨：近立A/B/C，下蹲任意键。●莉安娜：近立任意键，远立C，下蹲A/C。●拉尔夫：无此类CANCEL技。●克拉克：近立B/C（第二HIT），下蹲A。●WHIP：近立任意键，下蹲B/C，远立B/D。●麻宫：近立及下蹲任意键，跳A/C，过立A/B，超重击。●椎拳崇：近立任意键，远立A/C（第2 HIT），下蹲A/C/D。●镇远斋：近立A/C，远立C，下蹲A/C。●包：近立A/B/C，下蹲A/C。●KING：近立任意键，下蹲A/C。●玛丽：近立A/B/C（二段皆可）/D（限第2HIT），下蹲A/C。●香澄：近立任意键，远立C/D。●香绯：同香澄。●金家潘：近立任意键，下蹲A/C/D。●陈可汉：近立A/C/D，近立B，下蹲任意键。●蔡宝键：近立A/B/C，远立B，下蹲A/C。●全助：近立A/C，下蹲A/C/D。

**2. 属于第二类的特殊技包括：**

●庵：→B（二段皆可）。●京-1：→B。●K'：→A，→B。●马克西马：→A，↘C。●特瑞：→A。●舞：→B，↘B。●罗伯特：↘B（限第2HIT）。●琢磨：→A，←A，→B，↘B。莉安娜：→B。●椎拳崇/镇元斋/玛丽：→A。●香澄：→A（两段皆可）。●香绯：→A，→B（两段皆可）。●金家潘：→B。●凉-2：→B（第2HIT）。

**3. 属于第三类的通常技如下（包括特殊技）：**

●真.京：→B（第2段），超重击。●八神：→A，A（第1段不可空振CANCEL）。●京-1：→A，超重击。●MAXIMA：近立B，超重击。●红丸：跳B（可CANCEL↓D），→B，超重击。●真吾：→B（第2段）。●特瑞：↘C。●东丈：→B，超重击。●安迪：跳A/C，→B。●舞：小跳A/C/D，空中↓A，超重击。●良/尤莉/琢磨：超重击。●罗伯特：下蹲B/超重击，→B。●莉安娜：跳B/D，小跳A/B/D，超重击。

## 我这样看对手

无异可为分为——

①菜鸟级：基本是手握摇杆四键齐拍的那种，掌握一到两种必杀技，局局输，超杀在他们看来是一种十分华丽的东西。

②低脚级：使用大部必杀技，但对必杀技的熟练程度及使用特性还有待提高，会些小连技，其人属于那种以守为攻的那种类型。不会发起主动攻势，达人也在固定使用的几个角色之中，喜用阴人的招数将CPU屁死，如97罗伯特不住使用飞燕旋风脚→→↘↓↙←一重脚逼死电脑。

③入门级：这级别的玩家能熟练掌握各种必杀，会使用一些较强连击及组合攻击，往往会先发动一番攻击，有一定的攻击套路，是属于以攻为守的那种。清楚上、中、下段技的区别及防守方法，但不熟练，有悟性，经过修炼能练好一个人物的一套最强连技。

④高手级：这就是对战中最可怕的对手了。属攻守兼备且打心理战术的那种类型，熟练掌握特殊技的特性，没有固定的攻击套路，视对手作出的攻击而作出反应，超杀等大都融入连续技中而不是轻易打出，除非是被防住之后破绽很小的或易于反击对手的超杀在一定情况下才会打出。能够使用许多角色，有时还要使用一些无赖连技。

⑤达人级：一般地，大城市不超过10个，小城市不超过两个，与之对战，不感觉可怕，因为三局下来很少有取胜机会。达人一般贵精不贵多，善用特定几个人物，但对游戏中全体角色的特性了然于胸。对战时绝少喜怒形于色。这是天生擅长对战的人，心灵手巧，猜得准打得狠，其反击机会比 + r1 + 要多出三倍。（浙江兰溪三中　王奕）。

## 关于凉的格斗......

●天师，在下在百忙之中给您报一下街机战绩。（上海市东初中 + r1 + 赵嘉祎）

←新型SONY的彩监前与服部老师交谈在为凉配音的演员是松原和也

表面上非常简单，玩起来难难难。只有防（Y），拳（X），脚（A），投（B）四键，加上强力技（X+A），共五种基本攻击方式。将上面五种配合着方向键的前、后、前前、后后、前后、后前、前后后、后前前、跑动中等9种指令，即可组合出多样的技巧。这样便可不用死背招式出法（少数例外）。

凉在接受别人传授武功时经常被告以 + r1 + 或是 + r1 + 之类的，可见招式除名称优美外，连指令输入都蛮符合真实的。下面列出通过看 + r1 + 以及 + r1 + 后凉修得的招式。

水月突（前X，武馆福原传授）

重当（前X+A，商店街小公园山岸传授）

铠通（前后后X+A，武馆密室内得到后经贵章点拔）

双刃（后前前X，严屋内发现的技书）

旋风（前后后A，骨董屋购得）

土蜘蛛（后前前A，骨董屋）

列光闪（前后后X，骨董屋）

飞燕连割（后前前X+A，骨董屋）

胧无双（近身.前后后B，骨董屋）

虎身崩（近身.后前前B，骨董屋）

影狗（跑动中Y+A）

燕旋降脚（后A，贵章传授）

龙卷踢（前前AA，TOM传授）

燕旋摆柳（后XA，陈大人传授）

影身／影刀／残月（均为近身技，前Y+B／前Y+B. X／前前Y+B，退休码头老人传授）

有了这些技巧，可以组合出复杂的战术，甚至大段连续技。另外关于投技，还有 + r1 + 一说，即对手一拳打过来时（仅限于拳），按下B键，则凉会擒住对手的手腕将其摔倒，颇有返技的味道。实战中应多加实践投技。

连续技方面，有XXXA（对70人战的持棍者尤适用）、XXA，AXA，XXXXXXX，XAAA，AAA，XX←AAA，XX→XX，XX→XA，AXXXXXX等，但被对手防住的可能性很大。

所以凉要多练习，技巧的熟练度上升后，招式动作将被强化，有些原来一段动作的会变为两段发出，如→←A。

最后，凉要学会防御，这十分重要，当敌人出招时按下Y键方为有效防御，而Y键还可配合方向键进行定向躲。实在紧急时，凉应当以R键（在此设定在跑）配合方向键四处跑动，人多时或对BOSS时尤为适用。

 + r1 + 这款游戏，在画面上的进步无疑是其成功的一个必要条件，但编者认为它主要的诉求还是在于以电影镜头的视角来将事件交待清楚，而这个进步是值得大书特书的。这样一来，莎木就当之无愧地成为一部 + r1 + 。莎木权威综合大攻略请容后述。

由于故事内容及涉及黑白两道，所以格斗要素在莎木中的位置比较突出。FREE BATTLE中，玩家不但能够体会到相当强劲的格斗力度感（强劲得甚至有些过分），平常练习时你所掌握的那些技巧，在此亦可得以完全再现，什么影身、影刀、燕旋摆柳等，都可拿来打CHAI（最终BOSS）。值得一提的是投技，莎木中凉有两个最难打的对手，一是 + r1 + 的BOSS，二就是CHAI，此二人的杀招没有别的，就是投技。而凉在对付他们时也可能展投技，双手抓住CHAI的秃头，用膝盖狠狠地顶其面门，但见对手痛苦万分地匍匐住，只有使用投技才能得到这种感觉。虽然用 + r1 + 、 + r1 + 、 + r1 + 即可轻松解决他们，但还是建议玩家深入体会各种技巧的真味。
