##  + r1 + 顺调进行！

本期的投票量保持了强劲的势头，数量超过了一万，说明大家的积极性还是很高的。另外，由于许多玩友是第一次参加投票，对于一些细节不太清楚。主要表现在书写游戏名称的时候，不但没有按照本刊给出的名称，而且不注明机种。这样的选票不但无效，而且还给统计工作造成很大不便，请玩友一定要注意。举例来说，有些玩友投票时只写 + r1 + ，令我们统计时无从下手，或者写着 + r1 + ，也属于无法统计的，因为GB上的 + r1 + 有三款游戏，所以投票时一定要写清游戏的副标题。具体注意事项可以看本刊去年排行榜栏目中所刊载的那些，只有规范的投票才能起到支持自己喜欢游戏的作用，才会有更多的中奖机会。

本期的一万多张选票中，支持PS主机的玩友仍然占绝大多数。但是这期却没有像上期一样有那么多新游戏上榜，只是原来榜上的游戏上上下下，虽然投票量大，但并没有什么新鲜感。 + r1 + 经过上期的辉煌后，已经有些力不从心，下滑一位，下期能否保持前五的位置目前无法预测。CAPCOM的 + r1 + 再次上榜，将NAMCO的 + r1 + 挤出榜外。不过，ENIX的大作 + r1 + 依然没有上榜，但所差的票数已经不多了，也许下期就会上榜。

累计榜方面，GB的 + r1 + 势如破竹，一下子就占据了第22的位置，与世界范围内的走红趋势一致。令人感到意外的是：PS的 + r1 + 竟然受到迟到的欢迎，在推出很长时间后第一次登上 + r1 + 。看来心跳中众多的美少女，令无数电玩迷倾倒。

本月日本方面， + r1 + 销量依然保持领先，并且有望仅在日本国内销量就超过500万份。可是其它厂商的日子就不那么如意了，大家看看本期的销量排行榜就能够了解了。从最新的资料上显示，SQUARE年初的PS大作 + r1 + （ + r1 + 第三个满分游戏），一周也仅仅卖了不到20万份。
