## 放浪冒险谭（VAGRANT STORY）

机种 PS　厂商 SQUARE　类型 AVG+RPG　媒体 CD-ROM

RPG豪门与SLG巨子的最佳组合——史克威尔+松野泰己。前者有FF、圣剑、超时空之钥、前线任务、PE；后者曾一手创造史上唯一的、也许还是最后的百万级SLG。组合的结果就是这款 + r1 + 。

用现在的眼光来看，游戏很 + r1 + ，整个游戏只有片头的一小段CG，但仅是这一小段CG就已经把游戏舆论弄得沸沸扬扬，那个神秘妩媚的女郎究竟是谁？很多人为了这个问题而把游戏玩了个底朝天，游戏气氛烘托之妙可见一斑。

游戏最有特点的还是它深奥的系统，深奥得有些晦涩的系统自然不会影响到它的商业成功，但对于真正的玩家来说，这才是游戏最大的魅力所在。美丽的画面和感人至深的情节固然可贵，但能使人的头脑兴奋起来的系统同样不可或缺。在系统方面能和任何超大作匹敌的 + r1 + 之所以不被称为超大作，也许就是在画面上还存在着差距，以及——

商业成功上的差距。同为松野泰己作品的PS版 + r1 + 就曾取得巨大商业成功，同样在系统上相当有深度的 + r1 + 的销售超过了百万份，但这百万用户又有多少真正将其通关呢？一个为核心玩家层制作的游戏能否被普通接受，而有才能的制作人的价值要这样才能体现，这不能不说是一种悲哀。

截止到现在， + r1 + 还绝对谈不上商业成功，两周销量只有20万。不过在史克威尔公司内也算得上颇有个性的松野泰己想必是不会太在意的，他更看重的是人们怎样在玩他的游戏。

——武汉　张凌志

## 生物危机·代号维罗尼卡（BIO HAZARD·CODE:VERONICA）

机种 DC　厂商 CAPCOM　类型 AVG　媒体 GD-ROM×2

盼星星盼月亮终于盼到了DC版的BIO，应该说 + r1 + ，给人的震撼和冲击就是和一般的游戏不同。而且游戏中电影化的场面有增无减，这无疑大大增加了游戏的魅力和感染力，个人感觉DC版的BIO还算是比较成功的。

提到一个游戏，首先不能不说它的画面。DC版BIO给人的第一感觉就是干净，没有了马赛克和拮据的感觉就是和PS版不一样。背景虽是全3D的，但并不是像 + r1 + 那样可以随心所欲的变换视角。基本是类似于 + r1 + ，到了某些地方自动变换角度。至于感觉好与不好，只能因人而异了，不过总体的感觉还是不错的。

不能不提还有个人物鲜明的到[?]划，用 + r1 + 二字形容并不过分。尤其是人物的表情，可以说是逼真如生，让人不禁PS那边仅仅只是一张2D的贴图了，这一点的确是有了质的飞跃。单就画面质量而言，已经完全可以和PS版的CG一较高低了。

但是对于游戏的其它方面我不敢轻率，首先是情节编排的十分拖拉，远远没有了当初打2代的那种张驰感[?]，而且DC版的BIO似乎已经没有了任何恐怖的气氛。起因大概是我没有受到的惊吓太多，所以现在已经习惯了吧:）比起恐怖气氛营造得最好的一代，这次的BIO到更像是一个3D的射击游戏，缺少了恐怖感的BIO已经没有当初那样吸引我了。

惯性的提升并没有给游戏本身带来多么大的改变，也许是CAPAOM太专注于画面而忽视其它的方面忽视了吧？听说N64马上就要出ZERO版了，但愿CAPCOM这次能给我们带来一个真正的惊喜吧。

——北京　卡之琳

## 牧场物语·中秋之月（牧場物語·ハーベストムーン）

机种 PS　厂商 PACK IN SOFT　类型 SLG　媒体 CD-ROM

 + r1 + ，这个系列在N64和GB上的人气度很高，拥有一大批支持者，这主要是依靠其良好的耐玩度和有趣的事件。如今这最新一作放在了PS上，效果也是相当不错的。

其实本人一直比较喜欢这种经营型的SLG，因为这其中有一大堆有趣的事件，我喜欢凭着自己的头脑去解决。本作中可饲养的家畜和相植物多了许多，有猪、牛、羊、鸡、兔等，果菜更是数不胜数。首先你必须对整个牧场进行全面的打理，分配好家畜的活动场所，将它们安定下来之后开始种花种菜，来维持生计。

游戏中另一个比较有趣的算是与横上的人进行对话了。一开始他们对你很冷漠，但随着故事的发展他们渐渐地会对你做出不同的反应，其中之间会发生一些事件。本作中的谜题游戏十分之多，什么 + r1 +  + r1 +  + r1 + 了你能想到的都有，有的当然既联[?]经得了会到想恰不到的礼物。做为牧场的经营者，你必须每天都听天气预报，以便于知道是第二天的天气。如果明天是暴风雨天气就是万万不要把家畜赶出去，否则会冻死的！另外本作对应PDA，可以与同学友进行材料交换，可对于手[?]弃内的玩友就……

总之，喜欢田园生活的朋友及闺[?]中好友可不可错过！

## 波波救难队（レスキューショット ブービーぼー）

在今年软件商相对应的是1月份的游戏低谷，业界似乎一下子平静下来了，而本月也没有什么好作品，值得一玩儿的只有 + r1 + 和这十分有趣的 + r1 + 。

这个作品是NAMCO光枪游戏的最新作，前一作是 + r1 + 。与前作相比，本作更有童心趣味，相对也不用直接操纵主角，而只需用手枪射击，由于射的都是那个愣乎乎的 + r1 + 只会一味向前走走[?]，前方是什么也是个不要命的家伙，所以我们要时时刻刻保护它。基以连环画的方式表现的，像是用简笔画描绘的，给人以一种萧瑟的感觉，这种创意很很好。至少在CG飞的今天也他能体会到一种芦荟[?]的感觉。

## SNK VS. CAPCOM最强战士顶上决战（SNK VS. CAPCOM最强FIGHTERS顶上决战）

机种 NGP　厂商 SNK　类型 FTG　容量 16M

这款游戏算是NGP掌机上少有的大作了。很多玩友也是因为这几款游戏而下定狠心购买的NGP。

因为是手掌机，所以游戏在画面、音乐方面都没有太多惊人的表现，而且我对NGP的操杆十分不适。但仅就游戏本身而言，它并没有让我们感到失望，相反它却反而给了我们一份惊喜与欢乐。

惊喜的是游戏制作虽然是SNK公司，但在对CAPCOM游戏人物的性格等方面的把握十分到位。如雅典的沉默骗骗[?]、朱、隆的乾脆利落、大贺大闲等等，可见两家公司的交流不少，这也是我们此前最关心的问题。希望CAPCOM版的正式格斗版也能这样。

而游戏虽然是格斗游戏，但是搞笑的成分很大很多。给我们带来了许多欢乐。比如CAPCOM的搞笑火引爆的几个趣事，真是笑死人。 + r1 + 连续学看前、骂、摘花草碎石、轰兔几人的挑拨技，然后说到对方面边灰[?] + r1 + ； + r1 + 则是学院[?]像可能依赖飞不出去；还要在 + r1 + 游戏里下巴满地[?]。此人乃真，满满也……

## 街头霸王～斗士之梦～（STREET FIGHTER ALPHA～WARRIORS DREAMS～）

机种 GBC　厂商 CAPCOM　类型 FTG　容量 8M

笔者一直都是CAPCOM公司《街霸》系列的拥护者，因为笔者第一次接触到的街机格斗游戏就是它了。早就听说《街霸》将有最新作登陆GBC，经历了几个月的热切盼望，终于在伴随着欣喜和失望的同时，笔者拿到了这款游戏（STREET FIGHTER ALPHA～warriors dreams）以下简称《ALPHA》）。

这款游戏保持了真正的优良品质。画面精美！就掌机而言，色多而不杂，特别要说的是，《ALPHA》的人物设定不同于用于下掌机格斗游戏常用的Q版形式，以写实的风格保持了这个系列原有的不失夸，色块运用合理，虽然较简单，但一招一式都明明白白，实际用起来看[?]，实局面还不大。

包括隐藏的VEGA、豪鬼、DAN在内，《ALPHA》共收录了15名角色。系统和菜单令保持了真实的风格，相信大家应该很熟悉。游戏设有 + r1 + 和 + r1 + 两个模式，并可对速如时限、难度等进行设定。游戏操作比较简单，以按键轻重（长短）来区分轻拳（脚）和重拳（脚）。

当然，任何东西都不可能十全十美，《ALPHA》也有其令人失望之处。游戏背景虽然较细腻，但颜色暗一二[?]，看着蹊始还是不很好了[?]。操作上不佳。游戏配乐不错，但格斗音效不佳；也在一定程度上影响了格斗感觉。此外，这款《ALPHA》几乎是完全脱自《STREET FIGHTER ZERO》，给人以炒冷饭的感觉。

——北京　般
