## 最初的别墅事件

因为在浣熊市周围，某个地方的别墅出现了一群谜之僵尸，所以R.P.D.便派遣了特殊部队S.T.A.R.S.前往调查。在大多数队员都牺牲阵亡的情况下，这桩别墅惨剧后来得到惨烈爆炸作为收场……而之前的克里斯·雷德费尔德（Chris Redfield），过后追到了欧洲对进行调查。

↑  + r1 + 事件的发生地点——浣熊市。别墅事件发生两个月后，位于浣熊市市内的伞公司研究室，发生了T-病毒外泄事件，使得感染中的人们变成僵尸并引发了可怕的生物危机。在这同时，被配属到R.P.D.的里昂和为了找寻哥哥而来到浣熊市的克蕾尔两人，便因此而相遇，甚至更被一同卷入事件中。而之后打倒了引发这次T-病毒外泄事件的始作俑者威廉·巴金博士，并救出唯一的生存者——博士的女儿雪莉和里昂后，三人便逃出浣熊市。在浣熊市的事件结束后，克蕾尔为了追寻哥哥而前往欧洲，而里昂则是进入了美国政府的情报局。

之前曾经介绍过，玩家将可在 + r1 + 中使用视讯记忆卡（VM），以及在VM上还会出现显示玩家现在状况的心电图……不知各位玩家是否还记得？这次，我们将公开其画面并为玩家们再做解说。事实上状态画面将会出现现在显示的两种画面，一个是玩家在游戏中操作角色的脸部图案；另一个则是玩家在系列中，可以通过状态画面来加以确认的体力心电图。这两者将会交替显示，相信这样一来玩家们将可省下确认的繁琐步骤，大幅降低压迫感。

### 浣熊事件

从伞公司的研究室外泄出T-病毒，引起突发的流行感染事件。

### T-病毒

生物在感染后会失去知性并凶暴化的病毒。不但感染力高，而且美关[?]会引起一定程度的生物突变状况。

### 生物危机

以病毒等做为病源体，造成特定的生物发生异变的生态灾害。

### 威廉博士

伞公司所属，开发T-病毒与其发展型G-病毒的生物学权威。

### U.B.C.S.

Umbrella Biohazard Countermeasure Service的简称。伞公司的生物危机对策部队。为了解决象是生物兵器等由伞公司制作出的麻烦而设置的部队。

## B.O.W.

Bio Organic Weapon的简称。有机生命体兵器，是伞公司利用投入T-病毒或是DNA重组等技术所制作出来的生物总称。另外像是僵尸或是虫类等，经由T-病毒2次感染而出现的不规则生物，则是不被包括在B.O.W.在内。

## 病毒与生物

伞公司制作的B.O.W.打破了现有的兵器概念，主要原因就是其制作的基础T-病毒，而B.O.W.的效果在经历了别墅事件之后，证明了它已有缺点存在了。那就是生物在接受病毒后，其本身的智慧也急剧减退……也就是因为有这样的弊病，所以B.O.W.处目前仍属于不完全的生物兵器。而之后开发出来的新型病毒G-病毒，也是种会产生高智能下的病毒，所以说不能说是人类智能的成功的结品[?]。不过，在这之中也许会有人感到意外的事实。在这个体系图中，玩家除了可以看到曾在系列中出现的生物，还可以看到确定会在 + r1 + 中登场的生物，这可是初次公开的独家新闻喔！

## G-病毒

由威廉·巴金博士所开发出来的病毒型兵器第2弹。跟T-病毒一样具有相同的基本概念，不过G-病毒有高度知能的涅摩西斯可以引发细胞等级产生急速的进化。因为这个G-病毒的研究是经过博士个人之手，所以G类型的生物也不包括在伞公司正式的B.O.W.中。

##  + r1 + 中登场的新品种生物

已经确定会在 + r1 + 中登场的新型生物有伞公司制作的 + r1 + 和 + r1 + ，此外还有 + r1 + ，是经在 + r1 + 中首次登场的谜之第3势力所开发而成的B.O.W.。这是以伞公司的狩猎者作为基础，进而开发出比伞公司更加具有战斗能力的生物。虽然这些生物跟 + r1 + 系列登场的B.O.W.在外型上十分相似，但是能力却大不相同。当玩家与它们碰头时，千万要注意它们彼此不同的特性。

### Gulp worm / Sweeper

↑ 以蚯蚓的遗传因子配上T-病毒制作而成的B.O.W.，通常会躲在土里，待猎物方接近时再加以袭击。

↑ 狩猎者改的爪子具有毒性，是攻击力很高的B.O.W.。比起自行行动的那种，它的行动可以被控制。

### NE-α

由伞公司欧洲支部所制作完成的寄生虫生物。如果我们将这种生物寄生到以T-病毒制作出来的B.O.W.上面，B.O.W.便可受到一定程度的控制。关于这点，从涅摩西斯获得可以使用自独裁者那移植而来的重武器则能来看，可以得知这项实验是成功的。

### 狩猎者

由人类的卵子和T-病毒组合，造成DNA变化进而制作出的 + r1 + 系列代表B.O.W.。在各个品种中，量产类型的狩猎者β和在欧洲研究所中由人类和两栖类的DNA所制作出的狩猎者γ……都是为了各种不同的目的而制作出来的。

### Bandersnatch

↑ 稍微减小兵器性能，重视实用性的独裁者。这个是为了量产化而作出的试验体。

在这里，由于狩猎者改的制作，因此我们得知了有个 + r1 + 存在的情报。首先，这个组织不知道过了什么样的方法取得了有关于狩猎者的情报。并且因此制作出了更加强力的B.O.W.。就这点看来，我们可以推断出它是一个和伞公司拥有相同甚至更高技术能力的组织。再来，从它袭击 + r1 + 的舞台——洛克佛特岛这件事来看，它对伞公司是不惜使用武力。所以它一定是一个相当强大的组织。

而以上的情报显示出事情有两种可能。一个是 + r1 + 为伞公司的竞争对手企业；另一个则是它是从伞公司分裂出来，位于某个国家的组织……这两者是现阶段比较合理的推论。虽然目前并没有直接的情报显示，但传闻这 + r1 + 有派遣人员到一洛克佛特岛这件事来看，对以上的 + r1 + 的推测——你们觉得是相信呢？还是不相信呢？
