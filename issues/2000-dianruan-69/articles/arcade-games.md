# 街机仔——一舞惊人

——投币式(COIN-OP)娱乐器材——

## 一舞惊人

全国喜爱街机射击游戏的同志请高度注意，1945是如何打到200万分以上的，已有天津读者向日本人请教得出。下期此栏见。必见！！

天师，你好！我是一名南京的读者，看到第二期有刊登我的评语，兴奋不已。不过在这里还是有些后悔，因为上次来说 + r1 + ，但在我真正完整地玩了一遍末8后（投稿以后），我才真正明白FF8为何被称之为大作。时空之轮II虽然也很优秀，但是还是不足以拿来与FF8作比。

在南京，DDR很流行，可能不及上海等地，但高手绝对不少。这儿有日本原装的DDR 2ND混合机，脚感强过PS改装机百倍，我等常到此练习以提高技艺，不过每当遇到高手才发现自己实在太菜。第二期说S.P跳BUTTERFLY(HARD) 160连击，S级得22607600分，我实在不知把这个成绩放上来是什么意思，不会是封神吧？我也试着跳了一下，结果得了个SS级，3339300分，178全连击，170个PER非，8个GREAT。这种成绩在我们这儿随便拉个人就能跳出来。我的水平基本上就是八个脚的歌全都能过。AM-39 90%的把握拿A级，DUB I DUB 80%拿B级，I BLIEVE IN MARIDE 80%拿B级，其他的目前基本上都是C级。请天师注意，以上均为MANIAC的DDR 2ND混合机。

不过，最令我感到恐怖的是这儿有个人能将MANIAC的PARANOIA MAX 190（鲜红封面，就是PARANOIA 180右边那个最变态的）跳得A级，把歌曲中后部分部段毫无人性的舞步都基本上全吃了，令我目瞪口呆。这可是昨天我亲眼所见。这儿还有一位仁兄能跳过MANIAC的DUB I DUB的HIDDEN模式，也是相当了得。不过很可惜，南京还没有这方面的比赛，如果有的话，大家苦练，高手一定更多。

最后想请教天师一个问题，就是上次 + r1 + 的攻略中说游戏会在屏幕左上角显示HOME或ANOTHER，可我怎么找不到，以至于不看攻略的话，连自己是身处哪个世界都不清楚，这是什么原因？怎样判断我目前所在是身在哪个世界？麻烦你了。（南京紫竹林 卫伟）

●答：看来多到机厅去走走就能发现高手。其实厅主当初是个挺爱跳舞的汉子，可过了几个月他又不跳了，你大概不知道……哎哟……他那个成绩是PS版的，当然厅主也很爱跳街机，但 + r1 + 。莫说厅主，那个舞友敢说自己是最好的？据说，PEPSI公司欲举办一次动舞大赛呢（可能和SEGA合办）！

要说到原装机，还真得谢谢科乐美名古屋事务所，天津不少机厅就有原装巨无霸，家家儿有鼓（同鼓音箱），电吉他已经到了4TH以上，个中高手却并不多，因为，高版本DDR和DJ更到了4TH以上，个中高手却并不多，因为，高版本DDR难度有些大，打鼓、弹吉他还有DJ更是 + r1 + ，就这，也有众多半专业乐手或打或弹至3、4千万分的，当然，一律REAL模式。在天津打日本原装的全新或二手大型游戏机是全国最便宜的，简直是白玩。

北京方面，跳舞高手基本上集中在世嘉店的原装2ND等机台（盗版脚感差？），但的水平大概比信中的高手低一点（只有一个人跳得特别稳健，胜过信中），也是MANIAC8或是使用隐藏指令 + r1 + ，马马虎虎，跳的时候低手的身体不停转圈，自我陶醉（可那个高手的跳姿却明显稳、准、狠）。不过让世嘉进的那台杰里科STEPPING STAGE 2倒真是挺好，不但机台本身颜色美丽，而且音乐动听，音效也很专业，还带大屏幕真人画面，看起来十分过瘾，尤其是SCATMAN JOHN说唱，可说是绝嗨，但万万没想到此人竟然于1999年12月3日……唉。除了科氏的打鼓，杰里科的这款游戏也已经移植给PS2了（PS2版加后街男孩、BRITNEY SPEARS、STEPS等人气组合。当然更少不了经典的SCATMAN）。

原装机台最密集的社会主义现代化国际大都市上海，情况如何就不清楚了，肯定火。谁有听说或看到，可将情报寄来。

问题是不能搞盗版（又来了！？），原装机台的脚感就没法儿和PS改大型一样，前者能买后者10个还不止（STEPPING STAGE 2价格约12万元，PS改大型仅八千）。要别提家用跳舞毯了，一跳就产生 + r1 + 。看来街机还是无法被 + r1 + ，真的。
