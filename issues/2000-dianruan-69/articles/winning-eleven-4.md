### 阵型分析

一只球队使用什么样的阵型，一定要与该队的技术风格与打法特点相一致，只有这样才能最大限度的发挥出球队队员的个人能力及球队的整体实力。而荷兰队的优势是边路的速度与中锋的身高，因此进攻大多以边路传中，中路抢点为主。又因为有速度奇快的Overmars，因此进攻的侧重点更多的集中于左边，同时由于 + r1 + 阵型要求两名边前卫进行深度助攻，故此将两名边后卫的位置前提，而在边后卫的人选上，由于 + r1 + 中恐怕他也只有Zenden无论在进攻意识、奔跑速度上，都要比Cocu优秀，因此由他出任左边后卫是最佳人选。而他俩身前的Bergkamp传球与技术数值都是8，完全可以在担当杀手的同时兼任中场组织者的重任，因此我将他回撤到前腰的位置组织进攻。

后卫线可随R.F.DeBoer设为清道夫。记住他的位置千万不可以没有自由人，否则他会在角球时与Stam同时上前助攻，这时你后场将空空如也，一旦被对方打反击空如也，一旦被对方一个反击就……等死吧。

总之，这个阵型相对更侧重于进攻，防守人数相对较少。因此防守时一定要做好队员之间的协防，力争造成防守时的局部人数优势，同时两名中后卫绝对不要冒然出击，这样只会把身后空当留给对手。因此，使用该阵型一定要练好对防守位置的判断和整体防守的感觉。

### 攻防套路

如果你是一个真正的荷兰队球迷，仔细的看过荷兰队比赛，那你一定会发现，荷兰队员是很少盘带的。他们总是不停的进行大范围的转移，寻找球场的空当，然后进行快速的突破。这正是使用荷兰进攻的整体原则，那就是：一定要踢得开阔而简洁，避免不必要的盘带，一有空当迅速转移。而具体到他的进攻套路，则正如上文提到的，以边路进攻为主，靠传中抢点得分。下面，就借助图例来详细解说一下荷兰队在使用这套阵形时进攻的几种典型方式。

图2是边路突破边后卫互阵对的边后卫之间的配合。一般在阵地战中，如对方使用4或5后卫的阵型，那么边路留给你的突破空当是极小的，而在 + r1 + 中又不能如前作一般轻易摆脱对手，因此一定要借助配合。

以图中所示左边路为例，Overmars持球突破受阻后，应迅速带球向中路移动，此时对方边后卫会向中路靠拢，参与5防，而此时对方边后卫Zenden会高速插上，形成空切。此时Overmars应及时向边路传出脚下空当球，即可形成一次成功的边路下底传中。

也许有的朋友觉得 + r1 + 中下底传中后中路总是没人接应，其实这里面涉及到了 + r1 + 中的攻防转换技巧，也就是队员名字下方的那五个指示槽，它们由 + r1 + 键配合 + r1 + 来控制。当你突破下底时，应同时将攻击力度开到最大，这时不仅仅你的前锋将积极的去抢位，你的后腰队员也将投入至禁区外，寻找远射机会，而Davids和Seedorf二人都有极强的脚头，他们的远射是荷兰队进攻的重要手段之一。加大进攻力度将使他们获得更多的射门机会。但是在操纵队员的同时进行攻防力度的转换在操作上有一定的困难，不过使用荷兰队的朋友一定要掌握这项技巧。

前面曾经提到，荷兰的进攻应尽量打开，图3就是荷兰的大范围转移长传进攻的示例。这要求你有准确的长传技术，同时要注意观察场上局势。

仍以图中左侧进攻发起为例：Overmars带球至对方半场，对方守心偏向左侧偏移，这时长传给右侧边前卫R.Deboer，可选择直接突破或再次分球给套上的边后卫Reiziger，完成传中。值得注意的是，若对方迅速调整防守区域，造成右侧下底困难时，此时球恰好会落到助攻上前的左边后卫下面，自己决定了。

以上两图例所示的都是在阵地战进攻中使用的套路。图4所示则是反击中的进攻路线，在本图中所有的传球路线使用的都是过顶空当球。其中路线1为边路反击，由后腰队员出球给边路快下的前卫，直接造成边路突破。路线2则为中路反击，由球前锋直接拿球形成突破。要特别注意的是本图中路线前锋接球前一定要先按住加速键闪过追来球前锋跑直接形成突破，要特别注意的是在路线前一定要先攻击来球，在转身突破过程中，要特别注意的是，在前方没有防守队员阻挡时就是一定要快，用最简练的手段打对方立足未稳。

攻路线，在本图中所有的传球路线使用的都是过顶空当球。其中路线1为边路反击，由后腰队员出球给边路快下的前卫，直接造成边路突破。路线2则为中路反击，由球前锋直接拿球经边路反击。这的要特别注意的是在路线2中的反击，首先一定要先按住加速键闪过追来的后卫，直冲对方禁区。总之，反击的原则就是一定要快，用最简练的脚法最快速的达成突破。

如果队员球队的体力消耗过大，就会影像加速度，奔跑时进入最高速度的时间也会变差，此外传球、射门的精度，以及身体的平衡性都会有所降低，令攻防中出现明显的漏洞。下图为各种动作对于球员体力的消耗情况（左侧的冲刺滑铲是最为消耗体力的）。球员体力的剩余值可在状态画面中按R1键查询。

说到防守，并没有什么固定的套路，只需要遵循几点原则：

★冷静的保持对位置的判断。防守说到底其实就是防位置与卡路线，让对方技不到突破与传球的空间，你的防守就成功了。尤其是对于这个阵型，它的专职后卫实际只有两名，因此在防守时千万不能冒然冲上去抢断，否则把身后的大片开阔地都留给对手，一旦被对方打过，手，一旦反抢不成，被对方将球队员逼向边路，不给其进入禁区起脚机会。

★注意前场反抢时使用队员。最好不要让助攻的后卫去反抢，因为此时他的位置上并没有人，一旦反抢不成，被对方将持球队员逼向边路，不给其进入禁区起脚机会。

★尽量造成防守时的局部人数优势。 + r1 + 中可以使用 + r1 + 键呼叫临近队员参与围抢，此时一定要先死死盯住持球队员的出球路线，防止其分球到空当球员，再呼叫队伍一拥而上，将其截住。

★死卡对手内切路线。对于荷兰队来说，对手突入禁区后的直接射门要比传中后的抢点威胁大的多。因为后防线上有Stam的存在，一般的前锋很少有头球争得过他的，因此在防守时若暂时抢不到球，应尽量将持球队员逼向边路，不给其进入禁区起脚机会。

基本上，若能做好上述几点，你应该可以建立起一条稳固的后防线了。总之，使用荷兰队，就应该做到重整体、轻个人，打出速度，打出配合。若你用荷兰踢出这样的比赛，那恭喜你，你已经掌握了全攻全守的荷兰足球的含义。

最后，愿每一个喜爱荷兰队的球迷玩友都能驾驭着无坚不摧的 + r1 + 横扫球坛，成为 + r1 + 中的王者。好了，看我豪维斯达到GO，GO，GOAL！！！
