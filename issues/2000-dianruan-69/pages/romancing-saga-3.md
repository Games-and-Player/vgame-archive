# 浪漫沙加3 原画插页

> 彩色插页 / 插16-17 / PDF p.58-59

Romancing SaGa 3

本跨页为《浪漫沙加3》(Romancing SaGa 3)的官方原画插图，由小林智美绘制。

**插16（PDF p.58）**：女性角色的侧面肖像画。画面以粉红色和橙色为主调，人物被鲜花、鸟类和星辰环绕，充满梦幻色彩。右下角印有 + r1 + 标志。

**插17（PDF p.59）**：蓄须男性角色的侧面肖像画。人物头戴华丽的装饰性头饰，饰以蛇、宝珠与尖角等元素，背景有小龙和植物。左下角印有 + r1 + 标志。右下角注有 + r1 + 字样。

## 编辑备注

- 本跨页为纯美术插图页，无正文文字内容。
- 《浪漫沙加3》为SQUARE于1995年发售的SFC RPG作品。
- 原画作者为小林智美，以其华丽细腻的水彩画风著称。
