曾在街机版上造成轰动的隐藏角色，目前已经确定能够在PS版上直接选来使用啦！不过仔细一瞧，角色选择画面中，怎么还是有一个空格呢？真是令人费解！？

游戏开始玩家能够选择的角色总共有20人。其中有10位是隆、肯、春丽等大家耳熟能详的角色，另外10位则是北斗、史卡罗马尼亚等ARIKA公司的原创角色。在之前的街机版上，只要满足特定的条件，就可以选用七濑、加鲁达、凯利及夏朵盖斯特这4位隐藏角色。在PS版中，七濑一开始就可以选用，其余3位则还是要满足条件后才能选用。这回就来介绍这后3位角色的招牌格斗技吧！

满足条件之后→这些以外的角色→就可以选择

## 果然还是有隐藏角色！

### 凯利 KAIRI

**STORY**

受到不明人士的 + r1 + 攻击而坠落山崖的凯利，虽然后来死里逃生，却失去了记忆。然而却也激发出凯利身上特有的本能，让他能够拥有从恐惧岛中所产生的千气。还有当他再与北斗相会之时，不知情的凯利却感到心乱如麻。

 + r1 + 

凯利的造型和前作相比有很大不同。不但黑发变为白发，两手之中也多了青白色烈焰般的斗气，好象要将对手烧掉一般。其格斗技也和隆和肯有诸多类似之处。

**必杀技**

**魔龙裂光**
↑类似升龙拳的招式。打中对手会令之燃烧。

**神鬼发动**
↑从飞行道具开始在一定距离产生斗气的招式。

**超级必杀技**

**神气发动攻**
↑空中版 + r1 + ，变成多段HIT技。

**雅龙灭�的**
↑从空中跳下的腿技，接续膝蹴完结技。

**陨石必杀技**

**瘴鬼发动**
↑这是类似豪鬼 + r1 + 的新技。凯利全身会先散发闪光，然后就……

**朝对手奔过去的连续技**

↑→接连使出连续技！最后凯利就在倒在地上的对手面前抱头伤心。

### 夏朵盖斯特 SHADOWGEIST

**STORY**

夏朵盖斯特是出生在一个政府借法律之庇护，从事合法犯罪，致使人民只能靠一己之力来保护自己的时代。对于这般的政府，身为现代必杀工作者的他，应付对策就是靠人工强化自己的身体，甚至甘冒险将人工武器加在自己身上。

 + r1 + 

在本作之前，从未曾出现过的全新角色。在既存角色中，和他最为类似的就是史卡罗马尼亚。他也能够不断使出超乎常人的格斗技……

**必杀技**

**死亡粉碎**
↑对敌手的突进技有多段HIT。

**死亡剑蹴**
↑急降下踢技，可将跳跃通常技作为取消技使用。

**超级必杀技**

**死亡之梦**
↑类似 + r1 + 和 + r1 + 的招式，可做连续技。

**死亡能量**
↑将敌手吸入类似黑球的气中，再将之弹飞。

↓对手体内会产生能源球，他自己则在一旁观看，最后对手大爆炸。

**身体爆开！**

**陨石必杀技**

**超级死亡能量**
↑→当他全身放出光芒之后，会踏前一步出拳。被他打中之后……

### 加鲁达 GARUDA

**STORY**

加鲁达是产生于人类邪念之中的负面产物。而每个人都会具有的发怒、憎恨、嫉妒、背叛、失望，全都是加鲁达生存的原动力。加鲁达也会随着挑战者之不同，在外型上会有所变化。对手越强，他也会变得越强。对于复仇心切的对手，他也会以更强的憎恨心应战。

 + r1 + 

加鲁达曾在前作中登场，是非人类的角色。这次从他的全身各处也会飞出刀刃，让对手感到恐怖。除了前作中的必杀技，这次又新加了新格斗技喔！

**必杀技**

**鬼斩**
↑从双脚双足中突刀出刃象X字一样回转上升的招式。

**雷牙**
↑从双臂突出长刃的招式，可以 + r1 + 空中敌手。

**超级必杀技**

**鬼燕舞**
↑以双足为轴，旋转攻击的格式，空中也可以出。

**苍魂弹**
↑身边盘旋的鬼火向敌手进行三次攻击的新投。

**陨石必杀技**

**凶刃旋袭**
↑→朝对手突进的同时，会出乎意料之外地使出刀来大砍特砍一阵的技。

**再挥刀→挥刀→痛快地斩杀！！**
