##  + r1 + 首战告捷！

本期的投票量有了明显的提高，统计时的工作量也是难以想象的。最令人难以忍受的是居然还碰到了 + r1 + ，一度造成无法统计了。另外，由于许多玩友是第一次参加投票，对于一些细节不太清楚。主要表现在书写游戏名称的时候，不但没有按照本刊给出的名称，而且不注明机种。这样的选票不但无效，而且还给统计工作造成大量不便，请玩友一定要注意。举例来说，有些玩友投票时只写 + r1 + ，令我们统计时无从下手，或者写着 + r1 + ，也属于无法统计的，因为GB上的 + r1 + 有三款游戏，所以投票时一定要写清游戏的副标题。具体注意事项可以看本刊去年排行榜栏目中所刊登的那些，只有规范的投票才能起到支持自己喜欢游戏的作用，才会有更多的中奖机会。不过，虽然历尽艰辛，本期排行榜的新气象还是喜人的。

本期的一万多张选票中，有超过70%的玩友是投PS游戏的，不仅使得PS榜上变化显著，更有无力登上PS榜的游戏却登上 + r1 + 的事件发生，令人对PS的实力不得不信服。圣诞前后发售的PS大作，除了 + r1 + 和 + r1 + 没有得到足够支持来挤进当月榜外，其余都榜上有名。 + r1 + 还一举进入前五名。 + r1 + 居然升到第四名，同 + r1 + 仅差二十几票，实在令给球迷争气。不过，ENIX的大作 + r1 + 却没有上榜，让人颇感意外，不知下个月它是否能够上榜。

本月日本方面， + r1 + 销量惊人，已经超过400万份的销量，山内爷爷的嘴一定又乐得合不拢了。PS的 + r1 + 销量也超过了100万，虽然还比不上一代，但成绩也不错了。PS的 + r1 + 和DC的 + r1 + 也取得了不错的成绩。而在国内大受欢迎的 + r1 + 却连30万都不到，但相比DC的 + r1 + 它还算小有盈余。后者制作周期长达数年，经费无数，却只买掉5万份，销量榜上都找不到，恐怕连成本都收不回来，这次爱搞噱头的饭野贤治算是彻底栽了。
