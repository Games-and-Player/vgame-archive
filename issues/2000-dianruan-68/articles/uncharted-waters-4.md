# 大航海时代IV ～大洋冒险者的航海日志～

正如游戏中某位角色所说的，大海是人类的母亲，自古以来有无数的探险家、商人、海盗、军人在大海上演绎了无数可歌可泣的故事。海洋也是现代无数热血男儿所憧憬的世界，光荣的 + r1 + 给了我们这个纵横四海的机会。本游戏共有四位主角，其中中国的李华梅必须在有一个通关记录后才可使用。

**PS** 厂商：KOEI 类型：SLG
发售日：99年12月

## 普通航海士篇

● マヌエル·アルメイダ：船大工，能制造和修理帆只，才华横溢，ホドラム的起始伙伴。在サンニヨルヨユ港可以找到他。

● グルハルト·アーデルソカッツ：富官，德国人，曾领导过一支舰队，统率力高，讲究礼仪，是新大陆海盗バスタス的老师，ホドラム的起始伙伴。在ハンブルク可以找到他，对话时选择 + r1 + 还能打听关于海战的经验。

● ツヤルル·ロシユフオール：炮击手，喜好射击武器的研究（房中常有爆炸声）。ホドラム的起始伙伴，在ハバナ港可以找到他（选华梅时他是英国舰队的爪牙）。

● カルロ·シナート：船医，因家人病逝而弃商学医，治病救人，同时也是个生意精，在マラッカ港可以找到。

● サムウエル·カーン：料理长，游牧民族，跟父母作奶酪交易，擅外族美食，在カリカツト港找到。

● フエルナンド·ディアス：参谋，拥有优秀的洞察力，在ヴェラクルス落脚。赌硬币赢他（关键在于他取币时的币数为13、9、5）即可加入，リル的起始伙伴。

● チェザーレ·トーニ：见张，富家子弟，爱好天体观测，为寻找自己的船只而四处奔走，在アレキサンドリア可以找到。

● ヅヤム·ルドワイヤン：能力平均，性格豪放，因受人陷害而辗转于各地。在バタヴイア寻得，ホアメイ的起始伙伴。

● コリオ·エルネコ：测量士，舵手，退休海员，好烈酒。在ツェノヴァ港可以找到他，ラフアエル的起始伙伴。

● エミリオ·フェローグ：料理长，饲育员，是个副斗牛士。住在セビリア港，リル的起始伙伴。

● ツエナス·バサー：葡萄牙人，爱好机械，说得力高，在リスボン港可以找到，ラフアエル的起始伙伴。

● クリスティナ·フリオ：剑术高超的女剑士，一头红发，为男友的无聊追求而头疼不已。フリオ的孙女，由フリオ提及后，在ロンドン港寻得。

● アンヅエロ·ブシチーニ：没落贵族后代，为寻找失散的妹妹而四处奔走。生命力强，剑术高超，在イスタンブール港可以找到他。

● ユキヒサ·シラキ：斩达队长，日本会津人，为完成家传 + r1 + 而努力，在大阪找到，ホアメイ的起始伙伴。

● アル·シン：雇佣兵，战斗力强，爱好和平，在バスラ港可以找到他。

● イアン·ドウーコフ：商人之子，相貌英俊，剑术极高，在ンファラ港找到ホアメイ的起始伙伴。

● ユリアン·ロベス：一头金发，无特长，执着的追求杭州的酒吧女——美华，在杭州找到他。

● イファ·ンル：谜一般的朝鲜少女，修行道术，在京城可以找到她。

## 贸易心得篇

1. 这一代中，贸易难度相对简单，只要是本港没有的货物，一般都有利可图，但必须先和王宫或总督府签下合同（100%不可），一般钱币代表1%的占有度，占有度越高，每月产出的商品数量越多，商品等级亦越高。交易品等级分1～5级，1级最佳，名等级类卖价相同，卖价可相差许多。

2. 交易品在两地间相互交易的原则，以同类商品最佳，这样可以使物价稳定，以维持长远利益。在本作中，被卖出的交易品将不断降价，被不断买进的反之。

3. 在流行港口抛售该港流行品时，价格会是平时的5～6倍，另外可得到推荐，提高在本港的占有度（1份商品提升1%）。获得流行情报有以下几种可能：

A. 从酒吧女处

B. 由交易所指令 + r1 + 处（需金币50枚）

C. 广场处

D. 利用宣教师在广场作 + r1 + 也可制造流行，不过要等一段时间

4. 推荐黄金航道：

A. 如果カエンヌ港的烟草和香料交易到ヴェルデ港，利润非常丰厚。

B. 把原产物タバコ的苗木（从バハマ购得）交给クレタ交易所，增加特产品タバコ，然后用クレタ的タバコ和附近港口级毯进行交易，实为黄金航线。

C. ルワンダ港的琥珀，珊瑚与サンニヨルヨユ港的象牙交易。

D. 本作中利润最高的一条航路：在バハハ城门得到原产物——炼金术书，并交给西非ルワンダ港以增加特产品白金，用来和东非モガディツォ港特产——黄金相互交易（当然要事先力投资港口以提高交易品的数量和质量）。

注：以上航路关键是尽促使两个港口相互流行（广场使用 + r1 + ）

## 海战心得篇

1. 按照个人爱好，如果用白刃战，就要多修几个海兵治所，以提升水手上限；还要努力提升新达队长的剑击术，以应付单挑的情况；如果喜欢远战速决，则多建几座炮台（和海兵治所一样，最多四个），以及选择合适的航海士充当炮击手，用以提高炮弹的命中率。不过，如果需要大量金钱，还是白刃战为佳（取敌总资金之10%），但选ホドラム击沉敌舰可获得海盗王的倍金。

2. 白刃战时尽量以旗舰和敌人旗舰相遇。单挑如不成功，则需用其他舰船消耗敌人水手，待敌人水手不足时，再行靠近，以防旗舰水手全灭，损失舰船。

3. 海战时，将舰船侧面对准敌舰船头尾，即海军流行的T字型战法，效果最佳。

4. 许多大港口的发展度、武装度足够高时，都有暗藏舰船。北海的港口武装度到7000以上时可以购得本游戏最强舰船——战列舰；东亚港口武装度7000以上可以买到本游戏耐久度最高的舰船——铁甲船（但它只有一个帆且需要水手多，并不很实用）。

5. 消灭敌方势力的几种方法：

A. 海战胜利一次，敌势力所有港口占有度下降2点，50次后降为0，金钱大赚。

B. 投资港口至100%，使敌势力下降为0。告诉某港口只能投资至98%或99%，这时海战胜利一次即可挤掉敌人仅有的份额。酒场的不利传言和契收龈能收到同样效果。当然炮击敌方势力，独占契约港也是一个不错的选择，可提高占有度10%。但切忌不要在同一港口里面战，这样即使胜也无法在港口进行作为，只能补给出港。

6. 每次海战中敌我位置都不相同，因此在同一次战斗中有时可轻松获胜，有时却一败涂地。即使玩家拥有最强舰队，也不能掉以轻心，反复Save是最好的方法。

7. 于造船厂购买一般有三个帆的船，配给1个水手，出港后在港口围圈行驶。当水手疲劳度超过70可迅速升级，疲劳超过90后记及时在岸边漫步，用找到的温泉，上岸欢迎等方式降低疲劳度。温泉为东亚最多，杭州附件为最佳地点。在只有一名水手搜索时，不论如何也不能让水手逃走或死亡，笔者曾于4小时内将手下由50级升至200级！

8. 每位主角得到谜之小物的方法都不相同，窗者之证的位置也因人而异，应注意寻找并完成在同业工会委托的事务。
