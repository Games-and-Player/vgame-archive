# 电玩吉祥物集结篇

> 中部彩插 / PDF p.52-56 / 5页彩色专题

吉祥物，顾名思义就是赚大钱保平安的意思，也难怪各家电玩厂商最近都频频推出人气续作。其中有大受欢迎的陆行鸟、玛里奥、小精灵、史莱姆、古惑狼……等，不论是主角配角都是极有人气的角色。在此，就来跟您一起探讨各家电玩吉祥物的起源与其受欢迎的原因吧！

【图】集结插画：克拉修（古惑狼）、少年角色、炸弹人、索尼克、玛里奥、小精灵等吉祥物大合影。

## 续作+吉祥物=人气！？

对于喜欢电玩的您来说，在享受极佳的声光效果与丰富内容的同时，是否对于其中的电玩主角留下深刻的印象，或者已经不自觉地深深地爱上他们了呢？其实，这些游戏厂商当初在设计游戏的时候，并非皆以主角做游戏的卖点与诉求。因此有时会大红大紫都是无心插柳的结果，像陆行鸟就是一个很好的例子。

在这么多的软件厂商与游戏作品当中，我们特地挑选出几位最具代表性的电玩角色，由于他们的名气极大，也就顺理成章地成为这些厂商的吉祥物代表了（以致于续作一出再出）。

**吉祥物的无尽魅力弥漫业界**

## 任天堂吉祥物——世界最受欢迎的水管工人——玛里奥

【图】超级马里奥64游戏截图多张

在所有的电玩吉祥物当中，玛里奥可以说是最老也是最红的一位。这位原本在 + r1 + 中冒险犯难的苦命角色，当初并没有如此地大红大紫，直到后来当上了水管工人之后，才稍微打开了他的知名度。不过，真正奠定其巨星地位的作品，自然就是FC主机上面的 + r1 + 。不同于之前的作品， + r1 + 里的玛里奥，除了多了一位伙伴路易之外，他还得面对更多的挑战，更别说是游戏本身丰富多变的游戏内容了。

由于FC主机的三款作品连连突破百万销售的佳绩，使得任天堂公司在推出SFC主机的时候，玛里奥就成为同步发售游戏的不二选择。而且这款 + r1 + 也成为该主机的最佳展示作品。在进入16位元世界后的玛里奥，他的活动范围也一下子宽广了许多。除了原本的动作舞台之外，他也开始尝试起迷你赛车来了。更不用说他还会教你基本绘图技巧。后来，甚至与SQUARE公司的共同合作下，居然与宿敌库巴携手合作，一起为了救出碧琪公主而努力呢！

在进入64位元之后，其主机虽然不再是电玩界的霸主，不过玛里奥的演出依然是令人乱目相看。这次，玛里奥除了再度开着迷你赛车出征之外，同时也玩起高尔夫来。并且还与其他任天堂公司的电玩明星一起展开史上的大乱战。由此可见，其人气不减。此后，任天堂公司还打算让他成为RPG世界的主角，到各地去冒险呢！看来，这位水管工人可成了世界的知名人物啦！

→为了赢得公主的芳心，玛里奥不断地向困境挑战！

## SEGA吉祥物——让索尼克带领玩家进入速度的世界

【图】索尼克大冒险游戏截图多张

→有了新造型，看起来更帅气。

说到索尼克的诞生其实很有趣，当初在MD主机推出一段时间之后，SEGA公司决定要创造出一位可以跟玛里奥平起平坐的全新电玩主角，于是Sonic Team便以针鼠做为创作蓝本……索尼克就这样出现在玩家的面前。 + r1 + 这款游戏最大的卖点，就是可以让玩家享受到极为快速的奔驰快感，而不象 + r1 + 系列那样以挖掘隐藏秘密为游戏的特色。再加上游戏的操作相当地简单，除了方向键之外就只有跳跃按键而已。因此使得许多不玩动作游戏的玩家，也能够快速地融入其中。

在进入SS主机的时代之后，或许是因为公司的政策使然， + r1 + 的游戏产量一下子骤减很多，虽然一款 + r1 + 稍微让玩家解除部分干渴的感受，可是一连取消 + r1 + 与 + r1 + 的制作计划，实在是让人感觉到非常地失望。幸好，就在DC主机发售的消息确定之后，索尼克再度借着大冒险之名重出江湖，并且将该主机的硬件性能发挥得淋漓尽致，也难怪游戏会创下四十多万套的销售佳绩。现在它又进军NGP了。

## HUDSON吉祥物——以炸弹红遍天的炸弹人

【图】炸弹人游戏截图及角色图

您能够想象有人可以光凭着一颗小小的黑色炸弹，就这样子打遍天下无敌手，并且顺利地化解世界的危机吗？没错！他就是大家相当熟悉的炸弹人。最早在PC-E主机上亮相的 + r1 + 就非常受到欢迎，因为它的游戏规则简单易懂，就算是不玩游戏的人也能够乐在其中。不过直到后来与Irem公司推出街机版之后，才真正让 + r1 + 尝到 + r1 + 的滋味。

 + r1 + 起初只有一个人而已，直到后面的几款作品发售之后，HUDSON公司才分别给予他们不同的颜色加以区别。而且最初黑色面罩炸弹人登场的时候，还是以敌人的面貌出现的呢！在 + r1 + 出名后，HUDSON公司却没有特别积极开发其他以他为主角的作品。除了后来在一款射击游戏当中荣任主角之一外，就是在N64主机上以纯动作过关游戏演出的 + r1 + 了，实在是让人觉得相当地可惜。

→说到 + r1 + 舍我其谁？

## KONAMI吉祥物——劫富济贫的义贼——伍右卫门

【图】大盗伍右卫门PS版游戏封面及截图

除了鲁邦三世之外，世界上最出名的虚拟盗贼就要算是伍右卫门了。最早，在KONAMI公司推出 + r1 + 时，他可以说是劫富济贫的最佳典范。不同于其他的动作游戏作品， + r1 + 系列总想尽量呈现一个较为完整的游戏世界。所以玩家也才能看到有城镇与对话出现的设计，这可说是此系列游戏最大的特色与卖点之一。

伍右卫门一生最大的幸运，就是他总是有着许多伙伴热情的支援，因此他才不会每次都要自己冒着生命的危险孤身奋战，这也是他与其他电玩吉祥物最大的不同点。其实大盗伍右卫门的作品数说多不多，说少嘛也不算太少，几乎每隔一两年就可以玩到最新的系列作品。但与之前的风光比起来，伍右卫门的光芒已经不象从前那么耀眼了。KONAMI公司除了努力于音乐游戏的创作之外，也该回头好好地照顾这位可爱的大盗啦！

## CAPCOM吉祥物——英雄的化身——洛克人

【图】洛克人系列游戏截图

在所有的电玩吉祥物中，洛克人可说是唯一具有英雄特质的主角，因为他不但有一身蓝色的抢眼外表，而且还是正义代表的最佳化身。洛克人的发迹相当地早，早在FC主机时代，洛克人就以有着一定难度的游戏设定而著称。不过也因为这样的特点，反而使得热爱动作游戏的玩家更是不愿意服输，拼命地想要帮助洛克人一次又一次地阻止利尔博士征服世界的阴谋。

 + r1 + 的游戏几乎都是以世界为其表现舞台。除了正宗的 + r1 + 系列推出至八代之外，CAPCOM公司另外还发展出 + r1 + 系列。两者之间最大的差别，在于X系列的游戏世界乃是设定在未来时空。但在求新求变的潮流之下，洛克人也开始玩起赛车游戏。此外，在 + r1 + 系列中更以角色扮演风貌亮相。这款以3D方式呈现的作品其表现之佳，可以让玩家体验到洛克人的不同风情呢！

→那恶的坏蛋们受死吧！

## NAMCO吉祥物——重出江湖的小精灵

【图】小精灵世界20周年纪念版游戏截图

若是说起年纪最大的电玩吉祥物，那当然非小精灵(PAC-MAN)莫属啦！而今年正好是它20岁的生日不，2000年时它已经21岁了！！相信应该还有不少玩家记得小时候玩过的小精灵吧！一张小小黑色桌子的玻璃底下的电视萤幕，有着一颗又一颗的黄色小豆子，遇到危险的时候还可以赶快吃下黄色豆子，如此一来就可以反过来将敌人给吃掉分呢！其实我们很难将小精灵这款游戏分类，因为它同时有着动作与益智的双重特质在里面。也因为这样的游戏特质，使得它的历史价值一直被保存下来。

虽然小精灵已成为NAMCO公司的电玩吉祥物，不过以它为主的游戏倒是不怎么多，但它可是一出击就不同凡响啊！从SFC主机时代的 + r1 + ，一直到最新在PS主机上面推出的 + r1 + 。不但游戏本身的可玩性极高，而且带给玩家一般说不出来的冲动。在小精灵欢度20岁生日的今日，相信大家都相当期待NAMCO公司可让小精灵进入更为多彩多姿的游戏世界。

**小精灵以全新的风貌再度出击！**

## SCE吉祥物——以古惑狼为名的克拉修

【图】古惑狼赛车游戏截图

→觉得可爱？克拉修的样子真是……

在所有的电玩吉祥物当中，恐怕没有一个人比克拉修还要神经质吧！只是象它这样状况百出的主角，居然还可以成为本篇文章的主角之一，这真的是所谓 + r1 + 之最佳写照吧！同时，克拉修也是所有的电玩吉祥物里 + r1 + 的一位，不过您可别小看他年纪轻轻的样子，他可是有着别的主角所没有的上天下海的无敌功夫哦！而且它所遭遇的危险，也总是让大家不禁为它捏把冷汗呢！

克拉修可算是位相当多产的电玩主角，因为SCE每年都会为它 + r1 + 游戏作品。但是，在一连三款的 + r1 + 之后，克拉修再也受不了提心吊胆的日子，于是去年它便带着三代里头出现过的配角们，一起展开一场激烈又逗趣的赛车啦！不过俗语说得好，年轻就是最大的本钱，因此克拉修未来的前程实在是非常值得期待哟！

## SQUARE吉祥物——由配角晋升为主角的陆行鸟

【图】陆行鸟大冒险游戏截图

 + r1 + 系列游戏人人皆知，不过里头最红的主角到底是谁呢？很抱歉，其实它并不能算是主角啊！它不但只是个配角，而且还是只不会飞的黄色小鸟！没错！它就是陆行鸟。恐怕连陆行鸟自己都觉得红得莫名其妙吧！因为它初期只不过是在 + r1 + 系列里头亮相的交通工具罢了，直到后面几款作品才以召唤兽的身份登场。

探讨陆行鸟之所以出名的理由，除了它的外型相当可爱之外，当然它的高曝光率也是功不可没。不过在 + r1 + 之前，SQUARE公司也没想过要替它推出专属游戏。直到 + r1 + 大受欢迎之后，陆行鸟总算从 + r1 + 系列里独立出来。它除了将会继续冒险下去之外，还会参加赛车，并且还打算开个 + r1 + 牧场利用赛鸟来大发横财，更别说要在骰子世界里度过其一生呢！

## ENIX吉祥物——异类宠物史莱姆！？

【图】勇者斗恶龙系列中的史莱姆角色图及游戏截图

现在这个世界，越是不怎么起眼的角色，似乎就越会受到玩家们的热烈喜爱与欢迎，在ENIX公司推出的 + r1 + 系列中，我们就看到了这种奇妙的现象。不管 + r1 + 系列的剧情怎么变，不变的只有史莱姆。它那象水滴般的造型，相信您在游戏初期一定对它印象深刻吧！因为史莱姆最大的特色之一，就是它是一个体力极低的怪物，不过它却最擅长分身的动作，遇到金属史莱姆时更是难以将它打倒。

其实严格说起来，史莱姆的外型真的也设计得蛮可爱的，有种令人不忍砍杀的怜悯。但在 + r1 + 的游戏里，玩家为了快速地提升等级，也只有先拿它来开刀了。虽然史莱姆后来也在 + r1 + 中露面，而且以它为主的电玩精品也不算太少，可是ENIX公司却始终未曾将他设计为游戏作品的主角，这点也是它与其他电玩吉祥物最大的不同点。真希望以后也能够有机会可以跟着史莱姆上山下海，展开另一段冒险旅程。

**虽然作品不多，但却相当受欢迎！**
