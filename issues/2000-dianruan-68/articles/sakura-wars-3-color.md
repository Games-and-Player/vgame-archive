# 花开3度 回首望

## 樱大战3 发表纪念特辑

## 散发无限魅力的樱大战

在 + r1 + 这款游戏发售之前，大概谁也想不到它的影响力居然会如此之大吧！即使在SEGA与RED公司的联手制作之下， + r1 + 本来就注定会成为著名畅销作品之一，但却万万没有想到会获得玩家如此热烈的支持与回应。虽然SS版主机并没有籍着这款作品击败PS主机，或者是挽回大势已去的市场占有率，不过却也因为 + r1 + 的出现，证明了好游戏是绝对不分主机的。

### 天外魔境迷倒众生

在开始谈论 + r1 + 之前，我们不能不先来看看关于RED公司的一些背景。RED公司主要是由广井王子所领军，其大部分作品都是由HUDSON公司所发售，而最早发迹的作品就是 + r1 + 系列了。 + r1 + 不愧为创造话题之王，在一代作品推出时，游戏就请来了以 + r1 + 闻名的久石让为其担任音乐配乐；二代则是由闻名世界的音乐家坂本龙一所负责，因此光是这样的阵容就足以吸引玩家们的注意了。

光效果，实在很难将其与 + r1 + 系列联想在一起，就因为RED公司有这样的游戏理念，所以SEGA才会想要与他们一起合作。而这款由广井王子所构想出来的 + r1 + 也才会顺利地诞生。

###  + r1 + 成为绝响？

求新求变的PCE主机为了突破本身硬件的环境限制，于是就有了Super CD ROM²的出现，而第一款真正称得上强打作品的游戏就是 + r1 + 。也因为有了它的推出，使得PCE主机声势一下子变得锐不可当。难怪预计在PC-FX主机上面亮相的 + r1 + 会如此受到玩家期待。但是天不从人愿，PC-FX主机真的是处境尴尬，最后连HUDSON公司都放弃了这个平台，因此 + r1 + 也就从此成为绝响。

虽然在SS主机上面发表的 + r1 + 有着相当扎实的游戏设定。不过一方面因为销售成绩不如预期中的理想，另外一方面在剧情的安排上又遭到头重脚轻的批评；因此使得RED公司必须求新求变，重新思索一条全新的道路来，而 + r1 + 就成为他们重振雄风的最佳利器。

### 跨公司合作的典范

只要是细心一点的玩家都知道，RED公司一向都不会局限自己，他们常常多方尝试不同类型游戏的开发。当初同样在PCE主机上面推出的 + r1 + 系列，可说是一款不折不扣的横向卷轴射击作品，而且其一流的声

### 为帝国华击团欢呼！

在 + r1 + 的游戏世界里，时间背景是被设定在日本的大正年间，为了负责与保卫整个国家的安危，于是就有了 + r1 + 这个防卫体系的诞生。为了掩人耳目，帝国华击团平常是以剧团的身份自居，也因为这样的关系，团员们除了要懂得作战之外，更要十八般武艺样样精通才行。所以，在一代游戏刚开始的时候，大神在面对这样的帝国华击团而感到惊讶不已，因为他一直以为这是一个纯军事单位……

### 既简单又复杂的LIPS对话系统

说到大神，在一代的游戏中，玩家可以充分地感觉到他那犹豫不决的个性，就好像是 + r1 + 五代的翻版一样。但偏偏又最受女性们的欢迎，真不知道是该赞他也好呢！还是嫉妒他也好呢！RED小组为了更彻底反映出大神独特的性格，就大胆地采用了LIPS对话系统。玩家在游戏进行的过程中，经常会碰到必须选择对话的情况，而所有的选择都将影响到女主角们对大神个人评价与喜爱程度。

不过，这还不是LIPS对话系统最为特别的地方。在一般AVG的游戏作品中，玩家可以有足够的时间好好考虑，但是这点在 + r1 + 里可是绝对行不通。因为您必须在时间倒数完毕之前从选项中选择，否则游戏就会自动地帮您选择任一对话，或是引发额外的隐藏事件出来。

这样的设计是否跟真实的生活一样精彩！没错！这就是LIPS对话系统的精髓所在，就如同我们在日常生活中所说的每一句话，都会影响到交谈对象的评价与反应，将这个系统应用在游戏中，无形中也增加了不少挑战意味。

LIPS对话系统所带来的直接效应，使得 + r1 + 的剧情变得更为丰富，所以这款游戏若只玩一遍的话，是绝对无法体会到所有剧情内容的发展。在不参考攻略资料的情况下，一般玩家可能要玩个三四遍以上，如此，才可看到自己认为比较满意的结局吧！这一来，也就造出 + r1 + 的耐玩性居高不下的效应。

### 称职的语音演出与激烈的战斗交相辉映

在游戏的表现方面， + r1 + 最大的卖点之一，就在于细腻的人物设定与绘制的重现上面，一个个的角色简直像活生生的人物出现在玩家的面前，然后再配上对话时出现的真人语言，这样一来就好像是在观赏一出精彩的热血青年时代剧一样呢！

为了真实地重现帝国华击团艰苦的战斗生活， + r1 + 里当然少不了激烈的战斗场面啦！虽然在游戏中，它所占掉的时间并不太长，反而却有画龙点睛的功效。尤其是在策回合制的情况下，玩家必须控制整个帝国华击团的行动，并且发挥一定的战术以便将敌人歼灭。假如女主角们对大神特别有好感的话，还会在紧急的时候给予救援呢！象这种与剧情如此紧密地结合在一起的作品实在是不多见。

正因为游戏结合了这么多突出的特色，所以 + r1 + 的游戏一推出，马上就创下了50万套以上的销售佳绩，特别是采用AB两款不同包装的限定版本，更吸引了喜欢 + r1 + 的玩家争相珍藏。SEGA甚至还计划推出重制的限定版本，以便满足先前向隅的玩家们之收藏欲望呢！

### 无法一手操控大局的二代剧情

在相隔一年半之后推出的 + r1 + ，依然与前作一样采用原班人马。不过，这回又多了几位新的女主角加入，使得原本就相当错综复杂的人际关系变得更富挑战性。虽然整个游戏架构变化不大，但针对LIPS对话系统的设计确实做了相当大的改进。这回大神的个性将会随着对话选择的内容而不断地改变，女主角们轻浮或是正经的回应，也都会改变蓝色与红色光棒的比例，进而引发在接下来的剧情发展中，玩家所能选择的对话会因先前的选择而发生明显变动。而且，如果参考大神的光棒值来选出不同的对话选项，如此一来，等于多了两种变数影响整款游戏的进行。

在高达一百多种可能性的设计之下， + r1 + 更是显得完美无比。再加游戏的副标题大胆地采用二次世界大战期间出版的反战诗集，而增添了不少文学气息。此外， + r1 + 的推出更将这个系列游戏推向另一个高峰。随着连续两代作品的推出， + r1 + 很快地就拥用一群发烧玩家的热情支持。由于有强大的后盾，因此 + r1 + 除了这两款正传作品之外，更推出了许多相关的游戏作品哦！

### 樱大战无所不在！

在这些衍生作品当中，让玩家印象最深刻的应该就是 + r1 + 吧！虽然这不是一款正统的游戏，只能够算是以 + r1 + 为主题的电子刊物罢了。不过却相当受到玩家们的欢迎，也难怪此后又推出了趣味十足的 + r1 + 这款小品作品。另一方面，脑筋动得非常快的SEGA公司居然将 + r1 + 摇身一变成为方块游戏，所以玩家才能在 + r1 + 里头看到众主角截然不同的逗趣风貌。

 + r1 + 相关产品并不只限限在游戏上面，SEGA公司前前后后不但推出了游戏的原声带，而且因为樱花这个角色的大受欢迎，甚至聘请配音人员推出个人专辑呢！当然啦！因为帝国华击团这个设定的关系，SEGA公司同时也制作了不少歌谣专辑给玩家聆听分享，更不用说在日本当地举办过许多演唱会呢！从来没有一款游戏作品能够将其触角伸展得如此广泛，史上大概也只有 + r1 + 的林明美可以与樱花相互媲美吧！

至于 + r1 + 的电玩精品更是多得数不清，只要您想得到的东西应有尽有，包括一般的笔记本、文具、饰物、�的匙圈与玩偶……等等，都是 + r1 + 最大的商机所在。而且，为了满足那些拥有个人电脑的玩家们，SEGA公司更推出了多款 + r1 + 的插图光碟集，以及以樱花大战为主题的小型工具软体等等。更不用说在樱大战的Windows版本当中，以高解析模式重现 + r1 + 的完美世界。另外， + r1 + 还以OVA的方式在动画市场现身呢！虽然只有短短的四集，却赢得了樱花迷们极高的评价。

### 千呼万唤始出来的 + r1 + 

在SEGA公司推出DC主机之后， + r1 + 是否会转移演出舞台，一直以来都是玩家关注的焦点，而该公司为了不让玩家的期待有所落空，特别在十月份宣布了名为 + r1 + 这个超大型企划。 + r1 + 果然决定将在明年秋季重现江湖。在 + r1 + 一年之后的大正十五年，为了即将在巴黎成立的巴黎华击团之筹备工作，大神只身前往法国并展开了另外一段冒险的旅程。当然啦！在三代作品中，SEGA公司再度加入了五位全新的女主角，虽然无法与前两代的成员会合是有点可惜，不过反而带给玩家不同的游戏体验也说不定嘿！

在这三张GD-ROM的超大作品中， + r1 + 先是改善了以往因为读盘而造成游戏中断的缺憾，接着又在LIPS对话系统当中加入全新的设计，玩家可以在选择对话的同时表达出不同的感情强度，而且这回玩家还可以乘坐着光武在巴黎市区中穿楼寻找敌人。此外，还可以对应VM与歌剧下载等等新系统噢！

### 樱大战最新相关游戏纷纷出炉

 + r1 + 只是 + r1 + 的一小部分而已，因为SEGA公司将会推出五款与樱大战息息相关的游戏作品。而最让玩家感到兴奋的一点是，原来的樱大战前两代作品都将会经过重新制作之后，在DC主机上面再度登场。除了对应DC主机的打字游戏之备之外，与Windows版本一样的高解析画面最为受到玩家的期待。樱大战DC版本目前预计会在明年春季发售，为两张GD-ROM包装。而二代DC版本则是会在明年夏季推出，容量的安排恰好与准备在秋季发行的三代连成一气，想收藏的玩家可要开始存钱啦！

除此之外，备受玩家关注的樱大战GB版本则是会由Media Factory公司负责开发与发售。游戏虽然预定将在今年推出，不过可惜都没有公开进一步的详情，包括游戏内容是否会有所更动，以及将会以何种方式与DC主机互动等等。其他两款在DC主机上发售的游戏情报就比较明朗化了。预定在今年春季以两张GD-ROM推出的 + r1 + 是款将歌谣秀动作化的游戏作品。而 + r1 + 虽然早已公开，不过如今总算确定会在今年上半年发售。

### 完全满足视觉与听觉的樱花世界

 + r1 + 不只是针对游戏的部分。对于喜欢动画的玩家们来说，现在终于可以看到樱大战剧场版的诞生啦！而且最让玩家感到欣喜的消息，莫过于 + r1 + 即将推出电视卡通。虽然现在还无法看到任何画面，不过大家应该可以想像得到其某凑剧情与幽默场面不断出现的情景吧！而且呢，歌谣秀还是如火如荼地在日本各地公开表演，更不用说还会推出 + r1 + 的Windows版本，以及改编自一代与二代作品的樱花大战小说、音乐CD专辑……等相关商品了。

### 永远的樱花

 + r1 + 之所以能够成为成功的多方位游戏作品，RED公司当然是其中的每一功臣，不过SEGA公司本身展出不穷的点子更是使其受欢迎的关键所在。这种气势与深受玩家的热烈回响都不是一般恋爱游戏作品能轻易追赶得上的。

更别说是想要超越其成就，就连这类游戏的始祖 + r1 + 都甘拜下风！我们不知道 + r1 + 的未来将会变得如何，但我们可以预测的是， + r1 + 的故事将会一部接着一部地推出。

### 中文版本何时出线？

在这回的樱大战专题结束之前，相信有很多玩家都相当关切传闻已久的中文版本何时会出现！但相当可惜的是，太电玩实在的是保密到了极点，至今一切有关游戏制作的进度与发售时间全部无从得知。对于喜欢 + r1 + 的玩家们来说，真的非常希望可以看到亲切的中文对话能够出现在游戏里面，并藉此更进一步地了解与体验游戏独特的魅力，就让我们一起祈祷这样的梦想能够早日地实现吧！
