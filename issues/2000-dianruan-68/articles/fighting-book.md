## 97、99杂谈 北京 赵庚

### ■97为什么越来越火？

本来99年10月之前还没有几个人知道鬼步，但现在已经传开了，人们都在练习，想像96那样无限连。近立重拳>闷风>八酒杯指令>重拳>闷风的技巧，前提条件是不能有超杀能量珠。一旦没真的发不出八酒杯了。此法虽不难但真要练会还真不容易。不知为什么，打关时对手是大蛇很好连，但打到别人时连鬼步都出不来，有些人以为此法只可打大蛇，其实真正练会了是打准都行，对人战时也依然可行。为什么？因为我在99国庆放假期间曾被一位高手连了七个重拳>闷风，再>重拳>八稚女，我就是跟他学的。除他之外，我真正亲眼见过能够无限连的，只有两个人，本人最高纪录才三个（所谓无限连是到对手晕点或己方有能量珠时止）。鬼步后也可重拳>梦弹1段>葵花。

97一些很少人会的连续技在下面。

1. 莉安娜：超重击(COUNTER)或威武军刀后可接驳任意超杀。(限版边)

2. 特瑞：能量补充>轻碎石踢>上升击。(版边)此技除我之外未见过任何人在街机上连出来(MAX尖险外)，没见过的人都说此招连不上，其实碎石踢击中敌人时机正确便能连上，碗切点小跳，碎石踢应慢慢出，在那个 + r1 + 的时候击中对手(MAX时只需稍慢少许就行)，便可连接。

3. 八神：打对方背部时重拳后可打满两段梦弹，可惜在弹后跌八稚女外不能接任何招。但此招必须直接打，在百合折后即使打背也连不上。

4. 玛丽：①限陈可汉、两个克里斯。回身真落后倒对手后，不用在版边也能接驳垂直飞前(空交)>琛奇。②角落里俱乐部↓踹>踢倒对手后可接玛琉搔奇。

5. 草薙京：对方在角落时，MAX版大蛇薙发出后按住键，可将敌人用身上的火翼飞后再用大蛇薙攻击，得到4～7HIT效果，而七十五式。改后则不会用身上的火烧着对方。可是七十五式后若用下经拳解招(和特瑞无限击同问题)，便可烧着对方，但最多只能到6HIT，如果身上的火烧了4HIT，则后面3HIT的大蛇薙不会击中。

以上是我和其他几位同好对97的研究，请广大玩友放心练习，全部招式切实可行。

### ■99杂谈

一、关于空中追打：

都说97空中追打比98强，玩过99后，发现99的空中追打比97有过之而无不及，下面是一些实例。

1. 安迪：击壁背水掌>上面(↘A)1段>超破裂弹(版边)，在97中所有浮空投在追加地面攻击后对方在任何下落时，没有被攻击判定，而本作可以。

2. K'：第二居后可连接鸡鸣嗌/热动/一分钟，如果用援助攻击打飞版边敌人更可用2～4次第二居再接上述招式，而在COUNTER MODE下第二居可无限连击。

3. 全勐：空中↓B特殊技击中对手后可反复胜空中↓B无限连击，如果对手在角里，连几次后还可在超杀风圆天翻脚(3～4次后，双方将消失在屏幕中)。

4. KING：在97中，惊异玫瑰在角落里可接重龙卷脚，本作中也能，而且当自己有三格超杀能量增时，可反复用惊异玫瑰无限连击。关于不可防御投如下。

WHIP的跳重拳打点低时，对方防御不能；察宝健的跆重拳打点低时，第三段时不可防；庵宫用超能力球>超级精神透，对方防不住第一个飞行道具；八神暗勾手>属风及驹磨用虎煌拳>翔乱扣也是一样的效果。再有，全勐的太极极、察宝健的→A，驹蹄的猛虎无赖岩都是一招破防的招式，在CM点可接超必杀技，而察宝健除MAX超杀外，任何状态下都可接。

当己方援助角色为：草-1、2，真草薙、八神、全勐、坂崎良、金家藩，察宝健，可利用指令投空变使对方防不住援助攻击。而当援助角色是玛丽或慎元高时，除此法外，可用空变发翌攻击令对手不可防御。

##  + r1 + 传，笑开颜

大家对97中的无限连杀一定记忆犹新，特别是特瑞那招曾被人列为 + r1 + 。如今97仍是经典机台，很多人还在不断研究。以下便是由多名高手共同研究出的惊世成果：八神庵的 + r1 + 。

 + r1 + 如像是一样走路，即双脚不动而人物像向前移动，就像是 + r1 + 过去一样。由鬼步就能产生自KOF96以来失传已久的八神无限连杀—— + r1 + 。别说不可能，真的有这么一招。出法即是在 + r1 + 过对手时的，输入→↘↓↙←→A/C，八神就会迅速飘到对手面前，此时对手还处于不能防御不能状态，再输入 + r1 + ，再出鬼步，如此反复，即可形成传说中的无限连杀了。

这招看似简单，但做起来奇难，最关键之处在于掌握出招的时间与时机，简称 + r1 + 。建议一开始先练习用暴走八神出鬼步，他走鬼步速度有够快，比较明显，比普通八神给人感觉更酷。另外，在 + r1 + 后演中，也能使出鬼步，不过动作不明显且不实用（有些KOF99），但可以作为一项基础练习。鬼步的最大特征就是看得比平常快一些，当你看到一位高手使用鬼步无限连杀时，一定会发自内心地表叹惊叹——

太有型了！最后祝大家早日练成，神行天下！

另附送一套一击至 + r1 + 的连续技：

跳D>站C>鬼步>站C>属风>鬼步>站C>→A>葵花三段>晕点。（注意：对某些人物时鬼步后接不上 + r1 + ，原因尚不清，如韩国队的一大一小）

## 第七种连续技 图文并茂/四川南充 柴舟一号

笔者以为，援护型连续技的思路应是将援护代入连续技而非以援护启动连续技产生的追打。

①飞行道具接援护攻击产生的连续技。如：（相距较远时）八神弱 + r1 + >马克西马 + r1 + >八神前冲MAX斩爪柄，此类技还可变为 + r1 + 连技，即当飞行道具逼近对方时，对方若也以飞行道具相抵，此时放出援护，亦可成功。如不知火舞的此类技巧，（相距较远时）花蝶扇>红丸 + r1 + ，驹磨、特瑞之类同样适用，前提是需用自动追踪型援护配合。

②逆向按接援护攻击产生的连续技。实际上是发动援护角色后利用其攻击的时间差形成 + r1 + 。如：（己方版边，相距对手较近时）真草 + r1 + 发动>红丸跳逆向>跨重拳> + r1 + 援护攻击>MAX雷光拳，此技中二阶堂跃起的时机很重要。此类援护还有拉尔夫、包、�的元两等人，且拉尔夫可在对手未被击飞前继续追打，如，（相距较近时）拉尔夫 + r1 + 启动>八神百合折>拉尔夫 + r1 + >入站重拳>梦弹>MAX八稚女。这类的援护若配以投技角色可令对手因恐，其他角色在援护发动后引则要发到对手身后方能成为防御不能技。

③通常按接援护攻击产生的连续技。最酷的一种。如果说前面一二类技巧是利用援护产生多变的战术，那么这第三种就是 + r1 + 类的大杀伤力攻击，华丽而难成功。

（对手不在版边）八神百合折>克拉克 + r1 + >八神前冲至敢身后待机发出MAX版斩爪柄；

（对手不在版边）二阶堂红丸逆向低位重踢>特瑞 + r1 + >（此技巧若对手蹲更易成功）；

可在逆向攻击后作出此类援护的角色非常多，但关键是逆向时的打点要低。目前仅发现八神百合折和京 + r1 + 较易成功……

相信此类技巧还有大有文章可做……

看到这里，不知各位有无觉得，使用援护型连续技时不仅要熟悉援护角色的性能，而且对距离感的把握也十分重要呢？不光是许多援护角色在特写距离才能发挥最大能力，己方所控角色的攻击距离也十分重要，差之毫里也许会酿成大败绩。笔者以为，训练在实战中对 + r1 + 和 + r1 + 的控制，都是 + r1 + 之根本，如同武林中的手眼身法步，这一点在三维对战中尤为突出。

最后举几个特定距离才能发挥威力的援护的例子：真香 + r1 + 启动后若在小手画面1/3处接触对手，便无法相住发挥作用。八神八酒杯若不是连摊带 + r1 + 全数命中便无法追打，因此写对手距离要近。还有不知火舞、也利、WHIP、全勐、察宝健等人若不接触对手便无法使之浮空。再有玛丽、东文……

## STRIKE 连技 湖南大学 BAO

KOF99中很大一部分连技是围绕援护展开的，这类技巧一般具有限制条件多、复杂、杀伤力大的特点，因此熟练掌握之是成为高手的必备条件。而STRIKER由其自身的特性、使用方法，可分为四类。

一、攻击发动型：

以援护攻击作为连续技的起始动作，后面的攻击必须在援护击中后才成立，特点是突然性、快速性且判定均较强。典型STRIKER是罗伯特、KING、真吾。

注：下文中的 + r1 + 通常指对手，COUNTER MODE简称CM，ARMOR MODE简称AM，SUPER CANCEL简称SC。

■K'：近版边>呼罗伯特>站C>(扳击>第二居)N次>热动

近版边>呼KING>MAX热动>向回踢>MAX热动(一击必杀)

近版边>CM>呼安迪>(扳击>第二居)N次。此为无限连，利用 + r1 + 的浮空追击达成。

■二阶堂红丸：近版边，CM>呼罗伯特>站A>站D>弱雷韧拳>驳↓D>雷韧拳>SC雷光拳(不发动CM，则 + r1 + 一段取消)

■特瑞：近版边，CM>呼罗伯特>站C>碎碎石踢>站D>→A>弱能量波>SC高轨喷泉(小个子会打不到)

近版边>呼KING>站C>发动AM>闪避攻击N次>开龙(一击必杀)

■东丈：近版边>呼罗伯特>站C>→B>弱旋风拳>MAX史上最强之低踢(对手晕点)

■舞：近版边，CM>呼罗伯特>蹲C>↘B(1HIT)>龙炎舞>SC风凰之舞>强炎炎舞>SC超必杀祝蜂(第一次龙炎舞后对方应正好浮空)

■良：近版边>呼KING>站C>发动AM>闪避攻击N次>虎煌(一击必杀)

■罗伯特：呼KING>MAX霸王翔殴风拳>前冲>MAX无影旋风十段脚(类似K'第二式，关键差招要快)

这一类援护的特点是抓住对手处于受创僵直的每一瞬间进行块大破坏力的追击， + r1 + 是原则。限于版面，其他人再说。

二、攻击接续型：

在连续技中起着承上启下的重要作用，还往往能使自己不能再追击的对手重新产生被攻击判定。香澄、良是其中的代表性人物，有他们在，往往能形成一击必杀或无限连。

■K'：跳C>站C>(扳击>第二居)N次>前走两步>呼香澄N次>热动(一击必杀，关键是划线部分要走到对手身后)

■宗-1：近版边>空中C>站C>→A(七十五式攻>前走两步>呼香澄)N次>强赐车>暗勃(一击必杀)

■宗：近版边>空中C>(站C>七十五式攻>前走呼叫香澄>向后大跳>重铁铁)N次(重铁铁后必须在收招同时接C，始可形成连击)

上式的变形为：版边，CM>空中C>站C>七十五式攻>前走呼叫香澄>后大跳>重铁铁(1HIT)>SC大蛇薙

呼叫香澄要把握好呼叫的时机，太早不能使对手再次浮空，太晚也不行。其实拥有打浮空技巧的选手都可以一式，说不定会有惊喜。
