# 史莱姆诊所

问：VESA 制式是什么东西？　　 送文　黄柏庚

答：VESA 是 VIDEO ELECTRONIC STANDARD ASSOCIATION 的缩写，是一种 Super VGA 的统一协定，主板上的 VESA 是指 LOCAL BUS（局域总线）的一种，此外，还有 PCI、ISA、EISA 等总线。

问：①曾见电脑 486-33，“33”是什么意思？②视霸卡作用是否仅提高解析度？光驱有何规格差异？③声卡是否能用音响代替？④CG 如何戴出了？⑤3DO BLASTER 在国内似乎还没有售？　　　杨荣保

答：①“33”代表 CPU 的主频，DX50 DX、66 DX、100 也是此意。②视霸卡是用来与视解外设连接（如电视机、录像机、游戏机等），做图像采集、编辑之用，光驱从速度上有有单速、倍速、三速、四速、六速之分，从界面看有 SCSI、AT、IDE 之分。③声卡是音频输入/输出设备，音响可接于声卡上，但不可代替声卡。④采集 CG 最好用截图软件来完成，照相效果也差，直接用打印机的可能性不大。⑤3DO BLASTER 在国内似乎还没有出售。

问：我的硬盘比较小，听说 DOUBLE SPACE 可以增容硬盘，是吗？　北京　小黄

答：DOUBLE SPACE 的确可以增容硬盘，但真的数不了了，假的也真不了，增容后硬盘操作变慢，功你别用。

问：越玩越理机具的碰游游戏如何在 PC 机上玩　　天津　张冰

答：不可能，就好像电影胶片不能在录像机上用一样，超任的数据结构与 PC 机的数据结构不一样。

问：为什么有时候存储结后，DOS 命令行的字会变成很大的白字？　北京　陈浩

答：这是显示模式被误操作改变，如果此时没有死机，键入 MODE CO80 回车，模式就可改回。

问：我前两天发现我的硬盘上多了一个几兆的叫 386SPART 的文件，删除后发现 WINDOWS 起动后总有警告，这是因什么？　黑龙江　系力

答：那个文件的 WINDOWS 虚拟内存的交换文件可帮助 WINDOWS 提高速度，如果删掉了，进入 WINDOWS，在 Control Panel 中的 386 Enhanced 中修改。

问：听说有些游戏科技做了《天龙入部》（天龙八部），为什么到现在我也玩不到？　北京　中献

答：因为内地缺乏最新软件推导，新软件的消息大部分靠小道消息传播，您听说的可能是这类谣言，实际上智冠的确提过还会制作这两款游戏，但似乎还没有发布。

问：我有一块创造的 16BIT 声霸卡，可以游戏的，总觉得音乐不动听，难起来播放，我该如何修改？

答：您的算口真是很大呀？您只要买一块劲的 SB AWE32 正式版加一块 WAVE BLASTER，就能听到真实乐器演奏的声音了，如果您还嫌不够，在声卡上加挂个 28 兆内存会更好。

问：《三国演义 I》何处有卖？　北京　阿林

答：据可靠消息，在本文截稿之后（9 月 30 日），《三国演义 I》还未发行，许多邮购广告大多是虚假的。
