# 美丽的世界

## DEMO 与 Future Crew

文／软体动物

提到 DEMO，一般的玩家会以为是游戏开始前介绍游戏的动画，但我们今天讲的 DEMO 是一纸团体方参加国际性 DEMO 比赛而制作的 DEMO，展现出许多高难度的图形，带给欣赏者不少的惊叹。

### 什么是 DEMO？

“DEMO 是 demonstration 的缩写，在电脑上的 DEMO 常的说说是展示电脑图形与音乐的程序，所以游戏开始的动画画面也是 DEMO 的一种，在电脑公司，可以看到电脑上展示介绍电脑软、硬件的程式，这些属于高级性的 DEMO 这来说，这里面大部分是图形与音乐的汇表了，”

如果只是一般 DEMO 那就没有什么好看的了，这里要介绍的 DEMO 并非指的商业性 DEMO，而是在国际比赛，由多个参赛团体专门为 DEMO 比赛而制作的极为出色的 DEMO，比的是谁的力量、谁的技术更难以难度而更精彩，从来的 PC 上的种种，相比电脑展示也是超级出色的一种，颇为不同！

DEMO 主要目的是以一种超乎大家想像并且发挥电脑在绘图与音乐上的能力，也就是说 DEMO 结合了人看到目前以来的 CG（Computer Graphics）与令人感叹的音效第三目不纯的音乐，再加上 DEMO 制作者们堪称一绝的 DEMO 不假大，也可以播放 10～20 分钟，这也说明要 DEMO 运行顺畅，则需要一个运算快速的 CPU 通常 486-33 就可以了。

得到的优秀作品，在 DEMO 的程式设计时，进行程式化的立体声的音乐与音效。

DEMO 就是“示展化，我把 PC 变 SGI 工作站了”（此此，语出自某电影改编过来，SGI，即由 Silicon Graphics Inc 所生产的工作站，拥有快速绘图与快速计算能力）

将来的优秀作品，在 DEMO 的程式设计时，进行程式化，充分发挥 PC 的硬体潜力后，让本身普通的 PC 加算效率的组合，借 DEMO 产生新的功力，表现令人叹服的效果，DEMO 让人发觉 PC 并不是那个中坐人听，也充满满足效果，DEMO 的行色程式结合了各类型电脑的声光效果，包括：多多支持音乐，即时运算产生的着形（Render）图形、碎形（Fractal）、透明、电浆（Plasma）、3D 向量空间（Vector Space）、虚拟真实（Virtual Reality）、即时变形（Morph），前与向超凌凌旅游等等。

1. 使用汇编语音：要产生一个简单的 DEMO，用高

---

为了让 DEMO 产生有趣效果的进度与效果，这些 DEMO 的程式编写者（Coder）参以汇编语言（通常是 80386 的汇编语言）来编写程式，并将重要的算数法以汇编语言直接码进行最佳化（使用汇编语言，可以充分地处理与控制最细节的一些，拥有最细量的控制，令 DEMO 其至可以接近在 X 保护模式），把 DOS640K 的限制完全置于脑后！

2. 多声道的音乐，由于充分支持许多扩音卡（如 GUSB 或 SB PRO）Gravis UltraSound（GUS），在这些 DEMO 中都配合可到多声道音乐（Multichannel Music），超越一般来普遍的 4 声道，有些更高达 16 声道的立体声输出，因为 DEMO 援助此档的 MOD，S3M，等等的音乐格式，只要能支持这些格式的音效卡，都能发生此类 Wave table 的音效，也就是说 DEMO 的音效卡扮演了自身已总合强大的音效功能。

3. 突破传统的绘图分析度，在 PC 上经常 VGA 在 320×200 的解析度只能同时显示 256 色，且有着少少记忆体及多数限制，而 DEMO 往往使用特殊的模式（通常移为 X Mode）在这些模式下能达到 320×200 256 色，超高色记忆体，320×240 256 色，还有 640×480 256 色，还可以 320×400×3840 色，这些特殊模式都借用了一般 VGA 卡上就在速度上可以仿化，由于在多个绘制记忆体单元，进行动作时不需更加的激加输精，随着解析度的提高时画面上等各列出显示的图形将全部加细解。

4. 即时运算，在这些 DEMO 里大多有 3D 向量空

---

何、虚拟真实的部分，成是有许多的电脑上也效果，还有变形等，在 DEMO 中看到的这些特效都是程序执行的时候由 CPU 即时计算出得到的，而不是一般的动画，把事先作好的画面一张一张展示出来，由于即时运算的缘故，尽管一个 DEMO 不很大，也可以播放 10～20 分钟，这也说明要 DEMO 运行顺畅，则需要一个运算快的 CPU 通常 486-33 就可以了。

### DEMO 的创造者

DEMO 并非凭空产生，就象编一个游戏，任何的 DEMO 都需要有程式设计者、美术人员与搞音乐的花好几个月的时间（也就因为此），尤其对于 DEMO 有共同志趣的人就会集合组织成为一个 DEMO 的团体，借着每一位成员的才华，共同创造设计一个 DEMO，一个 DEMO 团体通常包含：

①领队（Organizer），统筹规划并且带领整个团体。
②程序人员（Coder），设计 DEMO 所需要的程序。
③作曲家（Musician），创作配合 DEMO 内容的音乐与音效。
④美工（Graphics Arts），创造绘制 DEMO 所需要的图形。
⑤其他人员：负责 BBS 或 Internet 管理与协助工作，DEMO 界的大集会。

### DEMO 界的大集会

“International Demo Competitions”简单说就是国际性的 DEMO 大赛，通常由组织较为庞大的 DEMO 团

体来主办，比赛项目包括 DEMO、INTRO（小于 64Kb 的 DEMO）、Music 及 Graphics，通常会不同的种如 PC、Amiga、Atari ST 和 C-64 举行不同比赛。各个参赛团体也借此机会，交流创作 DEMO 的心得，比赛的名次于参赛团体投票累产生，获得者可欣奖金或是赞助而提供的软、硬件设备。对于参赛的所有 DEMO，主办单位都会将作品发展在网络上，或是收录在 CD 内，由于这些 DEMO 都是免费的，任何人可以在不对作品修改的情况下，自由下载。

### DEMO 界之骄子：Future Crew

如果你也喜欢 DEMO，你一定看过 Unreal、Second Reality，这两款可以说是 DEMO 中的经典之作，也是我终向全世界 Assembly 大赛冠军的团体。

Future Crew（简称 FC），在 Asm'92 及 Asm'93 分别以 Unreal 及 Second Reality 获得 92、93 汇编术大赛 PC DEMO 冠军，当 Unreal 通过网络及 BBS 发行后，造成了大轰动，看过他们 DEMO 的编程人员，对他们技术的精湛都叫叹不如，他们充分运用汇编语言的威力，加上

多声道的背景音乐，再加上美丽的电脑绘图，以即时演算的方式将声音与图像同时展现出来，创造了前所未见的特效。Unreal 仅仅 2MB 多，就能播放 20 多分钟。等到 Second Reality 更是持到了 DOLBY SURROUND 的音效处！不仅如此，DEMO 的每一个都部都由 FC 精心策划，不论是画面呈现或是音乐的处理都得到好处，如付有 SB PRO 或 GUS，那绝对不值错过这是一 DEMO，一定让你不相信这是一块简简单单的音效卡能发出美丽的立体声的音乐与音效。

### 94 国际 DEMO 比赛

托高科技的福，着老已了解了 Asm'94 及 The Party 大赛的的结果，94 年的比赛可说是新入云霄，而留覆纳的 FC EXTREME 等都投有参赛，94 年的竞标 93 年以前的比赛有了不少变化，图形方面，由于 VR 技术的广泛应用，DEMO 中自然也少不了，94 年 DEMO 更加入了大量的类似《超模型（其里）》中的 LANDSCAPE，和类似《DOOM》的 3D 迷宫，93 年前 DEMO 中出现的 3D 物体（OBJECTS）都没有细图（TEXTURE），94 年 DEMO 中的物体已经贴上美丽的纹理，而且 640×480 高解析已经不再只用于 DEMO，真是新加加到不老不重的科技世界，以下是今年的几场比赛来，有超丰富的。

### 后记

为了大家介绍了平天的天 DEMO 的知识，真不知是否会爱欢迎，但有趣于让 PC 玩家，尤其是一些中小学玩家，电脑对他们来讲就是游戏机，多介绍点电脑艺术作品，会有助于提高 PC 玩家的素质，一个游戏往往玩一遍，但经典的 DEMO 几乎百看不厌的，PC 玩家们，TRUST ME！当你看看 DEMO 中绚丽的图形，听着超动感的 POP Music，你会大声的说，“It's Amazing！”
