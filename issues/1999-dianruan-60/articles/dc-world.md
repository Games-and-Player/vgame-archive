# 桃园结义中华区DC专题

> 桃·圆·景·桃 / PDF p.38-39 / 刊页 31-32

## SS出局后群雄逐鹿的配件市场大战

【图】DC手柄正面照片；SEGA BASS FISHING钓鱼手柄及游戏包装

### SS手柄与DC手柄的比较

随着SEGA在96年底推出DC（Dreamcast），SS（SATURN）的时代也正式宣告结束。但是，对于SS玩家来说，最关心的莫过于DC的手柄操作感是否能够延续SS的优良传统。

SS的手柄曾经被认为是当时手感最好的家用游戏机手柄之一。尤其是在格斗游戏方面，SS手柄的十字键被公认为格斗游戏的最佳选择。那么DC的新手柄表现如何呢？

DC的标准手柄在外形上做了很大改变。手柄整体更加圆润，握持感更好。但十字键的手感与SS手柄相比有所不同，这一点让许多SS老玩家需要一段时间来适应。

手柄上新增了两个扳机键（L/R TRIGGER），这在家用游戏机手柄上还是首次。扳机键的压感设计使得赛车等类型的游戏操作更加细腻。手柄正面还配备了一个插槽，可以插入VMS等配件。

### ■ SEGA BASS FISHING——SS钓鱼手柄的升级

【图】SEGA BASS FISHING钓鱼专用控制器“GET BASS”外观照片

SEGA BASS FISHING是DC上的一款钓鱼游戏，配套推出了专用钓鱼控制器。这款控制器的原型来自SS时代的街机版“GET BASS”。

在SS时代，SEGA就推出过SPORTS FISHING（94）和SPORTS FISHING2（95）等钓鱼游戏。而DC版的SEGA BASS FISHING在画面表现力上有了质的飞跃，鱼群在水中的游动表现极为逼真。

专用钓鱼控制器的设计模仿了真实钓竿的外形，上方有一个摇轮用于收线，控制器本体可以模拟挥竿的动作。虽然这种专用控制器的适用范围有限，但对于钓鱼游戏爱好者来说，使用专用控制器的沉浸感是普通手柄无法比拟的。

在街机中心，“GET BASS”也是人气颇高的大型筐体游戏。DC版的SEGA BASS FISHING完整地将街机的游戏体验带回了家中。

### ■ DC手柄周边配件大全，究竟有多少“变种手柄”

【图】多款DC控制器及周边配件照片——标准手柄、钓鱼控制器、格斗摇杆等

DC的周边配件市场可谓百花齐放。除了SEGA官方的标准手柄外，第三方厂商也推出了大量的替代手柄和专用控制器。

#### VMS（Visual Memory System）

VMS是DC最具特色的周边配件之一。这是一个带有LCD显示屏的存储卡，可以插入DC手柄的扩展槽中。VMS不仅可以保存游戏存档，还可以作为独立的掌上游戏机使用。

VMS的屏幕可以在游戏过程中显示额外信息（如格斗游戏中的出招表），也可以下载小游戏独立游玩。这种将存储卡与掌上游戏机结合的设计在当时非常前卫。

VMS使用CR2032纽扣电池供电，在脱离DC主机后可以独立运行。内置的小游戏虽然画面简单，但可玩性出乎意料。多个VMS之间还可以通过红外线接口进行数据交换。

每个VMS拥有约128KB的存储容量，对于当时的游戏存档来说已经足够。但部分大型游戏的存档会占用较多空间，建议玩家准备多个VMS备用。

#### ■ 究竟有什么“变种手柄”

【图】各种第三方DC手柄——包括ASCII专用控制器等

除了SEGA官方手柄外，ASCII等第三方厂商也推出了各种类型的DC控制器：

- **ASCII手柄**：ASCII是日本著名的游戏周边厂商，为DC推出了多款手柄。其中格斗专用摇杆的做工精良，十字键和按键的手感接近街机水平，深受格斗游戏玩家好评。
- **方向盘控制器**：配合DC上的赛车游戏使用，提供更真实的驾驶体验。
- **枪型控制器**：配合光枪射击游戏使用。

上述内容主要从外观设计和功能说明的角度介绍了DC的各种手柄和周边。不过需要注意的是，第三方手柄的品质参差不齐，购买时需要仔细甄别。

#### MEGA CD等老配件回顾

文章还回顾了SEGA过往主机的周边配件历史，从MEGA CD到SS的各种配件，展示了SEGA在游戏外设方面的丰富经验。DC的周边配件策略正是建立在这些历史积累之上的。

### 总评

DC虽然在主机大战中最终败给了PS2，但其在游戏配件设计方面的创新精神不可否认。VMS存储卡的概念、SEGA BASS FISHING的专用控制器、以及丰富的第三方配件生态，都展示了SEGA在游戏硬件方面的深厚功力。
