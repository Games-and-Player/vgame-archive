# 游戏机与游戏机

> 桃·圆·景·桃 / PDF p.40 / 刊页 33 / 大/OLD CAT

## 就目前情况来看在PC并不比新世代游戏机差多少

桃·圆·景·桃

大/OLD CAT 关注焦点

【图】大型机甲/机器人战斗场景截图

### 先说一下PS2。

关于SONY的PS2，近期的消息越来越多了。PS2作为SONY的下一代游戏主机，被寄予了厚望。目前已知的PS2规格令人印象深刻：

PS2搭载了SONY自主研发的“Emotion Engine”（情感引擎）处理器。这颗CPU的运算能力达到了惊人的水平，理论上可以处理的多边形数量远超现有的任何游戏主机。

关于PS2的POLYGON（多边形）处理能力，目前公开的数据显示其每秒可以处理约2000万至7500万个多边形（根据不同计算条件）。这一数字远超DC的约300万多边形处理能力，也超过了当前PC上最强的3D加速卡。

PS2采用DVD-ROM作为游戏载体，这意味着游戏容量将从PS时代的CD（约700MB）大幅提升到DVD的4.7GB甚至更高。同时，PS2还可以播放DVD电影，这使得它不仅是一台游戏机，更是一台家庭娱乐中心。

但是，需要注意的是PS2的发售日期尚未最终确定。目前的消息指向2000年初，但具体日期可能会有变动。

### INTEL与PC的发展

另一方面，PC阵营也不甘示弱。INTEL的新一代CPU正在紧锣密鼓地开发中。

目前INTEL的CPU已经从初代PENTIUM发展到了PENTIUM PRO、PENTIUM II等型号。最新的PENTIUM II的主频已经达到了相当高的水平。而INTEL正在开发的下一代处理器——代号为MERCED的64位CPU，性能将更上一层楼。

MERCED（即后来的ITANIUM）是INTEL与HP合作开发的IA-64架构处理器。这款CPU采用了全新的64位指令集架构，理论性能远超现有的32位处理器。不过MERCED主要面向服务器和工作站市场，对于普通PC游戏玩家来说，更值得关注的是INTEL在消费级CPU上的进展。

在PC的3D图形处理方面，当前最强的3D加速卡的多边形处理能力已经相当可观。虽然在纸面数据上可能不及PS2的Emotion Engine，但PC平台的优势在于：

1. 硬件可以随时升级
2. CPU和GPU可以独立升级
3. 分辨率和画质可以灵活调节
4. PC游戏的3D引擎优化水平在不断提高

实际上，如果以同样的画面条件（分辨率、多边形精度、贴图质量等）来比较，当前PC的3D性能并不比即将发售的PS2差多少。PC的高分辨率（可达1024×768甚至更高）在视觉效果上甚至可能超过PS2的标准输出分辨率（通常为640×480）。

### DVD与光存储技术

DVD技术的普及是另一个重要话题。无论是PS2采用DVD-ROM，还是PC上的DVD-ROM驱动器，DVD都将成为未来几年的主要存储媒体。

DVD的单面单层容量为4.7GB，双面双层可达17GB。相比CD的700MB，容量提升了数倍乃至数十倍。这意味着游戏开发者可以在游戏中加入更多的高质量FMV（全动态影像）、更高精度的3D模型和更丰富的音效。

不过DVD驱动器的价格仍然偏高，且DVD可录写媒体的标准尚未统一（DVD-R、DVD-RAM等多种标准并存），这些问题需要时间来解决。

### PC与游戏机的融合趋势

从长远来看，PC与游戏机之间的界限正在逐渐模糊。PS2本质上是一台专用的高性能计算机，而PC也在越来越多地被用作游戏平台。两者在硬件架构上越来越相似。

DC已经采用了Windows CE操作系统（虽然并非所有游戏都使用），这是游戏机向PC靠拢的一个明显信号。PS2的硬件架构虽然采用了专用设计，但其设计理念与PC的GPU+CPU架构已经非常相似。

未来的游戏主机可能会越来越像一台专用的PC，而PC也会通过更好的游戏优化来提供媲美游戏机的便捷游戏体验。在这样的趋势下，“PC与游戏机孰优孰劣”的争论也许终有一天会变得没有意义。

但就目前来说，PC的优势在于灵活性和可扩展性，而游戏机的优势在于统一的硬件标准和即插即玩的便利性。对于真正的游戏爱好者来说，两个平台各有千秋，都值得拥有。

SEGA的DC作为现在已经发售的次世代主机，在128位时代率先出击。虽然面对即将到来的PS2的巨大压力，但DC依然有着自己的一席之地。而PC平台凭借其不断进化的硬件能力，也始终是游戏市场中不可忽视的重要力量。
