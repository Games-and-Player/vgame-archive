## PS 回想之夜2·睡眠的支配者 (FP 4)

四个小孩的幽灵：在餐厅的桌子底下就有一个；若是在半夜12:00～4:00这段时间打开ジム的 + r1 + 也会发现一个；拿着油灯到ポート小屋中，便可以发现第三个小孩；到 + r1 + 将蜡烛点燃，便可以在桌子下面发现第四个。玩家只要装备 + r1 + 后再和这四个小孩对话，他们就会安心地开花了。

北京　手持 + r1 + 护符找幽灵的　风马

## PS 汉堡大王2 (FP 5)

好吃的汉堡：在标题画面上，按三次右、三次左、○键，开始游戏就可以参考点数来改变烹任方法，制作出更好吃的汉堡。

北京　比较爱吃汉堡包的　风马

## PS 虹吸战士 (FP 4)

可以选择全部任务：游戏中暂停，在OPTION中将光标与 + r1 + 重合，同时按R1、R2、L1、L2、○、×、口七个键，如此便可以选择全部20个任务。对于技术不高的人，可以利用这个技巧反复在同一关中练习了。

武器全部出现与弹数无限：在武装单面将光标与 + r1 + 重合，同时按R2、L1、L2、SELECT、○、×六个键，这样全部八种武器就会出现，并且可以使用，子弹也是无限量的。

可以看到MOVIE：在OPTION中，将光标与 + r1 + 重合，同时按R2、L2、SELECT、○、×五个键，就可以在 + r1 + 中看到MOVIE了。

北京　最喜欢打入敌人内部的　风马

## PS 机械帝国 (FP 5)

得到隐藏武器：在游戏中，要制作最强的武器レールガン，就必须找到四个零件。得到零件的地方如下：零件A在ACT 1的第9目标地点前的地区；零件B在进入ACT 2的击退直升机的车辆；零件C在升降梯的后面；零件D在ACT 8的第7目标地点的东侧。

欣赏动画：首先通一次，接着在标题画面中，选择 + r1 + ，就会出现MOVIE阅览项目，选择它的话，就可以欣赏到动画了。

北京　自认为在 + r1 + 中无法生存的　风马

## PS L的季节 (FP 6)

两个隐藏选项：在现实界和幻想界这两个世界中，分别都以GOOD ENDING过关的话，在角色的主题曲的选项中，就会出现 + r1 + 的选项；在出现此选项的状态下，看翡翠社的结局，这时会发现增加了观赏辅助角色登场面的选项，这个选项在 + r1 + 之中。

北京　达成率都没有达到100%的　风马

## PS 金田一少年的事件簿3·青龙传说杀人事件 (FP 5)

看到七濑美雪的事件簿。由于在此事件簿可以看到很多恐怖的画面，所以不可错过。

## PS 狂野兵团·战火二度点燃 (FP 7)

可以战斗避免：在游戏中遇敌时会出现一个惊叹号，此时按手柄上的△键就可以避免战斗，这对讨厌地雷的朋友有些小帮助。

上海　边在玩网络创世纪边打MAIL的　CYZZY

## PS 太阁立志传3 (FP 8)

赚钱大法：游戏中钱是非常紧张的，秀吉的私有财产可以用S&L大法。可是成为国主后城中的资金和米粮却很有限，感觉上总是不够用。其实有一个好方法可以使钱用之不尽，方法如下：在评定中让算术和内政高的武将反复买卖军粮。大体上1000石军粮可卖12000贯而10000贯可买到军粮50000石，这样每月月初就可白赚40000～50000贯，这样就有足够的钱发展壮大，建立无敌军团。

橘右京

## PS 劲舞革新版·二度混音 (FP 5)

隐藏模式1：镜面模式：MUSIC选项中，按SELECT，再按4次左右，箭头方向会相反；LEFT模式：MUSIC选项中，按SELECT，再输入8次左，箭头边时针方向旋转90度；RIGHT模式：MUSIC选项中，按SELECT，再输入8次右，箭头顺时针方向旋转90度。DOUBLE下无效；SHUFFLE模式：MUSIC选项中，按SELECT，再输入上、下、左、右、下、上、左、右，箭头顺序随机，DOUBLE下无效；HIDDEN模式：MUSIC选项中，按SELECT，再按4次上下，箭头中途消失；LITTLE模式：MUSIC选项中，按SELECT，再输入左、下、右、左、下、右、下，就会变为EASY模式；将已选出的隐藏模式1中的模式取消：MUSIC选项中，输入4次上。

隐藏模式2：全曲模式：PLAY MODE SELECTION中，连续右将 + r1 + 旁的4个箭头消去后开始即可；主菜单中间时按住左右键，START后选择：COUPLE模式（可跳STEP BATTLE舞曲），DOUBLE模式（一人同时跳二人舞步），VERSUS模式（两人跳同样舞步）。

使用隐藏角色：Konsent & Dsharezukin：主菜单中按住右，再按START；Afro & Lady：主菜单中按住左，再按START。

天津　ngpan

## PS 洛克人 (FP 5)

改编版的音效：当玩ナビモード之际，可以将某些特定舞台的音效变为改编版的音效。方法为按住SELECT键，来选择ナビモード，开始游戏就行了。之后在カットマン舞台、ガッツマン舞台、エレキマン舞台、ワイリース舞台的1和2，以及各关头目的登场音效，都会变成改编版的。

北京　觉得洛克人圆头圆脑不好看的　风马

## SS 街头霸王ZERO3 (FP 5)

环游世界模式的隐藏站：当通过环游世界模式中全部19站之时，如果总等级达到27，就会出现隐藏的美国站，在这里要向罗伦托、纳什、盖尔对战；当通过美国站时，总等级达到29，便会出现鬼牙洞穴站，最后出现杀气 + r1 + ；当通过鬼牙洞穴站时，总等级达到32，便会出现灾祸站，在这里最终会合真豪鬼对战。

终结VEGA出现：首先通过环游世界模式的全部舞台，在选择角色的面面按住R键再选择VEGA，就可以使用终结VEGA了。此技在PS版中没有。

北京　在编辑部舞台等待旅行格斗家到来的隐藏BOSS　风马

## ARC 侍魂2～阿修罗斩魔传 (FP 4)

## ARC 三国战纪 (FP 6)

无限复制宝物：首先将飞镖和炸雷的数目凑到十个，然后再拾一个火箭令牌；只将第二页，也就是B面的物品留下三样，包括火令前，丢掉一个已复制的物品，再去拾一个不能复制火箭令牌之前的物品。重要的是不能再拾B面的物品了，如果拾了就不能复制了，已可以拾A面的。还有，不能复制火箭令牌之前的物品，只能复制拾到火箭令牌后再拾到的物品。复制的物品有苹果、七星灯、酒……，不能复制A面的物品。

## PS 最终幻想8 (FP 5)

取得エネルギー结晶体的捷径：取得飞空艇后，让其飞到エスタエアステーション旁的降落阶上即可进入城中。按前、右、前的方向来到一个有三个通道口的地方，进入向前面内延伸的小路后再向左走。这时可发现通路的右边有一个黑影，上前与其战斗，打败它后就可以取得结晶体（与LEVEL高低有关，越高取得机率越大）。反复进出入这个画面，就能不断地取得结晶体（最好装备不逊敌的能力）。

## GB DQM·特瑞的神奇王国 (FP 4)

更改姓名：在游戏中时常会同样的怪兽出现，因为姓名也一样，会造成许多不便。这时，可到大树国最右下方的房间里（星之祠右面）与里面的神官谈话，庭可修改所带怪兽的名字。

预知怪兽的成长极限：在牧场主人曾普里奥（プリオ）的下方有一只大象，与之对话并选择 + r1 + （はい），它就会告诉你所带怪兽的成长极限。

わたぼう取得法：游戏中有一只特别的怪兽 + r1 + ，取得方法是通过全部31个旅行之门，再到牧场的地下室找它谈话即可。其成长速度与魔法的预力很高。它特别在哪呢？玩家自己去看吧。

关于捕捉 + r1 + ：到游戏后期用 + r1 + 来配合，而得到 + r1 + 是最基本的方法。但它和史莱姆妈妈的防备力高，速度快，很不容易抓到。本人有一种较容易抓住这种怪兽的方法介绍给大家：先选择几只容易使用魔法攻击的怪兽，然后在指令台中选择 + r1 + 中的第一与第二项指令。遇到 + r1 + 时先把出し + r1 + ，这时己方怪兽多半使用魔法攻击，而魔法攻击也对 + r1 + 不会造成伤害，这样就消灭了其它怪兽，接着再次抛出诱饵，消灭它旁的怪兽。此法成功率极高，同样可捕获其它怪兽。

## GB 圣剑传说 (FP 6)

瞬间转移：游戏初，打进一山洞，得到新武器くさりがま后，装备上，出山洞来到地图〔13，10〕处，屏幕左上方的河边有许多花，用武器くさりがま在横坐标，12与13交界处除花，且左右移动，即有可能瞬间移动到地图〔12，0〕处，〔雪山顶〕。

可回复大量MP的药：游戏后期，一村中得会下水的鸭子后，出村，从上方河岸下水，一直往东游，过雪山板陆，遇一宽阔水域，在此可找到一万块小岛。上岛，杀死周围水中怪物，岛中央便会出现一宝箱，药即在内（如反复进入；即「离岛一转画面一进入此画面」，且杀死敌人，则可无限得到此药）。

几处重要坐标：1. 道具さびトレール：〔11，16〕
2. 特别的椰子树：以皇宫为坐标原点〔2，-4〕
3. 对晶体意克语フレア处：〔8，0〕
