### 东京48小时——TGS特别报道

文/特约撰稿人 王骏生

#### PS2 想说爱你不容易

东京电玩展已经过去了。PS2作为 + r1 + ，在东京电玩展上首次公开了大量可玩的实际游戏。面对数十家软件厂商排山倒海的架势，看得人又欣喜又担忧——如果PS2游戏的实际水平真能达到那些DEMO所表现出来的境界的话，这台主机的市场前景简直不可限量。然而就像这种 + r1 + 式的话所暗含的意味一样，在一番眼花缭乱的折腾之后，或多或少也会产生一些疑虑。

首先给人留下最深刻印象的自然是SQUARE的《BOUNCER》。这款游戏曾在《最终幻想VIII》通关后出现过一段美丽如梦幻般的宣传动画，但是在TGS上可以亲手玩到的却只是一个非常初期的试玩版本（据说只有不到10%的完成度）。除了那些已经事先在录像中看到过的华丽的打击效果之外，这个DEMO几乎还谈不上任何游戏性。和当年《铁拳》试玩版相比，差距不是一点半点，不过这并不妨碍大多数人对此游戏寄予厚望。

 + r1 + 也是很引人注目的展品，因为它有一个曾经创造了巨额利润的前辈，再加上PS2的强劲机能，这款赛车游戏的前景令人期待。然而在TGS上公开的版本同样是非常初期的作品，画面中的赛道虽然与现实中一模一样，但赛车的建模水平还只能用 + r1 + 来形容。与此相比，NAMCO的 + r1 + 试玩版的完成度就要高得多了——铁拳的各种背景表现几乎令人怀疑这竟然是在家用机上运行的画面。

这些展品虽然在不同程度上都表现出了PS2的潜力，但真正完成度较高的作品几乎还看不到。在几乎所有的DEMO中都可以或多或少地发现一些贴图错误和处理速度不稳定等问题，这些问题到正式推出的时候应该都会得到解决吧——虽然谁也没有百分之百的把握。

#### 世嘉仍想充老大

和前几届相比，本次TGS上世嘉展台的气势有了一个质的飞跃。在SONY和任天堂两大巨人的夹缝中苦苦求生的世嘉这次似乎终于找到了属于自己的位置。DC上的独占大作一个接一个，《莎木》、《生物危机·代号维罗尼卡》和《索尼克冒险2》等重量级作品的展示令DC玩家信心倍增。

然而在这些令人欣慰的好消息背后，也存在着一些隐忧。首先是DC在日本的销量始终未能达到世嘉的预期，北美虽然开局不错但后续乏力。其次是面对PS2强大的硬件性能，DC在技术层面上的劣势将越来越明显。世嘉究竟能在PS2发售之前积攒多少实力，将直接决定DC的最终命运。

#### 结束语

从某种意义上说，本届TGS已经不再是一次单纯的游戏展，而更像是新旧世代交替的缩影。PS2的亮相预示着一个全新时代的来临，而DC的奋力一搏也让人看到了世嘉不甘沉沦的决心。至于任天堂的 + r1 + 何时才能露出庐山真面目，恐怕还要等上一段时间。

### 本刊与百事可乐首次合作 2000年共赞《美年达中国电玩榜》

本刊讯　经过长达近3个月的谈判，近日本刊和北京百事可乐草签协议，共同主办 + r1 + 。从2000年第一期起，共同推出以百事可乐饮料命名的 + r1 + 。投票方式由读者自制选票改为内附在杂志中的统一规格的名信片。排行榜亦根据读者的意见做了相应的调整，以便更适应玩家的需要。奖品也[?]为每月抽出50位读者奖励50元奖品，每半年抽奖一次，奖金3万元，全部奖品均由百事可乐公司提供。（详细办法见第12期《中国排行榜》）

经过对北京的青少年消费市场调查分析，百事可乐将其赞助的项目锁定于3个领域，第一是足球，今年的甲A联赛即被冠名为百事可乐甲A联赛，而不久又赞助了青少年街头足球比赛；第二是流行音乐；第三个就是电子游戏。根据百事可乐的调查结果显示，喜欢足球的青少年最多，大约为34%；其次是流行音乐大约31%；大约29%的青少年表示喜欢电子游戏。百事可乐当然不能漠视这一消费倾向，希望通过锁入电玩市场以提高其产品在青少年中的品牌效应。

除万方合作举办 + r1 + 外，还计划于2000年举办规模空前的电玩大赛。请读者关注进一步的报道。

### 美版DC销售再创佳绩 供货不足隐患未消除

本刊讯　曾经创下24小时近1亿美元销售额的美版DC，借着席卷美国的敛意风潮再次刷新业界纪录。美国世嘉5日前表示到9月23日为止，美版DC主机已经在2周内售出51万4000台，成为美国家用游戏主机历史上最成功的销售。

玩家狂热使世嘉开始感到难以满足。据世嘉估计，如果玩家的热情持续不减，世嘉将难以保证提供足够数量的主机。SOA的发言人表示， + r1 + 目前，DC的热火已经蔓烧到了欧洲，还有7成的零售商表示担心10月14日发售时，DC的供货量会有不足。

根据有关公司的调查表明，目前DC在美国侵占的主要是N64的市场份额，对PS的打击并不大。虽然N64在美国几乎与PS平分秋色，但对于世嘉原本的计划来说目标已有了一些偏差。

DC对PS市场份额的影响程度将关系到未来与PS2争夺的资本。当然，FF8和DINO CRISIS在美国的发售对牵制DC还是起了很大的作用。此外，同时发售18个游戏之后的空白期，也是造成目前情况的主要原因。

### 云山雾罩 微软X-BOX是假是真

本刊讯　也许是DC在美国的销售业绩激起了微软对娱乐业的兴趣，一个关于微软X-BOX的消息正在业界广泛流传。

传说中的微软主机具有比PS2更强大的三维处理能力（采用以AMD的满赫兹频X-BOX将采用他们的芯片），同样对应DVD和CD-ROM，不但可以浏览互联网，还内置5G的硬盘，供玩家下载各种资料和游戏，而售价却低得惊人（2000人民币左右）。据说微软认为X-BOX并不会对DC造成太大的威胁，因为其定位于家庭娱乐终端，甚强于PS2可能占领的市场份额。

需要注意的是，微软目前没有对任何有关的消息进行确认。发言人只是表示微MICRO SOFT将继续支持世嘉DC主机的发展。但是业界针对微软的这种说法指出，MS此举只是为了不过分刺激自己的合作者，而且PS2尚未正式发售，微软还有足够的时间来发表他们的主机。

据消息灵通人士称，微软总裁比尔盖茨已经专程赶赴了克利夫兰美国总部，对双方在未来的合作进行了探讨。据说史克威尔对微软的计划很感兴趣，同时表示一定将大力支持X-BOX在未来的发展。但双方都未正式公布消息。

### NAMCO欲全机种制霸 铁拳仍可推出DC版？

本刊讯　虽然玩家已经被满意的告知 + r1 + 将推出PS2版，但DC是否真的没有可能被移植上呢？

在过去一个电脑节目日前在北美推出了一个DREAMCAST特集，节目中赫然出现了 + r1 + 的字样！这实在令人惊讶。

在NAMCO正式宣布PS2版前，就有许多非正式的消息称DC版的铁拳已经在开发中，但出现 + r1 + 这几个字还是第一次。目前NAMCO和SEGA对此都没有任何表态，而如果此消息属实，这将会是PS2和DC首次直接大碰撞，DC究竟可以做到PS2移植版的几个百分点也引起了玩家的兴趣。

另据来自日本非常可靠的消息报道，NAMCO还有两个DC游戏在开发中：一个射击游戏和一个赛车游戏，目前这两个游戏的具体名称和内容均不详，但可以知道的是NAMCO将会很快正式宣布这两个节目。

### 日元上升 利润下降

本刊讯　任天堂于今天宣布调低当年利润预算200亿日元，这是任天堂在近几年来最大的一次调整利润预算。

事实上任氏的经营情况一直非常出色，远凌驾于其已二大游戏巨头之上。此次调低利润预算主要并非由于销售上的原因，而是由于近期日元对美元汇率大幅度上升所造成的。任天堂也希望投资者对此保持清醒的认识。

山内房家的产业最近一帆风顺，将于11月发售的GAME BOY COLOR + r1 + ，预计会有300万的出货量。这样，整个POCKET MONSTER系列任天堂就销售了超过800万套，超过了PS上最畅销的 + r1 + （全球750万部）。

任天堂目前的收益主要都是依靠GAME BOY的主机来达成的。

### 意吞欧洲 SEGA再施份额大手笔

本刊讯　方面，为了建立亚洲到美国和欧洲的大型网络游戏社区，SEGA宣布在欧洲推出了大手笔的新计划。

据悉，SEGA将于1月6日在欧洲发售DC，并推广其网络游戏战略。SEGA在欧洲方面以每年2亿美元在德、法等国推广DC游戏社区。SEGA的欧洲战略将综合43个发行商的游戏软件在欧洲市场推广。目前SEGA在欧洲的DC发行份额仅为8.2%，远低于SONY和任天堂。SEGA计划在欧洲发行超过200款DC游戏，其中包括SONIC、VR战士3等重要作品，并将开设在线游戏服务。

SEGA方面发言人表示，SEGA已经在美国证实了DC的成功，现在要将这一版本延续到欧洲。此外，SEGA还将在北美和欧洲推出DC的网络服务。

### 日本正式宣布 二手软件销售非法

本刊讯　今年5月，日本东京地方法院以游戏软件享有同电影等产品相同著作权的理由，判定销售二手软件合法，但是这项判决并没有成为此种诉讼的先例。目前，大阪地方法院支持SEGA、NAMCO、SECI、CAPCOM等公司要求禁止销售二手软件的诉状，宣布销售二手软件非法。大阪地方法院宣判的理由是，尽管适用电影产品的著作权法，但是零售商在销售时还必须征得开发商的同意。显然开发商是绝对不会同意这种做法的，因此事实上在日本销售二手软件已经成为非法。目前日本已有一大家零售连锁商店因此而宣布倒闭。
