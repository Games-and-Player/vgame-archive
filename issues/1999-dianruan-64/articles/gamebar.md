## 生物危机3·最终逃脱

(BIO HAZARD 3 · LAST ESCAPE)
机种 PS 厂商 CAPCOM 类型 AVG 媒体 CD-ROM

### 评论一

应该说， + r1 + 不愧于成功的游戏系列中的一员，不但保持了高水准的制作，而且玩起来有更加刺激的感觉，可能是由于上手是数量多了的缘故吧。

 + r1 + 比 + r1 + 简单， + r1 + 比 + r1 + 还简单。相信不少人在打穿之后都有这种感觉。谜题答案虽然是随机的，但老玩家早已熟悉其规则，可以说不存在什么难度；丧尸的速度虽然有所提高，又加上个追迹者作帮凶，但因为主角可以紧急回避，任何攻击都能消于无形；要拿S级评定也简单得多，太救判、超级武器随便拿去用，只要快点跑，少存盘就行。虽然游戏的节奏更加适合在人短暂的休息时间，不象以前的游戏一样占用大家很长时间，但对于喜爱破解谜题、迎接更高挑战的人来说，并不是很好的现象。

现在这类带有动作成分的AVG游戏已经不是CAPCOM一家独大，远的有《盗墓者》，近的有《寂静岭》。前者与《生物危机》类似，渐陷入套路无法自拔；后者则是新生力量，销量虽远不及《生物危机》系列，却有自己独到之处。隐藏在成功背后的危机才是CAPCOM最大的危机，不能另辟蹊径就只能在原地转圈。丧尸再多，杀了三代还不够吗？这回已经不能从心底深处体感到游戏所带来的恐怖感了。

优秀的画面，虽然与前作相比没有明显进步，但至少没有退步；便捷的操纵，与前作相比增加了许多动作；引人入胜的游戏环境；为了让游戏者多玩几次的而设置的附加模式，都是一部成功游戏不可缺少的东西。但快之新意也是系列游戏难以给人惊喜的重要因素。

### 评论二

乍一看之下，还以为这个 + r1 + 是二代呢！这也难怪，一开始的场景实在太象了。

确切地说，从故事的角度来看， + r1 + 有点象一个外传性质的东西——情节并不是原作的连续。不过这并不是说游戏也同2代一样——不，比2代刺激多了。

不知道别人是如何进行游戏的，反正我选的是 + r1 + 。那种被杀不光的丧尸围困，被巨大无比的 + r1 + 追杀，在恐怖的叫声与鲜血中奔逃的感觉……因为增加了紧急回避动作键和弹药组合！这个实际上不是新鲜的东西了，在 + r1 + 中就有了，游戏变得实在太刺激了！

比起二代来有显著提高的不仅仅是层出不穷的丧尸，场景方面也有很大改进——这次终于可以自由地在城市中移动，而不是拘束于封闭的房间中，同时还不影响读盘的速度……人物造型显然是系列中最棒的，给玩家的感觉也舒畅了许多。

说到场景，唯一有点不满的就是没有使用 + r1 + 那种全三维的环境——以PS的机能来说是太累了吧？不过现在也已经不错了。倒是解谜是一个比较大的败笔——其实根本就没有什么解谜的东西，与原作相比差得太远了。

爆机之后的过版游戏非常有趣，这好象是从 + r1 + 中来的灵感吧？不过用在这里还挺不错。

爽快、刺激、有趣。这就是对 + r1 + 最好的评价。这也算是后PS时代的一个佳作了。

## 极限运动（3XTREME）

机种 PS 厂商 989 STUDIOS 类型 SPG 媒体 CD-ROM

所谓极限运动就是在欧美青少年中流行的一些诸如早冰鞋、滑板、小轮车等之间的花样技巧比赛，本人对这种充满刺激与活力的运动忱心已久，无奈无良师指导，常泽身高胖而毫无进步，所以当我看到这个游戏时毫不犹豫的买下来，想回家当一回阿Q，过过干瘾。

一段充满动感的实拍与CG相结合的开场动画让我有些进入状态，可怕不及待的进入游戏后却让我有些失望，原来游戏并不是我想像的那样是技巧比赛，而是不折不扣的竞速比赛，而做花样只不过是挣附加分用来更新装备。游戏的画面比较粗糙，可以明显的看到远方2D的背景加上近处大颗粒的3D赛道和人物，在摇滚乐的伴随下还可以边比赛边向其他竞争者大打出手，没有规则，第一就是胜利，很有些美式的粗犷风格。

画面虽然不尽人意，但游戏中人物的动作很潇洒流畅，做动作还可以像格斗游戏一样 + r1 + ，丰富的花样组合可以让玩家有一套自己风格的动作。游戏中的音乐有几首听，狂放的重金属让人在比赛时热血沸腾。上层员的专业装备，在的道上风驰电掣的感觉就是 + r1 + 。

总之，这个感觉另类的竞速游戏，在等待GT2的日子中，值得一玩。

## 潜水员的假日2（アクアノートの休日2）

## 装甲机动队 L.A.P.D（装甲机动队 L.A.P.D）

机种 PS 厂商 EA SQUARE 类型 ACT 媒体 CD-ROM

好久没有玩到这么爽快的游戏了，这是我见过的此类游戏中最好的一个，EA终于开发出一个值得一玩的PS节目了。

其实你可以把它看作是3D版的 + r1 + ，且爽快程度不亚于其。可以说，只要你进入了游戏，就会无时无刻不在战斗。敌人从四面八方而来，有时还真应付不了。配合着游戏的难度，厂商为玩友安排了三种不同火力的双打，且都比较合理。主角的机甲可以变型为战车，变身后速度大增，非常适合被敌人包围的情况下迅速逃跑。本作无论在角色设定还是音响效果上，作得都十分出色，另外画面也十分细腻，手感上有点儿像美版游戏，而火爆程度比其他过之而无不及，读盘就更像美版游戏了——非常之慢！不过起起 + r1 + 不知要快多少倍！！

总之，本作是十分值得一玩的，希望EA以后能够推出续作。

## 星际战神（STAR IXIOM）

机种 PS 厂商 NAMCO 类型 3D-STG 媒体 CD-ROM

所以买 + r1 + 这款游戏是因为它是由NAMCO制作的，而NAMCO制作的 + r1 + 又给我留下十分好的印象，于是便不加思索地买了下来。

但是回家玩过之后便有些失望了，游戏的画面做得不算差，尤其是背景加上不俗的光源处理真的给人一种身处太空的感觉。因此刚开始的时候还是有些新鲜感的，但时间一长就觉得有些无聊。这是因为游戏的难度问题，游戏没有什么难度，很多时候娱乐机就像漂浮在太空中的什么东西，只等着你去打，很少主动来攻击玩家。再有就是雷达的问题，不知是不是在太空中不分方向，雷达上的敌机就用一小圆点表示根本弄不清远近和方向，对玩家造成了一定麻烦。此外速度感不强也是游戏的缺点之一。

我们习惯把坐架式的飞行游戏与 + r1 + 相比， + r1 + 完全没有 + r1 + 中被敌机包围时的紧张感，冲出敌机包围时的爽快感，和与敌机相互追逐时的畅快感，而这两款恰恰是一款飞行射击游戏所应当具有的。

对于对STG有较高要求的玩家来说，这款 + r1 + 就免了吧。

## 劲舞革新版·二度混音

(DANCE DANCE REVOLUTION · 2ND REMIX)
机种 PS 厂商 KOEI 类型 ACT 媒体 CD-ROM

趁着DDR强大的声势，KONAMI紧眼着又推出了DDR · 2rdREMIX，带动了另一轮另类游戏的狂潮。

此次的 + r1 + 加入了许多新要素，曲目比前作更丰富。仅其中ALL MILSIL模式共有30首省选曲目更精彩。从系统及操作上看，本作的难度较前作DDR要大些，两部作品同是HARD级，但操作上 + r1 + 却更难。且开始便可选择的MANIAL模式更是考验造诣的手脚协调度。在进入游戏后，1P与2P舞步不同也是一大改进。

特别值得一提的是本作的 + r1 + ，在用二版混合启动游戏联，换入前作DDR，就会出现16首二部作品中的舞蹈曲目。

本作有相当多的名曲也是其特点。有靠足球而声名大噪的《TUBTHUMPING》，以及被你狂翻唱过，被大多数非欧美流行乐迷所熟悉的《DUB I DUB》。如果你是欧美乐迷，如果你没有PS主机，那么本CD也十分适合，将之放入VCD机中，便可做音乐CD欣赏，虽然每首歌曲只有一分半左右，但却有35首喔！

笔者认为，本作唯一不十分完美的地方在于不能选择人物进行游戏，如KONAMI能加入此系统，则必将坐新舞蹈类游戏。总之， + r1 + 值得一玩。

## 特鲁内克大冒险2~不可思议的迷宫

（トルネコの大冒険2~不思議のダンジョン）
机种 PS 厂商 CHUN SOFT 类型 A·RPG 媒体 CD-ROM

## 奇异的吉它（GUITAR FREAKS）

机种 PS 厂商 KONAMI 类型 ACT 媒体 CD-ROM

最近没有什么游戏玩，值得说的好象也只有这个 + r1 + 了——尽管这是一个不算太新的东西了。

我一直坚信KONAMI创造这些 + r1 + 是为了向市场推销商品的缘故，不然一把连按都没有的塑料吉它你何以卖到300元的高价？不过话虽如此，玩这个游戏确实是少了这把模拟吉它的——用了它玩起来的感觉确实是大大的不同。

由于没有钱，所以事实上是不能真的弹出弹吉它的技术的，不过对节奏感的把握到确实是一种很好的模拟，希望左手五个手指变灵活的朋友千万不可以错过。

然而实际游戏的时又不是单纯地按在按旋屋便可以的，要想炸出真正的 + r1 + ，在高潮的时候还真的必须把吉它竖起来扭打一行——当然，你也可以一边作弹吉它状一边跳踢踏舞，那感觉实在是爽透了。

虽然模拟吉它的尺寸比起真吉它机要小很多，但感觉和手感还是完全一样的。以PS的水平移植过这种游戏也是绰绰有余，玩家大可以在家中苦练之后，再到街机房去尽显漓洒风流——把街机游戏搬回家，现在这个梦想是能够真正实现了吧？

## 游戏情结

文/上海　王骏生

初到日本的时候，感觉并没有什么特别：一样的文字，一样的街道，一样的高架，一样的人流。看上去，和国内的大城市毫无区别。

在第一印象中，东京似乎不比上海好多少，只是干净些、安静些、更绿些。

当然，这个印象立刻就被证明是错误的。

建筑和霓虹灯确乎是相似的，而且很容易建得相似。但是人却完全不同——至少对于游戏的情结，是完全不同的。

## 情节为主，音效与画面为辅——浅谈音效与画面在游戏中的重要位置

文/内蒙古根河　杨国峰

一想到音效与画面，就会促使我们想到史克威尔公司的《FF》系列、NAMCO的《魂之利刃》等大作。那无与伦比的音乐效果，细腻的画面，不论是玩过的、还是看过的，都不会忘记那一幕幕惊心动魄、感人肺腑的CG动画和听到就使人精神振奋的音乐。

就目前世界游戏业的发展方向来看，每一种类型的游戏都有各自的发展趋势，唯一相同的一点——音效与画面都逐渐华丽、丰富起来。首先，我们来看一看RPG型的游戏。RPG游戏人所共知最注重的是情节，而音效与画面是作为辅助的工具，使情节更吸引人。情节加音效加画面，三位一体，这固然好，但这只是对于那些懂日文、懂英文等外语的玩家而言的。尤其是日文，这不用多说，日本是世界上最大的游戏王国，而我们玩的大多数游戏都是日文对白、日文说明，日文……对于像我们这样只能靠中文与日文通用字来理解破解游戏的意思的玩家来说，除了从攻略中得来情节，那唯一的途径就只有从音效和画面中取得了。就拿《FF VII》中赤红十三与亡父会面的那一幕来说吧，赤红十三在山崖下邵断心裂肺的唧唧嘶叫（ + r1 + 好像不太贴切），震荡山谷，这是音效；紧接着，赤红十三部变为缅络的父亲遗尸在孤独的高崖上流出了晶莹的象宝石一样的眼泪，这是画面。音效与画面就这样由史克公司的巨匠们完美的溶合在一起，玩家马上就会感受到那种凄凉原始的气氛。

音效和画面不仅体现在情节上，在RPG游戏的打斗场面中一样伴演着重要角色。一个个变幻莫测的魔法，一只只所向无敌的召唤兽都在紧张、刺激的乐曲中华丽地登场，显示各自特有的魔力。

在RPG游戏中是这样，在《生物危机》、《寂静岭》这样的AVG型游戏中，音效和画面更是不可缺少。在AVG游戏中玩全是依靠音效效果与画面的效果实现恐怖效果的。《生物危机》中形象恐怖的僵尸愤怒而伴着沉重的脚步来到玩家面前，忽而破窗而入给玩家的心理承受能力来个小测验。玩家作出的第一反应是开枪，后便是心率加快，这也是随着剧情的发展，音效与画面给玩家带来的快感觉。《寂静岭》中漫天的雾给我们的印象最深，可见度很低，这就使玩家更集中精神，更心惊肉跳。

除了前两种受玩家庇爱的游戏之外，还有一种类型的游戏——FTG，对战类游戏也很受玩家青睐。玩家可以尽情地通过激烈的打斗画面，实现想象中打遍天下无敌手的英雄形象。在FTG游戏中，完全是通过人物的动作画面、激情的音乐（音效）来实现游戏的趣味性的。玩家可以尽情地享受音乐、欣赏画面。渐渐进入人机合一的境界，这时画面与音效也就达到了预期的目的。

以上三种类型游戏显然都离不开音效与画面，不用多说，其它类型游戏当然也同 + r1 + 相连了。因为它们都是电视游戏，都是利用荧光屏这一媒界，将欢乐与刺激带给玩家的，当然少不了画面。

说到这里我又想起了王菲的一首《eye on me》，歌声将游戏中的情节都集中到了乐曲中。 + r1 + ，所以音乐无论在什么地方都可以表达喜、怒、哀、乐。同样，人物说话的声音，即叫声也都可以说明他的心情。所以，游戏中少不了音效。

总而言之，游戏只所以那么丰富多采，那么美丽动人，那么悬念叠出，那么惊心动魄，那么……除了情节，最重要的就是音效与画面。
