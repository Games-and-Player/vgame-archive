# FINAL FANTASY 未知的世界 第3回

> 彩插 / 「FF」系列特别计划 VOL.3 / 禁断的？技巧

这次恰如其名，将介绍未知的宝物和事件。如此一来就请带着一份惊喜来读本文吧！

## PDF p.55 / 极难得到的究极宝物

下面介绍的是在一般通关过程中看不到的，也就是所谓隐密宝物。但无论是哪一个，也不过是按一定的顺序前进或是砍翻难得一见的怪兽便可获得。

就算不知道也一样能玩，但若知道了就会喜上眉梢，下面公布的情报你一定会感觉很爽。《FF精选集》已于3月11日发卖，在那上面试一下会有什么感觉……

### III 洋葱之圣物

在水晶塔中击败很少出现的龙，会得到洋葱战士专用的葱头装备，此乃III代中的最强装备。

- **葱头圣剑**——攻击力200且素早上升。可与之匹敌的只有VI代中的圣剑エクスカリバーＪ和 + r1 + 。
- **葱头圣盔**——防御力48。本系列中最高装备。装备后用不受任何特殊攻击。
- **葱头圣铠**——防御力48。防备所有特殊效果。通观整个系列，数值并不是很高。
- **葱头圣盾**——防御力48的盾。防备所有的特殊攻击。只有V代中的源氏之盾可与之匹敌。
- **葱头护手**——防御力48的护手。还可提升体力。若作为服饰，甚至不输给VII代。

### IV 亚当铠甲

在月亮上打败噗嘌公主后可得到粉红色的尾巴。将此交给尾巴老头可得到幻之铠——亚当铠甲。

### IV 圣剑エクスカリバー

将宝物 + r1 + 交给铁匠クロロ（克罗罗），他就会为你打造圣剑エクスカリバー。

### IV 菜刀

用从ヤン（杨）夫人处得到的爱之炸面包来敲杨，杨醒来后会送你一把菜刀，此乃投掷系武器。

### VI 河童（日本传说中的某名水怪）之圣铠

可从特定的敌人处盗取或抢到。此乃VI代中最强的武器和防具，但只有在カッパ（河童）状态下才能发挥其威力。

- **沙悟净之禅杖**——本系列中最厉害的枪系武器，攻击力高达253。从一个叫做 + r1 + 处盗得的稀有武器。
- **甲罗之盾**——防御力30。通观整个系列并不算太高，但可吸收水属性的攻击。
- **Ⅲ（盗的名字）**——防御力42。仅次于III代中的葱头圣盔。但在VI代中是最高的。
- **河童装甲**——与アダマンアーマー（阿德曼装甲）一样，拥有本系列中最强的防御力，100。而且还有魔法防御力。

## PDF p.54 / 相反的武器和防具

游戏中没有叙述的必要，但在设定时，将其设计为具有相反性质的武器和防具，下面将要介绍的，左边的属攻击力、防御力的最高等级之物，但右边基本上没什么用，大概是仅供收集吧。

- **V 圣剑エクスカリバー** ⟷ **伪圣剑エクスカリバー**——在I代至V代中登场的传说中的圣剑。攻击力属系列中的最高等级。伪圣剑通常只能打掉1滴血。
- **V ブレイブブレード** ⟷ **厨刀（チキンナイフ）**——每逃跑一次攻击力便会下降一次的武器。即使不装备也会有同样的效果。厨刀会随着逃跑增加攻击力。即使选择了 + r1 + （战斗），也会自动逃跑。
- **VI 英雄之盾** ⟷ **涂血之盾**——防御力最高级40。在装备有涂血之盾的状况下在255次战斗中获胜，涂血之盾便会变成英雄之盾。涂血之盾防御力为零，装备后会引起状态异常。此宝物可在倒塌后的 + r1 + 找到。

### 未知的魔法

这里再介绍3个基本学不会的魔法。在游戏中基本上没有任何提示，所以想学会不靠秘技恐怕是没有可能的吧！

- **IV 谜之召唤魔法**——在击败コカトリス、ゴブリン、ボム、マインドフレア之后，可随机召唤出其中的一个。
- **V 鸡蛋排骨（egg chop）**——在陆行鸟出现后，多次使用魔法灯（或是煤油），会学会叫做鸡蛋排骨的谜之召唤魔法。
- **VII 陆行鸟瓶**——给特定的陆行鸟三个叫ミメット的野菜，会学会一个叫做 + r1 + 的敌人的技术。

### 最后的小话题、情报集

最后是收集到的各种小道消息。虽然都是些提不上台面的东西，但实际操作后一定会有所触动。在最新的VIII代中应该也有些隐藏的东东吧。

- **I 附赠的小游戏**——在港口小城プラボカ得到船后，在海上按住A键后按55次B键，就会出现一个由1~15数字组成的移字游戏。
- **VI 抽水马桶**——在帝国首都ベクタ的厕所里按A键，就可多次冲马桶。好象帝国中抽水马桶非常普及。而在整个系列中可以冲马桶的却只此一处。
- **III 空手道专家的爆炸**——在战斗中让空手道专家连选三次 + r1 + （蓄），他就会爆炸成血。
- **IV 史克威尔开发楼**——进入ドワーフ城的隐藏通路上的房间。但在EASY中是不行的。
- **VI 帐篷的变化**——在使用帐篷时，帐篷会随站在排头的人物而变化形态。
- **VI 隐藏事件**——在与帝国举行完和平会餐之后，立刻赶去飞艇处，会有セッツァー的隐藏事件发生。
- **VII ALL 7狂热攻击**——在战斗中体力正好是7777时会发狂，会反复打出7777伤害的攻击，直到将敌全灭。此时其余的主角会呆在一边看热闹。
