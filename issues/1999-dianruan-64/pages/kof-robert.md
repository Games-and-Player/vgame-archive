# 格斗之王风云录

> 彩页·彩插

## 罗伯特·卡西亚

**家财万贯的翩翩贵公子 财团中的 + r1 +  华贵魅力在他背后升起！**

### 人物档案

- 格斗流派：极限流空手
- 生日：12月25日
- 年龄：22岁
- 身高：180cm
- 体重：85kg
- 血型：AB型
- 家族构成：阿丽·卡西亚（母）、帕特·卡西亚（父）
- 出生地：意大利
- 职业：下届卡西亚财团总裁（预定）
- 宝物：收集名牌
- 当前目标：除了继承父亲的财团外，当然是百合啦（笑）

### 我是可以富可敌国的！

**必杀技：**

- **幻影脚**—— + r1 + 的腿踢版本。
- **飞燕疾风脚**——出法很简单的攻击招式。
- **龙击拳（蛇击）**——类似升龙拳的对空技，根据作品不同，模样也有所变化。
- **飞燕龙神脚**——指令投操纵的类似幻影脚的招式。
- **覇王翔吼拳**——跳跃中飞出踢击的奇袭性招式。
- **极限流连舞脚！**——其与坂崎良赖以成名的必杀技。
- **龍虎乱舞**——利用翻滚转身踢击攻击敌手。
- **龍斩翔**——与 + r1 + 相似，但动作有所变化。
- **飛燕旋風脚**

 + r1 + 中的指令与其他作品有所区别。按B键会出现若干动作的不同。

**姿势：**

- **無影疾風重段脚**
- **失败**——背景变得全黑，很忧郁。
- **胜利**——扭腰的样子比较俊。

**衣着变化：**

- 龍虎の拳1·2——大幅变更衣着，更象公子哥了。
- 龍虎の拳3——衣服没大变化。令人怀念的第1代装束。
- 跳发——很受人气俏俏的姿势。

**各代形象变化：**

- KOF'98 口雑——红与黑的组合，一副主人的模样。
- KOF'97（ADV）——肌肉变得更壮了。姿势也变了。
- KOF'96——真红村衫登场。
- KOF'95——与前作几乎一模一样。
- KOF'94 92.P——与《龙虎1、2》相同的样子。

※以上资料及照片均选自《龙虎之拳外传》及《KOF'98》。
