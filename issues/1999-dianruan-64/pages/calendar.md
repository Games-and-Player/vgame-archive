# 月历

> 彩页·彩插

## PDF p.61 / 11月月历

1999年11月月历。整页为水彩风格插画，描绘一位身穿深蓝色水手服的少女坐在石阶上，手持卷轴，周围飘落粉色花瓣。底部为11月日历。

《电子游戏软件》1999年11月

## PDF p.60 / 12月月历

1999年12月月历。整页为水彩风格插画，描绘一位身穿淡紫色婚纱的新娘，手持玫瑰花束，头戴薄纱。底部为12月日历。

《电子游戏软件》1999年11月

## 编辑备注

- 两页月历为彩色插页，画风为日式水彩插画
- PDF 61为11月月历，PDF 60为12月月历
