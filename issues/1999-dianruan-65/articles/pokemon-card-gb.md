## PDF p.33 / 刊页 22

以 + r1 + 为题材的集换式卡片游戏在GB上登场了！本作并不是通常意义上的口袋妖怪RPG，而是将现实中大人气的 + r1 + （ポケモンカードゲーム）忠实再现在GB上的作品。

游戏中收录了228种卡片，包括基本口袋妖怪卡、进化卡、训练师卡和能量卡等。和实际的卡片游戏一样，玩家需要组建一副60张卡片的卡组来进行对战。对战的基本规则是：将口袋妖怪放到场上，附上能量卡后使用技能攻击对方的口袋妖怪。先击倒对方6只口袋妖怪的一方获胜。

游戏的故事模式以 + r1 + 为目标展开。玩家扮演一位刚开始接触口袋妖怪卡片游戏的少年，从 + r1 + 处获得初始卡组后，需要挑战8个 + r1 + 的 + r1 + ，收集8枚奖章。集齐奖章后就能挑战 + r1 + ，最终获得传说中的卡片。

和实际的卡片游戏一样，属性相克在本作中也非常重要。草属性克制水属性、火属性克制草属性、水属性克制火属性……善于利用属性相克来构筑卡组，是取胜的关键。

每赢得一场对战，玩家都能获得2包卡包作为奖励，从中可以获得新的卡片。卡片的获取带有一定的随机性，想要集齐全部228种卡片需要花费不少时间。不过这也正是集换式卡片游戏的魅力所在。

## PDF p.34 / 刊页 23

训练师卡是影响对战局势的重要因素。比如 + r1 + 卡可以让玩家将手中的卡片全部丢弃然后重新抽7张， + r1 + 卡可以从卡组中搜索一张基本口袋妖怪卡加入手牌。灵活运用训练师卡，能够大大提高卡组的运转效率。

进化系统也是本作的重要要素。将基本口袋妖怪进化为更强大的形态，不仅能提升HP和攻击力，还能解锁更强力的技能。不过进化需要消耗回合，而且不能在出场的第一个回合就进行进化，这就需要玩家在构筑卡组时仔细考虑进化链的搭配。

本作还支持通信对战功能。两台GB通过通信线连接，就能和朋友进行实时的卡片对战。用自己精心构筑的卡组和朋友一较高下，这才是卡片对战的最大乐趣。此外，通信功能还可以用于交换卡片，帮助双方更快地集齐全部卡片。

游戏还内置了卡组编辑器，玩家可以自由地修改自己的卡组。系统还会根据卡片的种类和数量给出卡组的评价，帮助初学者构筑出更加均衡的卡组。

对于口袋妖怪爱好者来说，这是一款不可错过的佳作。无论你是否玩过实际的口袋妖怪卡片游戏，都能在本作中体验到卡片对战的乐趣。
