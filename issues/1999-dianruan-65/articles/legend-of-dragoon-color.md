历时了3年的时间，动员了100位以上工作人员的RPG游戏 + r1 + 终于出现在玩家眼前了。为了给读者耳目一新的感觉，SCE使出浑身解术来描写这部有如伟大史诗的游戏。

【图】游戏CG画面：龙骑士战斗场景，展示精美的角色与怪物渲染

### RPG游戏的最终形态

今年冬天，一款堪称为RPG最新标准的游戏，即将呈现在玩家眼前了。这就是 + r1 + ，一部由4片CD组成，并点缀超真实音效的超级巨作。本作的内容是以中世纪 + r1 + 与 + r1 + 的世界为中心，一切的游戏设定都让人联想到欧洲的中古世纪，因此如果我们现在就断定它是部传统RPG游戏的话，可能还言之过早。此外，本作与过去的作品不同，整体的世界观相当完美，充满着美丽的画面。以下，就以游戏动画为中心，为各位玩家介绍本作。而本作的高品质动画，可由在今年8月于洛杉矶举行的世界最高荣誉CG大会中，被入选为 + r1 + 中获得证明。希望各位玩家能来鉴赏这个由感性及技术力结合而成的艺术结晶。

### What's SIGGRAPH?

这是每年8月，由美国电脑学会所主办的国际会议之略称。本作的CG画面通过严格的选拔而成为大会的入选作品，并且还赢得在会期中上映的荣誉。除此之外，日本的KONAMI公司也以 + r1 + 等作品入选，这证明其画面的品质可通用于全世界。

### 动画

这次的作品是强调过去作品动画所没有的存在感。本作的动画被赋予了统合整个游戏世界观的重要任务，而制作小组创造出来的动画也足以表达游戏的高品质。以下就以全长30分钟的动画来为各位介绍将在游戏里登场的人物，以及被细致手法所描绘的风景……

→故事开头的某片段——和人物的描写，从外在看来统一沉重的色调，以 + r1 + 感受到一调

### STORY

这个世界的所有生命都是由 + r1 + 所产生。但是，人类、龙族，甚至于君临世界的种族，却不断地重复过去以来所未曾发生的战斗。故事是从一万年之后，主人公——达特的旅途中开始的。他的面前出现了各式各样被命运戏弄的人们……

### 就从这里开始

**背景**

在本作中，凡是角色会抵达的舞台，工作人员均以精细的手法来修饰。不只是追求人物的动作姿势要有流畅的表现，更以独特的世界观做后缀，支撑游戏的制作。制作小组以制作一幅名画般的样子来制作本作，其完成度之高，请各位玩家别错过了。

【图】背景画面截图：形状有如水晶结晶的村庄。另一方面在岩石山上更布满了不可思议的光线。虽然场所不同，但全体气氛的表现是一致的。

### 为您一举公开游戏的精彩画面

### 游戏画面

在本作中，玩家将操作由多边形所构成的角色。就如前页所讲的，在美丽的背景画面下，各角色的动作都相当地流畅，玩家可以丝毫不感压力地操纵角色，到各地去探险。现在，就让我们从画面中，跟着达特长途旅行，一起进入游戏的世界。
