# SLG的变迁

> PDF p.47-48 / 刊页 36-37 / 游戏类型进化讲座 NO.2 / 编译/PACMAN、PERFECT

SLG（模拟游戏）：SIMULATION GAME的缩写。顾名思义就是 + r1 + 。SLG又可以细分为战棋类（S·SLG）、养成类（育成SLG）、经营类（经营SLG）等子类型。本讲座主要以战棋类SLG为中心，介绍SLG的发展历程。

## PDF p.47 / 刊页 36

### 重要游戏发售年表

- **1988.5.20·FAMICOM WARS·FC·任天堂**——任天堂制作的战棋类SLG。有红军和蓝军两支部队，以回合制方式进行对战。简单明了的系统至今仍被沿用。
- **1988.10.21·天下统一·FC·GAME ARTS**——以日本战国时代为背景的历史SLG，在FC主机上实现了相当高水准的策略性。
- **1990.4.20·火焰之纹章 暗黑龙与光之剑·FC·任天堂**——S·RPG的开山鼻祖。由INTELLIGENT SYSTEMS开发，加贺昭三设计。将RPG的角色成长要素融入SLG中，创造了全新的游戏类型。角色一旦在战斗中死亡就无法复活的设定至今仍是该系列的特色。
- **1990.8.10·SD高达 GACHAPON战士2 CAPSULE战记·FC·BANDAI**——将SD高达与SLG相结合的作品。在当时拥有极高的人气，系列作品延续至今。
- **1990.12.2·超级大战略·MD·SEGA**——在MD上实现了高水准的军事SLG。拥有大量真实的兵器单位和详细的参数设定，深受军事SLG爱好者的欢迎。
- **1991.3.9·半熟英雄·FC·SQUARE**——以幽默诙谐的风格著称的SLG。召唤怪物进行战斗的系统独树一帜，加上搞笑的剧情和台词，让玩家在欢笑中体验策略的乐趣。
- **1991.3.14·LANGRISSER·MD·NCS**——和 + r1 + 齐名的S·RPG系列。雇佣兵系统是本系列的最大特色，每个指挥官可以率领多名雇佣兵组成小队参加战斗。
- **1991.12.20·SUPER FAMICOM WARS·SFC·任天堂**—— + r1 + 的SFC版续作。画面和系统都有了大幅进化，增加了更多的兵种和地形效果。
- **1992.3.12·火焰之纹章 外传·FC·任天堂**—— + r1 + 系列的第二作。加入了世界地图和城镇探索等RPG要素，在系列中独树一帜。
- **1992.7.14·OGRE BATTLE·SFC·QUEST**——由松野泰己设计的名作。实时进行的战斗系统和复杂的剧情分支在当时引起了巨大反响。
- **1993.1.22·火焰之纹章 纹章之谜·SFC·任天堂**—— + r1 + 系列在SFC上的首作。分为前后两部，前篇是FC初代的重制版，后篇是全新的故事。是该系列人气最高的作品之一。
- **1993.7.9·三国志III·SFC·KOEI**——光荣的三国志系列在SFC上的作品。拥有庞大的数据量和复杂的内政外交系统，是历史SLG的代表作。
- **1993.10.29·TACTICS OGRE·SFC·QUEST**——高低差地形战斗、分歧剧情等创新要素令人印象深刻。被誉为S·RPG的最高杰作之一。

- **1994.3.25·超级机器人大战EX·SFC·BANPRESTO**—— + r1 + 系列的SFC作品。汇集了多部机器人动画的角色和机体进行回合制战斗，跨作品的组合令人兴奋不已。
- **1994.4.22·火焰之纹章 圣战之系谱·SFC·任天堂**—— + r1 + 系列中剧情最为壮大的一作。引入了世代交替系统——前半部分的角色结婚生子后，后半部分由子代角色继续战斗。
- **1994.8.5·银河英雄传说·SFC·TOKUMA SHOTEN**——根据田中芳树的同名小说改编。在宇宙中展开大规模舰队战的SLG。
- **1995.1.13·SUPER ROBOT WARS 4·SFC·BANPRESTO**——超级机器人大战系列的第四作。参战作品大幅增加，系统也更加完善。
- **1995.5.26·POWER DoLLS·SFC·工画堂**——从PC平台移植的机甲战术SLG。指挥女兵部队驾驶机甲进行精密的战术行动。
- **1996.2.23·火焰之纹章 托拉基亚776·SFC·任天堂**——系列中难度最高的一作。加入了疲劳度和捕获系统等新要素，是核心玩家的最爱。
- **1997.3.21·FINAL FANTASY TACTICS·PS·SQUARE**——由松野泰己设计的名作。 + r1 + 系列的系统精华融入了FF的世界观，成为PS上最受好评的S·RPG之一。
- **1997.10.30·超级机器人大战F·SS·BANPRESTO**——在土星上展开的超级机器人大战。分为前篇和完结篇两部发售。
- **1998.2.5·召唤之夜·PS·FLIGHT PLAN**——以召唤术为主题的S·RPG。角色之间的好感度系统和多结局设定是其特色。
- **1998.5.14·银河少女传说2·SS·NEC HOME ELECTRONICS**——美少女+SLG的组合。在宇宙中指挥美少女们进行战斗。
- **1999.3.4·超级机器人大战 COMPLETE BOX·PS·BANPRESTO**——将SFC上的2次、3次、EX三部作品合集移植到PS上的完全版。
- **1999.9.2·火焰之纹章 封印之剑·GBA·任天堂**——系列首次登陆GBA的作品。（注：此处为预告信息）

SLG的历史可以说就是一部不断进化的历史。从最初简单的军棋对弈，到加入RPG要素的S·RPG，再到融合各种类型要素的复合型SLG，这一类型始终在探索新的可能性。特别是 + r1 + 系列和 + r1 + 系列的成功，证明了SLG和RPG的融合是一条正确的道路。
