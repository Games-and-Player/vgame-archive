## PDF p.35 / 刊页 24

在 + r1 + 红·绿·蓝版大获成功之后，以动画版为蓝本的特别版 + r1 + 登场了！本作的基本系统和红·绿版相同，但加入了许多以动画版为原型的新要素，让玩家能够体验到和动画主人公小智一样的冒险旅程。

本作最大的特点就是初始口袋妖怪固定为皮卡丘。和红·绿版中皮卡丘被收在精灵球里不同，本作中的皮卡丘会像动画中一样跟在主人公身后行走。转过身和皮卡丘对话，还能看到它的表情变化，非常可爱。皮卡丘的好感度会根据玩家的行动而发生变化——经常让它参加战斗并获胜会提升好感度，而让它在战斗中落败或者用其他口袋妖怪替换它则会降低好感度。

和动画版一样，本作中的皮卡丘拒绝进入精灵球，也无法通过雷之石进化为雷丘。不过皮卡丘本身的实力也不容小觑，特别是在学会了 + r1 + 之后，攻击力相当可观。

故事的展开也参考了动画版。在旅途中会遇到动画中的经典角色，比如火箭队的武藏（小次郎）和小次郎（小次郎），以及小刚和小霞等。大木博士会在关键时刻给予玩家指导，故事情节和动画版一脉相承。

## PDF p.36 / 刊页 25

游戏对应GBC，在GBC上运行时画面以彩色显示。和GB上的黑白画面相比，彩色画面大大增强了游戏的表现力，口袋妖怪的外形也更加生动鲜明。

在故事进行中，玩家可以像动画中一样获得御三家口袋妖怪——妙蛙种子、小火龙和杰尼龟各一只。这在红·绿版中是不可能的（只能从三只中选一只），也是本作的一大魅力所在。有了御三家的加入，队伍的编成更加灵活多变。

和红·绿版一样，游戏的最终目标是收集全部151只口袋妖怪并击败四大天王。不过部分口袋妖怪仍然需要通过通信交换才能获得，单独一个版本无法集齐全部。

本作还追加了一些红·绿版中没有的事件和对话。比如火箭队出现的频率增加了，和他们的战斗也更加频繁。此外，部分城市和道路的训练师配置也进行了调整，即使是玩过红·绿版的老玩家也能感受到新鲜感。

冲浪皮卡丘是本作中隐藏的特别要素。满足特定条件后，皮卡丘可以学会 + r1 + 技能，在海上乘着冲浪板前进的画面十分有趣。此外还追加了 + r1 + 等小游戏，丰富了游戏的可玩性。

总的来说， + r1 + 是面向动画版粉丝的特别作品。如果你喜欢动画中皮卡丘和小智的故事，那么这款游戏一定能让你满意。
