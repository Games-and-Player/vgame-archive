★CAPCOM的冈本吉起在接受日本杂志访问时谈到有关PS2的性能，指出由于PS2的开发环境较复杂，在刚接触的时候大概只有1个 + r1 + 的能力，现在约有3个 + r1 + 的能力，若是能真正驾驭它应该有15个 + r1 + 的能力。（读者可能看不懂 + r1 + 是什么？ + r1 + 就是某主机的标志，懂了吧！）

★北美某周边开发商表示，在年底将有一张对应DC主机的加大的VMS记忆卡推出，而这张记忆卡的容量是16MB RAM（比原本大上16倍）。这张记忆卡除了可和其它的记忆卡连接交换资料之外，还可连接到个人电脑，玩家将可以把DC游戏的记录透过这张记忆卡直接存放在电脑里，也可以利用电脑去下载游戏的更新档案，再转到这张记忆卡来，这对于无法利用DC上网玩家而言是项好消息。

★日本Hitachi（日立）公司宣布未来将进行开发街机，目前计划将与NAMCO进行技术上的合作。日立表示将在这个合作的游戏设施上使用该公司的最新模拟技术，这个技术主要是利用密封型机体的转动配合模拟萤幕，让玩家感觉正在乘坐某一种交通工具的感觉。在开发完成后，NAMCO将会把这款游戏摆在NAMCO主题游乐园里。
