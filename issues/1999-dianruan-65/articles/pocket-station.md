### PS用 + r1 + 主机完全鉴定法

在PS的众多型号中，有一款被称为 + r1 + 的主机备受玩家关注。这款主机采用了改良后的光驱和主板设计，在读盘速度和稳定性方面都有所提升。那么，如何判断自己手中的PS主机是否为7501型呢？

首先，从外观上来看，7501型和其他型号的PS在外形上几乎没有区别。因此无法仅凭外观来判断。最可靠的方法是查看主机底部的铭牌。在铭牌上印有型号编号，如果显示 + r1 + 字样，那就是7501型无疑了。

不过需要注意的是，日本版和北美版的编号有所不同。日本版对应的型号为 + r1 + ，北美版为 + r1 + 。两者的内部构造基本相同，只是电源电压和制式标准有差异。

7501型的主要改进之处在于：
1. 采用了新设计的光驱组件，读盘噪音降低
2. 主板进行了小型化处理，发热量有所减少
3. 光驱的寿命更长，不易出现读盘困难的问题
4. 背面的串口和并口与前代相同，外设兼容性没有问题

对于想要购入二手PS的玩家来说，7501型是一个不错的选择。不过即使不是7501型，只要保养得当，其他型号的PS也能正常使用。

### POCKET STATION简介

POCKET STATION是SCE于1999年1月23日发售的PS用外设。外形类似一个小巧的液晶掌机，可以插入PS主机的记忆卡插槽使用。

POCKET STATION内置了一块32×32像素的液晶屏幕、红外线通信功能和闪存存储器。它既可以当作普通的PS记忆卡使用，又可以运行从对应游戏中下载的小程序（ + r1 + 程序），实现在外出时也能进行简单的游戏操作。

对应POCKET STATION的游戏在进行游戏时，可以将特定的数据传输到POCKET STATION上。比如 + r1 + 可以将召唤兽小游戏下载到POCKET STATION上，在外出时培育召唤兽，然后将培育结果传回PS游戏中使用。

POCKET STATION发售后一度出现严重的缺货现象，在店头很难买到。SCE正在逐步增产以满足市场需求。

POCKET STATION的主要规格：
- 液晶屏：32×32像素
- 操作按键：5个（上下左右+确定）
- 通信方式：红外线
- 电源：CR2032纽扣电池1枚
- 重量：约30g
- 价格：3000日元
