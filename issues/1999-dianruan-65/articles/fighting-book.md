## 连续技大观园（四）

**武汉 张伟**

天语：连续技乃格斗乐趣之本，需 + r1 + 。 + r1 + 从理论上武装玩家。

（续上期）在该状态中对手被攻击后僵硬值不变，而己方角色的收招时间则大幅减少或消失且对手一旦被此状态中的角色击中后除倒地以外一直保持被攻击判定。利用这个特性产生连续技。那么，如何在OC状态中打击对手便是研究的重点了。

贵刊也曾登过这方面的文章，但那篇文章似乎太 + r1 + 化了。我认为大众一些的出法应是先跳准所使用角色必杀技中攻击较强的一招，在OC发动后先将对手打浮空（也可视情况不打浮空），然后用这招必杀技以轻攻击键连续打击几次直至OC快结束时再以重攻击键出这招来结束OC，这样亦可达到较强的攻击。望大家试一试。到了ZERO3中OC则发生了本质的变化，角色在发动OC后，ZERO2中OC的几乎都不存在了。这样的话，大家就应以我说的第一类连续技为思路，加以研究。由于对ZERO3的了解尚浅，我在这里就不多说了。至于 + r1 + 、 + r1 + 等游戏的这种技巧，基本还是以ZERO2为方本的。

OK！我对连续技分的六大类就说到这里。看到现在杂志上连续技介绍越来越多之后，我产生了写这篇文章的想法。我觉得在了解了连续技的本质后通过一些 + r1 + 来举一反三自己创出强力连续技会比看了杂志后按步就班地练习来得有趣得多。要知道打出一个高难度的连续技，就意味着你的格斗水平又LEVEL UP了。

本文主要以KOF和SF为本而写成的，实是因其乃目前最流行的两大系列，是 + r1 + ！饿狼、侍魂等的爱好者们就请见谅了。好了，还是那句话，水平有限，错漏难免，望各位多多指正，本人感激不尽。（全文完）

## VF3（一）

**广州 野**

> 作者在信中说： + r1 + 。编者对此表示感谢。

由于MODEL3 STEP1.0基盘的原 + r1 + 版令以95%映像感和100%操作感之高格移植出现在DC上，尽管对角色的细部描绘尚有欠缺，但元气公司程序员们的天赋仍然表露无冥。今后若有机会将把本作一些鲜为人知的东西登出，有条件的玩家亦可在DC上练习。玩VF3时应体会其技巧判定的微妙所在。

### 结城晶

这是一位近身战十分有利的角色，其中近身↓P用来封位十分好用，而上步顶肘（→P）用来瓦解对手的防御也十分之有效，再加上各种突进技如马步顶肘（→→P），跌步顶肘（→→→P），马步冲靠（↓→P+K）的混合使用，绝对会打乱对手的方寸。而连环踢1HIT时（→→K），马步冲靠，扬炮（↘↓P）很容易形成连招。再加上各种当身投，使用得当将十分令人头痛。晶可列为上级角色。

1. 里门顶肘（→→P，对手浮空）> 里门顶肘 > 追加拔。威力15%，难易10%，命中100%。

2. 马步冲靠（↓→P+K）> 上步冲拳 >（←→P）> 铁山靠（←→→P+K）。威30%，难40%，中60%。

3. 马步冲靠或连环脚1Hit（→→K）命中 > 修罗霸王华山（↘K+G→P←→→P+K）。威40%，难65%，中80%。

4. 扬炮（↓↓P）> 上步顶肘（→P）> 里门顶肘（→→P）> 追加。威20%，难35%，中70%。

### 陈佩

是一位适合中距离战斗的女战士，出招快，威力也大，其中，苍下连推（↘PP），穿冲拳（→P），上步冲拳（→P）易造成对手混乱，燕阵旋风脚（←K+G）用来截击对手攻击后威力极大，但因其变化少，遇到当身投高手如结城晶时应小心。偏为中级角色。

1. 燕子双掌（→→P）> P > 燕旋踢（↗K+G）> 跳跃追加。威35%，难50%，中50%。

> 第一发是否真正令对手浮空，请确认下面技巧中，有些可能要在COUNTER态下才可连上。

2. 飞燕翻膀（↘P+G）> 虎燕煽脚（KK）> 跳追加。威30%，难30%，中80%。

3. 燕挂搪（← ↓P+G）> 飞燕烈脚（↗KK）> 追加。威35%，难30%，中95%。

### 陈洛

中远距高手，威力极大，操纵简单，易形成连招，其中穿脚连虎爪拳（↘K→P→→P）虽然较难形成连技，但用来打乱对手防御是一个不错的选择，而连环裁腿（↗KK）与连环拳（↘PPP）足可令你的对手觉得站防也不是，蹲防也不是。陈洛乃中级角色。

1. 双虎烈把（↘P+K）> 虎枪拳（←→P）> 大跳跃追加。威35%，难20%，中60%。

2. 飞虎转身拳（←→P）> 连环转身踢（PPPK）。威40%，难55%，中30%。

3. 柳手挂搪（← ↓P+G）> 飞虎转身拳（←→P）>（↗P）> 连环截腿（↗KK）> 追加。威45%，难60%，中30%。

### 莎拉

十分赖皮的角色，实战中连环下截（↓KK）与连踢腿（↘KKK）起动形式十分相似，而两者都是中了第1Hit就十分再难防住的招式。中级者角色。

1. 膝撞（→K）> 倒跌踢（↘K）。威25%，难30%，中45%。

2. 膝撞 > 连拳倒跌踢（PPP←K）。威30%，难10%，中30%。

3. 下式高踢（↙K）> 肘连旋风踢（→P←K）> 跳跃追加。威30%，难25%，中70%。

★一、蹲弱踢 > 近立弱拳 > CANCEL 红鹤之舞（↘B，1段目）> 小夜千鸟 > 以龙炎舞来CANCEL

## 反击技漫谈

**北京 SPECIAL**

→ ↓ ↘ 格斗天书 ↓ ↘→

天师兄：总觉得你很讨厌。嗯？首先你给人以以小卖小的感觉，且打街霸就用真豪鬼（无聊/赖），好像别人就赢不了似的。所以对你的评价是 + r1 + 。

SPECIAL兄：我打街霸从来只用肯、桑吉、亦杜壁[?]等几位。SUPER CANCLE达人现郑重向你宣战！（笑）

反击技是格斗技巧的重要组成部分，这里我所谈的广义反击技，即化解对手的攻击并能给予反击的技巧都应属于反击技。这种技巧更适合于喜欢稳扎稳打的实战派格斗家。以下为我对反击技的认识：

反击技包围的范围很广，主要包括三类：

一、当身投。说到当身马上会想到吉斯（其当身有 + r1 + 的前缀，尤其在RB2中三路防守密不透风）其实使用这类技巧的人物有很多，按照范围大小又可分为两类：

（1）普通型，这种类型很常见。招式本身不具备攻击判定，但可以抵消对手的一次打击（当然也有三次的。如RB2中洪福的必胜进袭拳↓↙←+C）然后出招，多属上中段，对空性能不错。如吉斯，玛丽（头部飞弹↓↙←+A/C）、比利（水、火、龙追击棍↓↙←+B/D）。

（2）范围型，与普通型很像，也可归为上中段，但用来对空实属找死。如大门五郎（根返↓↘→+B/D）苏杜姆（白刃→↓↘+趴踢）。这些技巧的使用率很低。此外，《天草》中柳生十�的[?]的架刀，弹岩或空手入白刃（↓↙←+D）仅对斩有效，而对踢攻击（D）无奇。所以柳生堪称最强的架刀也会被跌扑+D击溃。还有一些有效攻击判定的当身就不多说了。

使用这些当身投时应注意几点问题：

（1）当身投不能过于频繁的使用。你的当身从对手的攻击猛烈进攻中得手一两回后，对方一定会小心翼翼，不敢轻意出招。这就是扭转攻防局面的好机会。你应大胆进攻，使对手陷入被动地位。在进攻中也可夹带当身，若能乘机蓄满已久的超杀，肯定会在对手心中留下阴影。若频繁使用，低手尚可，遇上高手，被KO的一定是你。

（2）切不可依赖当身投对空。对手的空中攻击应尽量用对空技迎击而不是当身。否则高手会用极简单的方式诱你上当。你要问： + r1 + 好，下面就举一个例子：对方空中防御的姿势跳过来，之后的招术有三种可能性：

①不出招直接落地，然后上前摔你。这种对手猜测你会使用当身，之后尽量减小落地硬直，更使用当身无效的摔投。你应一直防持防御姿态，在其落地的一刹那用出招快的招法（以前经拳）封死。

②在你的角色头部附近出招。对手认为你在这样极短的时间内使不出当身（使出来不及），这时应防住后上前摔投。

③低段位出招。对方是真正的高手，不但认你使不出当身，增加了硬直时间，并强制解除自己的落地硬直。对付这样的高手，还是先乖乖防御吧！

其实对手还可以越过你的头顶或从你进行攻击等，都必须先以防御姿态应敌。当然你可以垂直或后跳使危险较大的普通投，迫使对手空中防御，落地后再用当身反击。

（3）使用当身最重要的一点——出招时机。由于是最重要的，反而觉得没的可写。诚然，如VR战士中的这一招术，终不能靠判断对手的行动来出招（否则后果自负），能且只能凭直觉（注意：是直觉，不是感觉，更不是朦胧。有什么区别？回去查字典！）所以只有靠丰富的作战经验和在短时间内了解对手的基本打法和最练的招术，因为在最后关头，对手往往会下意识地使用最熟练的打法（你也一样）拥有上述条件后，你自然会在恰当的时机使出当身。

这并不是说没的写了，有些事不一定要说明白，而且这些也说不明白。 + r1 + 

（4）心态保持冷静。这是最难做到的。

二、闪避。闪避并不是只用来躲避飞行道具的，也可以闪避对手的其它攻击，这样能更直接的接近对手。有闪避性质必杀技的角色有李烈火（月华2）玛丽（RB2）李香绯（RB2），尤其是李烈火的霞（A+B）可接4种追打，按B为蹲防不能，按C则为站防不能，在接触战中占尽了便宜。玛丽的头部飞弹（↓↙←+A）有三种追打，其中C可作近身摔投技使用。李香绯的酔拳（攻击判定出现一瞬间按着A拉↑、→、↓）可对上中下三路，上段有追打特性，若能熟练掌握，就可丰富你的进攻方式。当然，普通的闪避（如《天草》）使用恰当的话，能为连续技创造更多机会。

还有，闪避在进攻中发挥的作用远比防守时要大。如李烈火用 + r1 + ，对手防住空中B后，你可以用（↓+C）或摔投，而↓+C对手不可能防不住，上前摔投又极易被弱斩打回。所以在落地后略一停顿按着使用霞（A+B）。 + r1 + 的作用是看对手的反应，对手沉不住气的话想出招试探，你正好避过去，之后对手就任你宰割。若没有动静，可根据其防御姿态选用（B 或 ↘+C·C·→+C）来追打。相对而言闪避比当身的安全系数要大，所以可以大胆放心的使用。

三、普通反击技。即狭义概念的反击技。SFZERO 系列的防反，月华系列的弹档，RB系列的H·POWER都是这一类型。由于这些技巧攻击力较小，不宜作为进攻的主导，所以主要用于白热化阶段。特别是对付那种爱用超杀耗血的人十分管用。举个例子来说，《月华2》中，使创名方守纲与对方打得都没剩多少血了。这时把对方逼在角落里然后使出超杀耗血。对方防住后，可能在你使出超杀收招后适时按下D键，对方一定会对刚才的行为后悔。要是他预计到你还会用超杀，他会在你出招时（即手发光时）用弹档。这时你可以把超杀蓄住打个时间差。

到底对方会采取哪种方式呢？凭直觉啦！
