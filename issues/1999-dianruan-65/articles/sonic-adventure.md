 + r1 + 是SEGA也是DREAMCAST的世界级作品、玩家必备之作，该作拥有十分丰富的剧情，以及与情节结合得十分紧密的BGM。剧情与BGM均感人至深。

游戏一共有6名角色，全部通关后隐藏的SUPER SONIC出现，用它再爆机则赏你 + r1 + 观看各人通关后的ENDING（制作者名单）。此6人全是正面角色，各自拥有鲜明的性格特征，通过他们在游戏中的 + r1 + 我们可以明显看出。

游戏的进行方式是 + r1 + → + r1 + → + r1 + → + r1 + ……。即先要在某处寻找动作关卡的入口以及其他一些宝物（有时还需要发生一些对话事件），然后才能进入各ACTION STAGE。游戏基本是以即时演算的方式来完成上面过程的，唯一一处的CG动画就是OPENING那一段，也正因如此，在发生事件时，镜头视角的转换才能够达到电影级效果，进而表达出角色微妙的情感变化。

在游戏的AVG世界有这样几个主要区域，城市、遗迹、飞艇等。每个区域均有若干个出口，通向各ACTION STAGE。而区域之间也是可以相互往来的，通过电气机车、矿车、旋风号等交通工具就可到达，注意：从城市里的火车站乘车所到达之处，我们称之为 + r1 + ，从这里可以向遗迹和飞艇进军。而玩家应将每个区域内的坐标和方向弄清，这样可很容易地找到动作关卡入口。

鉴于本作的庞大，这里无法将每个细微之处——道明，只对AVG中的必备重要宝物加以点明，一些不必要的物件就省略了，同时对ACT关卡中一些不太好过、容易死人的地方加以解决。

### 让我们回到MD时代的快乐时光

在本作中，索尼克仍旧是爆烈无比的 + r1 + ；TAILS是个机械师，正在进行自己的 + r1 + 号飞机的试飞，是索尼克最要好的朋友；KNUCKLES则与索尼克互为敌人；艾米是女主角，为了保护一只小鸟而卷入许多事件；BIG是一只胖猫，在林子中以钓鱼为生，与他相伴的是一只青蛙；最后是机器人102，这个角色比较特殊，从为蛋头博士卖命到救出艾米的小鸟，机器人在游戏中被人性化了，其内心深处向善良一面转化的过程尤其感动玩家（他在监狱中与被困的艾米相遇之际，柔弱的小鸟从艾米手中飞出，就那样停在他面前……102于心不忍而开门）。

坏蛋就是EGG MAN博士他们了。这邪恶天才科学者搞了个EGG CARRIER军用飞艇，同时还在培养一个名为CHAOS水怪，妄图统治世界。因为这水怪需要汲取远古时代的宝石中所蕴藏的巨大能量才能进化至完全体，所以我们双方便针对着 + r1 + 来展开故事（随着情节的深入，角色会发生事件时有的要回到古代）。

有了以上的基础知识和你以前玩历代SONIC的经验，游戏时就容易进行多了，你会发现在本作中出现的人物、车辆、场景、事件等许多方面都 + r1 + ，只是变成了全三维的。

最后，大家不妨留意一下游戏中的配音演出，特别是TAILS、艾米、102三人以及SUPER索尼克变身前在五万市民的呐喊助威声中发誓保护STATION SQUARE时的悲壮。国际版中的英文配音不知能否达此水平？还是请设定成英文字幕日文配音吧！这款游戏绝对超乎你的想像。它没有动画，却像一场电影。

下面会来到一同有加速器的房子中，想吃宝还得加速，这时也不得乱按方向键，否则落不了脸。接下来还会遇到第三处风暴，顽强过之，来到第三批电子怪物后，用右边的弹簧台上去吃环。过了第七记后在右前方有一斜坡，冲上去后发现一排金环通向对面，冲过去后可吃2环并加1UP。过了第八记后机关火后使用大蓄力就过关。

＜FINAL BOSS＞ EGG VIPER（蛋头眼镜蛇）

这家伙倒不算特别难。你都打了10关了该对自己的技术有信心了。一上来，敢好似�的盘龙闪般用激光打你，我们只要在右移动或跳跃移动即可躲开，注意：要进着它跑，顺着跑快了它会连续攻击。这样反复攻击两次后敌人就出来啦啦……大笑，向你挑衅，SONIC趁这时可跳跃攻击它。

第二阶段敌退后伸直那条机械蛇的尾巴，这时索尼克应连跑跃攻击其尾巴上的绿色凸出地方（第一跳后不可乱动摇杆），直至打中EGGMAN，后SONIC自动退回至桥上。注意，索尼克在桥面躲避攻击的时应小心从缺口处掉下去。在这两个攻击阶段时，玩家需一个一个地吃桥上的散环以补充体力，打一次补一次。

第三阶段，敌正面从桥的一端向索尼克袭来，很容易躲过。接着敌退至桥另一端，使用两个爪从双方，看好层至索尼克的距离，按住A键来个大跳，跳到盾上后，让它带你回到VIPER处，跳跃攻击之。它中敌人后，它会继续拆桥，桥上的距离愈向短，VIPER离你愈近，但SONIC是天生的英雄，总是能看准时机的跳跃至盾上……

第四阶段，敌已爆，但还会冲向索尼克站的桥面，应机动避之。若能跳开那一冲，方能见到制作者名单。
