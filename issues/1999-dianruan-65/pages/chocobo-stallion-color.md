### 紧急公开 菌部先生的制作秘密！

这回编辑部特别访问到 + r1 + 的监修——菌部博之先生，并向他询问有关这款游戏的一些问题，以下就是与菌部先生谈话的摘要。

之所以要制作这款游戏，起初是由于SQUARE公司希望他能够活用在 + r1 + 中所得到的经验来制作一款游戏，因而与菌部先生接触。当时SQUARE公司是想延用 + r1 + 的系统，来制作陆行鸟的游戏。最初在听到这话时，菌部先生也相当地惊讶，但是经过认真考虑后，认为在 + r1 + 中就已经有陆行鸟的育成要素了，所以也并不是什么不可思议的事。后来，以这样的感觉一路思考过来，所浮出来的念头就是这款 + r1 + 了。不过，这款游戏虽然在某种意义上，可以说是把 + r1 + 中的马给换成了陆行鸟，但还是有不同的地方。在 + r1 + 中没有办法做到的地方就是 + r1 + 。在 + r1 + 中，所有马匹的血统都被现实的马所束缚，而无法有任何的自主性。但是在这方面，陆行鸟就完全没有限

### 也有牧场经营的功能哦！

右边的插图是在 + r1 + 中登场的牧场的想象图。从第一眼看来，这好象是个相当大的牧场，但玩家如果认为在游戏一开始，便可以使用这个大牧场，那可就大错特错啦！在游戏的序盘，玩家可用的不过是一个颇狭小而设备又不完全的小牧场。之后，玩家必须利用陆行鸟赢得比赛，再拿奖金来扩充牧场的规模与设备，然后才能成为如图一般的牧场。此外，建筑物与设备等的位置，都可依自己的喜好来配置。也就是说，玩家可以建造一个专属于自己的独一无二的牧场。

在此，将为玩家们详细地介绍各个设施的名称与功用。其中也有会让看之下不知道有什么作用的设施，不过玩家们只要好好地阅读说明之后便可以了解了。

【图】牧场鸟瞰图，标注数字1-8

❶ **诊疗所** 陆行鸟在比赛或训练中受伤的话，可以到这个地方获得治疗。

❷ **陆行鸟房** 简单的说，就是陆行鸟生活的场所。玩家也可以在这里看到自己管理的陆行鸟哦！

❸ **饲料仓库** 保管陆行鸟饲食的场所。由于陆行鸟的食量很大，所以需要储存大量的粮食。

❹ **事务所** 在游戏中若有不明白的地方就可以到这里来。希洛玛会亲切地告诉你。

❺ **游泳池** 可以从事不会参加比赛的陆行鸟身体部分负担的训练。设定之后也有放松的效果哟！

❻ **陆行鸟雕像** 只要玩家达到某一个条件就可以建造这座雕像，至于条件是什么目前还是个秘密。

❼ **广报事务所** 管理奖金及广告收入的事务所。在计算每个月的收入、支出金额后会提出报告。

❽ **农园** 种植农作物的场所。设置并经营这个地方的话，就可以减少陆行鸟的粮食费用。

### 新系统是这样子做出来的哟！

这款游戏跟 + r1 + 之类的赛马育成游戏不同，游戏中没有事先准备好的优秀的种 + r1 + 。游戏最初所准备的是数十种名为 + r1 + 的陆行鸟，将这些陆行鸟任意选择并且不断地配种之后，便会产生一流的陆行鸟。而游戏也是从这里才开始，如果能成功地生产出强力的陆行鸟，父方陆行鸟的名字及血统，便能够成为新的系统并且留下名字。这种感觉就跟赛马的世界一样，若是哪一匹种马的产驹能在比赛中活跃，身为父亲的马就会被称赞： + r1 + 嗯，对于这例子还不太了解的人，就请看看右边的图表吧！

### PDA内的交易买卖

自己制作的种羽，可以借由PocketStation与朋友进行买卖！卖掉的话，将得到金钱来扩充牧场设备。而买的话，则可以增加新的源流血统。无论怎么算都很便利，所以一定要好好地利用。

制了。由于陆行鸟原本就是虚构的生物，所以根本没有实际血统上的问题。

在这个游戏中，菌部先生只负责血统系统方面的制作。而育成系统方面，则完全交给SQUARE公司的人员处理。之后再交给菌部先生做最后的确认。下面再把话题拉回到血统方面，没有自己的血统也有缺点，那就是陆行鸟没有办法象 + r1 + 那样以血统来鉴别这匹马的特性。关于这一点，菌部先生也苦思了很久，虽然还没有定案，但他打算将陆行鸟的名字取得和赛马很相似，并且给予类似的特性。这样一来，玩家就可以一看到名字就知道这只陆行鸟大概具有怎样的特性了。而从另一方面来说，不给予任何提示的话，也可以让玩家们体会到发现血统特性的乐趣。虽然血统的配合还在研究阶段，但目前进展得十分顺利，诸位玩家们可以好好地期待这款游戏的后续发展。
