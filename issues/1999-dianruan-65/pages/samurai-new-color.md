世间新越和平， + r1 + 即将宣告结束。幕府政府为了让一些无法适应血时代的浪浪者躁重新收人，便在江户近海的孤岛上建立了一个 + r1 + 来管理他们。但有些人竟以「为上天所选之人创造的新世界」为口号，想在这里进行一场惊天动地的大阴谋。此组织名为「霸业三刀众」，当中的每个人都具有极强大的能力。他们利用这些能力收服浪浪者，使他们唯命是从，并且开始一步一步地计划颠覆整个幕府。不知从何时开始，这个 + r1 + 有了「高天京」的这个名称。而「高天京」也从一个让人重生的场所，锐变为只有强者才能存活的弱肉强食之悲惨地带。为了生存，每个人都互相残杀惨。「霸业三刀众」吸收这些负面的能量，确实地扩展了自己的势力。然后，20年过去了，浮到力量的「霸业三刀众」开始了他们的行动。同时，幕府也展开了对「高天京」的调查。

## PDF p.62 / 充满魅力的角色们！！

侍者，不灭的精神，在他（她）们心中熊熊燃烧不尽，战斗吧！

魂者，不灭的精神，在他（她）们心中熊熊燃烧不尽，战斗吧！

这次「侍魂」新作的角色设计，真是太糟糕了！每一个都帅到了极点，到时要我们选哪一个呢！SNK果然不是普通坏心眼！因为背景是设定在「侍魂斩魔传」的20年后，因此服装和武器也有很大的变化。来吧！您中意的是哪一个呢？

### 九葵 苍志狼 Seishirou Kuki

新主角，不但是一流的谈判高手，还拥有一身非常棒的好剑法及秘剑。承继亡父的遗志成为幕府的秘密剑士。为了从未死的义兄弟上寺殿只传子不传女的家宝「二刀对刀」，而向他挑战。

### 九鬼 刀马 Tohma Kuki

白肤赤眼，是男主角的同门师兄弟。谈判的高手，秘剑的性质是以直线型攻击及「一击必杀」为追求目标的 + r1 + 。他寻了自己已死的义兄弟父，并夺走「二刀对刀」的其中之一后，便消失踪影。

### 花房 逢卫门 Jin-emon Hanafusa

秘密剑士的其中一人，和男主角一同出生入死的老练战士，近乎出神入化的格法，足以弥补他身材矮小的不利点。因为接到了幕府的秘令，而和男主角一同前往「高天京」去进行调查。

### 霸王丸 Haomaru

看似豪爽又纤细，看似大胆却又慎重，不流于形式的剑法是其特色。在剑的世界中他的名字无人不晓，因为他身旁流传出的种种传说，使他有了「传说中的剑豪」之称。他的武器仍是 + r1 + 。

### 服部 半藏 Hanzo Hattori

最强的忍者，也是伊贺忍者群的首领。在隐密忍者装下隐约看得到的伤痕，透露出他还年轻的秘密，难道说他是第2代的服部半藏？

### 神统士波 Jushirou Sakaki

反幕府组织的秘密剑士，脸上的伤和黑色手套令人印象深刻。他手上的武士刀形状非常奇特，不知道会使出何种攻击？

### 七座 友人 Haito Kanakura

灰色的头发，兽异的气质。从他刺满全身的刺青和手上那把无鞘的利刃，可以看出他是一个具有反社会人格且特的角色。懒懒的笑容，自我风格强烈的服装，看来个性扎实残忍的。

### 十六雉 夜血 Yaci Izanagi

他包在鞘中的，是一把拥有锯齿状刀的利刃。腰间还插着另一把小刀，微懒慢的笑容，自我风格强烈的服装，看来个性残忍的。

### 吉野 凛花 Rinka Yoshino

拿着和身高不相称的长刀，这位少女到底有着什么样的过去呢？她也是一个命运不同于常人的剑士吧？

### 旋毛凤之卧龙 Garyou the Whirlwind

山贼的头目，是个独眼，全身伤痕累累的壮汉。一只手藏在怀中，另一只手则握着双击的武器。看起来似以一身蛮刀。

### 沙耶 Saya

暗杀性身材及透人姿势的金发外国人。不过她却是使用二刀锻的女杀手。被她外表给迷惑的话，凄惨的可是自己哦！

### 乱风 Ra po

爱着着大金锤的小野人。不讲求技术，看来在战斗中她都是以力取胜，个性轻狂而可拮。看来很难应付！

### 命 Mikoto

红色的瞳孔，拥有善异样美感的女性。全身散发的冷气和忠诚忧伤的表情，不禁令人想起 + r1 + 那位角色。
