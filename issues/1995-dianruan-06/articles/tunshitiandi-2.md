# 吞食天地Ⅱ之诸葛孔明传——指令和道具中日文对照

> 栏目：三国游戏专辑 ｜ 刊页 38-40
> 机种 FC　类型 RPG　厂商 CAPCOM

---

主要场景有灭袁术、白马战、过五关、三顾茅庐、赤壁之战、收服西蜀，与曹操的决战、与司马懿的决战。前后共有三十员大将加入刘备军，按登场顺序如下：刘备、关羽、张飞、朱灵*、露昭*、陈登、颜良*、文丑*、周仓、赵云、关平、诸葛孔明、鲁肃*、伊籍、魏延、黄忠、严颜、马良、马谡、廖化、孟达、法正、庞统、马超、马岱、吴兰、蒋琬、吴懿、姜维、张苞（有 * 者为暂时加入）。

## 基本操作（行军指令）（きほんそうき）

**行军指令菜单：**

| 指令 | 日文 | 说明 |
|---|---|---|
| 对话 | はなす | — |
| 调查 | しらべる | — |
| 队伍编制 | たいれつ | → 行军编队 / 任命军师 |
| 武将资料 | ぶしょう | — |
| 道具装备 | どうぐ | → 使用道具宝物交换装备 / 使用武器防具丢弃装备 |
| 计策 | さくりゃく | ← 计策（けい） |
| 阵形 | じん | ← |

1. 谈话：与人交谈，获得过版信息。

2. 调查：带队武将可得到金、道具、武器防具以及某些宝物。

3. 编制：①行军编队顺序的前五名武将才能作战。②任命的军师智力要在 160 以上，有了军师军队作战时才能使用阵形和策略。

4. 武将资料：察看武将的兵力（为 0 时无法战斗，不是死亡）。武力、智力、速度（攻击、躲避攻击的快慢）、攻击力、防御力、SP、武器类型、优势地形（战斗中兵力格颜色不同）、计谋、阵形、策略（以上三项军师才有）。

5. 装备：包括武器防具（见表 1）、道具（见附录）、宝物（见表 2）。

6. 计策：包括行军中计谋（见表 3）、行军中的策略（见表 5）、阵形（见表 4）。

## 战斗指令

**战斗指令菜单：**

| 指令 | 日文 | 子菜单 |
|---|---|---|
| 攻击 | こうげき | — |
| 策略 | さくりゃく | → 火計（かけい）/ 水計（すいけい）/ 落石計（らくせきけい）/ 回復計（かいふくけい） |
| 道具 | どうぐ | — |
| 総攻击 | そうこうげき | — |
| 武将资料 | ぶしょう | — |
| 撤退 | たいきゃく | — |
| 防御 | ほうぎょ | — |
| — | — | → 陣形（じん）/ 策略（けい） |

1. 攻击：游戏者设定指令攻击敌人。

2. 策略：包括火、水、落石、回复四种计谋（见表 3）、阵形（见表 4）战斗策略（见表 5）。

3. 装备：战斗中使用道具、宝物。

4. 总攻击：带队武将拥有，不需指令自动攻击敌人，途中连按⑧键可停止。

5. 武将资料：战斗中察看武将各项指数（敌我双方）。

6. 防御：退却不一定能成功，防御可使伤害减半。

## 附录　过版宝物

由于此卡硬性规定了游戏过程，过版不易。但游戏中有一些宝物是过版的关键。下面按顺序罗列出来，以助众“Game”友们早日通关。

- **ちず**（地图）：在徐州城东北的洞窟中，可看到中原、江南、东北的地形。没有它就不能看见费县。

- **エンショウへのてがみ**（信）：在郑玄家对面，交给袁绍，会派兵助刘备攻曹。

- **せきとば**（赤兔马）：关羽在许昌城刘备夫人处得到装备后行军速度会快一倍。

- **コハツへのてがみ**（胡华信）：关羽在弱华家中得到，在过第四关时交给郾喜便可出城。（如进堡半夜会遇到火攻。）

- **ひほうのしょ**（秘法书）：坐船渡海到日本，杀死山顶大蛇，得救的女巫会交给孔明，在赤壁之战中孔明用它来借东风。

- **しょうせき**（硝石）：在柴桑城东北小岛中的洞窟里。

- **うろばくのえだ**（老朽枝）：在柴桑城中央一位老者身后取得。做火药所必要之物。

- **かやく**（火药）：将硝石和树枝交给老者，他会制出火药，可将博望坡大石炸开，到达襄阳城。

- **ひゃくまんほんのや**（百万支箭）：在襄阳大殿地下秘室内得到，交给孙权在赤壁之战中使，为对抗曹操之必要。

- **てがみ**（信）：在荆州城大殿宝座下得到，可进襄阳城，和守城曹军交战。

- **ショクのちず**（西蜀地图）：在孟达华之子明处得到，可看到西蜀地形。

- **けっぱんしょう**（血判状）：李敢可用来说服马超，马弓加入。

- **せいどうのかぎ**（青铜钥匙）：可用来打开水门。

- **かやくつぼ**（火药壶）：可在绵竹关内引起大火，引张任军作战。使用火计所必要。

- **キッペイのしょうかいじょう**（吉平的介绍状）：用来请华佗医治维母亲的重病。

- **とこうじょう**（航海状）：可使在码头前的辛毗攻击，然后上船渡河去街亭。

- **さけ**（酒）：在司马懿家中取得，用来了解襄阳城西野盗森林的秘密。

- **てがみ**（信）：第二次在郑玄家中取得，可使洞窟中的孟达让开。

- **りっこうてんしょ**（六甲天书）：和操决战时，可用来破曹操的火计，能呼风唤雨的秘传之书。

- **おちゃ**（茶叶）：从天水城中一行人手中取得。

- **ちゃき**（茶壶）：从鲁城取得茶壶，将茶叶、水、茶壶交给隐士，他会教你破司马懿落雷阵的方法。

- **すいとう**（水壶）：从隐士处取得水桶，到黄家庄可向井旁的人要一桶水。

- **きゅうてんたん**（九转还魂丹）：用一千金买下，可获得走乱石阵的方法。

- **そっこうのくつ**（速攻鞋）：装备后可使速度提高 20。

- **ねんど**（连弩）：战斗中使用，可对每个敌人进行攻击。

- **じゃはこ**（蛇矛）：装备后可使攻击力提高到 235。

- **かしんのしょ**（火神书）：战斗中使用，可施出最高等级火计。

- **げきせきのしょ**（落石书）：战斗中使用，可施出最高等级落石计。

- **すいりゅうのしょ**（水龙书）：战斗中使用，可施出最高等级水计。

- **もうとくしんしょ**（孟德新书）：装备后可使智力增高 20。

- **へいほうのしょ**（兵法书）：战斗中使用可随机施出一种策略。
