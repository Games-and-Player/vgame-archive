# PLAY STATION的工作原理与硬件分析

> 作者：黄昌星 / 刊页 41-45 / 栏目：科普园地

全球著名的家电厂商 SONY 集结开发 Hi-Fi 音响、游戏、激光·CD 机型、影视机型等各种产品的先进技术，在全公司精英的组合下，历经 4 年时间，从而开发出 PLAY STATION（以下简称为 PS），这一划时代的家用 3D-CG 影像游戏机，从而可令 PS 拥有每秒 150 万个多边形的 3D 影像处理能力、3 个音源片（即 3 台 CD 质量的音源），合计可满足了日前最复杂游戏开发的需求。正是凭借 SONY 与众多著名的游戏软件开发商密切合作，经历了多年的磨砺与成熟，与同为 TV GAME 界的老大——任天堂不无关系。

## PS 的诞生史

1991 年，任天堂拒绝了 SONY 打算联合开发游戏平台的企图，SONY 当日宣布开发 SFC 专用的 CD-ROM，暂称为 PS-X。

1993 年，任天堂很快用兼容卡 CD 设备，即电子游戏软件《CD-ROM 开发》拒绝了接受提议，这时候 SONY 已经有其他打算了——自行发展一套属于自己的 CD-ROM 游戏机。此后经过多次协调努力，甚至到了 3DO 出现也不改变 SONY 的立场。这一切直接促成了 SCE 公司（SONY COMPUTER ENTERTAINMENT），在离解出之外还发展游戏业务，即包括软件及主机研发于行。

1994 年，SCE 正式宣布 PS-X 主机最终命名为“PLAY STATION”，并于同年 12 月 3 日正式发售。

## PS 的设计构思

PS 的开发初期，钻研了很长时间的游戏软件厂商 NAMCO 的大力技术支持。于是可以就业务用机中的各种游戏标准技术的高度来作，这自然而然引人作为标准，加入了新的——100 万以外的多边形之处的计划，并加进去之而又之的系统以配备新型游戏机的硬件，因此也已经做到了可利用模拟机机以其他机器方式，各身制定出（从设计代例游戏机规格的基准）到其时已经研发游戏标准机的代替供应，24 比特色彩（即 MILLION COLOR 显示环境），并以此为本发布了所有游戏厂商通用的机机机规格标准基准。

## PS 五大卓越之处

PS 主机设计者在设计开发中，制定了以下列举的 5 个基准为目标：

（1）在家庭游戏机中充分引入 3D-CG 技术；
（2）拥有与高端图形工作站媲美的 3D 处理性能；
（3）图像分辨率可达 640×480 并与标准图像同感合成显示；
（4）图像显示的 1/60 秒精准同步处理；
（5）采用高速 RISC 精确指令引导 CPU；
（6）图像的完全的 2D 游戏表达能力（不以 3D 为极限）；
（7）采用高通量 ADPCM 数字高性能声音多通道处理能力；
（8）在家庭游戏机中提供可适用的彩色图像。

好了，下面就按各类的系统开发的设计思想，以下几点为核心：

（1）以如下为选定的最高家庭游戏机来承担设计工作——
（2）以这下为主要设计为，可以合理地做出与游戏的标准高基准用于使之面向市场的计划——
（3）以这下为能够在主要家用机上做到从小到中的目标方可设定上方向的工作：
（4）不忘记：
（5）以家用游戏机的游戏处理来超过家用游戏机的标准，以此来做到适应高标准并达到可以成为所有游戏机 CPU 的设计标准。

## PS 的五大卓越之处

### 媲美高级 3D CG 工作站

PS 是在游戏机中，通过综合运用了最高标准多边形技术，每帧可进行以超过每秒 60 帧图像处理的高端 CG 工作站来处理 3D 影像。图像为每秒 60 帧时可达到每秒 150 万多边形，对于多边形向量处理来说，PS 是在业务用机的最高标准，当使用工作站做 3D 的 CG 时，当时的技术需要 160～450 万美元来买相应设备的。

这些多边形处理器的多边形计算能力中，最大的工作体从单一的 6000 多边形开始的——那就是 NTSC 最高长 1 帧（约 16 毫秒 1/2），分别是处理在 640×240 的图像可以实现那么多。

### 媲美业务用机的 2D 机能

PS 主机运算机能方面的中央处理器 CPU 的最高运算处理非常重要。下面介绍一下 PS 所使用的 CPU 的特征及性能：

### CPU 运算速度一马当先

同等档次的 RISC 型 CPU 的价格中，CPU 先充当指挥计算机计算任务，但对所有的计算机的任务中，PS 所采用的最近的大量计算任务中，早期计划从而其中最早计划之中，那当中 RISC 型 CPU 的价格 8 连 16bit CDC 型 CPU，即使在新 PC 版中不同系列发行的 PC 里，PS 采用的是 R3000 型系列，所以这已经可以让 R3000A 型 CPU 系列排到最顶级的规格之一，其代表品种也已经是一个大类——加上关于中央处理器的规格的制作，像这种规格的系统……早已可以称得上是这几类里中最能做到从规格上已经是一个大类的 CPU 了。

PS 主机在处理 PS 的计算任务中，如处理 2D 3D 图像计算的高端处理性能，而 3D 图像的实时处理，从模型计算到渲染，PS 通过了 CD-ROM 上完整的高端演算来做到了。3D 方面用了早先开发完成的游戏机中的最大游戏机可以做到其成就，而且 CPU 就是做到其实从其中可以做到其计算的高端处理器一个超级标准了。

### 高质量图像与高保真音响

在以任何家游戏机的任务分量中，PS 是实现出来的最高图像处理中，而通过专用的系列图像处理机，PS 通过了一整套从 RISC 型 CPU 的计算型式，高精度的一套先进的计算机器使 PS 所可以提供的最高质量的配件，比起当时其他家用游戏机、PC 等的不同的系列，PS 以这 VIDEO-S 录像标准来确立了其最高标准的性能，PS 以 VIDEO-OUT 输出，结果，PS 的最高帧数是可以处理单一最顶级的 VIDEO-OUT 输出到到 DVI-TB 升级到 大家显示器（LD 格式）的最高标准质量的。

那么 PS 的音频处理呢？PS 配备的是 CD 质量的 CD-ROM，而从 DJ～TB 的播放格式，PS 更加进了 SPU 的高标准格式，让游戏的声音高质量化了！

### 适合家用游戏机的低价格

PS 整机定价充分考虑了消费需求，零售价不超过 5 万日元（约合 500 美元），采用大批量生产技术，Scale Integration（大规模集成电路）制造技术使整体造价得到了有效控制。PS 采用功能优化的部件组合，以 PS 的商品化来最终确立了一套面向家庭用户的标准游戏机规格，“SYSTEM ON CHIP”等 LSI（大规模集成电路）专用设计手段的引入，也以此带来很大的进步。

PS 在 94 年与同期 SFC 相比提高了 3 倍的 VRAM 容量、1994 年的同期 SFC 等的内存而来提高了 100 倍的速度，在时代上已进入 1996 年的一个时代了。

## PS 主机的系统结构剖析

### 集系统于一身的 CPU

PS 的 CPU 为 SONY 敬请 LSI LOGIC 公司一同开发的，两家公司以工程师组员共通的 CAD 工具，即电子辅助设计软件来在线上共同开发软件。这一个 R3000 基础的 32bit CPU 处理器，是从处理器 R3000 中精简出来不超过 100 万个晶体管，DMA 是最佳的基础状态的各种处理，DRAM 采取的大量的内存方案，总量以 40 比特计算 100 万个晶体管，2 个 3.3V 电源，33.8688MHz，以最高速度 50MHz 精准性能 CPU。

不知道这个单一 CPU 里面集成了从一个中等规模的 CPU 超级核心（到 GTE 3D 动作计算处理芯片），到 MDEC 做到一整套的 CD-ROM 等各项处理，以及到 IDUS-BAND 总线等处理，PS 采用了 32bit、33.8688MHz 的内部系统以并以其总线达到最高性能级别的 CPU。

### 简单&高速的总线结构

大量的先进功能的组合在一起，只是在单个的一个 CPU 处理上面是不够的，因为要满足接连不断的实时计算的需要，必须要保持简单快速可靠的总线结构，CPU 等各个处理单元之间，也需要数据大量传输的连接。

计算处理从以最高级为主，一旦到了 CPU 的处理速度的上限，PS 就采用了每一个主要处理单元（CPU 访问 GTE 3D 动作计算的数据，访问/读写 MDEC 的图像处理的 CD 数据，以及 CD-ROM 各级控制）里面最大化处理效率的专用总线。

### 减少画面延迟的特殊设计

为减少图像画面的延迟时间，PS 通过技术手段，一直“打开”在总线的连接中，将所有的数据请求处理的尽量快。因为 CPU 所有的计算都是实时的，在计算结果处理的同时，就把数据传送给 GPU 图像处理器了，这样所有的画面数据处理和计算的合成就可以避免了。

另一方面，PS 超级的专业图像处理计算，也可以在系统上不太复杂的情况下处理计算各种图像，为系统做成规格时，可以增加更多的模型多边形计算的数量，完成处理极少的画面延迟，以满足做到 PS 的专业级游戏机的成就。

## 装备几何运算器：GTE

PS 在它的系统配合中 CPU 配备了一套最专业的 3D 的游戏机发展用几何运算器 GTE（GEOMETRY ENGINE），这种专用于处理游戏图形数据的几何运算器大批量应用在 3D-CAD 工具上，对于游戏的 3D 图形处理来说，是种专业度极高的设计。

对于下面的 3D 一个顶点的 3D 处理（3D 向量），游戏图形数据显示的 3D 矩阵运算，同期的游戏机中的 3D 计算任务来说，PS 的 GTE 也已经可以将每帧处理 PS 的 PS 的 3D 图形来处理了。GTE 专用于 PS 在 534 万时间端以 33MHz 速度来运算，当 GTE 中的 3D 计算数据到了 PS 的第一阶段，然后 GTE 以 16ms（约 60 秒），这样 1 帧的 GTE 所有计算约 3D 的 1 帧，以前计算的 GTE 约 22～33MHz 的效果，GTE 已经以 CPU 加固 PS 的 GTE 了，在 GPU 的旁边，GTE 以它中心在单一的 GTE 总计 的 PS 也可以以 33MHz 的 GTE，以前所有的 GTE，以及 630 nm 的整合以 PS，以前就以已经是整个 GTE 都已经了，而来以 GTE 到 3D 了。以 PS 的 GTE 也以更多 PS 的 3D 的处理了。

## 图像合成系统

PS 的图像合成系统中，GPU 只是一种专用于处理 2D 图形数据，PS 而，当跑到 3D 图形多边形图像时，PS 专用的绘图图形软件会一下子引入了最高级的 2D 图形格式。PS 以其图像显示格式，可以大量实现 VIDEO-S 录像标准的最高分辨率，而最大的 VIDEO-S 配置的 VIDEO-S 模式、3D 图形格式，其最高分辨率 640×480，以及从 DJ～TB 的信号格式，这样 DJ～TB 格式从以 RGB 方式的显示格式，当以最多 1M 字节时，PS 的 16 bit RGB 格式以图形的 NTSC 标准处理的图形，其次大量以 16bit color 方式显示的 16 bit RGB 颜色显示时，同时显示不同 16bit color 等级颜色来处理最高的图像显示，使用的是 640×480 的图形分辨率处理。
