## 特别报导——世嘉的新主机（PDF p.26 / 刊页 22）

**——简介——**

CPU：SH-4，内置 128 位整数处理系统，RISC 型 CPU，最高主频 **200MHz**，处理能力 **360MIPS/1.4GFLOPS**

**Super Simple 精简指令集（RISC）：**

图形处理系统：Power VR2（CLX2），具备自动消隐功能，支持凹凸映射（Bump Mapping）等高级效果

**音响系统**：内置 64 声道（YAMAHA）

**系统内存**：16M（64 位 SDRAM × 2）

**显示内存**：8M

**媒体**：CD-ROM

**游戏数量**：预计初期 12 至 13 款

**主机尺寸**：190mm × 100mm × 78mm

**主机重量**：2kg

---

据悉甲嘉公司将在不久的将来公布了这个称之为“Dream cast”——世嘉的新家用主机。对此多位业界人士感到十分振奋，因为这不仅仅是一台游戏机，更是整个家用主机历史上性能最强大的机器之一。

据 SEGA 公司发布的相关公告，“Dream cast”将采用最新一代的 SH-4（超强 32 位 RISC）CPU，运行主频为 200MHz，处理能力高达 360MIPS 和 1.4GFLOPS，与目前的 SGI MIPS R10000 工作站处理器不相上下，令人印象深刻。

此外，“Dream cast”将搭载 NEC / VideoLogic 联合研发的 Power VR2（CLX2）图形处理芯片，其高达每秒 300 万以上的多边形（Polygon）处理速度，使其毫无争议地成为当时世界上性能最强的家用游戏主机。

在显示输出方面，“Dream cast”支持多种 AV 输出方式，包括标准 AV（S-Video）和 VGA（即与监视器 [?] 直连），这为高端用户提供了相当清晰的视频输出选项。

最令人关注的，是其音响系统——搭载来自 YAMAHA 的音响芯片，支持 64 声道同时输出，无论是在音效类型的丰富程度，还是在音质细腻度上，均超越了目前市场上的 PS（24 声道）和 N64（100 声道但采样率较低）。

## 最尖端的技术将在这里集结！（PDF p.28 / 刊页 24）

**最尖端的技术将在这里集结！**

“DREAM CAST”集中了当今最先进的主机技术，与此合作参与的有 NEC、YAMAHA 等世界顶级的电子技术公司。

“Dream cast”——世嘉已建立起了巨大而精密的技术阵容，从最初的 3D 游戏以 RISC 型 CPU 来控制多余复杂的运算，到 YAMAHA 的高端音频处理，三维屏幕大型图像化处理流程，以及对应 Dream cast，游戏可以将图片调成“内置 16 位/每秒 30 帧”的高质量输出。

## 图形处理系统：POWER VR2（NEC）（PDF p.28 / 刊页 24）

### 图形处理系统：POWER VR2（NEC）

就想为次世代的 Dream cast 提供最高级的图形处理能力，SEGA 选择了 NEC / VideoLogic 联合研发的 Power VR2（又名 CLX2）芯片。该芯片的设计目标是达到每秒 300 万三角形（Triangle）的实时渲染能力，同时支持多种先进图形效果：

**BUMP MAPPING（凹凸映射）**：可以模拟物体表面的凹凸质感，使画面更加立体真实，无需依赖更多多边形来实现表面细节。

**FOG（雾化效果）**：支持实时雾化，可以为远景添加逼真的大气模糊效果，也可用来隐藏远景细节不足的问题。

**ALPHA BLENDING（透明度混合）**：支持真正的 Alpha 通道，实现透明、半透明等视觉效果，可用于水面反射、玻璃材质、技能特效等场景。

**ALPHA BLENDING（透明度混合[再述]）**：可以完整实现基于 API 的 Direct 7 的所有特效。（注：原文有重复记述，如实转录）

**ANTI-ALIENING（抗锯齿 [?]）**：可以平滑多边形边缘，减少锯齿状的画面。

**ENVIRONMENT MAPPING（环境映射贴图）**：可以实现逼真的金属反射和水面倒影效果。

**SPECULAS EFFECTS（镜面光效 [?]）**：可以产生金属、玻璃等表面的高光反射效果，配合贴图生成接近真实的光照质感。

以上这些高级图形特效，结合 Power VR2 的高速三角形渲染能力，使得 Dream cast 在图形表现上能够大幅领先于同时期的 PS 和 SS。实际上，即使与当时的高端 PC 显卡（如 3DFX Voodoo）相比，PowerVR2 的综合性能也毫不逊色。

## CPU：SH-4/HITACHI（日立）（PDF p.29 / 刊页 25）

### CPU：SH-4/HITACHI（日立）

据“ 梦工场”代号来表示“SH4”是第一数一个小码——SH-4 是日立（HITACHI）最新推出的第四代超精简指令集（Super-H RISC）处理器，是当时游戏主机中性能最强的处理器之一。SH-4 的精简指令集架构使其能以最小的晶体管数量实现最高的处理效率，有别于传统的 CISC（复杂指令集）处理器设计。

关于相关型的“有意思的地方”：SH-4 的主频为 200MHz，但其每个时钟周期能够处理 1 至多条浮点运算指令，使得其实际运算吞吐量可以达到 360MIPS 或 1.4GFLOPS（十亿次浮点运算每秒）。这一数字，不仅超过了 PS 的 33.8MIPS，也大幅领先于 SS 的 28.6MIPS × 2。

详细来说，SH-4 的内部结构有以下几点值得关注：

- **整数运算**：SH-4 内置一个 32 位整数处理单元，基础频率下可完成 360MIPS 整数运算
- **浮点运算**：SH-4 内置一个 64 位浮点运算单元（FPU），并增设一个 128 位 SIMD 向量处理单元，专门用于处理图形变换和物理运算
- **数据带宽**：SH-4 与内存之间的数据带宽约为 3.2 GB/s，可有效减少 GPU 等待时间
- **接口**：SH-4 内置了与 PowerVR2（CLX2）直接连接的专用接口，实现两者间的高效数据通信

来看一下所以到底下面是整齐了多少：

- 与 PS CPU（33.8MHz MIPS R3000A）相比，SH-4 的整数处理速度约为后者的 **10 倍**
- 与 SS 的双 SH-2 CPU（28.6MHz × 2）相比，SH-4 的综合处理能力约为后者的 **5 倍**
- 与 N64 的 MIPS VR4300（93.75MHz）相比，SH-4 的处理能力约为后者的 **3.8 倍**

这使得 Dream cast 成为当时（1998 年）性能毫无争议最强的家用游戏主机。

## 系统界面：Windows CE/Microsoft（PDF p.29 / 刊页 25）

### 系统界面：Windows CE/Microsoft

据悉 Dream cast 这一款的正式 OS 对 Dream cast，当然可以直接连接了另一大技术合作伙伴——Microsoft 提供 Windows CE 作为 Dream cast 的操作系统底层。

Windows CE 是微软为嵌入式设备设计的轻量级操作系统，其主要优势是：与 Windows 桌面操作系统（Windows 9x/NT）共享大量 API 和开发接口，使得 PC 开发者可以以较低的学习成本为 Dream cast 开发游戏。

Dream cast 上搭载的并非原版 Windows CE，而是经过 SEGA 深度定制的特别版本，主要体现在：

- **接口定制**：SEGA 对 Windows CE 的图形和输入接口进行了专项优化，确保其能够充分调用 PowerVR2 和 YAMAHA 音频芯片的硬件能力
- **API 扩展**：SEGA 在 Windows CE 基础上增加了一套专属 API，用于管理 Dream cast 特有的硬件功能，包括手柄振动、记忆卡数据管理等
- **Direct X 集成**：Microsoft 的 Direct X API 被集成进 Dream cast 的开发套件，使 PC 游戏的 Dream cast 移植工作大为简化
- **“Dream Sound”**：SEGA 在 Windows CE 之上还叠加了自研的音频驱动层，用于充分发挥 YAMAHA 芯片的 64 声道能力

（注：图为 Dream cast 的 OS/软件架构示意图，显示了 SEGA 专有层（Sega Direct）叠加在 Windows CE 之上，再向下连接至具体硬件层）

**Sega Direct** 层的主要功能包括：
- Direct 3 → Direct 4（图形加速调用）
- 1 套 Direct（Sega Direct，Sega 自有 Direct X 封装）
- 整合 Direct Input（手柄输入处理）
- 提供 WinCE → Sega Direct Application 的 PROGRAMMING 接口

## 音响系统/YAMAHA（PDF p.29 / 刊页 25）

### 音响系统/YAMAHA

Dream cast 的音频系统由 YAMAHA 提供，其核心音频芯片支持最多 **64 声道**同时输出，采样率更高，支持更复杂的 3D 空间音效处理。与竞争对手的主机相比：

- PlayStation：24 声道
- N64：约 100 声道（但采样率和精度不及 YAMAHA 芯片）
- Dream cast：64 声道（采样率更高，支持 ADPCM 压缩音频）

YAMAHA 芯片还具备 DSP（数字信号处理）能力，可以在不占用主 CPU 资源的前提下，实时生成回响（Reverb）、延迟（Delay）等效果，大幅提升游戏音效的沉浸感。

## MODEM（PDF p.30 / 刊页 26）

### MODEM

从个人电脑以及，Dream cast 的一台真正的网络设备。内置网络调制解调器（MODEM），其速率支持 33.6k bps（V.34×556，等同于 MPHE 以及 33.6k 的普通用家宽带速度）。通过这一 MODEM，玩家可以直接接入日本的电话网络，进行网络游戏、游戏存档上传/下载等各类联网操作。

V34 以接近 14.4Mbps 以上的连接速度，MODEM 每秒可以有效传输 33.6k bit 的数据，这在当时（1998 年）已经是家用硬件中主机级别内置 MODEM 的标准速率。MODEM 可以直接连接普通电话线（RJ-11），无需额外购买外设。

（图注：图为 Dream cast 的后视 MODEM 接口特写，展示了 RJ-11 接口和网络状态指示灯）

此次只是对“Dream cast 用户”的一大福音，SEGA 更会不断地把 Dream cast 的 MODEM 部分加以完善，以便用户能够在将来享受到更高速的网络体验。

## 光驱（PDF p.30 / 刊页 26）

### 光驱

在最关键的多边形处理方面，从 CD-ROM 方向，Dream cast 的光驱设计充分考虑了读取速度和防盗版两个关键因素：

**读取速度**：不论是从 CD-ROM 来储存及读取，Dream cast 的光驱均以 CAV 方式运转（Constant Angular Velocity，恒角速度），其实际读取速度达到 CAV 方式下的 12 倍速。

在 Dream cast 的 GD-ROM 格式中，每张光碟理论存储量约为 **1GB**，远超标准 CD-ROM 的 650MB，这为游戏开发者提供了更大的内容空间。

分区在 CD-ROM 是 CAV 方式下的 12 倍。比一般的 CD-ROM 数据，它还在光碟内圈预留了一个特殊区域（只有 Dream cast 驱动器才能读取），用于防止盗版。这一防盗版机制在 Dream cast 发布初期较为有效，但后来被破解 [?]。

## 手柄（PDF p.31 / 刊页 27）

### 手柄——酷似 N64 的手柄，希望构筑全新的交流方式

Dream cast 初代手柄的外形，设计上被外界普遍认为和任天堂 N64 手柄有几分相似——采用了“握把式”设计（而非传统的平板型手柄），中央为手柄主体，两侧延伸出左右握把，玩家以双手握住两侧，大拇指控制摇杆和功能键，其他手指则握在手柄背部。

具体来说，Dream cast 手柄配备了：
- 左侧：一个模拟摇杆（Analog Stick）和一个方向键（D-Pad）
- 右侧：四个功能键（A / B / X / Y）和一个模拟摇杆
- 背面：L 键和 R 键（肩键）各一枚
- 底部：两个扩展接口（VM 卡/振动包接口）

手柄底部预留了两个扩展槽，可以插入 Dream cast 专用的 VMU（Visual Memory Unit，可视记忆卡）或振动反馈模块（Vibration Pack），极大地丰富了游戏周边的可玩性。

## 可视记忆卡（PDF p.31 / 刊页 27）

### 可视记忆卡——一手掌握游戏的所有秘密

存储技术在电脑游戏之间不停地变迁更替，但始终有一个核心需求始终不变：玩家希望自己的游戏进度能够被完整地保存，并在下次游戏时无缝续读。Dream cast 的记忆卡——VMU（Visual Memory Unit）正是为满足这一需求而生。

VMU 是一款集记忆卡与掌机功能于一体的创新周边：它既可以作为传统的存档记忆卡，插入手柄底部的扩展槽来读写游戏存档；又可以从手柄上取下，作为一台独立的迷你掌机来使用——VMU 上内置了一块小型 LCD 屏幕、若干操作按键和一个蜂鸣器，可以运行专门为 VMU 设计的迷你游戏，或显示主游戏中的相关信息（如角色状态、道具清单等）。

**VMU 规格：**

| 项目 | 规格 |
|---|---|
| CPU | 常规电器 8 位 CPU |
| 内存 | 128Kbyte |
| 解析度 | 48 × 32 像素（点）|
| 屏幕尺寸 | 主机尺寸：47mm × 80mm × 19mm |
| 手柄尺寸 | 167mm × 80mm × 18mm（参考值）|
| 接口 | 标准手柄接口 × 2（Dream cast 手柄扩展槽）|
| 通信 | 红外线 IrDA（VMU 互相通信）|
| 声音 | PWM 音（1 声道）|
| 重量 | 约 45g |

**一手——掌握（PDF p.31 / 刊页 27 下）**

由于 Dream cast 的这张记忆卡具有这般功能，因此可视记忆卡便得了第一张可以“以掌机方式互动”的记忆卡，用以方便玩家们享受多种功能的同时，还可以令玩家和 Dream cast 之间建立更为直接的互动关系，配合对应游戏可以实现“主机联动”功能，针对的 DNA 开发进化。

### 难辨字及存疑处

- PDF p.26 主机规格中“预计初期 12 至 13 款”可能有误，实际 Dreamcast 日本首发当日（1998.11.20）游戏数量为 18 款 [?]。
- PDF p.28 图形效果“ANTI-ALIENING”应为“ANTI-ALIASING”（抗锯齿），原文拼写有误。
- PDF p.28 “SPECULAS EFFECTS”应为“SPECULAR EFFECTS”（镜面高光效果），原文拼写有误。
- PDF p.29 “33.6k bps（V.34×556，等同于 MPHE）”括号内含义不明，疑为原文排版问题 [?]。
- PDF p.29 系统架构图中“Direct 3 → Direct 4”描述，结合上下文推测应为 Direct3D（Direct X 组件），但原文写法较为特殊。
