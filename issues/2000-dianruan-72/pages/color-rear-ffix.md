## 特殊能力

### 透过道具的装备来引出潜在的能力！

由于 + r1 + 系列每一次都会推出革命性的战斗系统，因此相信大家一定都很关心到底这次的战斗系统变成了什么样子。虽然SQUARE公司已经先行公开战斗系统里和能力有关的部分，不过包含画面照片在内，目前还没有任何与能力、道具名称有关的具体资料。因此，这里只能将截至目前所获得的情报，通过图解的方式为大家介绍。另外，这里所用的装备品和能力名称全部都是虚构的，大家可不要信以为真哦！

在 + r1 + 的世界里，角色在装备了武器、防具或是道具以后就可以使潜在的能力出现。这种能力相当于存在于角色身体里的一种 + r1 + ，而透过道具等装备就可以将此种能力诱发出来。不过，在把道具装备上去之前，事前是无法得知会出现什么样的能力的。除了靠装备道具之外，角色本身也会依据自己的特性而拥有某些特殊的潜在能力。例如，身为盗贼的杰坦，他可能就会拥有和敏捷、盗取有关的潜在能力。不过，很可惜的是目前还没有任何能力特性或者是能力种类有关的情报。不过，只要一有新情报，我们就会立刻为大家报道，请诸位期待！

### 不同的道具会引出不同的能力

虽然可装备物品的详细名称和种类并不清楚，不过武器、防具和装饰品等，这些大部分都有引出各式各样的能力。看来 + r1 + 的战斗系统一定会非常地有趣。如果能象上图一样装备上很多装备的话，那么角色也就有可能同时拥有复数以上的能力。而玩家也可以制造出抢有复数能力的角色。

### 系统可做

就如左边的说明一样，本页所解说的这个新系统，可以经由道具的种种排列组合让玩家创造出自己喜欢的能力组合，并且订制出更加独特的角色。过去坂口博信先生曾经说过，玩家可以自由地订制出自己想要的角色。看来，他所指的应该就是这个系统。大家在看过这里的介绍之后，会不会觉得 + r1 + 这种可以自由订制角色能力的方式，真的是太神奇了？

### 越贵重的道具出现的能力也就越稀有？

潜在能力并不是以随机的方式出现在每样道具中。越是贵重的道具出现稀有潜在能力的可能性就会越高。另外，依道具不同，有些道具还可能会出现复数以上的潜在能力。不过虽然某些贵重道具拥有稀有能力，但它的攻击力可能并不高。反之，有些武器攻击力很强但是却不会出现很强的能力。到底要以攻击力为重还是要以能力为重呢？这将是一个很难处理的选择。

### 最令人期待的是就是由武器中二选一

就如右边的说明一样， + r1 + 的武器和防具有不少只是在性能或是能力上表现得比较暧。由于要重视哪一边完全是由玩家决定。因此从这里可以看出玩家的游戏风格。俗话说 + r1 + 不可兼得。那么这个 + r1 + 的新系统到底会为游戏带来什么样的新乐趣呢？真令人期待。

### 角色不同，出现的能力也就不一样

先前已经提过，出现的能力和角色的特性有很大的关系。即使是相同的一件武器，如果拿给不一样的角色装备的话，那么出现的潜在能力就会有所不同。因此，在获得一件新武器后，最好能给8位角色都装备一下，以确定出现的能力种类。由于角色不同，出现的能力也就不一样，因此玩家可以配合自己的喜好，将武器平均分配在每一位角色身上，或者是只集中在少数的特定角色身上。从另一个角度来看，这也是一种订制角色的方式哦！

### 各式各样的订制

#### 各式装备会出现各种不同的能力

出现的能力基本上在解除装备之后就会消失。不过，通过长期地使用而修得某项能力的话，那么该项能力就会变成角色自身的能力。此时，就算解除装备，能力也不会消失。另外，在修得能力之后接下去还可能会出现另一种更强力的新能力。

#### 你是用完即弃派还是苦行修得派

由于潜在的能力最好要完全修得，如此才不会于解除装备后消失。因此便衍生出 + r1 + 和 + r1 + 的分别。如果玩家是依照不同的合来使用不同的能力，那么这就是属于所谓的 + r1 + ；如果玩家在修得能力前一直使用相同的道具的话，那么这就是属于所谓的 + r1 + 。没想到连这种小地方也可以看出玩家的游玩类型呢！

这里所做的各种说明并不是 + r1 + 战斗系统所有的新要素。游玩对象不同，所产生出来的玩法也就千变万化的这个潜在能力概念，其实只不过是整个战斗系统中的一小部分而已。有关于 + r1 + 战斗系统的全貌，我们将会持续地做搜查和追踪，因此请大家期待后续的最新报道。
