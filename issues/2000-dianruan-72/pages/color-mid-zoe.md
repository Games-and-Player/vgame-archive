## 共同制作

由 + r1 + 监督小岛秀夫所监制的PS2用原创游戏登场了。本作是描写少年里昂在战场上成长的故事，堪称是一款壮阔的SF大作。游戏中，主角搭乘的是一种统称为 + r1 + 的机器人。本作虽是机器人动作游戏，但是却也非常重视游戏的故事性，相当值得玩家期待。

## STORY

故事发生在22世纪的太阳系中，一个木星卫星轨道上的宇宙殖民地 + r1 + 。住在这个世界尽头的居民，被人们称之为 + r1 + 。主角里昂是一个住在安提利亚的内向少年。有一天他被欺负他的人强迫到联合宇宙军的设施偷盗军品。而不幸地狂信派军事政权ZOE也在同一天开始扶持殖民地的作战。安提利亚一时之间成为战场。欺负里昂的其他少年也死在他眼前。善良的里昂虽然心想： + r1 + 但是，他的内心深处却为自己救不了他们而自责，伤心地想要离开这地方……却不知道自己所在的方向，正是这次事件的导火线奥比达伏雷姆·杰夫帝矗立的地方。

### 里昂·史汀巴克 Leo Stenbuck

本作的主角，居住在安提利亚的少年。个性内向而胆小，经常被欺负。ZOE袭击安提利亚时，偶然发现奥比达伏雷姆 + r1 + ，之后乘坐杰夫帝和ZOE军战斗。14岁。

## 小岛·新川·冈村 开发者倾力制作的大作

KONAMI公司秘密开发的机器人动作游戏 + r1 + ，终于要露出真面目了！这款由小岛秀夫监督，率领所有智慧与力量的本主力游戏成员，集合是什么样的作品呢？我们这次将要为大家一举揭开它神秘的面纱！

### 杰夫帝

里昂所搭乘的奥比达伏雷姆。上面搭载了专为如何发挥机器的最大破敌效率而开发的人工智慧ADA。

→上面的图片是剧场艇。究竟是上面某装着的一个小房间。右边则是比较里面，在比较大的战斗舞台。那剧场艇在闪光中新航行时的场面。

### 拉普达

ZOE军的量产型奥比达伏雷姆。附图中的是标准装备型。此外还有象是装备大型匣捕等。其他种类的敌人吧！在游戏中位这算是小喽罗级的敌人吧！

### 何谓奥比达伏雷姆？

所谓的奥比达伏雷姆就是游戏中宇宙型机器人的一般名称。大部分都是以人型居多，但是也有象暴风号那样造型特殊的。

### 暴风号

ZOE军的奥比达伏雷姆和拉普达之流的比起来就特别巨大无比。以异形的外型合看起来就特别的人心生恐惧。以这它的外型实在会让人心生恐惧。中变身的变态登场。

→杰夫帝可以在空中自由飞翔，而且飞起来的速度与爽快感简直是爽歪歪的哦！

↑光线刀将敌人一刀两断。里昂在战争中会看到什么？又会失去什么？

→基本的战斗都是以杰夫帝单机对付复数的敌人。

###  + r1 + 的角色设定是西村诚芳

担任这次角色设定的是KCE日本的西村诚芳。如果你是一个动漫迷的话，听到这个名字应该会觉得似曾相识，因为他曾经担任 + r1 + 、 + r1 + 、 + r1 + 、 + r1 + 等作品的作画监督，并在 + r1 + 中同时兼任角色设定和作画监督。在日本动画界可是赫赫有名的人物。据说西村诚芳本人进入KCE日本是出于自愿，在这次作品中除了担任角色设定外，在剧情场面的图像构稿、演出面等等地方也都不遗余力。有了这位动画界的专家加入制作，相信本作的动画水准一定会大大提高才是。就让我们一起来期待西村先生笔下的角色们，在游戏中活跃的日子早日到来吧！

### 女主角与竞争对手

**塞尔薇丝** 本作的女主角。14岁的少女。在战斗中和里昂相识相知，和他一起劲敌。搭乘的奥比达伏雷姆叫 + r1 + 。22岁。

**克莱因** 隶属于ZOE军的女性飞官。是里昂的劲敌。被卷入战争当中。

**贝奥拉** 里昂的竞争对手。虽然起来就有意无意地嘲弄着里昂的奥比达伏雷姆的可怕。事实上，看起武器的可怕也恐惧多种。它的红色的机体外表让里昂觉得强力无比。

**混特** 比达伏雷姆的竞争对手贝奥拉拥有的奥比达伏雷姆。
