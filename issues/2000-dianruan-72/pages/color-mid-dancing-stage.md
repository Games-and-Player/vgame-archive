## 梦想在这舞动的舞台上定会成真

KONAMI制作的 + r1 + 系列，其最大卖点是其中所收录的歌曲都是知名歌手的作品。这次的 + r1 + 更使用了乐风独特且在日本乃至全亚洲都相当受欢迎的3人团体 + r1 + 所演唱的歌曲。由于每一首都是十分耳熟能详的曲子，相信大家一定会不知不觉随着旋律摆动身体哦！

### 想玩的类型也可以简单地自由选择

以往在 + r1 + 中选择难度、游戏形态时，玩家一定得输入指令或是在歌曲选择画面开启另一个视窗。但是，在这次这款连初学者也能轻松上手的游戏中，每当游戏开始前，便会先显示游戏类型选择画面供玩家选择。而且在此处当然也能够设定游戏的难易度及玩的人数，甚至是箭头移动的速度及箭头的形状也都可随个人喜好加以选择。让玩家可以随心所欲地跳吧！

↑箭头的形状有心形的，也有恶魔的蹄蹄等形状。每种形状都好可爱哦～～

自由选择吧

↓难度的设计共有4种，从适合初学者到高手玩的等级都有。首先还是从最基本的开始玩起吧！

## 以下介绍13首耳熟能详的收录曲

相信大家最关心的莫过于游戏中到底可以跳到哪些曲子吧？这点小事玩家大可放一百个心，因为我们已经为您做了一番详细的调查。游戏中所收录的正是以下列出的13首，而且从慢板的曲调到超快节奏的歌曲都有，相当齐全哦！连最新曲 + r1 + 也包含在其中呢……真是太有够给他厉害呀！

### SNOW DANCE

↑中板速度，跳起来轻轻松松。是首不会过于剧烈的歌曲。由于并不会很困难，初学者可以试试看哦！

### RING! RING! RING!

↑同时踏的箭头随时会出其不意地登场，实在是快得不得了。不过只要跳过之后便会乐在其中，不可思议。

### 明天依然会来

↑去年年末发售的最新单曲。虽说旋律听起来有些悲愁，实际上可是首快节奏的歌曲呢！记得要配合着跌声跳。

### GO ON BABY

↑述说女性心情的歌曲。虽然并不是首困难度高的乐曲，但是偶尔会有几个漏发的音符跳出，可别掉以轻心！

### 如果天晴

 + r1 + 是首让人感到轻松愉快的曲调，与其喧闹地蹦蹦跳跳地跳，还不如悠闲优雅地轻轻跳着。

↑和着旋律轻轻跳跃，途中出现的 + r1 + 声是关键所在。一瞬间决定好要摆的姿势，好好表现一下。

### 歌词也会同时显示

游戏在进行时，除了箭头会流动外，歌词也会配合乐曲显示在画面上。其他的人还可以看着歌词边唱歌，这种设计很不错吧！

### PEACE!

注意这里

### It's so DELICIOUS

↑小心不要被一开始强而有力的合唱部分给绊住哦！只要跳过此处，其他地方就没有如此困难的箭头配置了。

### LOVE TIDE

↑由于各种8分音符与16分音符散布在各处，因此难度相当高！如果能够通过这项考验，你就是上级程度哦！

### 青兴！欢喜！最喜欢！

↑充满吉田美和深具魅力活泼嗓音的歌曲，也许不规则的旋律有些难掌握，不过只要听出鼓声的节奏就没问题。

### 为什么恋爱

↑以第3张单曲发表，堪称初期的名曲。如同曲名一般，也许快快乐乐地跳舞是致胜的关键。难易度适中。

### 2月14日发售畅销精选集值得购买

 + r1 + 于2月14日发售的畅销精选集《GREATEST HITS（THE SOUL）》。这张专辑应该是先买起来放着比较好吧。噢，为什么？嗯，详细情形就等玩家自己去买来看看吧。我们在这只能稍稍透露一点点，就是只要拥有这张精选辑，在游戏中就会有好事发生。总之别想太多，赶快选去买这张精选集就对了！

### opening theme ～the monster is coming～

↑游戏中唯一没有歌声的曲子。虽然如此，也不可因此轻松哦！因为这首是本游戏中号称难度最高的曲子。

### 决战星期五

↑慢板的抒情歌，因为箭头流动的速度缓慢，非常适合初学者。参与庞大的合唱中，一边不停跳舞。

↑难易度适中的乐曲。答答答～、答答答～的旋律很多，整首歌易于变得 + r1 + ，导致抢拍。要注意。
