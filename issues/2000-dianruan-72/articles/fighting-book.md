# 格斗天书

## 99告白天下（二）

KOF系列无限连击实为不少，留给玩家印象最深的要属草立。但前几作的无限连较之99可谓相形见绌。（吉林省辽源市矿物局一中二·一班 祖强，136200/ 天语：作者在江湖上人称 + r1 + 。欢迎小有名气的 + r1 + 们投稿！）

99的无限连大体可分为六类：

一、于CM后产生：如K'的CM后拔击＞第二击打到角落空中的对手后可无限连；KING的CM后角落怪异玫瑰等。

二、AC后产生：K'、包、良、蔡利、邪魔等AM后，闪避攻击打到角落空中对手可无限连；而WHIP在AM后只需用闪避攻击打到角落对手即可。（其实这种连击不只限以上几人，凡闪避攻击出招快、收招快的都可一试，有待玩家自己考证）

三、特殊无限连击：说难听点就是无赖技。如拳崇穿号腿打到角落空中对手可无限连（但要有节奏才行）；全勤空中下B击到空中对手可无限连（必须击中对手头部且注意节奏，否则只能击三四下）；

四、达人级无限连（非达人级水准禁练）：

金家蕃①近C＞弱雾气脚＞CANCEL←→B/D……此技对输入速度要求极高。②空中�的飞翔蹬打到角落空中对手后可无限连。此技虽属无限连，但极不实用且难度也极高。

五、STRIKE无限连击：任何打击浮空技＞接驳香蕉提取＞打击浮空技……（玩家可自行研创无限连击）

六、浮空后的无限连击：请参看上期。

无限连击往往是扭转局势，反败为胜的法宝。玩家应努力才会出成果。至于今对手浮空方面，玩家可自选援助必会浮空。

### 99无赖技浅谈

●包：已方版边CM后，↓↙←A＞SC ↓↙←×2A/C。
此招使出后，对手有两个选择，一是原地防御，直到CM结束，这时血已费半；二是摇杆跳跃，这时已方变招为SC ↓↙←×2B/D，保证令对手牵到极点，其中有些对手用飞行道具抵消已方↓↙←A想费一点血，那是妄想。已方可直接出↓↙←×2A/C，吃掉其飞行道具而攻其人。这样保证令对手老老实实地防御，不敢动一步。

●蔡宝健：普通投（C键）不可解拆，→A打破防御后可接超杀。跳跃C键打点低判定范围且常为迎向，破纵小，第三段不可防御。常用这三招，蔡宝健实力当上倍增！另附一套 + r1 + 人连续技。

### 99必修攻略

我是一名高二学生，希望大家读了我的研究后能吃一惊（黑龙江省木兰县高级中学二年一班麦吉，151900。）

■99必要攻略

一、K'的CM下用拔击＞第二层击中空中对手，然后利用第二层令对手浮空N次，形成无限。这一成败关键在于第一击拔击必须用C键发出，否则连不上。

二、全勤，利用香蕉使对手在角落浮空＞空中↓BN次，形成14HIT将对手打死。此技关键在于①空中↓B第一击须打对手头部（目测要准）②拉住↓，节惯性投B才可。

三、以前贵刊曾经用香蕉使对手N次浮空的双人合作技，如：K'、KYO1、KYO作主力。关键在于对手浮空后，你走到对手身后再吟蕉：①叫暂时，摇杆回中；②让香蕉对方飞过。

四、拔略老头有一技：C＞→A＞龙虎乱舞（龙也用A让出才可）

五、拉尔夫出场，KING援护：近版边呀KING＞用空中飞翔蹬踢击中对手＞KIM先落地用（→↓↙←→↙B）N次或几次后再用（←↓↙←→×2B）KO对手！

六、KIM出场，KING援护：近版边呀KING＞KIM用空中飞翔蹬踢击中对手＞KIM先落地用（→↓↙←→↙B）N次或几次后再用（←↓↙←→×2B）KO对手！

### 99的简化更深入

一、KIM近向飞翔脚＞B、B（站立）＞↓B＞→→B（不用蕉，即为凤凰脚）

二、拉尔夫：→↘↓↙←C＞→↓↙←→B/D（第一式为空招，将乘马机爆拳分解）＞若用良援护的话，呼良＞MAX乘马击爆拳。

三、 + r1 + 的IORI：隐凤＞→A（一段）＞八穿女，若经常发出，VERY HARD。须简化如下：隐凤后＞（按住C）＞八穿女前半程指令＋A（假手后，松开C）＞八穿女后半程指令＋C，TOO EASY！注：第一加续处须按住C，否则在第二加续处八神会发波的。

四、MARY：用→↘↓→D将对手打至角落＞→看→B（对手倒地）＞上前（→↘↓↙←→×2B/D）。

以上所有技巧，100%发出。愿与天下玩友，共同切磋。

近立C＞→B＞远立A＞→A＞CANCEL凤凰脚。请玩家自想此技为什么能连上。

## KOF全系列怪异处（一）

94已来，苦练连技便成了本人每年的必修课。而当99玩完后，忽发奇想，希望让大家都知道一下在正统的游戏方式外出现的种种 + r1 + 玩法，给玩家们一点菜料到也是新鲜。本人还将对浮空间隙性。（陕西泾阳县八四九零九部队教导队 王启，713700）

### 1. 普通投后接续超杀（94）

使用者为麻宫和镇元斋。使用方法为在二人的普通投（←/→C/D，注意，镇仅限D键）后立即使出二人的超杀 + r1 + （→↘↓↙←→BC）或 + r1 + （←↓↙←→C）成功的话，对手会在被投出程度上再加上中了超杀的损失。由于那时的超杀威力颇大，改此技一次消耗体力50%以上（最少），极为恐怖。

### 2. 抢先攻击（95）

当八神VS镇元斋在特殊的场地时专用。当刚一开局时，二人会先从桥下跳下，由于镇的体重较轻， + r1 + 小，所以会在空中停留较长的一段时间。而对于先落地的八神来说，此时便可立即使用 + r1 + （↓↙←A/C）来打击仍在空中的尚无法防御的镇，而镇也就白白地挨了打。

### 3. 轻腿＝重腿（96）

在二阶登红丸对投技高手克拉克的用。当红丸将克拉克击倒后，迅速靠近，在克拉克尚未起身时使用下蹲轻腿，克拉克会在倒地的状态下再次被击倒。如此反复，克拉克直到死也无法还手。

底技（不要来打击对手，只要向时按BorD不放）＞待对手已浮空＞立即松开B/D（此时拉尔夫踢已完成）＞MAX乘马机爆拳

六、KIM出场，KING援护：近版边呼KING＞KIM用空中飞翔蹬踢击中对手＞KIM先落地用（←↓↙←→↙B）N次或几次后用（←↓↙←→×2B）KO对手！

## 华山小论剑

自《街霸》起，笔者在学校附近就难遇敌手。因此高手寂寞之感油然而生。长大后，去的地方多了，见识过各路人物后，方知天外有天，人外有人。回想夜郎自大之时，真是可笑可叹。笑的是天下之大，如我一般的井底之蛙又不知有多少？叹的是街机不像PC GAME那样能联网，无法使天下英雄来一华山大论剑，真乃一大憾事。因此突发奇想，欲从理论上与大伙过过招，以解华山论剑之渴。（湖南衡阳31号信箱退休办转王苏霖，421003）
