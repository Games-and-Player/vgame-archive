# CLAMP最新作品《天使阶层》

> 彩页插16-17（PDF 58, 60）。CLAMP漫画《ANGELIC LAYER 天使阶层》彩色插画两页。

## PDF p.58 / 彩插 插16

【图】整页彩色插画。红色背景上巨大的“ANGELIC LAYER”黑色与红色标题字样纵贯画面。右上角为一个天使人偶（Angel）角色，白色头盔、黄色发光眼睛，身上缠绕着紫色电缆与红色线缆。画面中散落着白色羽毛，左下方有一个黄色锥形装置（天使蛋）。整体画面充满科幻与幻想气息。

CLAMP最新作品《天使阶层》

## PDF p.60 / 彩插 插17

【图】整页彩色插画。画面主角为粉红色短发的少女角色，身穿红白配色的战斗服装，佩戴红色头饰与黄色护甲，白色手套和白色长靴。身后有红色的天使人偶，缠绕着紫色与棕色电缆，白色羽毛飘散。黄色锥形装置（天使蛋）上写有“HIKARU”字样。红色背景上隐约可见英文水印文字。

《电子游戏软件》杂志社　1999年3月
