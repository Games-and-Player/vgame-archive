# 大墙画廊 彩版 VOL.26

> 彩插（PDF p.56-57 / 插14-15）。读者投稿彩色绘画作品展示，SHADOW与PHOENIX点评。本期主题「不相见不如不怀念」。

## 不相见不如不怀念

“烟火”后的天空已经被温柔的春风吹拂，但愿这被人为及非人为因素所造成，被各种欲望横流所笼罩，被不再清澈的传说所浸漫的大都市能早日回复往日湛蓝的天空，明媚的艳阳，和谐的气氛以及人与人之间不变的守望的时候……

S：一年一度的春节又莫名其妙地在一片平静中逝去了！许久没有闻到那从小就令我嗅觉“冲动”的烟火味道，许久未有在被烟花照亮如同白昼的除夕夜的钟声响[?]后感受万众欢腾。整个京城为之倾倒的“灿烂”回忆；许久的许久了，但也许真的这样可以减少这个城市的“负担”。可我还是宁愿……愿有一天能够与自己心爱的人到自己憧憬的海滩去将烟花……放逐在那漫长的海岸线上，去将梦想随升腾的希望洒落在万里长天……

P：似乎变了身的S·P总爱说些与画廊无关的话，是啊！

太多的东西恐怕也不是件好事，打开柜子，整柜的画稿已数次令我产生一种近似“负罪感”的情结，太多的朋友寄望于此，而我又一股脑“承担”下来，但毕竟Page有限啊，被雪藏的，被埋没的，被遗忘的，被……抱歉了，相信我吧！

画嘛，大家能从稿子上一看清，但我们之间的交流呢？！听着《Hawk Never Fly》，莫名被其中那隐约散发的气氛所感染，所打动，尽管我这个Phoenix也是无法Fly的，但…承受了

相见的感觉也许也是诗 / 不相见的感觉也许也是诗
怀念的道也许也是风景 / 不怀念的道[?]也许也是风景
相见不如怀念 / 任待随风
不相见不如不怀念 / 今风从诗

——廊主SHADOW & PHOENIX

※注：凡图名后加“(计)”均为计算机绘稿。
