在游戏过程中暂停后，画面上将出现 LEVEL 水平、Hi 最高分数、Sc 现在分数，还有十二种宝物的图。宝物不是一开始就有，要不断获得，一关内有一至两个。

在游戏过程中，每打死一个敌人会飞出三件宝物，其中有：加经验值的宝物、加体力宝物、加分宝物、加魔法值宝物、加命宝物、减血酷髻[?]等。每种宝物有大小之别，所加数值不等，其中最重要的是加经验值和加魔法值的宝物。

每当游戏中主人公全死光了后会出现两个菜单：CONTINUE，选此项为继续；END，选此项为结束。

### 2. 要熟练掌握主人公的各种腿功用法

主人公在游戏中武功不断增长，腿功越来越高，经验值每够 1000 点水平长一级，招数就多一个，共有以下几种招术：

| 名称 | 出招方法 | 水平段 |
|---|---|---|
| 高腿 | →B | 0 |
| 弹簧腿 | ↓B | 0 |
| 扫堂腿 | ←B | 0 |
| 铁腿 | ↑AB | 1 |
| 铲地腿 | →↓B | 2 |
| 二踢脚 | ↑→B | 3 |
| 连环脚 | →B | 4 |
| 飞腿 | →AB | 5 |
| 旋风腿 | ↓→B | 6 |
| 霹雳腿 | ←↑B | 7 |

### 3. 尽量多吃长经验值和长魔法值的宝物

这样主人公不仅可以使出多种腿功，而且由于体力和魔法值也会随之增长，便可以使用更多魔法。

### 4. 需要熟练掌握各项魔法的性能和用法

最好在不同情况下用最适宜的魔法。

魔法使用方法为：在暂停时用 A 键选定宝物，然后用选择键使用。

## 第一版　魔幻森林（PDF p.19 / 刊页 17）

殊死的战斗在王国外的森林中展开了，原本美丽的大森林已被魔王的军队占领，所诺兰首先遇到了骷髅兵，骷髅兵会用手中的剑劈刺，如果硬拼会吃亏的，要利用那些高台，把骷髅兵引过来，然后站在下面用高踢法将其毙命。还有许多会飞的骷髅，他们会从上面或下面垂直飞向所诺兰，不要紧，只要站在原地就可以把他们全打死。消灭这些怪物后向前走踢开一个宝箱，所诺兰拿到了第一个宝物**意念弹珠**。（图 a）

> **意念弹珠**：可以反弹打击敌人，消费 3 个 MP。

继续向前走，出现了许多蜥蜴，怎么这么多呀！不要紧，这些蜥蜴根本不会攻击所诺兰，只要到它前面踢死它就可以了。除了这些蜥蜴精以外，还有一些小精灵，它们蹦蹦跳跳地向所诺兰进攻，踢死它们向前走，一个会飞的女巫出现了，她还放出两个会满地滚的小鬼作帮凶，所诺兰打死两个，她又放出两个，真讨厌！不理小鬼了，还是直接向女巫进攻吧，所诺兰打定主意，立即跳起用弹簧腿攻击女巫，女巫一会儿便消灭了，所诺兰得到了第二件宝物**小还魂魔壶**。

麦克瑞和所诺兰一起向魔王驻地出发了。所诺兰对哥哥说："有一堆泡沫中有一具活骷髅，我们要先去找。"

突然，一名手执利刃的骷髅兵出现在他俩面前，"好好想想，你是伯里兹的对手吗？你连我都打不过！"骷髅兵冷笑着对所诺兰说，"让我来会会你！"麦克瑞冲了上去。

唰！骷髅兵的利剑无情地砍在麦克瑞身上，骷髅兵便得意地跑了。

所诺兰正要追上去，躺在地上的麦克瑞喊了一声："所诺兰，等等！"麦克瑞喘息着说："我的兄弟，我的利剑是不管用了，只有用你的腿功来对付这些走狗了。"

"麦克瑞，不要死！"

"救公主，你是最后的希望了……"

麦克瑞死了，远处又冲上来许多骷髅兵，所诺兰怀着满腔仇恨与愤怒，向骷髅兵冲了过去……

## 第六版　穿越沼泽（PDF p.22 / 刊页 20）

所诺兰来到一片沼泽前，巢丛中隐藏着毒蛇怪，要去魔巢必须通过这里，无论多么艰难都要继续努力，所诺兰毫不犹豫地进入沼泽，这时空中出现了许多绿色飞死神，他们忽隐忽现地向所诺兰放毒叶镖，远处又来了许多长着鸡头、鸡腿、鸡翅、蛇身的鸡头蛇身怪，这些家伙一跳一跳的样子笨得很，所诺兰把他们放在眼里，不一会就把它们消灭了。突然从沼泽中钻出了几具僵尸，僵尸伸直双臂向所诺兰走来，别看他们动作僵硬，可真难缠！他们会突然向所诺兰发射子弹，还挺不容易躲，所诺兰用铲地腿和铁腿消灭他们。来到一座石桥上，前面有一个宝物，可是石桥用大石头封住了，用脚踢踢不开，怎么办呢？他忽然想起了那件在古堡中拿到的不知何用的宝物，不妨试一试，看它能不能起作用。

## 终章·决战与结局（PDF p.24 / 刊页 22）

断变成各种模样，所诺兰经验丰富，这些怪物都是他的手下败将呀！四名灰甲骑士都被所诺兰消灭了。魔王终于坐不住了，他狂叫着要杀死所诺兰，所诺兰知道魔王非比等闲之辈，也不敢大意，小心迎战。魔王动作奇快，忽上忽下，一落地便用魔法向所诺兰射出一道长长的闪电，所诺兰丝毫不到他，所诺兰真急死！怎么办呢？对，也用魔法对付他，所诺兰使出了浑身解数，神奇闪电，振荡波，等等，效果都不好，魔王也没受到什么损伤，所诺兰使出地震术，把魔王震得晕头转向，动弹不得，抓住时机上去几脚，结果了魔王的性命。

- **p.16 招式表"连环脚 →B 水平段 4"**：与"高腿 →B 水平段 0"出招方法相同。原刊确实如此重复，**疑为印刷错误**——连环脚出招应为不同组合（如"→→B"或类似）。已在正文按原刊保留并在此处说明。
- **p.16**：原刊"水平值"作为屏幕下方数据之一，与正文"经验值每够 1000 点水平长一级"中的"水平"为同一概念（即等级 / Level）。
- **p.16**："魔法使用方法为：在暂停时用 A 键选定宝物，然后用选择键使用。"——与前文"按 A 键……开始键和选择键基本不使用"略有矛盾。本游戏的"宝物 / 魔法"使用确实通过暂停菜单 + 选择键执行，**矛盾仅是描述性的不严谨**，不改。
- **p.17**：原刊"麦克瑞"——主角之兄名。文中亦直接用"哥哥"代称。
- **p.17**：开篇"魔幻森林"段落有版面交错——左中两栏先讲麦克瑞遇骷髅兵临死托付，右栏紧接讲所诺兰击杀女巫得宝壶。本文按**先正文叙事 → 击败女巫 → 麦克瑞之死前置 → 仇恨进入 →第二版**还原；如果原刊另有阅读次序，可能需要再调整。已尽量按视觉版面（"●版面介绍"小标题在左中，剧情对白在中下，宝物图说在右栏）拼接。
- **p.17 / p.18**：原刊"绝境洞府"末句"水潭上有许多台子，这"——直接断到 p.18 起首"里有许多吸血蝙蝠"。本文按跨页连贯阅读保留。
- **p.18**：原刊"所诺兰用利他飞来但还没到地面时"——疑为"所诺兰**用弹簧腿**等他飞来但还没到地面时"，"利他"应为"等他"或"待他"。已在正文保留原貌。
- **p.18**：原刊"所诺兰丝毫不到他"——p.24 决战段，疑为"所诺兰丝毫**伤**不到他"或"丝毫**奈何**不到他"。已在正文保留。
- **p.18**：原刊"上飞下飞"——指魔王上下飞行，保留。
- **p.18**：第五个宝物原刊未给名称（"??这到底是什么呢？"）。这件宝物在 **p.20 第六版沼泽石桥处** 才被使用——即"那件在古堡中拿到的不知何用的宝物"碎开石头，得到第十件宝物**防光圈**。所以第五件宝物的实际功能是"碎石"。原刊故意悬念处理，不直接命名，**保留原貌**。
- **p.19**：版面图注右上"五大船遇到一只绿色鱼鹰怪"，"五大船"疑为"**在大船**"或"**坐大船**"形近误植；本文按"**在大船**"还原。
- **p.19**：原刊"消费 3̲0̲ 个 MP"（振荡波）字迹模糊，按"会发出不断扩大的电波"语义判断，应为 30 MP 量级。
- **p.19**：原刊"道路的断口"应为"**断处**"或"**断口处**"——表达不流畅但不影响理解。
- **p.19**：第七件宝物（大还魂魔壶）与第八件宝物（振荡波）的获得段落在原刊中部分**重复**——版面右栏与版面下栏均提及"长铁链女妖 + 幻影功 + 魔法闪电消灭 + 取宝物"片段。疑为编辑排版时将同一段两处插入。已合并叙述并在编辑备注此处说明，便于校对核对扫描件。

- **p.22**：原刊"亲和恩爱"——疑为"**相亲**相爱"或"**恩爱**"二字之缺漏。
