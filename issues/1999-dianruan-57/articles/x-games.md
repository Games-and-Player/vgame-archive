# X-Games 职业滑雪者

> X-Games Pro Boarder / PS / 厂商：EA SQUARE / 类型：SPG / 发售日：3月11日 / PDF p.113, 112（对调）/ 彩插

<!-- PDF p.113（介绍页）-->

特别收录了“极限运动”的倡导影片的正宗滑雪运动游戏，即将在PS主机上登场了。

## 可以自由奔驰的滑雪运动游戏

这款“极限运动～职业滑雪者”是由世界顶尖滑雪好手们出场比赛的滑雪运动游戏。由3D造型所呈现出来的立体跑道，玩家将可以在斜坡上的所有地方进行滑雪哦！另外，其BGM也有收录了富法达司与蓝西多等知名乐团的最新原声音乐哟！

**所有的场所都可以滑行**

**没有所谓的禁止进入**

## 登场的8名世界滑雪好手！

玩家们所能选择的角色，是8名拥有最高超技术的顶尖世界级滑雪运动好手。在游戏中，玩家将可成为他们的分身之一。再度展现他们风靡世界滑雪场地的不凡身手。当然，在游戏当中也可以制作出属于玩家自己的原创型特技。而且，可以练习的各种超高难度的大小技巧至少有两千种以上呢！

**比特·莱恩** ——因为讨厌别人向自己挑战，所以并不会在一般的比赛出场。但是，他在电影等中所看到的大型蝶旋滑行比赛上，具有相当的知名度。

**迪娜·巴希吉** ——在属于女性的滑雪选手中，是少数具有世界级知名度的选手。其最擅长的技巧是在争夺速滑之后，飞向空中的大跳跃。

**托德·理查斯** ——是一个擅长圆坡道的滑雪选手。采用正规的站立姿势。除了滑雪运动以外，平时他的兴趣是在玩滑板与电子游戏。

**摩根·拉丰第** ——拥有K2、Daikine、Mother Nature等赞助厂商的女性滑雪选手。她的擅长项目是需要特技与跳跃技术的花式滑雪比赛。

**杰米·林** ——虽然一般性的比赛几乎不会出赛。但他可是一位广受世界瞩目的超级巨星。同时也有参与了滑雪板的制图与服装的设计。

**夏农·唐** ——从她可爱的笑容当中，无法想象她是一个攻击型、充满魄力的滑雪选手。与迪娜一样是少数的世界知名女性滑雪选手。

**泰瑞·哈根森** ——以17岁的年纪就夺下世界冠军的第一滑雪高手。在半圆坡道的技术上，被认为是已领先其他顶尖滑雪好手数年以上。

**塔尼尔·弗兰克** ——最擅长的特技是半圆坡道与垂直跳跃时的花式技巧。在其出身地的国家——挪威，是继泰瑞之后，又一知名的滑雪选手。

## 操作方法有两种选择

游戏的初期设定虽然是设定在“业余设定”，但是在开始画面的选项中，也可以切换到能够进行复杂特技表演的“职业设定”。这两个操作都是以按下X钮来准备使用特技→放开X钮后跳跃→跳跃中以十字钮加□或△钮，就能够使用特技。

**业余设定** ——以简单的操作就能够使出特技。首先要习惯业余设定的操作喔！

**职业设定** ——会显示出十字形的威力参数，而且还可以调节特技的回转速度等等。

## 能够欣赏特技影片！

在游戏中能够欣赏到在运动影片中具有影响力的“极限运动”的推广录影带影片。让玩家也能亲眼目睹由滑雪选手们所表现出的华丽技巧与精彩刺激的动作镜头。所谓“百闻不如一见”，那些在静止画面或是说明文中无法解释清楚的特技，只要看过影片就可以一目了然。

## 9种充满个性的舞台

在本作中，总共收录了9种不同的舞台。玩家们可以在这些充满富有变化、且极具个性的各个舞台中，尽情享受在滑雪时的各种乐趣。游戏一开始时，能够选择的只有5个舞台，当玩家达到某种条件的时候，便能够选择新增加的4个舞台。这次，就先来介绍一下刚开始时的5个舞台吧！

**午夜竞速比赛** ——以最快速度到达终点的比赛舞台。在跑道上所放置的旗子只是表示最简单的行进路线，并不见得是最短的路线喔！

**半圆坡道** ——半圆坡道（半圆状的路线）的舞台。利用速度，以左右的墙壁进行跳跃，一边使用特技，一边朝终点前进。在限制时间之内无法到达终点的话，就算出局了。

**体育场** ——在四分之一圆坡道（只有单面墙壁的半圆坡道）上，比赛技巧分数高低的一关。但不是以特技的合计分数决胜，而是以一次滑下的最高分数来决胜负。

**深山跳跃竞赛** ——放置了大型跳台与障碍物，地形起伏不定的一关。以在这两个地方的特技合计分数比赛。在限制时间之内，争取第一名吧！

**斜面型** ——利用设置在跑道上的铁路或是圆木、野餐桌等等障碍物来使用特技，朝着终点前进的一关。在比赛结束之后，如果分数是第一名的话，就算优胜了。
