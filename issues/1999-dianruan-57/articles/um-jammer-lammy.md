# 拉米乐队

> 拉米乐队 / PS / 厂商：SCEI / 类型：ACT / 发售日：3月18日 / PDF p.110-111 / 彩插

<!-- PDF p.110 -->

喂喂喂！大家眼睛都照过来！在这里，要介绍给您的游戏就是“啪啦啪”的第2部作品“拉米乐队”！相信这款游戏又会让您惊喜不已！废话不多说，看介绍吧！

## 可爱的拉米姑娘要挑战摇滚乐曲了！

配合着音乐节拍来按按钮，也是节奏动作游戏始祖的“啪啦啪”第2弹终于登场了！在这次的游戏中，将和前作乐曲以Hiphop为主有所不同，而变成以挑战摇滚乐曲为主哦！不过，依然有着和前作一样可爱的角色，还有相同的乐趣。最重要地就是这次还能够两个人一起玩！好令人期待喔！

←和前作一样，您只要跟随着画面所显示的节奏，掌握好节拍按钮就行了。然后再加入热情澎动的即兴表演，试着演奏出动听的旋律来。因此您的节奏感很重要。

【图】看到这个画面，相信您一定十分惊喜！竟然……竟然会有2个人在画面中玩！您没有看错！跟一个人玩相比，两个人一起玩，绝对是更加有趣。

**拉米**——她是住在某个小镇的小羊，平时是一位温柔又腼腆的娇娇女。

**凯蒂**——她是个喜欢将自己打扮得漂漂亮亮的猫，对流行音乐很有兴趣。

**玛儿小姐**——她是个喜欢算命占卜的女孩子。我行我素地行动，让人不敢领教。

↑平常虽然胆缩细小，不过一旦摸到吉他，可会180度大转变，变成炫颈热情的吉他手喔！

**ラミーにおまかせ！**（交给拉米吧！）

↑3人并排一起成为少女乐团，名叫“MilkCan”！热情畅快地摇滚起来！

<!-- PDF p.111 -->

## 热热闹闹快乐地一起随着乐曲来玩吧！

在游戏中担任主角的拉米是吉他手，而玛儿是鼓手，凯蒂则是少女乐团“MilkCan”的主唱兼贝斯手。在这页中，将为您详细介绍她们的活动。她们将会遇到各种不同的人，还会有许多的学习课程一再来到拉米的面前，有时还会遇到某些蛮眼熟的人！……嘿！那个人不就是……洋葱老师吗！？

**洋葱老师登场！**

←这不就是在前作“啪啦啪”中非常吃苦的洋葱老师吗，台词也要得很「Ba」十分抢眼吧！

【图】MilkCan乐队演出场景

←洋葱老师真魄力十足！

这里这里，就在这里！洋葱老师真迫力十足，会指打拉米浅样去感受的摇滚乐曲

**还是老样子，迫力十足！**

←老师在传授拉米姑娘吉他手的心法，以及正经八百地回答老师问题的拉米。……没想到这只是梦！？

**究竟哪边是梦哪边是现实世界？**

## 可以两个人玩了！？

在前页已经有说明过，在这次“拉米乐队”中，可以两个人一起拼个高下，似乎会更好玩。若能够让几个死党好友、情侣，甚至全家出动来玩，更是令人感激不已！

**多么令人欢喜的新要素**

←消防队员的达尔梅西安，好象也会教给拉米吉他的精髓喔。只是，拉米手上怎么拿着灭火水管！？

↑在飞机场也传出摇滚乐！和滑稽的老伯伯在驾驶舱中一起跳起来，角色的台词也很炫吧。

**从现在开始好好期待吧！**
