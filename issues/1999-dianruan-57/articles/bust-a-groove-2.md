# 纵情跳跃2

> 纵情跳跃2 / 舞蹈天国混合版 / PS / 厂商：ENIX / 类型：ACT / 发售日：4月15日 / PDF p.107-109 / 彩插

【图】游戏LOGO：Bust-a-Groove 2

开创新型态对战游戏的“纵情跳跃”，让人期待的续篇在增加新角色、系统、舞台后，即将登场！

## 再次引爆对节奏&舞蹈的狂热、兴奋！

由玩家操控各种不同类型的舞者，以舞蹈来进行对战的对战动作游戏“纵情跳跃”，其续篇即将登场。本款游戏是让玩家配合音乐、输入指令，操纵画面上的舞者角色，展露又帅又酷的舞姿。只要比对手正确地输入指令，就能展现美妙的舞姿，也就更能获得胜利。这次除了负责音乐的成员有所变更外，本作的制作小组完全与前作相同。在舞台、音乐、舞步动作……等全都焕然一新了，请您好好期待这款全新的舞蹈对战游戏吧！

## 还有许多新角色的登场喔！

在这一次的续作中，加入了许多新舞蹈类型的角色。例如：直排轮舞，跳日本传统舞蹈的角色。在此，我们就要为您介绍这回的新登场角色——蔻梅特。脚上穿着直排轮鞋的她，在狭窄的场地来回滑动、舞蹈，真是帅呆了！而除了蔻梅特之外，还有许多新角色也将陆续登场；而曾经在前作中出现过的角色们是否会再次登场呢？舞出日本传统舞蹈的又会是哪个角色呢？

**蔻梅特 COMET**

踏着直排轮鞋滑出优雅的舞步，吸引众人的目光，这就是这次新登场的角色之一，蔻梅特。她是曾在前作登场的佛莉达的妹妹。目前是以寿司吧“寿司行星”为主，四处打工。登年16岁。

## 各个角色都准备有专属的舞步指令！

由于这一次的“纵情跳跃2”依据前作的游戏系统做了数点的改善；因此玩家在“纵情跳跃2”中将更能充分地享受其令人赞叹的舞蹈画面。其中，最大的要素就是原创舞蹈指令的存在。

在前作中，所有的角色都是以相同的指令展现舞步，在本作却是配合各个角色的舞蹈动作，分别设定了极具个人特色的舞步指令。所以玩家与画面中角色那种融合成一体的感觉将更加提升。另外，每个角色的舞蹈个性更加被突显出来！

## 以输入指令时机决胜负的新系统

这一次，本作采用了随着按下按钮的时间不同，各个角色的舞步就会变化、产生分歧的全新系统。

这也就是说，玩家若配合着音乐的4个节拍准确地输入指令，和输入指令的时间稍微有点差异的话，下一个出现的动作就会有所变化。而这一次刊载的虽然是尚未公开的开发中画面；不过听说在完成版时，一起和舞步指令显示的计时表会同时显示出来。真是太完美了。

## 决定舞蹈让舞台有丰富地变化！

在本作中，每个角色都安排有专属自己的对战舞台。如果，每个角色在属于自己的舞台中跳出华丽绝伦的舞步的话，将会出现比前作更多样的变化。其中会有像是漂浮到宇宙空间，大佛像飞出来、寿司都出来一同跳舞，安排有隐藏特殊效果的舞台喔！您一定不能错过！
