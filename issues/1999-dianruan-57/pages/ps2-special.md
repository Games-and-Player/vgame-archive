# 最新特报·直击PS2画面

> 最新特报 / PDF p.114 / 彩插

这里的图片都是3月2日SCEI在关于PS2的新闻发布会上公布的。既有静止的图片，也有小段的CG演示。下面就请大家欣赏一下PS2的效果。

【图】多幅PS2技术演示画面：
- 几何图形渲染演示
- 粒子效果演示（无数由中心向外射出的针状物，形成一个小绒球，这些可都是用POLYGON描绘的。）
- 花朵渲染演示
- FACIAL ANIMATION REAL TIME DEMONSTRATION（面部动画实时演示）

←这是「FF9」[?]CG电影片段的人物，此图段由即时演算生成，而面部表皮的刻划极为细致。据称是「FF9」中的人物，此图由即时演算生成，对面部皮肤的刻划极为细致。

**全部画面和CG均由即时演算生成**

←这是大家熟悉的「FF8」学院舞会中的CG片段，在这里是由PS2即时演算完成的。

←左图为「GT赛车」的计[?]划面，预定在99年夏天推出的「GT赛车2」不知是否会出现在PS2上。

←右图为「R4赛车」中的女主角——永濑丽子。
