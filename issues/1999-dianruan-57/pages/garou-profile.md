# 饿狼风云录——安迪·伯格

> 饿狼风云录 / PDF p.65 / 角色档案 / 安迪·伯格（Andy Bogard）

## 好吧，来呀！

### 必杀技

**激飞翔拳** — 从手中突发出巨大气状冲击波，对敌手起牵制作用。

**飞翔拳** — 发出“气”的飞行道具，比起“激飞翔拳”是会“飞”的。

**斩影拳** — 有残像，用时攻击的突进系招式。

**残影** — 从空中而来的残像突降的招式。

**空破弹** — 在空中以放射线状突进的招式。

**超·裂破弹** — 回转上升的对空技。

**男打弹** — 「男破弹」的豪华版。

“RB2”中的新技，颇具朴素味道。

### 衣装变迁

**饿狼1** — 负面感觉，令之更像忍者。

**饿狼2** — 与“饿狼2”一样，2P颜色会有变化。

**基本装备** — 腹部好像很丰满的样子，样子有些俊。

**白与红的基本装束** — 不一样的感觉。

**黑色夏衣** — 与《97KOF》中KOF的EX2P颜色一样。“黄金版”？

**腰上的赘肉消失了！**

**2P的绿色很像盗版？！**

**KOF'94** — 后悔得全身都有所表现。

**KOF'94 姿势** — 手不断舞动的胜利姿势。

**KOF'96** — 继承96装束，热血的赤红颜色。

**KOF'96 姿势** — 全身给人一种强悍感觉，特别是腿。
