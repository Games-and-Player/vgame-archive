# 灌篮高手角色卡片

> 彩色插页 / PDF p.58-59 / 《灌篮高手》（SLAM DUNK）角色卡片集锦

## PDF p.58

整页为《灌篮高手》角色卡片散落拼贴的彩色画面，各卡片上绘有不同球队的球员角色及其英文名、位置标注。

可辨识的角色卡片包括：

- **流川枫**（KAEDE RUKAWA） — 身穿蓝色背景的湘北队球衣，持球投篮姿态
- **小暮公延**（KIMINOBU KOGURE） — 身穿红色湘北7号球衣，戴眼镜，运球姿态
- **�的场浩司**（EIJI SAWAKITA） — 身穿红色球衣5号，运球动作
- **南烈**（TSUYOSHI MINAMI[?]） — 丰玉队球衣，持球进攻
- **岸本实理**（KAZUNARI FUKATSU[?]） — 身穿蓝色丰玉TOYOTAMA 4号球衣
- **樱木花道**（SHINICHI[?]） — 身穿红色湘北球衣，夸张的扣篮姿态（中央大卡片）
- **�的场浩司** — 另一张卡片，山王工高9号球衣，运球突进
- **宫城良田** — 白色球衣，运球
- **神宗一郎**（SOICHIRO JIN） — 左下角卡片
- **赤木刚宪**（TAKENORI AKAGI） — 持球投篮

页面底部印有：《电子游戏软件》 1999年4月
