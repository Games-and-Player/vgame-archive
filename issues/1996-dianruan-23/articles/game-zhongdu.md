# GAME中毒症例（上）

**是迈进了快乐之门，还是走向地狱入口？**

**危险！现在，GAME 中毒患者在日益增加！**

如今人类身上有一种新病症出现，这就是 GAME 中毒，这是定理明电视机越来越普及的现如，玩了之后就越来越上瘾，而一直远离现实生活的，而下面就是带你带你充分了解这已经超出了正常的行为！

## GAME 中毒是什么的彻底研究！！

过分热于于电视、到处每不管次不管、不断地络绎不绝，GAME 越来越成熟，每天沉迷于 GAME 的游戏场面都都有过这样，这就是 GAME 中毒，如果不采取措施，它就会侵入你的身体，让现在医学和精神病科学，到目前世界中一些中毒经历者都有多少多家有异常别报警，有一自己已不知。

▲中毒患者数的发展（折线图说明）

GAME 中毒者是随着 GAME 的普及而增加的。时代：游记时代→电视时代→PC 时代→SFC 时代→SS·PS 时代

## 报告① 时间·场所与发病的关系！

玩 GAME 时成瘾 GAME 时，各种症状随时间都会发生，当了一天之中，可能会有各种时间以在一起被发现！

## 你是中毒者吗？

以上的症状你有吗？如果一半以上的现象你都有，那就已要小心不行了，今后可要多加注意……

对了 GAME 中毒是那早单的发现，早就痊（游戏瘾），早诊断，如果你能把不良症状在最初阶段，不用在末时间，不让这之晚越来越后更好。

## RPG 中毒症例

RPG 游戏的中毒者可以说是最多的，游戏状态也是多种多样的。

玩平面RPG那，提醒点在家家面前继续玩，能过各种 60 的背景，一直在 GAME 游戏，必是在一直出现了问题，也同时同一看待于 GAME 中的名声。

### 【中毒 GAME1】《勇者斗恶龙》系列

**RPG 时代名词**

## SLC 中毒症例

SLC 游戏的中毒症状与 RPG 比较型，戒整型也不太多，但却将世界引入到现实之中……

### 中毒症状（复选框形式）

- □ 与 GAME 中较之间的笔记、记录
- □ 经常阅读游戏中出现的书刊指南
- □ 现实生活与游戏中的标准相照惯
- □ 特别喜爱搜集色彩鲜艳的物品

### 【中毒 GAME1】《三国志》系列

**求知病**

对三国时代历史的非一般了解欲望越来越大，沉迷最完全于游戏前，以至于各种三国相关书籍（赵山纪载和史记），稍稍相似或者大约两方面的，是大量的这方面的史实画…… 求知之心不停的涨……

**偏种病**

上述病症都进一步恶化便会较多来字中于三国的，而后代的 GAME 与更多的历史 "加之" 以及学者的历史地位……那么所有的一种时……而下面……一段……一段……

### 人材收集、选拔癖

在无处可以证明其真实……如果你是真正没有过的，会开始收集的，就一旦一某些……不只是这个……时时有人到来，确实已经找自己带来了……

### 自我满足症

这自己已经已经找了这为自己人，而且，一直将实生活应用而来正下，

### 【中毒 GAME2】《超级机器人大战》系列

**究极 LV 症**

这是与 RPG 中最相近的戏相性的，因此打进超 BOSS 的时候机，是最终以上面没有的别是否到极限机的，是全 60 的者，者 LV 极、是极最终目 BOSS，使以这个结果……

### LV 修正癖

不过你已中以心认定的角色的等级高低顺序有变化，就我们自己想方设法去修定过来。

**游戏中居然也可以得到较高的评价，若在现实中可以……**

### 【中毒 GAME3】《模拟城市》系列

**权力膨胀症**

完全不承担任何建设之外，随时随地任意多数来对所有城市中的一切事情都……

## FGT 中毒症例

对 GAME 不论是在家用机还是单用方方面来大家打方，，类型，许多中毒者在攻防伤敌中人物的必杀技，其判交主，台间，其军服除了，档档……

### 中毒症状（复选框形式）

- □ 努力人物的必杀技变态及模仿人语。
- □ 随时带着卡带与手柄到处挑战。
- □ 对战时不先行告退。

### 【中毒 GAME1】《街头霸王》系列

**超 PERFECT 症**

从第一位打手到最后的敌角（VEGA 或盖尔？），一辩格都是超越 PERFECT 的，一到一个你都要把一定量的一条，一有打上，就会去投 RESET 键而结束掉自己。

**行为模仿症**

对格斗游戏姿势，肌肉发达，力击范围划过到分数差。

**华丽必杀症**

在对战时是不用普通技，而从头到尾都是要用华丽的必杀技……

### 【中毒 GAME2】《VR 战士》系列

**口诀背诵症**

对于"P P K"及"G + P + K"当忘掉日起来不能忘掉的……

到别人说不好的结合出现" K，G"等字时也不时不时来……

## SPG 中毒症例

SPG 是特殊而的游戏类型，因为它为方面与现实关联上，以利之会产生多岁月寻寻的时间……

### 中毒症状（复选框形式）

- □ 看体育比赛转播时会在上面做控键的动作。
- □ 每天与朋友在各体育俱乐部时会谈 GAME 中的观念。

### 【中毒 GAME1】《实况足球》系列

**守门员进攻症**

对阵方而目的头天时会义什事，让己下一直来守继，这些对于这种变化很大的，完全不能接受完整对手向……完全能不接受出来对于游戏中而……

**平衡症**

对 COM 比对方手将比分进入队球速前射在自己的一侧时，就比较总力的……不过，也以比较不到一这种稍……也稍时候动了大写！

### 【中毒 GAME2】《超级火爆摔角》系列

**关节技达人症**

在开玩四四时间把非常非常的抢先角前，热烈于此而不输，直到是个危险的姿势。

**EDIT 角色症**

编辑自己以自己想由，编辑完自己人，让自己人成为格斗手，开以此为止……

▲"龙哥"

### 【中毒 GAME3】《NBA JAM》系列

**帅姿灌篮症**

在本节目中坚持使用具帅的姿态为赢取比赛，却是否真的能够获胜呢！
