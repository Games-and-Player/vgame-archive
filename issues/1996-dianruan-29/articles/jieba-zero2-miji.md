# 《街霸 ZERO 2》秘技大公开

## ARC/PS/SS

### 电脑乱入条件

除了原本的8位对手外，如果玩家选出第八人或以上的对手的话，这超就有机会乱入一名——ROUND 之后，选出8人之后，ROUND 之后，会合上三 ROUND PERFECT，都与第八人好友被当有一些，跟机器较真战战（注：如果用普通走法击完对电脑乱入人与真·豪鬼乱入那人的条件，刚会先打乱人的系，等待真·豪鬼，最后才打到八人）。

### 真·豪鬼乱入

如果玩家以2P使用1P画，能够在第八人之间结到到——ROUND 之后，连胜上三 ROUND PERFECT，都与第八人好友被当有真·豪鬼进入来战战（注：如果用普通走法出电脑乱入人与真·豪鬼乱入的话），刚会先打乱人的系，等待真·豪鬼，最后才打到八人）。

### 选台台

2P乱入时，将光标先移到想进入的角色，然后按2P的 START 键不放，再移动到想进入的人物上按方向键，此秘技只能在对战战机的模式中进行。

### 使用旧版春丽

在选人画面中，将光标移到 ZANGIEF 后按一次 START，然后移动选春丽，此时选出的将会是旧版春丽（注：选旧版的春丽在选人画面会显示为特大点，是在对战中用 SELECT(START) 键二次或以上选角色后运用的）。

## SS

### 选真·豪鬼

在选人画面中，将光标移至豪鬼身上，按住 SELECT 键停留一段时间，再按一次 START，然后再按键盘人。

### 选旧版盖尔夫

在选人画面中，将光标移到 ZANGIEF 处按一次 START，再移到 DHALSIM 处按二次 START，最后选回 ZANGIEF，此秘技的最大特点是没有叠幅打出飞头动作。

### 选旧版达尔锡

在选人画面中，将光标移到 DHALSIM 处按一次 START，再移到 ZANGIEF 处按二次 START，最后选回 DHALSIM，此秘技的最大特点是没有叠幅打出飞头动作。

### 选冲差隆

在选人画面中，将光标移至冲差隆处按一次 START，再随便按→，最后按住 START 来选定的角色，不然选定的将是普通式的选择人物，必须在对战游戏中，配到各个角色的出场以及随后触发，随后的会乘来者，阿修罗飞到空中，并重新从人头叠前选择触发的顺序。

### 在 SURVIVAL MODE 中使出场人物次序改变

在 SS 地送入4个角色4个地就在 SURVIVAL MODE 可以看到的总是固定，不过我们在还没选择胜利的图画面的，这个人物相在画面中用 L 或 R 键来选择人物，这样就会有不同的出场人物次序。

### 无限 OC 和高速连发

在 TRAINING 模式中，在选人画面时，按住 L 键 + START 来选进入TRAINING 无限 OC，接任 R 键 + START 来进入高速连发，两秘技可以不同时间。

### 罗比的一击致死

在 SS 地选角色进入 GAME 调至 LEVEL 3，将画 RECORDING MODE 进入游戏，按键盘置于战角色之后，使用BOSS 任意选的一侧的超流的进天，握在手半身与时下，以此的各超的路和一击一中，竞到真后出出向力的超超超力，打出此超必会各超所有剩的必要的路和时，以此打向后叠出向后超超超力的动作，在竞此后出出向力与动作在后后超超超力将向后后对超必，在 SATURN 上用该秘技的注意！

## PS

### 选择隐藏背景

于街机模式中，作出2基本对战时，只要在选人画面中将光标移到 SAGAT 或 VEGA 上，按住 SELECT 键一下以上再在选择使用该角色，此时背景画面轻易选出了两个隐藏背景。

### 选真·豪鬼

在选人画面中将光标移至豪鬼身上，按住 SELECT 键停留一段时间，再按一次 START，然后再按键盘人。
