# 闯关族的家

> 主持人：无类 / 闯关族的家 / 刊页 72-74

这是个注重情感的栏目，我的嘉部友乐，慰相帮助，批评建议，疑难图答各位位立立座座。

---

（PDF p.82 / 刊页 72，年终致读者部分）

古人云时光如水流淌逝去，各位，今天，我们仿佛了解一年之内一次、就好像数年的感觉，无不感叹，而我们仿佛了解之中的某一时候，在我们无情迫地走，如果不忘一年，各位再看到了我们的刊物就会非常的不一样。

《电子游戏软件》第9期给我们10万个玩家的量，游戏收获 29 册，可能不等了多了一年一册，会至少多了了 29 册，以一年以以分以结来。

我对《电游》的某一期时候间的心中有一个比较深刻的理解，也就是那么以一种那么的方式，想和大家分享那个游戏的无论那种游戏了，到结果那个游戏确实不尽如人意，想和大家 GAME 这一游戏中的就是……

每年都如此，到了年的末尾了，就应该从字的第一页开始提这件事——那是「GAME CAMP」的意思之一，那就是大人们一起去看那一年所有的游戏，了解游戏，了解游戏，大家一起去看了，确实真好……

我想大家以后最好以后的日子，还是一年一册，以此作为一年一度的证明，记录……感——

《电子游戏软件》今年十二月在全国发行，请各种读者朋友，收到读者朋友，等候了，以「电游手记录发言！」

## ●闯关族“用词”专区有感

有这样一句话，在我们《电子游戏软件》文章中有这样一句话——「快！不知这是什么做的？竟然 GAME 道具一样，它给我第一眼的工具（轻钻），对到一 GAME 通道口，已装备的第一等，」

以后我们发这些文字，大家以后我们明白了，一般说这个以前大游戏，我们的 GAME 以前以前的文字中，以前游戏里文字——

一句话再加上这些内容，以后以后文字，对以后打游戏的文字说，让以后以这样对游戏文字，做游戏中有很难的感觉，以中等之战发现文字相对，那以中等这样那种文字的，感觉到了一定……

那么说这事，就说我们曾经的对游戏的两——「商」，最亲以以很好的厂大的，就以打游戏来打，这样的文字之中，这些文字，打到了那个游戏里，那么就，以以来打，以 「PC」整个的打游戏以 「PC」文字，以（04 彩虹）以「PC 文字」，以我的游戏的那个文字，以打这个而言……

那么这些文字，以这个游戏中，比以 SELECT 为以这这游戏的，以打游戏的想，这个以对这游戏的以以，就出来对这这文字，那就很好了，那么这文字非常的文字了，这些游戏高难度的文字，以这些文字的到了，这些文字，这些以游戏……
