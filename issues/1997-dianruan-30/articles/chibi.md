# 独家报导国内业界最新消息

## 正在开发的大型仿真游戏——赤壁

编者按：一直以来，本刊都十分重视对国内业界的报导。但由于国内业界尚属起步阶段，事件不多。自去年下半年起，以北京前导公司为代表的国内开发商逐渐增多，真正属于游戏范畴的软件也逐渐增多，本刊在前几期中大量地报导了这方面的消息。今次，将向大家介绍《赤壁》的最新情报。

## 千年传奇进电脑，万古英雄入荧屏

目前，北京前导软件有限公司正在开发大型仿真游戏《赤壁》，这是该公司计划中《三国》系列游戏的第二部，也是继今年 6 月推出大型仿真游戏《官渡》后该公司的又一大举动。

游戏的故事无须多讲。雄姿英发的周公瑾，羽扇纶巾的诸葛亮，横槊赋诗的曹孟德，还有黄盖的忠心耿耿、阚泽的随机应变、蒋干的自作聪明……，多少年以来都是人们津津乐道的人物和故事。需要指出的是，《赤壁》游戏包括的内容还不只这些：它是从刘备找到徐庶开始的。于是，火烧博望、长坂坡等一系列热闹故事就都成了《赤壁》的一部分。

《官渡》和《赤壁》都是前导公司《三国》系列中的作品，因此有不少类似的地方。二者都是战略模拟游戏——游戏者指挥自己所属的部队与敌人展开战斗，通过完成一系列的任务而夺取游戏的胜利。二者都突出了战役指挥特色，都是实时战斗模式；二者都是以 Windows 95 作基本的开发平台；二者都大量使用电视连续剧《三国演义》中的实拍画面，堪称气势恢宏。甚至二者的开发方式都很接近：都是先把某种外文版的海外发行权出售给外商，从而减少了投资的风险。

多年以来，三国故事一直是我国文学、戏曲历久不衰的主题。在亚洲的偏家文化圈里，三国也是老幼皆知。在游戏领域，这一题材已经让日本人占了先手。日本人对三国自有他们的理解，而中国人当然更有中国人的想法。为了体现这些想法，《官渡》已经迈出了艰难的一小步，《赤壁》是不是更大的一步呢？

至发稿之日，《赤壁》的开发度已有 50%，有两个重要的方面没有进一步的消息：音乐音效和人工智能。前者是很难用文字描述的，眼睛和手无论如何也起不到耳朵的作用。笔者只是听到了 demo 中的音乐和音效，感觉还可以，据称《赤壁》是全程语音。雄壮的喊杀，威严的命令将会令游戏增色不少。至于人工智能，由于游戏还在开发中，无法亲自上手试一试，所以也无从谈起。只知道在《赤壁》中，开发者有意将人工智能分为若干层次——将领有将领的打算，士兵有士兵的主意，主帅自然要有深谋远虑。

## 这是一款真实时向制的策略游戏！

与《官渡》不同的是，《赤壁》是一款像《魔兽争霸Ⅱ》一样即时策略游戏。那么，究竟有何不同呢？

首先是“关”的设置。《赤壁》分二十余关，游戏者可以在指挥刘备或曹操二者中选择其一。每方共有十五关，各关有不同的故事，每关任务各不相同。假如你指挥刘备一方，则第一关的任务很简单，消灭吕吕[?]，翔率领的 5000 人就可以过关；而到了第十四关你就得在赤壁上演火烧船的好戏，等等。在战斗中，既有小部队行动也有大兵团作战；作战的区域从陆地到水面无处不在；作战的方式从单打独斗到大军混战无一不全；兵种有步、骑、水、弓等多种。

其次，在《官渡》中兵力、武器、马匹都是不会增加的，而在《赤壁》中，由于有了“建造”这一操作，没有枪，没有炮，都可以自己造。

《赤壁》对阵法也大大动了一番脑筋：十种不同的阵法，每种都有一个十分有“中国味”的名字，比如“九宫八卦阵”之类，据说，在作战中这些阵的每一部分都可能有不同的战斗力，至于如何不同，目前没有详细资料。

对实拍画面的应用上，《赤壁》与《官渡》也有不小的差别。《赤壁》画面气势宏大的特点，在细腻方面也在“细腻”二字上下了不少工夫。《官渡》的画面，强调的是大部队、大战场；《赤壁》则在逼真方面更胜一筹。动画上也不惜工本，笔者曾看过《赤壁》的 demo，小到单个的士兵倒地毙命，大如战船着火沉沦，都有很不错的动画来表现。据说做到 demo 中的动画，还只是《赤壁》所有动画中的一小部分，看来只有等到制作全部完成，我们才能确切地知道，《赤壁》中的美术设计究竟能取得怎样的成绩。

▲在赤壁中演火烧船的好戏，等等。在战斗中，既有小部队行动也有大兵团作战；作战的区域从陆地到水面无处不在；在南方的水乡作战，战船是必不可少的。

▲有些关就是以攻城为过关条件的，这……开不到的，以前的三国游戏中是绝对找不到的，攻打空寨具是真奇，可正式的游戏中可没有这么容易对付的敌人。

▲这是游戏中的图素，可以看到人物的入种姿容。

《赤壁》人物众多分文官和武将，主角自然是刘备、曹操、诸葛亮、周瑜。不同的人物不下 100 个。每个人物的“能耐”由他所具有的参数决定。参数中当然少不了体力、武力、智谋等老生常谈，但也有几个比较特殊的参数。比如官职，这是一个在 RPG 游戏中经常出现的参数，但在策略游戏中出现就比较少见了。

赤壁的操作和著名的《魔兽争霸》很有几分相像。选中的对象即可对它发号施令，多个对象执行相同的命令，只需用鼠标“拖拽”一下就行了。选中部队的方法有两种，既可以用鼠标，又可以用键盘，以适应不同习惯的玩家。执行不同的任务，有不同的按钮，请大家着着这些按钮，猜猜按钮的功能，估计费不了多大劲。

《赤壁》依然是在 Windows 95 下运行的游戏，与《官渡》不同的是，不仅中文 Windows 95 的用户可以玩，而且在西文 Windows 95 环境下，《赤壁》照样可以运行，并且正常地显示各种各样的汉字信息。《赤壁》还具备网络功能，支持连机对战。当你指挥手下的千军万马把真人指挥的对手杀得一败涂地时，心里那份快感，恐怕是任何词句都无法形容的。
