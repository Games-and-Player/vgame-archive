# 移位的自行矫正——土星激光头

> 文／SATURN，责编：PINSER｜科普园地｜刊页 82

土星激光头的位置，是以整个机器的主体位移为参照物的，因为激光头的位移，大部分时间是由于光学镜头组件于机械传动系统整体移位造成的。这好像一大群人按顺序排成队伍行进，只要是领头的方向正确，其余部分都不至于走错。

所以，现在玩家经常问到的“激光头磨损”其实是一个不太准确的说法，即使是说“激光头损坏”也欠妥当，真正的说法应该是“激光发射器老化”或“传动系统磨损”。

不同一批的机器中的激光头组件，它们的型号和参数各不相同，甚至有时同一个品牌同一型号的机器，其内部零件也可能是不同的。以土星为例，原装土星的CD或VCD光驱（NO DISK显示则为NO DISC），用的是SANYO（三洋）的激光激活定位系统，而在国内组装的非正品机则采用的是不同的方案，价格有高有低，差别很大。

以下的情况，当玩家发现激光头位置不对或者出现了读碟问题时，请先别忙着维修：

首先，如果将机器放在平稳、干净的平台上后，按一下读碟盖关合的按钮（不是电源），一般情况下，激光头会有一次自行矫正运动。这是因为光驱系统在每次盖子闭合时都会进行一次归零复位操作，以确保每次读碟都能从正确的起点开始寻道。

但如果发现在单独合上盖子之后，激光头仍然停留在一个看起来奇怪的位置，则不要轻易用手推动它。激光头是有精确的步进电机控制的，任何人为的强行推动都可能损坏步进电机的齿轮或传动导轨。

正确处理方法是：

1. 打开后盖检查电路板。

2. 检查激光头驱动的供电是否正常（对应红色线路）。

3. 检查传动导轨上是否有异物或是否需要添加润滑油（一般建议用缝纫机油，每半年一次）。

4. 如实在无法判断，请到专业修理处进行检修。

反复复位几次，如有改善则说明是软件问题。如无任何改善，则说明是硬件问题。

一般而言，激光头的移位也有以下几种情况：

一、系统从盖子关合到第一次探测光盘之间存在时间差，导致在非正常情况（如有碟片或有碟但放置不标准）下，激光头超出了正常范围。

二、一般以“探头归零”来说明这种状态，如果探头在探测5～8 后没有收到完整的轨道信息，则会重新从零点开始。

**注意事项：**

1. 以上均为普通情况，部分机器可能有特殊情况（如线路板短路等），请及时关机处理。

2. 如以上步骤均不能解决问题，则可先尝试打开机器，将激光功率调节旋钮（有的机器需要配合十字螺丝刀，细心操作）向顺时针方向调小约三分之一圈，测试是否有改善。

3. 本方法适合于一般技术状态良好、但读碟时经常小故障者，技术基础薄弱的爱好者请在老手指导下进行。

4. 此仅为内部技术交流，如因自行维修导致问题扩大，本杂志不承担任何法律责任，请读者自行评估风险。
