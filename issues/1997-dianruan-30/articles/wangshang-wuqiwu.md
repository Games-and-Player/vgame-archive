# 网上武器屋

在 RPG 游戏中，主角们通常要到村子或是城市里的武器屋中买一些强力的装备，使自己变得更加强大，也使今后在未知领域里冒险变得更加顺畅。所以，道具屋、武器屋几乎成了 RPG 主角们一入新城便直奔而人的地方。玩 INTERNET 也是这样的，如果不找一些强力的武器（软件），探索时将会遇到很多的麻烦，浪费了大量的时间和金钱（没错，在 INTERNET 上奋战数小时却没有任何收获的确是浪费金钱）。如果您是 INTERNET 新手，您需要什么？还不明白吗，您已经身在其中了，这里是网上武器屋。

在网上可以做的事情有很多很多，比如看 HOMEPAGE，用 FTP 下载文件，玩 IRC，阅读 NEWSGROUPS 等等，都是很简单的玩一玩；又比如架设自己的动态 FTP 服务器，自己写 HOMEPAGE 等一些稍微高深一点的玩法等等。不过，就是在这一定是在网上，把铁锤当做魔法杖去施法一定不会有好的效果，是不是？

下面请看一下货架

## HOMEPAGE

看 HOMEPAGE 几乎是上网者必做之事。微软与 Netscape 公司的 WWW 浏览器之战已经搞得全世界沸沸扬扬，如果您 INTERNET 经验丰富，却又说不知道 IE 与 Netscape 之争，那您一定是在开玩笑，或者是在说谎。没错，目前世界上被使用得最多的两大浏览器就是微软的 IE 和 Netscape 公司的 Netscape NAVIGATOR，其它还有一些浏览器，如 Internet In A Box 中的那个，功能不强，名气又小，实在是不招人喜爱。

IE 与 Netscape 到底谁比较比较好，世人众说纷纭，各有各的拥趸。在《每周电脑报》上曾经有过一篇关于 Ziff Davids 的实验室关于 IE 与 Netscape 的性能进行试验并且评测的文章，我看完后大有一种“天哪，看起来 Netscape 就要完蛋了”的感觉，而且立即去微软的主页下载了最新版本的 IE。原本我是一直用 Netscape 的，但是那篇文章实在无法让人不去赶紧了解一下“未来 WWW 浏览器主人翁”的风采。IE“试飞”的结果并不十分让人乐观，因为除了感到将自己以后在未知领域里冒险变得更加顺畅之外，我这个铁杆 Netscape 用户几乎没有找到 IE 的任何闪光点（这也有可能是因为本身对 IE 就没有任何好感而产生的偏见）。试着到平时经常去的 HOMEPAGE 转了转，并没有那种文章里说的速度快的感觉，而且因为没有下载和安装对应 IE 的 ActiveX，往日在 Netscape 下看起来动感十足的页面变得十分呆板，给人一种没有跟上时代的感觉。要想让自己的 HOMEPAGE 在 IE 中也能象在用 Netscape 时那么生动，就还得去下载一系列对应 IE 的 ActiveX，已经有了一个全副武装的 Netscape，犯不着为 IE 浪费更多的硬盘空间。而且 IE 做为微软家族的一分子，与 WIN95 和 NT 的联接非常紧密，安装 IE 时可以看出，属于 IE 的文件并不只放在两个目录中，在 PROGRAM FILES 中，在 WIN95 的 SYSTEM 目录中，在很多地方 IE 都插入了“自己人”，搞得你别想轻轻松松的就能把 IE 卸掉，因为那样往往会留下不少后遗症。最后让 IE 从我的硬盘中消失的是这么一件事，我在著名的免费 HOMEPAGE 提供公司 GEOCITIES 公司的 WEB 服务器上做了自己的 HOMEPAGE，里面有不少的动态 GIF，还有 DIRECTOR 5.0 的文件和一首 MIDI，发 EMAIL 请朋友去看自然是必需的。第二天一个朋友告诉我说我的主页慢得让人无法忍受，我想他用的一定是 IE，结果我真的猜对了。从此，IE 从我的硬盘上消失了。（后来我才知道，当时我的 HOMEPAGE 慢的原因不完全是因为 IE，复杂的 FRAME 结构和过多的效果也是原因之一）。然而 IE 也并不是一无是处。Kurt Cobain 说过；“去攻击一个自己并不是 100% 反对的东西是很傻的，如果在那东西里还有一点希望之光，我们就应该去支持它”。IE 也有一些令人激赏的地方，比如它的界面看起来很不错，按钮设计得也比 Netscape 有新意，尽管用过 Netscape 的人需要一点时间去熟悉这种新玩意儿。因为 IE 是微软的产品，所以在与 WIN95 合作的很好，运行起来比 Netscape 要稳定。另外 IE 对于运行 JAVA 指令要比 Netscape 明显快很多。

Netscape 是我长期以来一直在用的 WWW 浏览器，从古老的 1.X 版一直到最新的 3. 01 GOLD（以结稿期 12 月中旬为准）。Netscape 通常分为两个版本：普通版与黄金版，区别就在于黄金版内设一个 HTML 编辑器。最新版的 NETSCAP 里带有 LIVE 3D（一个 VRML 浏览器）和 CoolTalk（一个语音聊天软件）。各个公司为 Netscape 制作的 PlugIns 也很多，如 LIVE UPDATE 是一个 MIDI 插放的插件，SHOCKWAVE 是 MACROMEDIA 公司为方便 Netscape 观看该公司的 DIRECTOR，AUTHORWARE 和 FREEHAND 所生成各种文件而制作的插件，还有什么 Quicktime PlugIns，RealAudio PlugIns 等等。另外 Netscape 还包括了 Email 与 NewsGroups 的功能，但是因为这并不是 Netscape 的主要功能，所以显然不如 EUDORA，AGENT 等一些专门的软件好用。

IE 与 Netscape 之战还在继续着。IE 4.0 的 ALPHA 版现在已经推出，而 Netscape 的 4.0 版也已经放出风声来，究竟鹿死谁手，目前无法预测。微软尽管实力雄厚，但由于太过霸道（想在各个领域里都做老大），目前反微软的呼声很高，用户们虽然无法拒绝使用 WIN95 或是 NT，但却可以轻易为 IE 找到代替品。萝卜白菜，各有所爱。IE 与 Netscape 都是浏览器中的矫娇者，用哪一个您自己决定。我喜欢 Netscape。

## IRC

IRC，是 Internet Relay Chat 的缩写。玩 IRC 的用户要用客户端软件连接到散布在世界各地的 IRC 服务器上，然后再加入不同的频道（CHANNEL）中，就可以与世界各地的人聊天了。聊天的方式以输入字句为主，因为要与众多的用户（一个频道当时多数数十人）交换信息，以语音和影像方式在目前是绝对不可能的。如果光是“白纸黑字”的打字聊天，未免太无聊了，所以后来 IRC 软件的作者们给软件加入了颜色，用户可以给自己输入的句子加上颜色，而且还可以用 /ME 的命令加一动作的描述，如 “EastEvil Gives Joshi A Big Apple”等，别人看到这样的句子（这样的句子通常以不同于一般对话的颜色显示，非常醒目），脑子里就会想象出一副 EastEvil 正在给 Joshi 一个大苹果的画面。这样颇有点象 MUD。光凭文字的描述，看起来十分空洞枯燥，但比图形更容易引发人的想象力。在彩色字符之后，软件的作者又加入了一些其他的新花样，比如播放声音文件。举个例子，A 与 B 的 IRC 软件里都有播写“啊”的一声的文件，这时 A 输入“/ME 用军刀砍了 B 一下”和播放声音文件的命令，B 的屏幕上就会显示“A 用军刀砍了 B 一下”的字样，并且音箱里发出“啊”的一声，嗯嗯，怎么样，声光俱佳吧？IRC 里面的频道是有分类的，比如“Chinesecafe”是华人聚集的频道，“Newbie”是给 IRC 新人上课的频道（Newbie 一词指新人、新手），“PIRCH”“mIRC”这两个频道则是给 PIRCH 和 mIRC 这两个 IRC 软件的用户提供交流的频道。每个注册了的频道都有一个 BOT OP，也就是服务器上的机器人频道管理员，还有一些人类管理员。管理员的作用是在频道里负责维护秩序和主持正义的，因为随时都有可能有一些“不受欢迎的家伙”跑来捣乱。遇到这种情况，管理员就可以把这些家伙（KICK）出频道，然后再禁止（BAN）他再进入。

玩 IRC 的软件有名的有很多，但被使用得最多的还是 mIRC 和 PIRCH。mIRC 是我第一次玩 IRC 时用的软件，功能很多，界面也不错。不过后来遇上了 PIRCH，发现 PIRCH 的功能更是有趣，于是便改玩 PIRCH。PIRCH 目前最新版本是 0.85（32 位版）和 0.80（16 位版）。它的优点在于界面简洁，尤其是在用户已经加入多个频道时，频道间的切换非常简单快捷。而且 PIRCH 比 mIRC 先加入了字符效果的功能，还有音乐播放器，用户照片簿等功能。另外 PIRCH 防 FLOOD 的功能很强。FLOOD 是在 IRC 中一种很恶毒的给别人捣乱的办法，而 PIRCH 防这个很有一手。mIRC 目前正在紧追 PIRCH，很多地位有模仿 PIRCH 的意思，比如最新的 4.7 版中就有类似 PIRCH 的窗口切换快捷纽，不过真是做得不好。在字符颜色方面，mIRC 4.7 的 15 色明显超过了 PIRCH，但这并不是好事，因为很多 mIRC 的忠实用户以前没有玩过这么新鲜的玩意儿，给字体加上颜色这件事，对他们来说（一点也不新鲜），疯狂的过度试验这个新功能使 mIRC 4.7 推出后几个小时后，不少频道便挂起了“COLOR = KICK + BAN”（玩颜色就会被踢出频道并且禁止进入）的 TOPIC（主题标语，每个频道都有，并且更换频繁）。mIRC 的作者本想在字符颜色方面超过 PIRCH，给字体加上颜色功能还不到一天就被禁止使用了，真是惨！mIRC 的优点在于由于它出现的比较早，所以为 mIRC 写 SCRIPT（一种预先写好的 IRC 设置文件）的人很多，在网上有很多 SCRIPT 可以下载，而且最新版里的功能也比 PIRCH 多很多（不过很多象都没有什么用处）。PIRCH 的作者如果不是很忙就是很懒，因为自从 9 月分推出了新 PIRCH 后，到今天已经 3 个月没有任何动静了。品味不同，见解不同，选择亦不同，PIRCH 与 mIRC 都是优秀的 IRC 软件。
