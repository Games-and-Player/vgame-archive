# DC硬件大检阅

> Dreamcast本体硬件规格详细解析，涵盖SH-4 CPU、PowerVR2显示芯片、GD-ROM高密度光碟、手柄与虚拟记忆体、MODEM等全部硬件构成。

## PDF p.52 / 插10

### Dreamcast 本体

「DREAMCAST的开发概念在于“游戏与通信”的概念上，其中，在游戏表现的部分这款主机远远地凌驾于目前所有的主机之上，而通信部分，利用专用MODEM，也提供所有的玩家们一个全新的“家庭游乐场”。」

根据目前所公布的情报看来，这款主机在外观及SEGA的标志上已经与四个月前所公布的模型机资料有些少许的不同，同时在主机的右侧也第一次的装上了散热风扇，这在家用主机上可是创举（图片下），而这风扇的声音也不会太吵。

### 「目的是要凌驾于目前市面上所有的家用游戏主机!!」

### 速度快、性能强、价格又便宜的 SH-4 晶片

这台主机的CPU采用了由日立利用最新的RISC技术所开发的中央处理器，这利用最新CPU制程技术（0.25μm制程）所开发的处理器以低廉的价格提供超高的性能，同时在利用其搭载的作业系统后，更能够发挥出最大的效能，其主要任务是处理各程式及对周边晶片发出命令、计算多边形等。

↑ 图片中的处理器与实物有些些差异。

### 记忆体的大小是SS的2.6倍 MEMORY

这款主机的内存除了在五月所发表的16MB的主机内存之外，其他还包括了8MB的显存（TEXTURE RAM、FLAMEBUFFER）、2MB的音效用内存等，因此共搭载了26MB的内存。尤其是这次所公布的音效用内存更是SS的94倍。同时，在利用ADPCM的压缩技术下，其资料量将成为原本的四分之一，因此音效用内存可说是原本的16倍之多。除此之外，在显存方面他也采用了Power VR2的VQ压缩技术，因此在显存中可容纳其四～五倍的容量（相当于32MB～40MB的容量）。因此，这款主机也完全不必使用类似扩展内存的记忆卡来储存游戏进度，而成本也可以此而降低许多。

### 家用主机的必备机能 PowerVR2

这是由英国VIDEO LOGIC公司与日本NEC公司所共同开发的GRAPHICS ENGINE，它与SH-4相同，都是在最先进的0.25μm制程技术下所完成的，在这一片小小的晶片中具备了高速演算处理、计算高度复杂的画面的能力，这晶片本身可在每秒内处理三百万个多边形，这可是PS及SS的十倍以上哦！

↑ 由发表会上的展示看来它具备了各种特殊效果的处理能力。

### 「再一次与关系良好的YAMAHA合作」SOUND

这回Dreamcast采用了内藏特效用DSP的YAMAHA高效能32位元RISC音效处理器，这处理器不但支援XG规格，同时也对应ADPCM模式，除此之外这个音效处理器也支援可在一定周波数区域产生特殊音效的TIME VARIENTFILDER，相信这在声音上的演出一定是非常的不错。

### 何谓XG规格

这是负责开发SOUND ENGINE的YAMAHA公司在1994年所提出的一种有关电子乐器等音源的统一格式，它具备了与世界的标准规格GM相容的特性，同时也与ADPCM相容在利用考虑扩张性的音色配置、更多的音色数及各种特殊效果下，它可以发出有别最新电子乐器及电脑音乐的声音。

### 设置了四个插孔，可连接各种周边设备

「虽然在这款主机设置了四个的手柄插入孔，但这并不只是单纯针对让四人同时游戏时所做的设计。利用这四个插入孔，玩家们将可同时连接各种不同的周边设备，例如键盘、鼠标、可辨识声音的麦克风或摄影机等，日后只要有新类型的游戏登场，这时这些相关的周边设备便会成为游戏的必需品，而这些插入口的设计就是针对这项需求所预留的。」

### 手柄用端子

在主机上设置四个分别独立的序列输出入端子端，这款主机的手柄将比SS的手柄插端更快速的对相关资料进行存取的工作，而手柄的形状也是依耐用性的考量来设计的，在打开电源时也可以辨别插入或拔出的动作。

### 手柄

在这款主机的手柄上配置着与一组过去相同的十字键及一组以类比方式来收发信号的摇杆（可同时使用），在手柄的表面还配置着A、B、X、Y等四个按钮，而在内侧则配置着L、R两个按钮，虽然这些按钮在某些格斗游戏上可能会不敷使用，但到时候还会发售相关的专用手柄，专门用来管理手柄Direct Input的资料库其基本容量便可同时管理三十二个按钮及六组摇杆，因此在游戏时将不会受到指令输入的限制（Dreamcast的Direct Input与Windows用的并不相同）。这手柄除了单纯的控制游戏的按钮之外，还配置了两个可插入虚拟记忆体等周边设备的插端，这样一来玩家们便可带着记忆着自己游戏记录的手柄到各个好友的家中与他们游戏了。除此之外，这款主机在本体上并没有RESET的按钮设计，因此到时候可能会采取利用手柄上的按钮组合来进行软性的RESET动作。

↑ 这款手柄在START按钮的形状、列在按钮上的文字（A、X）的字体等都有所改变，其材质与主机相同，因此重量很轻。

### 「确实、正确、高速地传送资料」VIRTUAL MEMORY 虚拟记忆体

虚拟记忆体是一种对游戏的进度进行存取的记忆装置，这个记忆装置本身便可以当成情报的携带末端。虚拟记忆之间还可以彼此进行情报的交换，它之所以未采用最近市面上十分流行的红外线传输方式，其理由在于使用直接连接的方式以便在传输时达成「更确实、正确、高速」的目的。由于利用红外线的传输方式在速度上会受到相当的限制，如果为了解除这个问题，他们必须采用高价的光波授信模组才行，而如果利用目前所采用的直接接触传输模式，在进行资料传输的过程中受传输的资料也较不会发生流失的情况，同时还可能以较高的速度来传送资料。目前已有VMS的「哥吉拉怪兽大图鉴[?]」在市面上发售了，玩家们可以去买来玩玩看，先体验一下那种感觉。

### 以一般通信线路的品质来看，33.6Kbps的速度最值得信赖

### MODEM 可拆卸式的设计

在现代这个资讯发达的社会中，家庭中的电话线路和移动电话已经成为相当重要的通信工具了。为了让电话线路除了能与他人交谈之外，再增加一项「传送电脑资料」的功能，MODEM就变成一种不可缺少的必备工具了。所谓的MODEM是一种可以将在电话线路中所传输的类比讯号与电脑及游戏主机所使用的数位讯号作交互转换的「转换机」。如果没有MODEM的帮助，电脑或游戏主机的资料便无法透过电话线路来进行传输，而电话线路中的讯号也无法转换成电脑或是游戏主机可以使用的资料。

这回在Dreamcast上已经把未来会成为电话主机标准配备的MODEM搭载在机体之内，其传输的速度为33.6Kbps，也就是每秒钟可传输33.6Kb的资料。以现在电话线路的能力看来，这已经是最高的传输速度了。虽然目前市面上看得到更高速的56K的MODEM，但是因为目前电话线路的效能不足，因此33.6Kb的传输速度，无论在速度及品质上都是最稳定的。由于这款MODEM对应了V42数据通信的错误修正规格，因此可以高速而又正确无误的进行传输。除此之外，因为它也具备MNP5(MNP COM NET WORKING PROTOCOL CLASS5)的通讯协定机能，因此可以解决为了修正错误所产生的速度不足，以资料压缩的方式来提升传输速度，虽然这规格在未来也会被淘汰，但它也对应ISDN（总和数位通信网路：64K~128Kb/秒）的数据TATERMINAL ADAPTER：ISDN线路专用的接续末端）。所以在短时间内，它还能够完全满足玩家的需求。

**MODEM SPEC**
33.6Kbps：V42及MNP
支援MNP5通讯协定/可拆卸式设计

在通信的领域上与半导体的技术一样，是以倍速的速度在成长，而且相信日后也会不断的进步。过去的大众通信网路系统将来可能全面转换成ISDN或者是卫星通讯，以目前的一般电话线路的通讯品质来看，33.6Kbps的传输速度可说是最值得信赖的速度了。而为了让日后数据机有升级的机会，这个数据机也采用了可拆卸式的设计，如果未来的通讯网路进化到以ISDN为主，大家只要将此媒介转换为ISDN的媒介即可。

SEGA在16亿元的MD成长到配备数据机的DC，而在利用了数据机的通讯功能之后，电玩游戏的领域也变得越来越广。现在，SEGA终于将数据机列为DC的标准配备了，同时为在日后通讯技术进一步时能够加上升级，因此采用了可拆卸式的设计。

### 背面的端子为序列埠

配置在主机背面电源下方的小小扩充埠经过SEGA正式的发表后，终于证实这是一个序列埠，虽然目前这序列埠的用途尚未公布，但拥有相同序列埠的SS（下方图片）可利用他连接通信对战线，MIDI转接装置、打印机、磁碟机等周边，有了这些周边之后，搭载了Windows CE的操作系统将可发挥出更大的功用，但是在目前还是无法连接USB（UNIVERSAL SERIAL BUS 12MB/秒，最多63个周边的连接规格）周边。

### MULTI AV端子与电源

其他的端子还包括了AV端子与电源端子。AV端子与SS相同，连接头可连接VIDEO、S端子、RGB端子及音源。然而这次连接头的形状却经过了改良，在连接时不必担心因接触不良及损坏的问题，而电源则采用了是内藏式电源。

↑ 这回所公开的便是与其他端子所有不同的DC端子[?]，千万不要在未经SEGA的同意下另外发售。

### 因应大容量的需求，GD-ROM的容量高达1GB

### GD-ROM 高密度光碟

DREAMCAST ORIGINAL FORMAT GIGA BYTE DISK-ROM

过去我们曾经为大家介绍过DC所采用的是高密度的光碟片，而在这里将更详细的为大家介绍这光碟片。

「这回DC上采用了称为“GD-ROM”(GIGA BYTE)新格式的光碟，光从其英文名称来看，相信大家便可以了解，每一张这种光碟的容量为1GB。换句话说，其容量大约是普通光碟片的两倍左右。为了将大量的CG动画以及系统资料使用在游戏之中，大容量的储存媒介是绝对必要的，因此这回的光碟片的资料容量提升为1GB。相信一定有很多人会问为什么不使用DVD呢？其实理由很简单，在考虑成本的前提下，这是很难达成的，DVD的相关周边的成本都很高，再加上开发环境的考虑，于是不采用DVD。」

**GD-ROM DRIVER SPEC**
最高12倍速/MEDIA GD-ROM
(ORIGINAL FORMAT CD-ROM)

由于由YAMAHA所开发的GD-ROM采用了高密度的1GB容量格式，因此其记录密度为传统光碟的两倍左右，换句话说我们可以将两倍容量的资料记录在这光碟媒介上。在高于传统光碟片容量两倍的帮助下，读取光碟资料的PICK UP LENS将必须更为精密才行。除此之外，虽然成本较传统的CD-ROM来得高，但是大容量却也是其不可多得的一大优点。

由于此光碟媒介采用了独特的高密度格式设计，因此无法使用传统的PC或者是光碟机来读取，这样一来这种特殊的光碟媒介也具备了防止盗拷的功能。除此之外，因为这是一种以传统光碟为基准所研发出来的新一代媒介，因此各机材、环境只要将原本的设备稍加升级即可读取，因为目前所完成的影像等与过去可压缩的影像有所不同，因此在软体开发工作进行时，必须实际的输入磁片内的资料进行检视才行。根据佐藤博士所提供的情报看来，要将DVD导入目前的工作环境内还必须克服相当多的困难，这也是为什么采用GD-ROM的原因了。

除了容量之外，它还采用了十二倍速的CAV方式的DRIVE，这样一来也可以缓解光碟片读取速度太慢的问题。所谓的12倍速指的是可以12倍的速度读取资料，每秒大约可以读取1.8MB的资料，是SS的两倍。

### CAV与CLV

GD-ROM是在游戏主机上第一家采用CAV记录模式系统，能够大幅缩短资料读取磁头的移动时间。过去游戏主机的光碟机多采用CLV这种密度固定的记录模式，所以为了能够稳定地读取资料，读取内侧资料时就不得不加快碟的转动速度，而读取外侧资料时就要减慢转动速度，这样一来读取资料的速度便会因此而减慢。CAV则是采用了角密度一定的设计，因无论读取内侧或是外侧的资料，光碟片的旋转速度都是固定的。

**CLV**
外侧的记录密度较低 / 外侧的回转速度较慢
内侧的记录密度较高 / 内侧的回转速度较快
回转速度一定 / 记录密度固定
