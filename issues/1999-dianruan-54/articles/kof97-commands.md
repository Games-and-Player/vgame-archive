## KOF'97 指令输入技巧

**广东湛江 平平　　安徽蚌埠 柴华**

在 97 中，有一部分必杀技和超杀的指令是可以简化的：如←↙↓↘→指令可简化为↙→，→↘↓↙←→↘↓↙←→则可输入↘←↘←，等等。简化输入在对战中表现其重要性。

八神的屑风（←↙↓↘→A 或 C）和百八式·暗勾手的指令（↓↘→A 或 C）非常接近。当对手接近你时，你想用屑风来攻击他时，对方已跃起，这时你会用出暗勾手而露出破绽，被对方有机可乘。指令↙→可简化屑风，当出现上述情况时，输入简化指令即可屑风，不会用出暗勾手，使攻击完善少破绑。

简化指令还可将出招之间融合在一起，简短连招时的时间，使连续技更容易使出。

如大门之超必杀技岗之山（←↙↓↘→←↙↓↘→B 或 D）也可简化为（↙→↙→BD）。当对方作中段攻击时，大门可作下盘防守（↙），等待时机到对手攻击结束时开始还击：向前疾走到对方前面→←↙↓↘→+B 或 D 即可使出岗之山，因防守时已开始输入指令（↙），到疾走时→已把一次←↙↓↘→指令简化，把防守和攻击落合并简化在一起。整个防守攻击过程就是↙→←↙↓↘→+B 或 D。这招用于远距离攻击十分有效，令对手防不胜防，但要注意疾走时间不能太暂，大概二分之一秒内有效。

下面列举两个样板例子，都是要求把招数简化，溶合连贯，才能一气呵成的。

例一：大门五郎的一击必杀（两个气点爆掉一个）——空中 D·地面 D·特殊技（↘C）·地狱极乐落（近身→↘↓↙←→↓↘→×2+A 或 C）——此招需要极快的动作输入指令，而且容易出错，极难一气呵成，其同主要是地狱极乐落极难连贯超特殊技（↘C），但运用简化程式就可解决这个问题，可以这样表达，将地狱极乐落的指令和特殊技的指令溶合在一起并简化：空中 D·D·↘C·←↘←C。这样可以很轻松地使出，干脆利落。

例二：山崎龙二的一击必杀（两个气点爆掉一个）——空中 D·蹲下 B·蹲下 A·砂�的（→↓↘B）·蛇使（↓↙←B）按住·D（取消）·C·射杀（近身→↘↓↙←→↙←×2+A 或 C）。此招较例一中的更加难于运用，更复杂。但只要把射杀简化为↘←↘←C，其实很容易使出，表达为：（空中）D·B·A·→↓↘B·↓↙←B（按住）·D（取消）·↘←C·↘←C。

以上技巧均为本人之经验所得，如有漏点，欢迎广大朋友批评指点。

KOF97 中精彩的连续技成为此游戏的魅力之一，我想强调一下连续技的实际操作，愿大家能从中得到些什么。

大门、二阶堂、真七柳社：他们拥有一个相同的连续技——小跳跃 D·下蹲 C·必杀技（→↘↓↙←→A 或 C）——这个连续技看似简单，但较难掌握。通过指令简化，在实际操作时可将其变化为：↗↘（按住）D·C·←→A 或 C。这样就比较简单了。

真克莉丝、麻宫：他们拥有一个相同的连续技——小跳跃 D·下蹲 C·必杀技（←↙↓↘→A 或 C）——此连续技与上技差不多，但也有些许不同，实际操作时为：↗↙（按住不放）D·C·←→A 或 C。在必杀技之后，他们还可以连一个超杀。拥有此连续技的还有玛丽。

比利、拉尔夫、莉安娜：他们也拥有一个相同的连续技——小跳跃 D·下蹲 C·必杀技（←蓄→A 或 C），具体操作与上技相同。比利和莉安娜在此连续技之后还能再追加一技。

特瑞不是有个无限连击吗？——下蹲 A 或 C·能量补充（←↙↓↘→B 或 D）——这个连技要求手上的动作要快，也较难掌握，但在实际操作中却变得非常简单。只须连续输入 A·→B 就可以了。

类似上面的连续技还有很多，这里就不一一介绍了，但只要广大玩家开动脑筋，就一定会创造出更好的连续技。不过，我还有二条小技巧：96 模式中的山崎龙二使用蛇掌然后再取消，只要反复使用，他的气就会马上集满；还有，当我方在角落里，CPU 进行空中攻击时，我进行空防，然后使用指令投、必杀技或超必杀投。我方可不断跳起，引诱 CPU 使用对空技（注意：空中防御防不住超必杀技），这就是对 CPU 的无赖技法。

## 98 部分连续技

**霸王塾**

**高打乐园——街机仔之天堂　　广东开平 关丹丹**

98 最大的变更是近身指令投在远处也可以放出，但只是一个失败的动作，对对手根本造不成威胁，还会遭到反击，令到以投技为主的角色力量被削弱。再来就是能量槽，97 中爆掉一个，接首放超杀，放完后能量槽还在，这时还可以再接着放 POWER MAX 版的超杀。但 98 就不同了，放完超杀“气”就立即消失，所以最好在“气”用到差不多的时候再放超杀。但如有大好机会，还是不要“孤寒”为好。

## 97 部分连续技

**桂林农机校 伍怡**

走进机厅发现八神的使用率很高，但 97 中有一些人物是令八神十分头痛的，如：七柳社的跳跃重手根本不怕八神的必杀技入雉女；大门的站立重脚轻而易举地就把八神的跳跃攻击给破坏掉……这些初时我也觉得头痛，后来反而让对手头痛不敢使用了，因为七柳社的跳跃重手可以小跳跃防御后以屑风或葵花进行攻击（视距离而定），大门的站立重脚可以用八稚或暗勾手来破解。总之，使用八神一定要灵活，尽量发挥其重脚对空，重手对地攻击的优势，击中后可以重手加葵花来加大破坏力度。对于这样一个绝对进攻型角色，当对手一味防御，要用屑风破坏；对笨重角色不妨用暗勾手来对付，对手跳起可以用下蹲重脚来破坏，对手前回避则可以使用投技。另附上一些未刊过的部分角色大破坏力连续技（96 模式）。

■草薙京：B·→B·毒咬（↓↘→C）·罪咏（毒咬中→↘↓↙←→A/C）·罰咏（上式中→A/C）

■特瑞：下蹲 B，A·↘C·高轨喷泉（↓→→×2+B/D）※：高轨喷泉接前三招时只用（↓↘→B）即可，且按 B 比按 D 成功率高

■东丈：下蹲 B，A·→B·爆烈飓风猛虎脚（↓↘→↘↓↙←A/C）

■金家藩：→A·凤凰脚（↓↙←↙→B/D）

■克莉丝：下蹲 B，A·→A·滑触（↓↘→C）

■真克莉丝：下蹲 B，A·→A·（↓↘→×2）A/C

※：同特瑞一样也可只搓一次（天语/作者自然而然地融合并简化了指令），但攻击力大，一次去 3/5 体力

■八神：①下蹲 B，A·→→A·葵花三段（↓↙←A）②下蹲 B，A·→A·八稚女（上述这些招式难度稍大，请多加练习）
