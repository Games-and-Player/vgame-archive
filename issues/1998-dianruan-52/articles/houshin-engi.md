# 封神演义

> 次世代天空 / 刊页 25-26 / PS / KOEI / SLG / 98年9月10日

酒池肉林 生灵涂炭

**讨伐暴君商纣，歼灭邪帝蚩尤，拯救乱世中的芸芸众生！**

光荣制作的战棋游戏向来是中规中矩，难度适中。不过今作多多少少有些借鉴“FF”之处[?]，而且整个流程有着单一化的问题[?]玩到[?]后面[?]让人感到枯燥。相比之下，还是“封神演义”的传统题材更加吸引人。

### 解说“洞府修行”

“洞府修行”是游戏中比较新颖的系统，其中不仅可以提升级别还能够获取特殊道具，因此对玩家来讲是至关重要的。如果准备提升角色的级别或是修炼类似“乾坤术”等不易成长的能力，就要进入“元始天尊”的洞府修行。这里简介一种修炼的委任模式，待战斗开始后，先按“×”，把光标移至地面按“START”，调出菜单选第一项即可。这样战斗便会基本上自动进行，电脑将控制主角的行动。玩家大可看杂志、喝杯茶，待战斗结束后如此往复，便可在比较轻松的状态下完成修炼。

除“元始天尊”外，其他随剧情出现的仙人洞府是获得宝物的最佳去处。玩家每次去打败神仙都会得到一件特殊道具，随之该神仙的级别也会大幅提高。一般来讲，神仙的级别增到40多级便达到了顶点（磐灯道人为48级），如果玩家再次将其击败，那么洞府将永远无法进入了。不过，由于仙人的攻防能力均很高，而且在一轮行动中可以攻二至三次攻击，因此玩家要想将全部仙人击倒，除了超强的能力以外就要靠运道了！

### 有关隐藏道具

游戏中的绝大多数战场均藏有宝物，只要我方队员在该位置上待机便可取得（类似雷霆子的浮空角色无法获得）。在游戏进入中盘后可收到杨任。他有一项“掌中目”的特技，作用是看到战场上的隐藏物品和陷阱。玩家利用此特技就可以轻易的发现隐藏道具了。不过，“掌中目”无法分辨是陷阱还是宝物，有时经过一番苦战冲过去后，发现只是陷阱，那种感觉实在是……

### 隐藏角色赵公明的加入

在赵公明和其妹首次出战时，要先击倒赵公明。这样就可以把生腐泥匙，而后选择第一项将其释放。之后，击破九曲黄河阵再次活捉赵公明后再次将其释放。那么在随后的情节中赵公明便会加入。赵公明各项能力均不错，是类似黄飞虎、杨戬的主力角色。
