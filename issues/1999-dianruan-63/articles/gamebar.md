## 世界足球·实况胜利十一人4

(WORLD SOCCER·实况 WINNING ELEVEN 4)
机种 PS 厂商 KONAMI 类型 SPG 媒体 CD-ROM

### 评论一

在比预期的日子晚了几天之后，终于拿到了期盼已久的WE4，一看光盘上的数据量，心里就认定本作将超过前作，KONAMI没有每年糊弄一次。

为了不让老玩家对本作产生 + r1 + 的感觉，KONAMI这次在系统上下了很大的功夫。首先，游戏的基本操作系统和风格继承了前作的风格，画质在原作的基础上有了一定的提高，球员的动作更加细腻柔和，基本达到了PS的极限。其实，在真实感上又向前迈进了一大步，真实地体现了足球的感觉。与前作相比，短传的威力大大降低了，而利用长传进行大范围的转移显得很重要。通过各种导弹配合，进攻的路线更加多样化，但是在比前作更加容易的防守面前，几乎没有特效的进攻方式，任何一次进攻，都要根据场上的情况来选择方法和路线。

说实在的，KONAMI所有的改变，使得本作对于没有玩过此系列的人来说，也许能更快上手。而对于象笔者这样的老旧星来说，反而有一时适应不过来的感觉。但这种不适感很快就被强烈的真实感冲淡。可以说，目前还没有任何一款足球游戏的仿真程度能达到这么高的地步。当然，这里所说的仿真程度指的是踢足球的感觉。

KONAMI的足球游戏，常常被认为设定过于简单。本作大大丰富了球员的数值设定《但笔者认为许多数据原来就隐藏在前作之中》，游戏模式、编辑改变球员的能力等等辅助功能。看来KONAMI在潜心探索足球游戏真谛六、七年之后，也开始关注边边角角的东西了。用句成语来形容，叫做 + r1 + 。

### 评论二

大概从发售前的一个月起就开始盼望的游戏。但是拿到手、欣赏完令人感动的片头之后，发现就有点不太对头——是不是买错盘了？菜单结构太象EA东西了！乱得要命。而进入游戏之后，更是有一种手足无措的感觉——这是那个令我欢如知醉的WE系列吗？KONAMI难道损了一批好汉？

这当然还是原来那个WE4。之所以感到不适应，是游戏的系统与前作——主要是WE3的FINAL版——相比有了很大的差异：速度优势不再明显、过人不再容易、身后传球不再精准、射门不再有力……

这种改变，相信是KONAMI为了使游戏更真实而作的。事实也确实如此，游戏的节奏、中场的争夺、传球、配合、等等等等，都与真实的球赛相差无几。哪怕你把速度调整到最高，也不会出现那种打冰球般的感觉了。从这个角度来说，WE4有了长足的进步。

但是回到游戏本身来讲，WE4不如前作有趣，队员在场上始终是有气无力地奔跑着，与WE3充满活力的动感相差甚远。游戏的操作也过于冒险简易[?]（与之相对应的是密密麻麻翻来翻去网来翻去的数据指数）。

今国内玩家满意的当然是可以选中国国家队和国奥队的！虽然数据被日本佬子设得出奇地低，但是还是可以通过EDIT功能改得比较实际一些。而且可以编排英语名字，同时又不影响解说时报出的队员姓名，加上队员外观的设计功能，实在是比较贴心的安排。

游戏中还可以继续精彩的进球过程和预设阵形战术，从此要不必为暴露战术而头痛了！出许，随着对该作品的进一步熟练，对它会有更多的感触吧！

## 前线任务3(FRONT MISSION 3)

机种 PS 厂商 SQUARE 类型 S·RPG 媒体 CD-ROM

等了这么久，终于又玩到了如此正统的S·RPG，也让我们看到了史克威尔的实力。也许有人觉得这次做得太中规中矩，其实玩起来还是很过瘾的，至不仅体贴而且细致，到位，也只有最具经验的名牌厂商才能作得如此精准。

本作特别强调了战斗的流畅性，攻击发动时只做视点的拉近，比起SFC上的I代还简练，但是因为全部采用了多边形，所以视点变化的自由度很高，而空魂似乎在这方面的手法又很老到，再配上丰富、精细的机甲动作，用短短的战曲的同时保留了一个硕大力十足的战斗场面。另一大优点在于它的地图做得不大而又充为真实，有着多种多样的 + r1 + ，形态也不一而足，随着视角的远近变化表现出了身临其境的效果。这也是我见过的经过精心绘制的最为精美的多边形地图。

其实，这种机器人战争类游戏大多着眼中的轻人大头，毕竟最精彩的战斗过程全是机器人在表演。 + r1 + 对此还进行了探索性的改进，每驾员出于主动或者被动可以离开升机甲，可以移动、攻击、被攻击，甚至某上其他没有驾驶员的机甲，揭开了机器人战类游戏的一个新的局面。不过现在的做法还是比较初步的重叠机甲，驾驶员的双向活动力大胆，不够实用化。

沿袭下来的 + r1 + 系统变得更加提到了手里，除了收发e-mail还可以浏览BBS站点、下载软件等（当然是在游戏里的虚拟下载），感觉一些幽默机甲乘员也因此比较通感。

## 劲舞革新版·二度混音(DANCE DANCE REVOLUTION·2ND REMIX)

机种 PS 厂商 KOEI 类型 ACT 媒体 CD-ROM

自从花200多块钱买了跳舞板之后，每天早晚各1个小时的练习时间就被DDR牢牢的霸占了。这时的我才深深地意识到——人真是 + r1 + ！

这样说的原因是反管普者身材比较丰满，但是一直不愿听从劝告进行健身活动，自从DDR横空出世，才愿意为了一个游戏而挥身骨冒汗。

这次的混音版，收录了不少新的曲子，从感觉上来说与前机更接近了。而且也有一些好曲子仍然保留，虽然很可能也没有特别好听的 + r1 + ，但是可以通过前作片件的功能达到，所以不能算作很烦恼。

由于增加了节奏的编辑功能，使得这个原本很简单有趣的跳操游戏变得更有趣了。且是由于我对舞步并没有什么研究，所以经常被自己的 + r1 + 玩死。

不过这游戏也有令人不太满意的地方，有的难度高得有点过分。比如有支曲子，上下左右反复同时出现，如果并没有玩家用双腿来完成它的打算。此外还有一些的节奏之快也是很难用脚来完成的——或许是我的水平还不够出色？

## 狂野兵团·战火二度点燃(WILD ARMS 2nd IGNITION)

机种 PS 厂商 SCEI 类型 RPG 媒体 CD-ROM×2

几乎是怎么玩过一代。之所以会买，一是因为近来没有游戏可玩，二是被它的片头动画所吸引。

PS发展到现在，似乎是看动画的时候多过玩游戏的时候。不过联系着是索尼自己的作品，其质量还是应该可以保证的。游戏可以说是用足了PS的机能（索尼出品，又是PS发售后第5个年头，再没有用足机能也许说不过去吧？），画面效果可以说是没有的全新的了，三维与二维之间的切换也很流畅和，还新增了多米诺牌阵的 + r1 + 机能——也许是想和DC争一把吧（DC确实支持SCREEN SAVER）?

游戏的读盘速度，因为有了巧妙的设计而显得 + r1 + 很快。此外，游戏还可以使用一代的记录来找乐（果然找得我有趣），如果有玩过前作的玩家，建议再来尝试一下这个2代，一定有更好的感受。

3D次世代RPG的外包装，2D正统派RPG的内涵，经典的故事，多愁善感的情节、性格顽强的人物，加上两个不错的动画片头和好听的主题歌，似乎已经有很多理由去买这个 + r1 + 了。

当然它离自己的佳作还有不少距离，但已经蛮不错了。

## 真实系机器人战线(REAL ROBOT 战线)

机种 PS 厂商 BANPRESTO 类型 S·RPG 媒体 CD-ROM

 + r1 + (SRW)系列在我国玩家心目中有着崇高的地位。而我虽非铁杆狂热的SRW迷，但对此系列的3D化作品还是抱有不小的兴趣，因此购买 + r1 + 时并未犹豫，可没想之后大失所望。

尽管本作将SRW系统中的精髓如机体换装、多段攻击、精神分支情节等全部继承，但并没有给玩家什么新意。几个动画名作中的名人名机和一些并不太有名的人物、机体以及强加于人的原创人物、机体、剧场度构构成了整个游戏的世界观。虽然片头制作得相当不赖（数码公式化！），但一到战斗画面，粗糙的背景及机体人类呆板的动作实在有些不堪入目，这样的3D画面在今天的PS上，水准就太不够了。我在玩此游戏时几乎不看战斗画面，想来有不少朋友与我相同。难度、战术不高，读取速度也不快，我已经很难在这样一款游戏中找到闪光点了。

BANDAI及其子公司还将量产数款机械人体数的游戏，而 + r1 + 不免令我对其后续作品的品质产生了忧虑……

## 做个软件要几钱？

**文/特约撰稿人　王骏生**

开发一个软件到底要多少钱？比如说，在PS这个平台上？

这是许多玩家都有兴趣的问题。有些玩家——比如说3、5年前的我——对中国软件开发小组一直不愿开发家用游戏主机软件一直耿于杯，既然知道电脑游戏更加容易被盗版，既然知道PS或者SS的用户更多，为什么却总是无动于衷？既然这样，那么不如我们自己上吧：找几个伙伴，找几家赞助商，为中国的游戏事业献出一点红白白的随便什么……

到底要多少钱？开发一个PS软件？

在这个宏伟计划实施以前，大概所有的赞助商都会提出这个有点 + r1 + 的问题。这也难怪，既然是赞助，就不是无偿的，总得要前投资有所回收吧？虽然开口就说淡阿堵物有辱知识分子的身份，不过看是没有这个扎么，什么样的理想都是万万不能的……

最近比较烦，因为谈起钱来总是基本上比较难。好吧，譬如说我们已经万事具备，譬如说我们只等东碱[?]，不妨以开发一个PS二流RPG为例，看看做个软件到底要几钱？

### 劳动力支出

开发小组的人员基本配备：

主管（即小组长，一般由发起人担任，有时也被尊称为经理或者老板）1名

游戏设计2名（游戏结构设计，游戏剧情设计，等）

美工4名（人物设定，场景设定，动作设定，菜单设定等）

程序员4名（设计相关程序，开发游戏引擎）

音响设计2名（音乐及音效，语音效果等）

这13人的编制，对于一个初出茅庐的公司来说，算是非常节约的了。软件开发行业的含金量是比较高的，工资给少了，那些特才自傲的 + r1 + 当场就会散伙走人，克扣粮饷当然更大大的不行。按照目前北京、上海等大城市的白领收入情况来看，整个小组的月工资支出大概要4万7千元。

从设计到制成第一个样品，就要手下下将得力、大伙尽心尽责、都有开发PS游戏的经验，最快也要超过6个月，加上测试、整合等步骤，没有10个月是不行的（这还是建立在咱们中国人吃苦耐劳、任劳任怨、天纵英才、唯我独尊基础上的，一般国外的厂商要超过1年），这样整个小组在开发期内的工资支出就差不多是47万元人民币。加上聘请临时的测试员的费用，前劳动力的开销是50万元人民币。

### 硬件

人手一台电脑是最最基本的要求，这配置还不能低了，质量也不能差了。别人PⅡ233、15或17寸显示器、64～128兆内存还算过得去，但是对于美工来说，显示器一定要21寸以上，内存不能少于256兆，处理器不能低于PⅡ400。如果还要做几秒钟清晰度不高、不算太流畅的CG动画，就得配上PⅢ以上的机型，加512兆的内存和12、3个G的硬盘了——而且你千万不能对2天出1秒动画的速度表示任何形式上的慌恼。

这些电脑设备的支出，按现价的话还要花掉不少于25万元人民币。而作为投资方还要做好思想准备：这些电脑对下一个软件的开发基本没用。

整个小组中拥有两套PS的DEVELOPE KIT是很基本的，此外还要至少4个PS测试专用主机（DEBUG SYSTEM）以供平行测试使用。这些硬件的花费都不便宜，索尼会要你再付出超过25万元人民币。如果游戏想对应振动手柄、PDA或者其他一些奇奇怪怪的配件，就只有自己去买了。

对于音响开发人员来说，没有200万元人民币的设备是根本不可能完成工作的。这里的子儿 + r1 + 。

还没有算上传真机、打印机、扫描仪、复印机等等附加的设备呢！这些虽然也是必不可少的附件至少还要10万元。而如果不用MOTION CAPTURE技术的话，可以节约100万元人民币，但付出的代价将是滑稽而僵硬的角色动作——当然，对玩家的批评我们可以继续熟视无睹。

### 软件

美工必需的3DS MAX黄金版；原版价格是美金3000元，合人民币2万5千。如果希望玩家买正版游戏，开发商也不会盗版软件来开发游戏吧？这样再加上其他名目各样的应用软件、办公软件、杀毒软件（这是最重要的），以及用于 + r1 + 的PS、SS、DC等其他的游戏，总支出大概是5万元。

### 办公、吃饭、考察、上网

开发组在露天办公是不允许的，虽然那样比较容易和群众交流。一个比较僻静、具备空调、地板、电源、电话、水、厕所和一定面积的办公地点，在租金方面并不会过于便宜，在这方面的开销，10个月将超过60万。而且水、电、电话、厕所清洁等等费用还是要另外支付的，这里并没有计入。

吃饭也是让开发组头疼的问题。整天吃盒饭的代价很有可能是让肝炎病毒拖延了软件开发的时间，从而造成更大的预算外支出。早晚饭可以让员工自己解决，但是午饭这一关是怎么也过不了的。如果我们的运气比较好，找到一个心脏并不很硬的食堂（譬如某小学或者中学）搭伙，5元一顿也就凑强可以了。因此这10个月中，开发组还要向这个食堂缴纳2万2千元。

考察费用最好的情理处理，可以仅限于主管级人士。国内外有什么展览、去看看、交流、学习一下是必须的，也不能说这就是公费出国旅游——再说这笔费用也是羊毛出在自己的身上。生活费、补贴、飞机票、签证费、入场券、住宿费、泡温泉、逛新宿、黄石公园、迪士尼……按照一年3个国际大展、3个国内小展来计算，花费应可以控制在5万元人民币之内。

上网是另一个费用大头。对于开发能力处于初级阶段的国内人士来说，很多素材我们只能去网上 + r1 + ，此外休息时提供联机对战游戏也是游戏开发小组最基本的福利之一，下班以后的上网和游戏虽然有点 + r1 + ，但似乎也不是应该管制的。这样一来二去，10个月花在网络上的软硬费用加在一起，少说也要20万。

### 审批、广告、关系协调

辛苦半苦，恭喜恭喜，10月怀胎终于可以见分晓了。不过成品首先要送交索尼检测，而且一定会被退回来要求改进——无论我们甘心不甘心、愿意不愿意。为了显示诚恳和重视，小组总得派个代表去一趟日本吧。审批的工作是比较麻烦的，而且批准书用钱是买不到的。就算索尼不十分刁难，就算我们的产品卓越超群，就算我们只送审3次，也已经花了不下3万元。

上帝保佑，索尼终于恩准我们的游戏可以正式出版。于是开始做广告、拉关系、找门路。电视广告？中国从来没有这个先例，不过要做也可以：黄金时段10秒1万5千元，效果如何不知道，还是算了吧！电台的广播广告不够直观，花费也不小，也不必加以考虑了。

这样我们的宣传阵地就只剩下电玩杂志。平时就已慈眉厚的总编先生这个时候更是屡开提笑，望着我们就象望着北京烤鸭一样式乳猪。为了支持国产软件总编摩拳擦掌带劲蹦跃，黑白版一页只售5000，彩色版一页只售10000，特别专访和讲评价，楼上面谈。为了达到宣传效果，怎么也得每几本杂志齐头并进，奋起直 + r1 + ，一个季度下来至少20万。

### 算帐

秋天是收获和算帐的日子，赞助商们满腔堆笑地来看着我们宏伟理想的实现。在经过了这一番折腾之后，究竟要卖掉多少软件才能做到至少保本呢？一五得五，二五一十……把所有可能发生的费用全部加起来算算，我们已经用掉了差不多520万元人民币，这真是够便宜的了，换成美元不到70万，这种奇迹真不知道除了聪明勇敢的中国人民，还有谁能够办到？

为了打击盗版并响应玩家的号召，我们决定把软件的销售价格定在28元。除去所有的零件开销（压盘、塑料盒、纸盒、说明书、销售推广、销售回扣，等等），每张光碟大概可以有8块钱的收入。蛮不错嘛不错了！

真的不错？可就这样，我们也必须要卖掉65万份软件才能收回成本！我的天！！且不说全中国有没有65万台PS主机，且不说出版部门是否允许我们压制这样多的塑料片，反正我们的经验便是：正版一出，盗版即随后排山倒海般地绝杀过来。如果开发组里有个把职员食图小利，把测试版出来卖给外人，盗版早于正版发售那也是稀松平常的事，不值得大惊小怪。我们压价，盗版商也会压价。28元和5元，玩家会买哪一个已是不用多说的了，哪怕平时口号嘹亮的他们对待盗版……才怪！

算来算去，我们其实真正是在给盗版商做软件、做广告。谈论节、各个节，花去500多万元开发，我们想什么也得不到。

当然，玩家是快乐了——别急着高兴，他们也许并不那么快乐，因为我们的东西只是他们几百张盗版藏戳中不那么重要的一张，无足轻重。他们一边玩着游戏，一边对国产游戏大摇其头，然后用个笔名在杂志上发表几篇 + r1 + 、 + r1 + ，谈谈我们的RPG离FF8究竟有多远，讨论一下520万元人民币与3000万美元之间的差距，责怪游戏开发商吃里爬外而且一毛不拔，同时再对中国游戏的未来未来发出无限的感慨……很快的，我们便会被遗忘，被淹没在越来越多的盗版之中，再也无人提及——直到永远。

唉，做个游戏，到底要几钱？
