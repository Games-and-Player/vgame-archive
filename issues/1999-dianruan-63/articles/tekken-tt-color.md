## PDF p.112 / 这次将是两个人的战斗！

今夏的新作，NAMCO的大型电玩版 + r1 + 已在开发当中。采用了所谓双人战斗这种新系统，毫无疑问的将比之前的系列作品更能展开前所未有的热门！

每次都持续在进化的3D格斗对战 + r1 + 系列最新作，就是这次的 + r1 + （以下称TTT）。目前为止所有系列作品中的角色大概都会出现，到时玩家要在这些角色当中选出两个人来进行双人战斗。在系统方面，除了旧有的摇杆＋四个攻击钮的操作之外，最大的特征就是加入了新的交换钮。这次就让我们来介绍系统方面和再次复活登场的角色吧！

## PDF p.113 / 系统与角色

### 双人战斗才有的新系统

 + r1 + 基本上是沿袭前作 + r1 + 的，而所谓的双人战斗也只是在对战中增加了一个交换角色的按钮而已。但是从什么时候该交换角色的选择当中，也产生了新的战略性，很有可能因为一次的角色交换就分出胜负了。

### 掌握胜利关键的角色交换

在双人战斗当中最重要的就是伙伴的交换。角色的交换是以新加入的交换钮来控制的，而能实行替代的情况则是在站立状态、倒地之后的起身、采取守势的时候这几种情况下。顺带提一下的是，在后方等待的伙伴多少会回复一些体力。

↑趁对手被击入空中时进行交换，然后以交换的伙伴攻击也是可以。

### 其中一个同伴被打倒的话……

在 + r1 + 中，只要同一组的伙伴被打倒的话，就会被对手取得胜利。即使是后方等待的伙伴体力完全没有损失，比赛还是输了。要掌握住正在战斗角色的体力以及对战的情况，正确地做出角色交换的动作是相当必要的。

↑即使后方等待的伙伴体力满满，但只要有人被擂倒，就会被对方取得胜利。

### SYSTEM 系统

### 防守姿势更多彩多姿

在前作的 + r1 + 时只有两种防守姿势，这次的最新作就加入了上述的交换防守、倒下时瞬间起身、在对手身后翻滚取出距离等几种新的防守姿势。因此，双方的攻防也会变得很热烈唷！

↑不管是攻击方还是被攻击方都有很多选择，要从技体上来推测对手下一步的行动。

↑一流及击术玩保卫士凤间难以风来进行战斗风。

### CHARACTER 角色

→花郎的师父贝克。有着超越其弟子威力的足技。红鹤等特殊招式也依然存在。

→在 + r1 + 中和妮娜风格不同的安娜。这次的登场将有更明显的特色。

### 总数20名+α的登场角色

现在已知在 + r1 + 中登场的角色就是上面的20位。不论哪一个都是在 + r1 + 系列中出现过的角色。但是，只要看到角色选择画面就会发现有好几个 + r1 + 存在着。虽然知道之后会有更多的角色出现，但是到底会是谁呢？

→花和妮娜虽然在外观上完全一样相同的捷法：X和N和铁甲相打斗技也有相同之处，但入技特殊动作对手打。
