### 合金弹头X 隐藏人质宝物

● MISSION1：在打完3个炮台后（一阵大爆炸），用枪打真度蠍的一个洞可打出金币或者蘑。在可通路处处，打其背景房屋的2个窗子，可打出许多食物。在打完送兵车（就是留下可乘坐的那个），打其背景房子，靠右边第2个窗子可打出一封信。

■ MISSION2：在一处有个宝箱下有3处喷毒液的之间，即有人质给你 + r1 + 枪处；向左边的石像开枪可打出金币。在有拿胜大旗的人都底，用枪打背 + r1 + 的牌子，打出金币。（后面字迹太潦草，无法辨认）。另在4阶梯的那里（去拿神灯之前），打最后一个 + r1 + 牌子也有金币。

■ MISSION3：火车没有开动前，可看见一敌人躺在椅子上。杀死他后，在其尸体上猛砍几下，可得一苹果。在旁边有一个信号灯，向上打可得一人质。

■ MISSION4：在一处 + r1 + 的台子，向上打可得人质。

■ MISSION5：在写有 + r1 + 的路牌处向上打可得人质。在有火车部段有标有6-0的信号灯，打它们都是人质或宝物（有的要在打完火车后才有的）。

■ MISSION6：刚一开始就可以看见一个 + r1 + 牌子，打它可得许多肉。在前面的 + r1 + 牌子，打之可得了人质。打小BOSS的画面中，在2个台子中间向上开枪可打出人质。

北京DREAM读者指出，利用技巧的特性使敌人浮空，再加以追打；或利用必杀技抵消普通技的收招形成连续技，这些FTG的连技要素在三国战记中同样适用。

### 各关BOSS应战心得（续）：

● 孟优（孟获）：首先把他击倒在地，若击向左方则站在其侧下位置的在下或左上方；击向右方则为右下或右上方。只要站位正确，他起来时便会冲向与你相反的方向。马上冲上去击倒他。如此循环，可轻松干掉。但要注意他全身发红时跃出时，要立即跑避冲车。（难度：C，相克：X）

● 夏侯淳：他出现时绝不能站到城门前的中央，否则立即掉半管儿血。首先不能打灯下方的木桩。然后将其逼至两个木桩之间。接着让诸葛亮用火剑打他（他被困至木桩间后不能行走，否则会帮他 + r1 + ），若无诸葛亮，是要将四个木打桩，以增加对敌活动空间。（难度：B，相克：雷）

● 张辽：他一跳下来便会向左方发起冲杀，可站在尽头偷袭他。道具 + r1 + 对他有特效。他全身发红并有光柱护体时，要马上向左或向右逃走，否则必中打板。建议用 + r1 + 后，爆POW打他。（难度：A，相克：无）

● 貂蝉：若有张飞或张辽，可一下一下地打她（但不要形成连击），只要保持好按键的节奏，则可轻置 + r1 + 她。若无上述角色，则又要累了。可爆POW打她。（难度：B，相克：X）

● 吕蒙：擅击他有板回、筋，九节鞭可挡，只要看清楚他行动的规律，不难打败他。此关关尾容易联机时间，因此要准备好 + r1 + 。（难度：B，相克：X）

● 沙摩柯：若有冰剑，对付他会很容易。若无，则要放 + r1 + 。他全身发红时要立即跑杀，否则会给烧烂所伤。（难度：C，相克：冰）

● 吕布：若有火剑在手，只要站在画面中央。待其现身后，马上用火剑打他便可满血不费地杀虫击倒。要注意在吕布刚出现时，画面会不停闪烁。此时在行至画面中央处，不然会被雷击。若无火剑去打吕布，多少条命也不够死。（难度：C或S，相克：火）

● 陆逊：先用 + r1 + 把他迷晕，然后爆POW，用连续技打他（难度：B，相克：X）

● 左慈：待其出现后，把 + r1 + 逼个炮出来烧他。若自己及同伴都持有 + r1 + 时，可用 + r1 + 来射他（因下关有 + r1 + 可拾）。待暗器用完后，拿出火剑或 + r1 + 去烧他。若情势为危急，可用 + r1 + 护身。（难度：B，相克：火）

● 许诸：首先用暗器与他周旋。放光后，看准时机，放出 + r1 + ，又耗其半管血。最后用能令他无法行动的道具困住他，爆POW用连续技打之。（难度：B，相克：毒）

● 黄盖：先用 + r1 + 将其变为小动物，然后用冰剑或爆POW后用连续技打他。用完后，若冰剑可慢慢地与之周旋；若无，则要用 + r1 + 。（难度：A，相克：冰）

● 魏延：有雷剑可不放道具。若无，则要用 + r1 + 打死他。（难度：C，相克：雷）

● 司马懿：把除 + r1 + 、 + r1 + 、 + r1 + 之二等道具全用来打他。打此BOSS就只靠放道具，硬拼只有完蛋一条！其 + r1 + 那招要在画面下方不停跳跃才能避过。（难度：S，相克：X）

● 曹操：把剩余的POW用大必杀技打他。然后放 + r1 + 。然后使用 + r1 + ，爆POW后用是用连续技疯狂地操他，不要让他有加血的机会（曹操可补三次血，三次共计1-2管血）。来到此处时一般能剩3个 + r1 + ，足够耗他半条命的了，因此不用担心。但切记要注意他的 + r1 + ，此招一中就没一条命。当画面内出现门曹操的分身时，就代表恶梦的开始。此时，一定要连续发大必杀技或连续使用耗时很长的道具（例如天书）才能避过。当然，若使用 + r1 + 就不用担心了。（难度：S，相克：冰）

### 小秘技：

1、打击敌人时，若一下一下地攻击（指不形成连击）在达到对手得到的分数高，且敌人也不易死去。如此一来，此法将对数值增加攻怒气积累都有极大的帮助，特别是在游戏初期。但貂蝉若马超两位角色是无法如此偷分的。

2、只要玩家每1个COIN通关（所有加入游戏者都没在途中GAME OVER），在观看完结局画面后，电脑便会给出三位隐藏人物其中之一的调用方法。

3、在游戏进行至取得 + r1 + 后，使用一次 + r1 + ，此时， + r1 + 会减至9张。然后将 + r1 + 倒满9个。接着使自己的道具表的第二版面刺下被调成无限使用的道具（待剑角色也可围一把神剑作底）。先把该道具使用，然后将 + r1 + 全部切出（使用），再检查道具表的第二版。此时画面中央会打出一行程序函数，即表示成功。此后，该道具可以无限使用了！诸如 + r1 + 、 + r1 + 、 + r1 + 等都是用此法调出的。但切记调出以后千万不要拾加那些只能拾一件或全游戏只有一件的道具，否则会破坏读无意。

### 游戏心得：

1、游戏中应不断自己所处的位置，以免沦入敌人的包围之中。

2、使用道具的通则是： + r1 + 。性命紧急关头要敢放敢取用。对付难缠的BOSS时，某些道具是一定要放的。如张辽之处就该要放掉两把 + r1 + 。

3、多人游戏时，若同伴一没，是会剩命损一条的。所以，既没时间又没 + r1 + 时，就要发作出大无畏的精神，生命数多的同伴先去给敌人打死，时间会补满。其他同伴的 + r1 + 将得以保存了。

4、多人游戏时，不仅要保护好自己，也要顾及同伴的安全。这是多人游戏的关键。
