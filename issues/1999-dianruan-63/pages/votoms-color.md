## PDF p.9 / 插 7

### 装甲骑兵终于变成了战略模拟游戏！

 + r1 + 是为了纪念卡通 + r1 + 十五周年所推出的第二款游戏。游戏中，各位玩家可以操作被称为AT(Armor Troop)的战斗用机器人完成各式各样艰难的任务。

为了这款游戏，制作小组特别聘请了卡通中担任机器人设计的大河原邦男，以及担任角色设计的监山纪生。因此，原作中真实的世界观和严谨的设定，在这款游戏中也可以清楚的感受到喔！

←↑战斗和移动画面都是3D。玩家们可以尽情享受有魄力的多重视角画面。

【图】多幅3D战斗截图，展示AT机体在雪地和城镇中的战斗场面、Arm Punch近战攻击动画。

### STORY

就在百年战争即将划下句点的时候，新的战争又悄悄的揭开了序幕。在 + r1 + 的口号下，这颗星球的命运被这钢铁军势的战略所左右着。阿斯特拉奇斯史上从未发生过的激烈战争就此展开，但是这场战争幕后的黑手到底……

↑游戏中还有介绍装甲骑兵世界观的简介功能，不熟悉的人也能安心的玩喔！

↑一久违的Arm Punch！这可是AT特有的双击。

**炸裂！！**

### 游戏中出现的新AT！

AT中最受大家欢迎的就是 + r1 + 。用来开发Scope Dog的ATM-04、06、08等试作机，在游戏中也被重新设计过。另外还有很多雪地用的AT和新装备也会登场喔！

【图】底部排列8种AT机体设计图：

- **ATM-09-SNC Scope Dog**：装甲骑兵中最受欢迎的机种
- **Aparench Dog**（ATM-06-ST）：机关枪在短距离之内，是一台具有高度通用性的机种
- **Mascarel Dog**（ATM-06-ST）：将Scope Dog本身作了大幅改良的机种
- **Igreel Cattle**（ATM-04-ST）：专门为Scope Dog之前所作的初期试作机
- **Spending Wolf**（ATM-08-ST）：可替换各种的足底，也就是机关枪类型的机种
- **Srege Totus**（ATM-14-9SC）：用BS(Standing Totus)的强化型
- **Blocker**（B·ATM-02-ST·Fatty）：战的前卫，镇压中坚机
- **Aelt Spinel**（X·ATM-01）：这次的新AT，名为 + r1 +

## PDF p.10 / 插 8

### 操作AT(Armor Troop)打破不利的战况

这款游戏中为了增加驾驶对战斗的影响力，增加了两项特殊能力。第一项是 + r1 + 。使得战斗不再只是单纯的移动→攻击，只要点数足够，攻击→移动→攻击等等多彩多姿的战术都可以办到。第二项是 + r1 + 。每次行动都会增加疲劳度，如果太累的话可是无法发挥战力的喔！

### 自由自在的改造

↑AT具有高度的可调整性。可以瞄着敌人规模和作战内容调整武器和装备。

### 作战流程

**开场动画** → **作战名称**（精锐救出作战）→ **敌人登场**

↑看到作战名称后，就可以欣赏开场动画。

### 移动

移动的方式分为 + r1 + 和 + r1 + 两种。步行可以朝各个方向移动，而冲刺虽然限制在前后左右四个方向，却可以高速移动。

**步行**　　**冲刺**

### 攻击

随着使用的武器不同，可以攻击的范围（距离）也会有所不同。每位驾驶员都有最拿手的攻击距离，要懂得好好利用。

**机关枪**　　**飞弹发射器**

### 完成任务

**Mission Complete!!**

↑无论作战成功与否，游戏都会进行下去，但是故事内容会不同。

**多重剧情**

↑可以任意调节部分的设定。战斗画面也可以关掉

↑对于高强的敌人，就要靠同伴支援和用双人战略攻击

### 有高度差的地图

由于采用了3D地图，因此可以在山岳地带和城镇等等有高低差的地方进行各种策略。善加利用高低差和障碍物，一定能创造出对我方有利的战况。

↑利用地形的话，就算是小部队也有可能击退大军。毕竟作战的成功与否决定在策略。

**埋伏和迂回攻击都能办到！**

### 伴随着同伴支援的双重攻击

战斗中可以和自己的同伴要求支援，如此一来，便可以同时对一个目标进行攻击。如果运气好，还可以一次就把敌人解决掉。在战斗的3D动画里，连担任支援攻击的友军的行动也会一五一十的出现喔。

↓→Aparench Dog的Arm Punch，接着是Srege Totus的近距离攻击！

↑Aparench Dog（白色AT），向同伴Srege Totus（土黄色AT）要求支援。敌人则是雪地战用Fatty（粉红色AT）。

**重创敌人**
