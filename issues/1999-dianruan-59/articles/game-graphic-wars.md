## 《沙加开拓者2》转向2D，走出自己的道路

> 机种：PS / 厂商：SQUARE / 类型：RPG / 发售日：99.4 / 媒体：CD-ROM

散发狂热魅力的《沙加》系列这次不再3D化，走向朴素。

《沙加》系列的标志——命运之石，在系列之后的作品中是否会继续存在，这本身也是一个不确定的命题。而在《沙加开拓者2》中，并没有以主角之命运作为中心的剧情展开方式，系列作品从表现手法上来说，要将所有的事物用3D来呈现也不是件轻而易举的事。本作之所以选择2D，不是一味追求写实，而是根据发展方向，让人感受到其新鲜的意味和氛围。作为表现手法来说，2D图形出来的和风格，让人还是觉得好，毕者的看法——这就是一种创新的“美丽之绘”。

不能相提并论的是，两者在游戏世界方面可是有着相同的世界观的。虽然那是另外的故事了，然而本作的画面中充满了一种新的气息——“纯粹的美丽之绘”，这就是本作的目标。

插图令你一眼就可看出来是小林智美的笔触。

## 《最终幻想VIII》3D POLYGON

> 机种：PS / 厂商：SQUARE / 类型：RPG / 发售日：99.2.11 / 媒体：CD-ROM

对于《FF》系列，自97年的《VII》到今年的《VIII》，短短一年多时间，2D的各种手法已SFC时代的《V》到多如金子般的角色比例，SFC的2D图形到PS的3D Polygon，所经历的变化都是不同阶段的表现。以后，2D与3D究竟会走到什么阶段呢？不能相提并论。

可以说到了《FFVIII》中，图形的运用已到了level up的步。

### 游戏中的图形何去何从？

计算机游戏是在电视画面上形成图形运动所展示的，色图形的大前提，从过去到现在的规则，换句话说进化就是让画面所追求游戏的光影化变成不同级别的变化。

游戏从最初的黑白色的画面到了许多色的游戏画面——是多幅画前大分家用机游戏的光色的变化——从机的进步让大量的颜色可以运用。在2D阶段中是角色的部分配合着各种特效被表现出来，用上背景，加上之后使用的电影化的手法使之无限丰富。

在《FF》到《VII》，2D与3D化已经完分发展；是指“有不同角色的感觉”，计其中机建筑山水配置等各色需要的一定必须变形到3D位，此感觉必须感应到而变，其中角色等素因素。在2D与3D化无分发展《V》到《VII》，20与3D已达到展。

### 游戏图形未来会怎么样？

游戏RPG也有浓浓的血统。列举元素的“维度”来说，从到来的3D游戏，人物不像现实但运动画面上配上2D表达性，则会有很好的效果。变故事的“剧”方面——中最为重要的，是制作一部好的“影片”。至于那些新地区的3D游戏系列化的描述是“移动”至“角色”之间的差别。

## 再现那曾经的三度感动！[最终幻想 COLLECTION]

从87年登场至今已有10年以上历史的《FF》系列，除最新作《FF VIII》外，今年3月11日在PS上发售的[FF精选集]，包括《FFIV》、《FFV》、《FFVI》三代作品。此三部作品是《FF》系列在SFC上推出的该系列中兴期的名作，作为旧作的移植版，游戏中的名场面均被3D MOVIE化，是FF迷不可错过的“礼物”。

## 天野喜孝与[FF]

作为《最终幻想I》到《最终幻想IV》，2D时代的角色设定者的天野喜孝先生相信大家不会陌生吧，本刊曾多次采用其真的作品作为封面及插页。对于3D化，天野先生曾经说过十分为难，他曾经说：“用自己最早的绘稿的3D角色感到十户的手中”！

分亲切，但就好像有种隔了一层的感觉。许多可爱的角色（如莫古）并不能忠实再现自己的绘画……

2D时代的《FF》系列的功助筑话者之一的他对FF的3D MOVIE的评价是“现在及用户”！

## 向“作家性”的拘泥和向3D的疑问

### 《向北·白光》

> 机种：DC / 厂商：HUDSON / 类型：旅行通信 / 媒体：GD-ROM / 发售日：99年3月18日

原作·总制片：广井王子

● RED CO.代表作有（天外魔境）、（樱大战）。

《向北》是在3D机能很强的DREAMCAST上推出的游戏而被称为CG的卖点却是其根本的2D风格表现，“被注意到的只是CG”却是很多运用3D美术出来的作品，是因为那些“简单来表达出CG业界最专业水平”的东西，本来美术的制作的事实，变成出美工的劳动了。

挑战在3D中使用林新式的2D画风的尝试，是用简单的3D来表现2D的不同。在2D中来表现景色和人物也具有创新的意味。在所有3D的作品中，从各方面来看都代表了“2D的较好性”。

想想世界中有那么多的优秀作品都是建立在2D的基础之上的，自然其作家性及其世界观的传达力也是很出众的。

从这个意义上来看，《向北》在3D世界中使用2D技法来表现故事的人物、画面和氛围是非常具有意义的——自然而不做作的新风格。

### 《莎木》超越3D的“时间的要素”——4D

> 机种：DC / 厂商：SEGA / 类型：FREE / 媒体：GD-ROM

制作：铃木裕

● 世嘉第二AM研究开发部部长，世界第一部3D对战格斗游戏《VR战士》的开发者。

开创3D大门的铃木裕的游戏《莎木》，是不打算生产那样的东西而要在“代质量”以高品质给予最好的3D。另外，令人注目的是尝试加入“时间”的概念，而几乎没有什么多余的3D MOVIE画面，是目前《莎木》的特色。不能被忽略的3D——投入大量的人员和集众多人的力量来制作的3D游戏，其要求、追求近现实的3D画面，基本上反映出最多的游戏开发方向。

和要求相比较的是，真实的3D——铃木先生比较之前所有的3D作品，中用户所身处下的游戏深层画像中，《莎木》基本上是DREAMCAST上也尝试的，也许目前许多游戏《莎木》究竟3D是否还有那么多要3D还是需近现实也好这是目标？

描绘出的世界，哪里都是真实。

## 《电脑战机 VIRTUAL ON》游戏必须实现3D化？

> 机种：DC / 厂商：SEGA / 类型：ACT / 媒体：GD-ROM

### 继续摸索的制作者们

此次被这样采访被运用。3D的特性均匀地应用，例如《莎木》中的3D特性均匀应对，此外这款实际映像相对的，如Polygon也是在前文中的3D和用户的操作而形成的，角色的运生和用户的操作而形成，反过来2D正如在前文中所述量的MOVIE的游戏也是有的，不使用3D图形却是运动也不再只穿于3D图形的Power也是无法表现的。

关于2D，变得并不重要，重要的是游戏图形的要求，仅以美丽表现方法就够了吗？！

设计　有井伸孝

● 以设计《VR战士》的图形闻名的世嘉AM2研人员。有终极技术。

超过自身真实感——3D的创意在于自身所感到的3D本身的魅力。第一代始被认为的30年代以来作品中真的充实的感觉，然而要做到这到一步的地步，就是令人惊叹的东西了！纵使3D之后的世界观的3D MOVIE大部分都是由有CG以及制作的高级技术万的要求的3D是非常难，大事情。看到那个MOVIE来说，这样的这些人的需求是对游戏的3D的水平有很高的要求。大到开发者们的3D创意设计，甚至是难以想象的。

设计　尾崎弘一

● 世嘉AM3研究人员，以开发街机为主。

步一步进化发展的关键。外来游戏就是“现在”之后的家庭娱乐。步，技术上面仅仅是已经8D的游戏。没有特别的关注改良。另外，与公众大众多数的关注度不同，一个与现实之间——不存在了。有对比的性质已是确认的了。

设计　森康浩

● 参予制作本作街机版的世嘉AM3研人员。

真实的CG视觉高的基础3D要求的真实感，比较方向——同一步的发展的关键。什么看得都取决于手的能使用。与《电脑战机》那样的用上2D绘制的形态来表现人的精神和行动方式是很不同的。一方面是令人期待的，其中大部分原因就是因为2D的插画上色的。

视点会随背景而改变的。

## 《勇者斗恶龙VII》追求“有趣”是一个到达点

> 机种：PS / 厂商：ENIX / 类型：RPG / 媒体：CD-ROM

艺术指导：真岛真太郎

● 从《VI》起历代CG都出版在其手，当然《VII》也是啦！

好地利用2D与3D——在众多的角色让人感觉十分和蔼可亲的游戏中，对于很多作品来说本来不必用3D的，但为了“2D的进步”的立场而去利用3D也的确很重要。另外，方的话游戏开发就会变为3D，但有趣的一是个人会觉得，占用利用的空间，一的制作的地步，要的过程一，仅仅是3D的追求——“真正的”而自身变得简单。制作中最为重要的因素。是在作品《DQ VI》中被引用的3D地图等方式也被利用了的模样，3D的元素在2D游戏中的运用也是一般性的。

那种朴之所以“踢”出一个《沙加》并如今追了3D化是游戏图形的主流发展趋势。来不再繁琐，里不再杂乱来。

★注：有关RPG史氏的广告及介绍，左边有介绍，这拓者》，系列是本不之所以然，3D化是游戏图形的趋势方向。

### 游戏图形的进步之首要因素

高形态，从而加强会引发制作者本身产生某些实化不断，而这又推动着图形不是多么伟大的设计。仅仅是3D的追求到达的“真正的”一步。模仿达到现实——一方面是直到目前已到了的阶段，一个消失的角色。如《DQVI》面东西会从这些有发展的方向。之后会有什么呢？立体这样的物角色之间的差异性会越来越大。
