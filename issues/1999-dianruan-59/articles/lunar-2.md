# 露娜2·永远的蔚蓝色——纹章组合与纹章特性完全解说

> 游戏研究所 / 刊页 26-27 / PDF p.36-37 / SS,PS / 角川书店&ESP / RPG / 发售日：98.7.23(SS版)、99.5.27(PS版) / 文：朗波 / 责编：PERFECT

“Lunar2”共有纹章25枚，各有其特殊的用途，若只粗略一提实在可惜，因为这是“Lunar1”中所没有的设定，若只凭玩友去探索其用途，也须费一番功夫。而且，不仅纹章，就连戒指也与一代不同。不过，戒指的作用不那么复杂。下面就一一列出各枚纹章的作用，以及两两搭配起来之后产生的效果。

**各种纹章的系统研究，搭配效果完全曝光！**

### 两枚纹章搭配效果

下面将要介绍的是纹章两两搭配起来之后的效果。但首先要说明的是，并不是所有纹章两两搭配都会产生特殊效果，一般情况下，纹章装备以后，如果没有特殊效果，那么，搭配在一起的结果就是各自的效果的相加，例如：地の纹章+魔眼の纹章，则出现：攻击-10，地系攻击魔法使用可能，精神+10。但是，助攻性魔法纹章与辅助纹章搭配，则会出现装备负效应抵消、追加作用相加的结果，例如：地の纹章+战士の纹章，则出现攻击不减，精神不减，攻击+10（战士），地系攻击魔法使用可能的特效。

下面我就介绍两两搭配会出现特殊效果的纹章组合：

|---|---|---|

| 地+水 | 攻击-10 | 地水复合攻击魔法使用可能 |
| 地+火 | 攻击-10 | 地火复合攻击魔法使用可能 |

| 水+火 | 攻击-10 | 无属性攻击魔法使用可能 |

### 魔剑の纹章与女神の纹章

下面为您介绍魔剑（后面称M剑）和女神の纹章。M剑の纹章可以和所有助攻性魔法纹章搭配，除可使用相应魔法外，普通攻击时亦有该属性的追加打击。虽说装备会因为攻性纹章会降低5点攻击，但由于有追加打击，所以也算值得。但碰到不怕魔法打击的敌人，追加打击也就效果不大，但若碰到有该魔性魔法特效的敌人，相应效果就会较理想。实际运用中玩友应考虑一二。我建议使用M剑+眠の组合，因为攻击睡眠后，防御力减半，相应的我方打击力加倍。助攻魔法纹章包括地、火、水、风、雷、封魔、魅惑、毒蛇、梦魔。

女神の纹章是所有纹章中最理想的，可惜的是在龙之遗迹中的宝箱里，所以游戏早期不能用。女神の纹章可以与任何纹章组合产生特效，不仅抵消所有负面效果（除鬼神、魔神外）而且对其追加作用均有不同程度的强化。女神の纹章和所有助攻性纹章搭配时，普通攻击均有该属性的追加打击，还可使用相应的上级属性魔法。

| 组合 | 效果 |
|---|---|
| 女神+魔眼 | 使用魔法时MP消耗减半 |
| 女神+报复 | 敌通常攻击时反击率高 |
| 女神+圣光 | 通常攻击时敌HP大量吸收，玖装备无效 |
| 女神+战士 | 攻击+50 |
| 女神+チロ | 敏捷+20，每回合最先行动 |
| 女神+狩人 | 器用+10，クリティカル率上升（我不太清楚这是什么意思） |
| 女神+シロ | 移动+10，回避率上升 |
| 女神+鬼神 | 攻击回数-2，通常攻击即为全体攻击，玖装备无效 |
| 女神+M剑 | 各能力均少许上升，战后胜利品入手率上升（移动+2，幸运不变，其余各+4） |
| 女神+白龙 | 防御+15，敏捷+5，魔防+15 |
| 女神+赤龙 | 攻击、防御、器用、敏捷、移动均+7 |
| 女神+青龙 | 精神+15，魔防+15，移动+5 |
| 女神+黑龙 | 攻击+15，器用+5，精神+5 |
| 女神+疾风 | 攻击回数+2 |
| 女神+魔神 | 攻击回数-2，通常攻击即为全体攻击，玖装备无效 |

除此之外，装备女神の纹章更可令攻击回数+1，提高治愈的戒指的回复点数，因此相当珍贵。

最后，还有点应告诉各位，战士+战士，攻击+30；魔眼+魔眼，精神+30。有些纹章是可以叠的，只要你有钱铁，戒指也有类似4枚。还有シロの纹章是从拳手口袋上拿到的，治愈の戒指只有4枚。迷宫盲中，狮子山是最难的，最好给妮姬装备唯一的一枚“全状态”の戒指，给魔法装备“狩人+疾风”的组合纹章，在打“魔殿圣”的时候给魔法换上“白龙の纹章”，（若你嫌麻烦，可以不换了“白龙の纹章”，由于隆法的移动力很低，一般都派不上用场）因为这种怪物既不怕物理攻击，也不怕魔法攻击，只能用魔法的“殿”子（消耗MP2点）和雷米娜的最大攻击魔法才打得动，这得看运气及道具不足的情况下，还是不去为妙。

因为这家伙除了第一回合是全体攻击外，后面只要攻击，都是单体必杀攻击。我第一次碰上这家伙的时候就已了5，这装备白龙之纹章，导致利下雷米娜一人，还是用逃跑的。最有心得的，还是希洛迪装备唯一的一枚“绝大防御力戒得”的戒指（因为装备这枚以后，每次碰上敌人，首先你就只有1滴血上阵，物理攻击和魔法攻击全部的命！）和一枚治愈的戒指，这样足够了。因为你只需第一回合用“ホの宝”＋200点HP，以后每回合均可以自动回复100点HP，而且龙之纹章翅下雷米继五只“回旋天舞翁”，它说Bye-bye了，我依然一丝回没费，MP也还剩许多，足够我再斩它2个回，五回的。真蠢机面面确实不错。
