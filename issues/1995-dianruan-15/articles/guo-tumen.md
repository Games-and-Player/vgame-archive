# 过屠门而大嚼

文／雪晨

年初电脑界一股冷场后，各游戏公司都准足了力量几乎同时在二月开始向 PC 玩家们发起了猛烈的攻势，正如“软体动物”所说，玩电脑游戏的趋势越来越进化，无 GAME 对大家宣；但如此批发般若不让人措手不及，让不少好 GAME 走进不实多的。

## 一、磁片游戏

哦！磁片游戏真是投资了！诸多美国公司已直播不再被磁片游戏了，而台湾几家公司的速度又一推再推，象大宇的《仙剑奇侠传》、《楚留超人 LEM》，佳密帝宇的《魔月幻影》及《征龙》等，让人望穿秋水却不见影影，看到如此一推再推的出品方式，让人完全搞不明白什么时候轮到自己，但代理汉化的《兰巴达王传奇》（一、二）这些冒险游戏的玩家留下了很深的印象，这次第三部的风格以日本著名的风格开发制作了《楚留香传奇》并已推出了第一部（直觉飞棋），带着游戏的头有十分钟的动画，虽然粗糙了一点，却把古龙笔下的侠士刻画

得淋漓尽致，该游戏盘套积也不好错过的，因为由华艺国际汉化的美少女普或游戏初开山柏柏——（半章）们于近期同时：比些差评，在笑工厂的台湾游戏不久于日本，天意鬼选出了八女神游戏出没失败的，以导得格选了这一点，开始向华艺国际看齐，大量出化 H — GAME（成人游戏）、至少 18 岁限游），从有好而整个 PRG 结合的（魔少纪录）、（甚少女奇特动物）和在日本连线数数也都在 TOP10 定常位（甚至如回盘上），三段益游戏要分为普通级和限制级两种，至于华艺国际如何对待 18 岁以住作品（少女少就），由来游戏的主角某某学园的一位驰[?]少年郑田仅，搭乘五位英雄向人的少女对可能阻止目前中间的是，沿长有大群到开，作此力的申请有大群，这就是原来大群，长大的超大群 DK 世界到达奇幻想，玩家知而且当游戏中出现大幅绑票游戏（马群），则来抬脸道，我们将来的游戏中，不知和以我们游戏地带同搭乘飞机到处取得。电脑将多少体电脑，相伴随着游戏的普及，加入 PC 游戏的后方的款会越来越妇！

## 二、光碟游戏

天哪！今年的 CD GAME 实在是太多了！多得让人难不过气来，且都向互动式发展，不都对硬件提出了高要求，386 已成强弩之末，486 — DX/33 + 4M RAM 已是最低配置——先别急，软件的要求远不止，硬件的价格也因声卡（音效板）价近千元，帮不见一行老产做对叫声音机（虚容机）也比卖一万二）

五月份的集中里里，“软体动物”曾提到过的 CYBERIA 是由 INTERPLAY 出品，运用 War Front 的 3D 绘图工具由 Silicon Graphic 公司（工作站）发展出来的，音效部分由著名声乐家 Thomas Dolby 合录，同样由 Tool Works 出品的

《The Dragon Tale》（龙的传说）有着更以于 PRG 结合的术技，两者对比之下，CYBERIA 的人物动作比较到客要活跃精彩多，没有 3D 中懂有的嗖嗖感，但要论说多的宏伟气势，清晰逼真的问题以及互动的影片效果，后者毕竟对于更好者，不过毕竟免对我是指了两把光盘，差老听举图把下这些游戏的图片，以中不难看出这两款游戏的区别。
