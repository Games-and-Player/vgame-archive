## 世嘉精品 MEGA CDX

文/特工兵

土星同世嘉在家用机上的重点战术有所转移，但却远未忘对 MD、MD-CD、32X 的软硬件开发，这款撒播而生的 mega-cdx 一体机将更显示出世嘉公司无与伦比的整体实力，因此以其精湛灵巧、功能完全的特点令受益玩家寻声。

### 小巧秀丽　功能完全

MEGA-CDX 约 20cm×13.4cm×5cm 的超薄流线外观外型不仅打破 mega-cdx 的机型不到 MD 的 3 分之二，超时间具有另一台 MD 遊戏卡专机功能，令人印象深刻如同以为为了一台 MD 遊戏卡专机（功能），一套以以为以为台以 CD 遊戏卡专机。在数量方面中电于 MEGA 大量画面上可以为为一台 MD 遊戏播放机，冷却而一套以为以为台以 CD 大量遊戏更大的数量，由于采用插上 CD-G 格式的游戏与 AV 端口对接便可使用，直接游戏电视如切换到 MEGA CD 模式，模式顶部还拥有相同红色的背光显示。

### 对应众种 CD 规格

MEGA-CDX G800（12.5MHZ）+Z80（4MHZ）的制式表面规格，可对应 MEGA-CD、MUSIC CD、CD+G 功能，另有 CG 专辑表 CD 格式，也属于超任 CD 的 CD-G，其 CD 碟体超薄全金。

| 参数 | 规格 |
|---|---|
| CPU | 68000（12.5MHZ）+Z80（4MHZ） |
| RAM | 64Kbit（音频） |
| | 512Kbit（视频 VIDEO） |
| BOOT ROM | 1Kbit |
| | 1Kbit CD GAME, MUSIC CD |
| | CD-G BIOS |
| 音频 | PCM 8 音 |

### 无可挑剔的设计

在游戏者对于其内部电路的 CD 与 迷你 MD2 +MD CD2 的进行了分别、分析、前以追求超 32X 以的 MEGA CDX 的 CPU 方面，以的 MEGA CDX 全球特别制式游戏卡甚至 9 头 AV 的端为功能，家用机享受之一体的功用外，另外为了迎合缺乏充析的游戏玩家，主机享受之一体的功用外，另外为了迎合缺乏充析的游戏玩家，主机享受之一体的功用，另为了迎合缺乏充析的游戏玩家，主机享受 1.5V 电池能提供游戏时所有个时间上享乐 CD 的游戏，音源极也音源方面 7 AV 立体声输出，音频方面还包括完全一体 AV 输出，含有配音选择器，另于 6 通道六音频游戏机提供个小时输出上享乐 CD 的游戏，音源输出面向方面 7 AV 立体声，这六输出，含有配音选择器，含有六道的输出 AV 音频 含有音频信号 RGB、S 端子手柄，有图案按钮上与 MD 一样使用，RGB、S 端接入手柄，在图案按钮上与 MD 一样使用。

正是如此 MG 和 CD 唱机的朋友，不消考虑一下今秋 MEGA CDX。
