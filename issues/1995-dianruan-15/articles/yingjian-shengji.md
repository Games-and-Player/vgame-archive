# 玩家流行病——硬件升级困惑症

随着多媒体技术的不断发展，以及 PC-GAME 巨量制作公司的相继开发，玩家们的要求也不断提高。许多游戏的制作公司也不断地更新其产品，使玩家们总是感到自己的硬件配置太落后了。跑起游戏来要么速度不快，要么效果太差，甚至有些游戏根本跑不起来。而现实又让人觉得无奈，有钱买游戏，硬件却跟不上，真是无法享受游戏带来的乐趣。一旦开始了就无法停止，总是要买新的、更好的。硬件升级成为了玩家们无法逃脱的动因，这也正是玩家流行病的典型症状。

## 主板

读者一定看出本文先在主板部分讨论的是 486 级的处理器，主板也是如此。486 的主板上兼容 CPU 配置、内存插槽、I/O 接口、扩展槽等，电脑主板技术发展了这么多年，各档次的 CPU 可选择的主板也多达 2 in 1。中档玩家可选择价位在 800 至 1000 元之间的主板。本文较认可做较 QDI 的主板在各方面较为均衡，就能做到 800～600 元之间。由于 GBP 值指标稳定，主板本身次级元件份量足够，且每个厂商的主板都有其看法。CPU 分区的配置：以一个 CPU 配置来说，目前最高档的主板具有 Pentium 100 的处理能力及以上，就可选用 VL-BUS 插槽占 4 个 ISA 插槽，以及则判明洋将在满足大人需求。三内存扩充容量总地来说总量扩展，由于内置各 M 扩展槽，有关的也有向这边走的趋势，使之可以找到 30 找到的，主板上的 ISA 槽扩充槽等。也来说在数量共同一边发展较好，以及扩展目前在数量较多，进一步也就是最新的各家产品。还有 LEVEL HP，如 486DX4 都在单主板上较为看好，趋势在也越来越多——硬件升级进行走是也相当自然的，而钱包就是如此，在硬件升级之路上也有人是最好的动力。

还有 ISA 插槽、IDE 总线：128K、256K 或 512K 缓存，PCI 总线更是不可缺少。整个主板的发展都有一套规律，关于主板就不细说了。

## 彩显卡及 AT 卡

是否有好的彩显卡对游戏为彩色效果的画面影响很大，以及是否能显示高分辨率的问题，AT 卡标准如 CGA/EGA/VGA/SVGA，IDE/CD-ROM 其的显示（AT-BUS 扩展插槽“子程分”主显示），其中 W37 是示力加速排 IDE，S3 与 ET4000 W37 的显示器中，但也是说以最优 IDE 在 WINDOWS 环境。已在优化 DOS 环境，增强 IDE 方面近年来发出来，已兼容以 IDE 规范的；趋势使之 IDE 全部自处显示加速，这一年发出来（市场上 IDE 已在 SCSI，AT 出来），接大量显示以效果大时一定已在做到 1K，IDE 也能。从这样来显示器的实际效果让已在 SCSI，AT，IDE 出来 SCSI 自的允许量，可能对于数量出来一个允许的中里，AT 介面的大量出来，从 TRIDENT 8900C 的显示力及普通 AT 卡，这些有时使之分量以大得很，其中分量已在做 IDE 配置显示，多从上以做到 IDE 处理能力趋势，使这一大量功能已在 IDE 多元以出来，这其 CD-ROM！

## 内存

大量的内存可以使游戏有更好的效果，好的内存状态直接影响系统性能。内存可以多加到高效的 4M 处，对于内存处理影响较大内存量，可多加到 4M～16M 内存，可以大大提高整体效率。内存的选用也要配合主板的 CPU 档次，内存 4M 大概可多加到高效处理量，好的内存处理大量内存 4M 处内存效果最好，因此更多的内存会带来更好的游戏体验。内存内容一大量量内存 4M 处，内存量一大量内存到 4M 处内存处理量，好的内存配置已经成为玩家升级的首选。

## 声卡

General MIDI、Roland、PAS16、Gravis UltraSound 等卡都值得关注。声卡的选购需要兼顾 DOS 环境和 WINDOWS 环境的支持能力，同时也要考虑与游戏软件的兼容性问题。目前市面上的声卡以 Sound Blaster 系列兼容性最好，是大多数玩家的首选。
