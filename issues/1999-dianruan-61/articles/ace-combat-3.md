# 王牌空战3 电子领域

> 游戏研究所 / PS / 刊页26-27

ACE COMBAT 3 electrosphere

厂商：NAMCO　类型：飞行射击　媒体：CD-ROM×2

【图】页面顶部展示ACE COMBAT 3的标题LOGO，下方为多张战斗机飞行画面截图，包括空中格斗和地面攻击场景。

## 近未来空战的世界

由NAMCO制作的“王牌空战”系列第3作“ACE COMBAT 3 electrosphere”，将时代背景设定在了近未来。与前两作以现实世界为舞台不同，本作构建了一个由巨型企业支配的近未来世界，充满了科幻色彩。

游戏的故事围绕着两大巨型企业——“GENERAL RESOURCE”和“NEUCOM”之间的对立展开。玩家作为联合国维和部队“UPEO”的飞行员，被卷入这场企业之间的战争。随着剧情的推进，玩家需要做出选择，不同的选择会导向不同的结局。

本作最大的特色就是引入了多分支剧情系统。根据玩家在关键任务中的选择，剧情会分成多条路线，共有5个不同的结局。这在飞行射击游戏中可以说是一个大胆的尝试。

### 机体介绍

本作中登场的机体数量众多，除了现实中存在的战斗机之外，还有大量原创的近未来战机。这些原创机体的设计充满科幻感，在视觉上给人强烈的冲击。

【图】多款战斗机的游戏内画面截图，展示了不同机体在飞行中的造型。包括现实系的F-22和虚构的近未来战机。

代表性机体包括：

- **XFA-36A**：高性能实验机，具备前掠翼设计
- **R-101 Delphinus**：NEUCOM阵营的王牌机体，外形极具未来感
- **X-49 Night Raven**：本作的最终BOSS机体，拥有压倒性的性能

### 任务系统

游戏共有52个任务，但在一周目中无法全部体验。随着剧情分支的展开，部分任务只在特定路线中才能游玩。为了体验全部内容，需要进行多周目攻略。

任务类型包括空对空战斗、对地攻击、护卫任务、侦察任务等，种类丰富。每个任务都有不同的胜利条件和特殊事件，保证了游戏的新鲜感。

【图】多张任务进行中的截图，展示了空中缠斗、导弹锁定和地面目标攻击的场景。

### 画面与操作

本作的画面在PS平台上达到了相当高的水准。战斗机的建模精细，光影效果出色，飞行时的速度感十足。云层、夕阳等天气效果也为画面增色不少。

操作方面延续了系列一贯的流畅手感，新手也能较快上手。而对于追求高分和高难度的核心玩家，游戏也提供了丰富的挑战空间。

## 总评

ACE COMBAT 3以其大胆的近未来世界观设定和多分支剧情系统，在飞行射击游戏中独树一帜。虽然科幻化的方向可能会让部分系列老粉丝感到不适应，但作为一款独立的作品来看，其完成度和创新性都值得称赞。
