## 圣战士传说(Dunbine·圣战士传说)

**机种 PS　厂商 BANDAI　类型 S·RPG　媒体 CD-ROM**

知道圣战士完全是从SRW开始的，而玩这个游戏也只不过是想作为即将发售的 + r1 + 的开胃酒以及充分了解圣战士来更好地玩SRW。不过这个游戏和我先前所想稍微有点不同。

因为不是在BANPRESTO直接开发下，所以 + r1 + 和 + r1 + 及 + r1 + 相比带有的SRW味道要少很多。战斗系统不是回合制而是所谓的半真实时间制，有点像 + r1 + 。因为战斗地图 + r1 + 化了，所以会有方向的概念，值得一提的是如果从背后进攻，这种方向的概念会在战斗动画中表现出来，当然侧面还不行，但这在 + r1 + 和 + r1 + 里都看不到。特殊场合会发生单挑战，比较新鲜。战斗动画很漂亮，但为没有台词出现，所以极其迫力。机体能力的提升不再是改数了，圣战士独特的昆虫式机子会踏踏搭乘者等级的上升而增强，又增加了开发的要素(虽然大多数的情况下可开发的机体是随剧情而增加的)。令人惊异的是 + r1 + 里像RPG一样可以理论上无限练级。

故事很容假，似乎是以动画为蓝本添加了主人公这个角色，同样作为被召唤而来拥有很强奥拉力的地上人。也就是说，主人公和我们熟悉的座间同等人在一个故事中。这样做就可以好看惯原作动画的人也可使玩家更为投入。

BANDAI的游戏向来似乎有不太好的名声，不过从 + r1 + 到这个圣战士我们也可看出它所做的努力。虽然 + r1 + 并不是一个让人惊喜的作品，但也是可以玩玩的。

## 口袋妖怪·金银(POCKMON·金银)

**机种 GB　厂商 NINTENDO　类型 RPG　容量 8M**

《金银》在内容上做了很大的创新，新妖怪、新地图、新剧情，大部分是原创。在妖怪种类上也是空前庞大，在原先150种的基础上多了100种之多。总数达到了250(怪数)！并且每一种妖怪还有了睡雌之分。但故事大概流程没有什么变化，主要就是主人公在旅行中收集训练自己的妖怪，打败N个BOSS，与宿敌的决战，并完成妖怪图鉴。

我个人认为《金银》的最大卖点就是它新加入的系统。例如：模拟时间——游戏有自己的时间制，并有白昼、星期之分。这个模拟时间对收集妖怪有很大影响，如不同妖怪根据各自习性在不同时间出没；电话系统——主角可利用移动电话与游戏中的人物联络，如与交过手的妖怪训练员约好下次决斗(对练级和攒钱有很大帮助)。还有就是繁殖系统这项新设计——玩家可用雌雄妖怪配对生出小妖怪并加以培养(很有趣的系统)。像这类新添加的系统还有许多，它们不仅丰富了游戏内容也增加了游戏性，但同时也使《金银》比前作略显复杂，可好在对游戏的整体影响不大。《金》和《银》的区别并不是很大，但也有其各自特点，还须玩家自己体会。

在游戏画面上，《金银》可要比前作强了不少，必竟是可同显56色的彩机作品，很明亮，画面精细了许多。游戏音乐质量比起前作来也有相应提高。总体来说《金银》在掌机游戏中属上成之作。如果你有GB的话，赶快去买盘玩玩，如果没有GB，那你不光要买一盘《银》(《金》)，还要同时买台彩色GB，我保证你会很满意。

## 爱与破坏(LOVE & DESTORY)

**机种 PS　厂商 SCEI　类型 3D-ACT　媒体 CD-ROM**

一直以为SCEI做动作游戏要比文字游戏拿手得多，这个 + r1 + 的动作游戏证明了这一点。

桂正和的人物设计相信大家都没话说，除非是对这位大师深感精疲的朋友。因为桂正和的招牌毕竟是恋爱漫画，所以这个游戏有很强的恋爱因素便不足为奇，只要能顺利在时限内将TARGET DELETE掉就会出现与Q-TRON交往的动画，并有选择来提高好感，战斗中Q-TRON的身份类似于Fatima，辅助操作，也正因为这样，可以在战斗中不断与她们联系以提高好感度，详情让大家去自行游戏理解。

游戏最值得称道的恐怕是键位的设计，在战斗中完全使用□、△、○、X四键恐怕是PS动作游戏的首创吧。左摇杆控制移动方向，右摇杆控制转身和视角，L R四键则分配了跳跃、联络、攻击与侦察器，充分利用了震动手柄的键位安排，将PS手柄的优势发挥到了极致。也许在一开始会有不知所措，但习惯后就会深刻地感受到键位设计的合理性。

游戏最大的缺点恐怕在于流程过短以及可以追求的Q-TRON实在太少，根本不过瘾嘛，而且可惜的是各关数内人的弱点太过明显所以难度也较低。何时会出2代呢，不要告诉我是在PS2上！

## 格斗之王R-2(KOF R-2)

**机种 NGP　厂商 SNK　类型 FTG　容量 16M**

SNK在掌机上创作的KOF系列大概也就只有这款KOF R-2了吧！人物是以KOF 98为蓝本制作的。虽然主机的键位少，但不影响每个人的出招。这款KOF R-2中的人物的出招是和街机上的一样的。而且以终极卢卡尔、真大蛇队95草薙京等此类的隐藏人物也可以直接选了。

游戏中的角色普遍IQ值比较低。所有人用跳跃重拳·必杀技·超必杀技这种的连续技，电脑连防都不会防。就连草薙京的近身下蹲重腿都可以形成 + r1 + (其实电脑不会蹲防)。

在系统方面，本作继承了KOF系列传统的紧急回避和CG攻击/攻击，KOF 98的转盘选人系统，但优势系统却没有继承。不知是SNK的工作员忘记了，还是他们想偷懒？

本作的结尾很搞笑，在打败了卢卡尔之后，他望着狂笑起来，然后拿出一个大炸弹。画面一转，一般耀眼穿的船像泰坦尼克一样，沉入了海底。

## 牧场经营的卡牌游戏

**(牧場経営のボードゲーム　うまぽりい)**

**机种 PS　厂商 KONAMI　类型 BOG　媒体 CD-ROM**

这款游戏真是另类的，不同于一般的赛马游戏，而以大富翁桌上型游戏的方式来经营赛马，整个地图都绕着马来进行，连LOADING画面都有马的出现，十分有趣，风味较新鲜，很适合全家一起玩。刚开始并不是很能接受，待熟络之后反而爱上这样的赛马游戏。行动格数并不是很多，但麻雀虽小五脏俱全，没有的都有了。其中包含了牧场的购入、马匹的生产调教等育成功能，当然也有银行的借贷、牧场或其产业的拍卖竞价等。结合牧场经营和大富翁的游戏，可以让玩家体验成为真正的马场大亨，必要时还得向其他玩家借钱，否则是无法生存下去的。

本作在人物设定上有蛮笑的脸孔，当然也少不了漂亮妹妹和帅哥噢！在 + r1 + 处理方面更是一绝，虽然没有人物的语言背景，但不需玩家做什么举动，因一些细节的部分，都配上了有趣的声音来衬托其可爱之处。

当作轻松小品来游戏一番，还蛮不错的！若您是 + r1 + 的FANS的话，不妨试试本作！

## 心跳纪念品2(ときめきメモリアル2)

**机种 PS　厂商 KONAMI　类型 SLG　媒体 CD-ROM**

在玩过这个游戏若干个小时之后，终于明白了KONAMI为什么这么迟才推出这一取得巨大商业成功的游戏的续作，因为KONAMI明白这一续作将是设法再超越 + r1 + 了，它将成为 + r1 + 成功的终结，为这一创造出奇迹的系统画面一个并不完满的句号。

在它爱好SLG变遍成灾的今天，即使反观 + r1 + 也很难找出什么成功的必然原因。设定各种不同个性的女孩、固定事件、送行游戏、涌真的人声、漂亮的CG，早已被其它同类型的游戏用烂了。在欢乐登峰的系统相对上的，是无法在代替时给我们带来的那种感受的——用一件相识见父的看起相知的女孩们，这种 + r1 + 的模式无疑是质不适于或是SLG的，却包含它的作品多半是 + r1 + 而 + r1 + 的玩家带来的是相当的失望(笔者的感觉亦是如此)。

但II也有它的闪光点，即就是在技术上具有重大意义的 + r1 + 系统，这一系统在刚刚开始就给玩家的笔者相当不已，能够声音应合并合合成出类似真游员的人声，虽然只有仅仅的6个音节，但在游戏中实到这样的效果是太令人吃惊的。试想一想在玩游戏里有人不停的喊你的名字，那是多么样的感受。如果把这一系统应用到其它游戏里，可调焦和亮不可限量。

总之，去年底的大作效应至今余数，许多游戏还未通关，所以要想让我对本作产生想玩的念头，实在是难。

## 美职冰球~钢刃2000(NHL Blades of Steel 2000)

**机种 PS　厂商 KONAMI　类型 SPG　媒体 CD-ROM**

作一看这款游戏，就知道是美国KONAMI的作品，因为无论从包装设计、操作界面到现场解说都是美版游戏的风格。

但在体验了游戏的爽快感和真实感之后，我确信这是KONAMI的作品。游戏中3D人物虽略显粗糙，但在动作模拟上却毫不含糊，如滑行、争球、摔倒甚至斯打都真实自然。游戏的操控也较简便，容易上手，但要打出精彩流畅的配合则要多加练习，这大概纯于冰球运动本身的特点：拚抢激烈，攻防转换快。游戏时场上你争我赶，人仰马翻，冰花四溅，加上解说员慷慨激情的解说，场面更是火爆。打上一场比赛下来，大呼过瘾之余，恐怕手指也因不停按键而酸疼不已了吧(笑)！？

尽管如此，这款游戏还是有不少缺点，如游戏模式太少，仅有三种，至少应该加上训练模式和全明星赛模式(这可是NHL的传统项目)！在细节上也有些马虎，如体育馆的场景太过简陋，进球后该有什么狂喜的表现(只是站在原地欢呼？)，回放镜头太短等等。

总的来说，这是一款还算不错的SPG，有兴趣的玩家不妨玩一玩，体味冰球的特别魅力吧！

## R·TYPE DX(R·TYPE DX)

**机种 GB　厂商 IREM　类型 STG　容量 8M**

随着彩色GB(GBC)的推广，大量经典GB游戏被厂商推出怀旧版，其中当然少不了R·TYPE这样的经典之作。

在这一R·TYPE合集中，后三个R·TYPE作为强化版是GBC专用游戏。其画面效果着实令人大吃一惊；特别是R·TYPE II Color后四关，效果甚至超过了MD！看来，只有GBC专用游戏才能将其机能真正发挥。GBC的双倍主频、四倍内存和其它一些特效机能也只有在专用游戏中才能有效使用。至于GBC的同屏显示56色，恐怕也只有在这时才能达到。实际上，在GB/GBC共通游戏中，GBC同屏最多也不过16色！这一点玩者可自行验证。从GB到GBC，机能上其实有一个飞跃。

R·TYPE作为经典二维射击游戏，有着优秀的角色造型、独特而引人入胜的外星生物世界观、富有感染力的音乐和一流音效，玩者很投入。游戏中有不少狂澜而激烈的场面，例如二代中第三关宇宙巨舰子爆之后巨大的战舰炮弹之间的意觉，令人难忘。游戏风格上，许多地方已开飞行机的活动空间都限很狭小，体现出家用机原创游戏和街机游戏的不同风格。

R·TYPE的GB版可以说是本身的起点就很高，这次的R·TYPE DX借助GBC的机能使这一系列作品几乎近完美，是值得GB玩友收藏的经典之作。

## 狂热节拍GB(BEAT MANIA GB)

**机种 GB　厂商 KANAMI　类型 ACT　容量 8M**

早在看(绝对GB读本)时，就已对此游戏流口水，这下终于买到了。回到家迫不急待地玩起来，可是这并不是一款容易上手的游戏，连最EASY的都过不了。不过一旦适应了，就会体会到这个游戏的乐趣。

这KONAMI可真厉害，如此制作的游戏总是很让人惊叹——从《心跳》到(DDR)，无不让人为之叹服，虽然为了专利权和其它厂商大动干戈，给玩家们留下了一些不好的印象，但这公司的创意和游戏制作水平的确值得肯定的。

这款游戏选GB音乐机能几乎全部发挥，与其它GB游戏简单而且单调的BGM比起来，大不相同是本不像是GB游戏；歌曲十分耐听，十五首普通曲目加上五首隐藏曲目，从GB的角度讲已经很不少了，曲也十分丰富，其中有3首重新编曲的古典赤十分精彩(第3首为随机！)。游戏进行时，右边动画曲目的动画面更十分精彩。此游戏对应GBC，用GBC游戏时会看到一些黑白机上没有的特效哦。另外，本游戏还支持对战。

这款游戏中有两首歌难度可评为SSS级！其中有一首歌对了四星，就准得不像话！！一般玩家根本坚持不了多久。可惜。(注：2代已经有卖!)

## 计算机游戏文化何去何从

文/上海　比尔·拨死

看了贵刊一月杂志中有关计算机暴力后一篇文章后，感想颇多，故写此一文，与小编交流一下思想。

电子游戏发展至今已经形成了一个庞大的产业，在美国，游戏业已力压电影业，成为了老大。被称为 + r1 + 的它正以迅猛的速度进入人们的生活。然而，就如同所有新生事物一样，电子游戏也有它的不良一面，即暴力、色情等内容充斥其中，对青少年造成了较大的影响。这一问题随着计算机图形技术的日益发展而日益体现出来。《计算机暴力》一文用大量的事例为我们展现了这些不良面。

对于这个问题我认为应该一分为二来看待。即要充分认识其中不良问题的影响力，又要肯定游戏文化中好的一面。不可以偏概全的因一些暴力游戏而否定整个游戏文化。

以前的社会没有电子游戏，但枪杀、抢劫、强奸等犯罪问题依然存在，而现在社会的这些犯罪行为也没有因为电子游戏而猛增多少个百分点。社会问题与电子游戏文化之间并没有多大关联。

当然，日益严重的美国校园枪击事件或多或少与电子游戏有关也是不争的事实，但那毕竟是少数，大多数情况还是良好的。

但另一方面，也不能说《计算机暴力》一文的作者是杞人忧天，电子游戏确有不良影响是不得不承认的。我在这又要明确一个观点，即应如何解决这一矛盾，在下认为这完全在游戏制作公司以及社会管理体制身上。只有制作出优秀的游戏，我是指能起和极一面的那种游戏，这样才能对青少年起推动作用而不是教其学坏。离开了社会监督同样也不行。

我是从小玩游戏长大的。从7岁到22岁，玩龄15年。不能说是骨灰级也能算是资深玩家，从雅达利到DC到刚升好级的PIII+128M+GeForce256，玩过的游戏千千万万，对游戏应该说也有点发言权的。在我心中优秀的游戏很多，这也是我对游戏文化充满信心的源泉所在。有些优秀的游戏如FF系列等，都是相当不错的。如果哪个敢说FF8只是一堆垃圾，我立马将自己所有的游戏主机和PC一并砸了(笑)。在我心中，它们丝毫不比电影中的所谓的大片差。优秀的游戏不会去带来负面影响只会帮助人们愉快地生活。譬如火热的DDR不就可以帮人们边锻炼边游戏吗？胜利十一人不就让许多青少年对足球技战术有了更深刻的了解吗？帝国时代不就让我们经历了一次文明的演化史吗？(其它优秀游戏恐不一一列举)虽然这一切只是虚构的，代替不了现实。但如果哪个人说这些都有害，我是决计不肯同意的。

《计算机暴力》一文中所提到的一些游戏我也玩过，有些确实不宜给儿童玩。但指望所有游戏公司都能像史克威尔一样是不可能的，小公司只能出这种游戏招生意，作为一个成年人，紧张的工作学习之余，我也不希望一些适合成年人玩的刺激游戏被禁止(现在再让我等十几岁的青年去玩马里奥是无论如何也提不起兴趣的)。那么问题的关键就在于管理：让少年人避免过早接触暴力内容。在这方面，西方发达国家做的较好，已经限制级别了，这可以使不同年龄段的人找到适合自己玩的游戏。虽然执行情况并不乐观，但我相信这是个好办法，会有成功的一天。再看我国……不提也罢。

综上所述，游戏文化做为一个新兴文化，必将以其强大的生命力在21世纪发挥重要的作用，只要我们加强管理，弘扬美好的游戏节目，电子游戏一定会有美好的明天。
