##  + r1 + 顺调进行！

本期的投票量保持了强劲的势头，数量超过了一万三，说明大家的积极性还是很高的。另外，由于许多玩友是第一次参加投票，对于一些细节不太清楚。主要表现在书写游戏名称的时候，不但没有按照本刊给出的名称，而且还不注明机种。这样的选票不但无效，而且还给统计工作造成大量不便，请玩友一定要注意。举例来说，有些玩友投票时只写 + r1 + ，令我们统计时无从入手，或者写着 + r1 + 有三款游戏，所以投票时一定要写清游戏的副标题。具体注意事项可以看本刊去年排行榜栏目中所刊登的那些，只有规范的投票才能起到支持自己喜欢游戏的作用，才会有更多 + r1 + 和 + r1 + 两款，而且加起来也只不过三十几万。算上没有进榜的几款随主机一起发售的PS2软件，也离主机的销量相去甚远，给人感觉是有一半购买主机的人根本没有买软件，仅仅用PS2当作DVD影碟机了。三月底，PS2新一批的软件也发售了，其中包括 + r1 + ，不知它的销量能不能让NAMCO满意。

在PS榜上，本期居然有两款新游戏上榜，分别是 + r1 + 和 + r1 + 。虽然游戏推出时间已经不短了，但是第一次登榜，说明这样的游戏耐玩性很好，会逐渐吸引玩家的注意。上期说无法预测 + r1 + 能否保持前五名的位置，本期 + r1 + 在众多球迷游戏迷的支持下，又上升一位，变成第四名。最近听说各地都有 + r1 + 的比赛，看来这款足球游戏真正得到了玩家们的认可。在累计榜方面， + r1 + 快速上升，从第22名上升到第16名，相信在过一段时间，名次会到达十名之内。ENIX的 + r1 + 虽然仍然没有登上PS榜，但却登上累计榜，不知下期它是否能登上PS当月榜。

日本方面，PS2软件的销售情况已经反映出来了。虽然PS2主机在发售三日内售出近100万台，但在前三天的软件方面的销量却不如人意，登上销售榜的PS2软件只有的中奖机会。
