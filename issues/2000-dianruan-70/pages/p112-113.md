## 特报：皇家骑士团外传——赛诺比亚的皇子

**伝説のオウガバトル外伝　ゼノビアの皇子**

**NGP　厂商：SNK　类型：S·RPG　容量 16M　发售日：2000.4.27**

曾在 + r1 + 第14章出现的圣王特利斯坦，现在摇身一变成为 + r1 + 的主角。在这款以特利斯坦为主角的游戏中，玩家可亲身体验主角冒险及学习成长的过程哦！

让外界投以相当期待的S·RPG游戏 + r1 + ，终于决定在一个月后于NGP上发售了。在这回的报导中，我们就要让玩家对新登场的角色有更深入的了解。

本作承袭了 + r1 + 的基本系统，并充满了原创要素。新的角色及故事固然值得注意，但是NPG版的要素则可使用通信�的线的对战方式。玩家不但可以和自己培育出来的角色对战，游戏还设定成�的方可以将对方的角色抢夺过来，整个对战过程真的是非常惊险刺激，因为要是输了，你是会因此而失去伙伴的哟！

此外，在对战胜利的情况下，还有可以得到道具的设计，以及预计会在这款 + r1 + 中出现的新要素……玩家们可千万不能错过喔！

### 新角色陆续登场！接下来的故事是！？

由13个复合构成的 + r1 + 。玩家可在散布于地图上的都市和教会中打听到各种情报，就以破关为目标吧！

### 为了相信还有明天

由女王安朵拉所率领的海拉纳霸占了整个大陆，并将其改名为费提基内亚……之后的915年间，各地恶声载道。治安也极度恶化。就在此时，半岛上一个名为艾斯特亚的村庄因为内盗贼横行而苦难。原赛诺比亚骑士团之一的艾斯特拉达见到村内如此窘状，便决定带着他的学生们一起去讨伐强盗。艾特利达的学生，一位是剑术相当高明且性格开朗的凯恩，另一位则是自幼失去双亲而由妈妈芭尼雅养育成人的少年特利斯坦。在击溃了一个又一个的盗贼据点后，他们隐隐感觉到这些盗贼的背后存着不为人知的秘密。

**角色介绍：**

- **凯恩** ——特利斯坦青梅竹马的好友。个性讨人喜欢，脑筋也相当灵活。
- **艾斯特拉达** ——原赛诺比亚骑士团的一员。因为受托于妈妈芭尼雅而教特利斯坦剑术。
- **阿克艾利亚斯** ——自从懂事以来便失去了父母之爱，由名为芭尼雅的奶奶将之养育成人的少年。
- **娜娜** ——北方利希特佛洛斯国的公主，是一位行事非常率直任性的女孩。她的父亲利希特佛洛斯国一也对这个女儿十分头疼，但她还是有着身为王族应有的自觉。
- **妖术师** ——据说能够使用魔法让人复活，并操纵他们。宣传部他这样做的目的是什么？妖术师是一个被重重谜团包围的人。

### 迎向波澜万丈的旅程——特利斯坦、前进吧！

**POINT 1 移动路径还可设定为3个所在地**

编成队伍，一切准备就绪后就可以出发了！还可以将一次移动目的地指定为范围内的3个所在地。先考虑好各个队伍的移动方向，再将旗子插在目的地突然被攻击。玩家最好先预测敌人的行动，仔细思考队伍的配置场所再做移动。此外，在此次的外传中，玩家在各式各样的角色加入游戏后，就可以培育战斗的团队、进行战斗。只要解除地图上的各个都市和教会的危机，打倒头目，这样便算是过关了。在这里，我们为您介绍有关于本作的基本流程及系统，除此之外，在携带型主机上映力满分的战斗画面也很令人注目哦！

**POINT 2 掌握命运的混乱计量**

 + r1 + 的最大特征就是，在画面右上方会有个 + r1 + 。藉由这个民心量表，玩家便可知道自己所率领的军队在民众心中的评价究竟是如何。民心量表也会随着玩家行动的增减而变化。而当据点的危机解除时，玩家所得到的塔罗牌种类也是改变民心量表的因素之一。另外，民心量表的数值高低同时也决定了故事和往后结局的变化。玩家若是想看到所有不同的结局，那么保育教民众的期待与支持……等事也就非做不可了。

**POINT 3 发现隐藏的据点了**

游戏的地图上藏有不为人知的隐藏据点以及埋宝的地点哦！在这个隐藏的据点里，不但会有可成为同伴的角色，还有握有重要情报的人物存在，玩家一定要注意。千万别忘了把它找出来。而在行进途中若是经过宽广的平地，这也是值得怀疑的地方，玩家可以利用队伍在地图各角落间移动，以便找出藏宝的地方。

**POINT 4 战斗**

战斗的进行是采用基本的半自动战斗模式，玩家只需要给予全体团队指示，再选择 + r1 + 就可以了。而且塔罗牌的使用更可以达到间接地帮助我方军队的目的哦！

### 左右命运的塔罗牌

得到塔罗牌后，玩家便可随时在往后的战斗中使用。随着玩家所拥有的塔罗牌种类会产生变化，这时的玩家就须注意不同的民心量表而做放弃。一旦接近被扣，会出现被受自控制的准备的就会巨目标示，战斗就会强迫结束。使用塔罗牌就还可以发动强力的魔法！玩家若能善加利用，在战斗中会有很大的帮助哦！
