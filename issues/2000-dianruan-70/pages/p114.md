## 特报：萨尔达传说——奇迹树果之力之章

**THE LEGEND OF ZELDA　ふしぎの木の実　力の章**

### 使用神奇的树果　让众多的谜团明朗化

这个 + r1 + 系列，会因为从哪一款游戏开始游戏，而使剧情和事件产生变化。这一次，要介绍给玩家使用四季之杖改变季节来冒险的 + r1 + 最新情报！这款游戏，要使用 + r1 + 和 + r1 + 等具有各式各样效果的树果，来解开谜团。而且在 + r1 + 里，已经知道会有同伴登场来助林克一臂之力！值得好好注意。

### 季节错乱的镇！

林克来到突然变成多天的小镇。这个季节错乱的小镇，能不能回到原来的状态呢……？

### 用亚奇奇之实来点火！

这个游戏中登场的神奇树果，每一种都隐藏着各式各样的效果。而亚奇奇之实就是拥有炎之属性的神奇树果。所以假如发现烛台等物品的话，请马上使用亚奇奇之实来将火点上。

**发现烛台的话使用亚奇奇之实吧！**

使用弹弓，可以将亚奇奇之实射出去，将敌人变成一团火球。即使是手甲不到的地方，也可以像这样的使用弹弓！如此一来，就连遥远的地方，也能将火点上。

### 用好痛呀果实来解决敌人！

 + r1 + 正如名字所表示的，是拥有攻击属性的树果。比起一般的攻击，好像能给予敌人更为强力的伤害。但是 + r1 + 好像有着什么秘密。这个秘密到底是什么呢？

一围住在 + r1 + 四周的凶恶怪物们。这个树果所隐藏的秘密是……！？

**用弹弓来打倒敌人！** ——使用 + r1 + 的弹弓攻击，威力真是超群！从远距离也能确实打倒敌人。

### 林克同伴登场了！！

在这里为各位介绍，会在林克的冒险过程中帮助林克，值得信赖的两位同伴。一个是以强力的龙卷攻击将敌人打倒的姆修，一个是让林克坐在背上可在水上移动的威威。藉由他们的帮助来解开谜题吧！

- **姆修** ——长了翅膀的熊(？)一姆修。能载着林克到处移动。
- **威威** ——坐在威威背上，即使是水中也能轻松的移动。行动范围变得更广了。

坐在矿车上的林克。有许多分歧的路线，到底会到达哪里呢？

所谓勇者的洞窟是？
