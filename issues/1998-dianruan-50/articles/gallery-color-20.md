# 大墙画廊 彩版 VOL.20「夜色」

> 主持：SHADOW PHOENIX——读者投稿游戏同人画彩版专栏

## PDF p.48 / 彩插 插6

凤凰在廊中舞罢三巡，接着又是我向大家许下的誓言，这回又该是什么了呢？

经过一个假期的休整，许多朋友又开始紧张的学习生活了吧！一时间不知该对大家说些什么，耳边萦绕着北京摇滚群星翻唱邓丽君的《夜色》，尽管窗外还未变暗，但一股似水的感觉涌上心头，好像上期对大家的誓言还未许尽似的，别被我的誓言所迷惑哦，否则…（开玩笑，这一次我是当真的哦！）

画稿方面虽有源源不断的画稿“袭”来，但优秀画稿确实不太多，特别是“还我河山”的画廊接力行动的来稿少得真是让我心痛，大家请来捧捧场吧，不然我会……

夜色正阑珊　微微萤花闪闪　一遍又一遍
轻轻把你呼唤　阵阵风声好像对我在叮咛　真情怎能忘记
你可记得对你许下的诺言　爱你情深意绵

——廊主SHADOW PHOENIX

**《相约九八》** 辽宁抚顺 郑春明

据说左下角的人是你，看不出来你究竟是男是女哦！画面的色彩处理有些乱，否则也许5星大赏非你莫属了，可惜呀！（麻POWER：★★★★）

## PDF p.49 / 彩插 插7

**《格斗之王》** 南宁 吴奇

好像三人的面容都其美似，难道三人都有吸血鬼的血统吗，否则为何都有颗尖尖的獠牙呢？不会是《恶魔拳皇》吧！（麻POWER：★★★★↑）

**《KING》** 哈尔滨 靳全超

搞不清楚究竟在吃什么的京样子很纯[?]，笔法不错。（麻POWER：★★★★）

**《侍魂之帅哥靓女大串烧》** 安徽蚌埠 李博

你说花脸酷的霸王丸是因为遇到了厉极丸了，其实他心仪女子是静啊，这“霸王别姬”的传说在他身上可可应验吗？！牙神却和查姆查姆[?]在寒地！（麻POWER：★★★）

**《侍魂海滩版》** 秦皇岛 詹勇

很搞笑的一个版本，远处海面沉下的是TITANIC吗？被7位大汉压倒的加尔福特居然“土遁”到了卿可靠身边，而右京却看《大友》不看咱们志……（麻POWER：★★★★）

**《真侍魂·永远的霸王丸》** 哈尔滨 任重

尽管你打印的效果真算是不错，但这《真侍魂》的结尾画面似乎难逃抄袭嫌疑，究竟能加强创意，否则霸王丸是无法FOREVER的。（麻POWER：★★★）
