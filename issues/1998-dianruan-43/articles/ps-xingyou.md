#### 开发访谈

**我们对“（FF）”日后总相同的前景感观如何？**

我们打（FF）好日时当初同样投入了一大批努力，不过那时的 SFC 向游戏市场已推进了大型厂，正当此时，我们也以 GG 部分合力投入，但如今这 2D/3D 画面结合的强大局面，我们我们实际上是以 2D 发了一批全新制作力，我还有心思就是至今仍像往昔 RPG 给的那样玩法大力回归到 SFC 的当时玩法，这样我们两方都能够贯彻原来的理念，也并非要偏向这种 3D 化的倾向。

**现在的次世代游戏均为常注重画面？**

今年中，对镜头以及 CG 部分有我们了、我算是 ENIX，并非一贯着眼于 CG，其实就 3D 大格局来，2D 仍旧保持着的这部分，同时、如果次世代看看，镜头看看——我们之所以深思这个问题的，是出于我们这款新作的需要——确认的方式是完全不同的，我们目前我们在 3D 上已经做出了有意义的布局格局，只在 2D 方向也不想落后，因此于此进行的同时，两方以上的立场相互联合。为了最终实现精彩演出效果，以 2D 传统方式——又在全新 CG 手段中，以其最终精彩方式，以大手段联手运用后，最终集中打 2 种手段总力、从而实现了各自 2D 之最精华的那一部分的格局，我们这次相当的于有了把握！

**对画面处理以什么样的方式来解决呢？**

我们这次是 2D 和 3D 同步技术上的研发，2D 中 BG 技术当然已经是成熟掌握的，而此次是两者合并运用——以 3D 的运用，可以做到 2D 时做不到的那些效果，其实在 SFC 时已有所尝试，而今以新的结构以新的实际方式去实现——当然如果看看 FF VII，看来那是 3D 完全做到头了！但我们的是另一方面，还是以 2D 保持了、用 2D 方式去营造出氛围，这本来就是我们的拿手好戏的本质吧。

**整个次世代均为高质地展开的局面如何看？**

就 SS 来看，实际上我们倒也注意到了 SS 时代，倒现在，大 RPG 那时已经非常盛行了，而今 PS 时代已经由 FF VII 之类大作开局，而我们也想在此联手 PS 系！当时SS 上的次世代 RPG 的游戏本身已几乎等于 100%上的这个阶段，从此之中——也就是说我们这款游戏出现以后——那个时候已经全部都充满了游戏来玩的玩家们了！然后此后，在 97 年某月，时间到了 SS 出了这款游戏，此后在 98 一年之中确认投 PS 市场一批，在此之中，已有确认之处，使到 PS 领域以其一种 RPG 的品格的玩家数也有了一定的积累，所以接下来就是——整体认为是时候了！然然，这也收录了在正在进行中的玩家，大家的玩家之中还有不少，此后大家以玩 RPG 乐此不疲的那些玩家也彻底成了稳定力量。
