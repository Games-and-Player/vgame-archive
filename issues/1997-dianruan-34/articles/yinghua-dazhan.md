# 樱大战

> 次世代天空 / SS AVG+SLG SEGA 2CD / 第34期 PDF p.24-28

## 偶像剧团帝国华击

如果评选 96 年度最佳游戏，《樱大战》绝属于最有资格参与竞争的，它于去年销售超过百万（アドベンチャー上的最高记录），而且又将于今年带来全新续集，真的是名不虚传啊！此次为大家带来这款游戏（OF 篇上上篇的攻略），能让大家最直接最细致地了解这款游戏，更加系统地掌握它的精妙！

**SS　厂商：SEGA　题型：AVG+SLG**
**光盘：CD-ROM×2**

## 超强的制作阵容，令游戏具有超一流的品质！

《樱大战》的购买量真是惊人。通常一款普通SS游戏能卖个十几万就不错了，但《樱大战》一上市就售出了六十万，其后更是持续热销，连续三个月（アドベンチャー游戏的第二位），可以这么说，《樱》是历史上最卖座的SS游戏，也是有史以来第一款突破百万的AVG游戏——这实在令人难以置信。

那么，《樱》是凭什么征服了百万玩家的心的呢？当然，最重要的是它的内容，但更进一步分析的话，它的成功有以下几大原因。

①制作班底极为豪华，制作人为红®本广嗣，角色设计为国内著名漫画家“藤岛康介”，音乐为田中公平（其优秀的音乐令人陶醉）。可以说，这些人都是在其各自领域内顶一流的人才，汇集这些大师的结晶，当然不会差到哪里去了。

②从最新的游戏技术上分析，像“あぁそ父よ”这样的感叹词及特殊符号（涕泣的拟声词）并用于游戏之中，还加入了多支音乐和精彩动画，使游戏在感情表达上有了一个新台阶。《樱》中出现真人动作，给人一种亲切生动的真实感受，玩家在玩游戏的同时，更多了一份当明星的体验！

不管怎么说，但制作人们的确有极深的根基，他们在游戏里的技术上做得非常精，每一段情节都能令玩家十分投入，每一个地方都能体现他们独特的风格，其中无不包含他们的苦心与创意。这款划时代的AVG+SLG成功地让游戏进化到了一个全新的境界。

此外，游戏的另一个引人入胜之处在于它结合的AVG+SLG形式（通常AVG部分与SLG部分是相互影响的），由此让玩家感受到历史和文学的双重艺术享受，从而留下了永恒的印象。

上述的这些特点，其实还不是最重要的，让百万玩家钟情的最大原因是——美丽动人的剧情与多姿多彩的女性角色，这一点是成功的最主要原因！

（《樱》）中扮演的主人公“大神一郎”将在帝都（东京）服役，途中遇到了帝国华击团，这是一个集合了多方优秀才能的由女性组成的歌剧团，大神一郎将负责带领她们进行战斗，与此同时也和这些女性在日常生活中有着不少有趣的情节，让人在体验激烈战斗之余，也感受到人情世故的温柔一面……

这就是这款划时代的AVG+SLG，不愧是颠覆了时代的作品！

## 攻略基础

接入游戏，必须要将自己最开始的部分设好——

其次，日文版本上你会有两个初始 SAVE 档案供你选择，1 和 2，只有在初始菜单里才能正常进行选择，当然进入游戏后也有 SAVE 位置，记得 SAVE 时不要选到之前的档案。（最初最好 SAVE 一遍，方便之后对比用）

对话中，有些对话会允许你用 LIP（Live Interactive Pictures System，即互动式对话系统）选择的，这时你就能通过对话选项选择自己想说的话，不是一般的游戏，此处的对话也影响到后面的故事走向，所以各位在选对话时尽量选择「OK」出现的那个，这时结果通常不会使后续游戏剧情出现大的偏差！

## 攻略基础（战斗篇）

游戏分为 AVG 和 SLG 两个部分，AVG 部分负责推进故事情节，而 SLG 部分就是战斗了。

每次出战之前，先要在帝国剧场里准备好出征队伍，每个人的装备及状态都要留意，因为战斗时有人倒下会影响故事走向。

## 第一话　帝都·花之华兵团

我们的主人公大神一郎奉命在帝都服役，踏上赴任之路，乘坐电车时，有一位少女（真宫寺樱）在车上站立，车行中发生了一段小插曲，由此引出了接下来的故事……

**帝国华击团**
冲绳帝华团
弹翼组迎战队——茜组
奋力联合歌舞台联合会
进击上野公会全

真宫寺中将

（正文开始）

初次出发直接在帝国剧场中，女孩们在排练，这时有一段游戏介绍，大神一郎作为新来的队长，必须跟每一位队员打招呼，先和大家交流，然后才能进入到正式战斗任务。

“欢迎大家……你们都是帝国华击团的成员了。”大神一郎道，“希望大家能配合我。”

“好呀，大神队长——！”真宫寺樱首先响应，笑着打了招呼。

“哦……我来了。”大神一郎望了望大家，“希望我们能合作愉快。”

“真的吗……我……也没想到……”旁边的李红兰也开口了，低着头有些害羞。

真宫寺樱向大神走过去，将一封帝国剧场的信交给他，那是一份出战的命令……

“怎么，这么快就要出战了？！”大神一郎读了信，“我才刚来……”

“没关系的——我们一起！”樱说，眼睛里充满了斗志，“大神队长，帝国华击团从来不退缩！”

“那好……既然如此……”大神一郎也鼓起勇气，握紧了拳头，“让我们出发吧！”

于是第一话的战斗任务就这样开始了——

在这一次战斗中，敌人不太强，场地也较为简单，适合新手熟悉操作，记得灵活运用LIP系统选择对话，不同的对话选择会让队员对你的好感度发生变化——这是整个游戏的核心要素之一！

注意，此时绝对不能让真宫寺樱的HP降至0，否则会引发不好的后续……

其实若想搞清楚，艾莉儿便会主动帮你推荐：

“两个人要整整了！”然后她接着告诉你：

“有没有……我也需要你的一个帮助，一个女……”

“你好，我先进入了……”

“那就……不行的……你不知道……”于是开始先找——
“那你说……也算吧……你看……”对方问道，一问不知道……

“没问题——就那边——我去了！”艾莉儿大声说道，往前一步向对方走去，“那不是……不行的……”

“不知道的话，让我……也算了！”

然后转而对他们一看：“那边……那边……那边！！”一边叫一边就向一个方向跑了——

那边是道路，但路上已经堵了很多人，很多人在那里聚集，艾莉儿和你一起站到了人群外面，往里一望……

“这里……太多人了——先想想办法！”艾莉儿一边嘀咕，一边盯着那堆人看，“有没有别的……”

“我知道了——我们绕路！”艾莉儿说着，和你往一旁绕了过去，走到了一个巷子里，这里还好，相对空旷，从这里穿过去就能到另一边了——

“快走——别墨迹了！”艾莉儿走在前头，催促道。

两人穿过了这条巷子，到了另一边，发现这里是一处广场，很多群众聚集在这里，看起来是在围观什么……

“什么情况——！？”艾莉儿走近看了看，突然叫了一声，“快来看……是……！！”

广场中央出现了敌人——那些穿着黑色制服的人，从道路的另一侧冲了过来，将人群冲散！

“怎么……这里也有！！”艾莉儿转身叫道，“大神队长……快命令大家出击——！！”

战斗又要开始了——

要注意，此话目标是把敌军的领头人干掉，但别忘了保护周围的群众，如果百姓伤亡过多，会影响之后的故事走向。

### 第三话　我没客从长的资格？！

初战告捷之后，大神一郎和帝国华击团的成员们开始有了更多的接触，日常中的一些小事也让大家彼此更加了解——

“喂，大神……你……这是个什么东西？！”樱拿着一件奇怪的物品走了过来，脸上带着疑惑，“这是你的……？”

“那个……”大神一郎挠了挠脑袋，有些不好意思，“那个……是从老家带来的……”

“哦……老家……”樱看了看，“是什么老家的？”

“栃木……那里的特产……”

“栃木？！”樱惊讶了一下，“那你……也是从……？”

“那边的……”大神一郎点了点头。

“真的——！！”樱突然高兴起来，“那我们是老乡啊！！”

“是……是这样吗？”大神一郎也有些意外。

“那你……知不知道栃木的……”两人开始了一段亲切的对话，从此彼此之间的距离又近了几分……

在帝国剧场里，有时候也会遇到其他团员们的一些有趣情节，每段情节都有对话选项，记得选择正确的，不要伤了大家的心——

这一话的战斗任务相对较为复杂，地图也比之前大了很多，出战前记得做好准备，合理分配队伍……

战斗中记得要配合队友，不要让任何人落单，特别是HP低的角色，要重点保护！

### 第四话　轰炸！轰炸！大轰炸！

“一般情况我们只有……”他说，“那就是……还没有！！”

那是艾莉儿圈定自己、先把自己最大的行动展开，发现，那个之前的敌方已经出现了！

“等一下！先用完那次就可以了……然后还有……那个有所改变的就不会……”

出击命令传来——大神一郎和帝国华击团的成员们已经准备就绪，这次的敌人比之前要强了许多，要做好充分的准备！

“都在吗——！！！”大神一郎大声呼唤，“那大家……出击——！！”

“收到——！！”队员们齐声回应，斗志昂扬！

战斗开始了——这次的敌阵阵势相当大，要集中力量各个击破，不要分散队伍……

记住：
- 先处理掉有远程攻击能力的敌人
- 保持队伍阵型，不要单打独斗
- HP不够时及时撤退补血

有一句话要谨记：所有最佳的队友并肩前进，尽量激活每个角色的个人特技，这样能发挥出最强的战斗力！
