# 漫画地带

> 作者：杨柯来、刘鹏 / 刊页 12-13 / 栏目：世嘉世家 / 责编：PINSER / MD ACT by SEGA，16M

某某，在纽约的一幢大楼中，年青的漫画家 TURNER 正在废寝忘食地工作。突然，漫画里的人物神奇般地无法无天的另外人出现，却制造了有趣的漫画景观，但他给他的形象带来了——一卷一卷扑到了了，漫画把了——TURNER 在漫画场景中的冒险就此开始了……

## 一款绝对不可不玩的美版 ACT

《电子游戏》杂志每期时都有通过反映游戏类型的打击乐享受乐趣作为方式，很多以前 ACT 游戏在以一种接近格斗打击的形式出现，很多游戏通过不断地打击反应——在以前一段时间内通过游戏 QL 各种游戏通过 QL 以体现游戏 QL 这种可及以前的游戏打法风格。

好，为什么要以这段话作为开头，就是本刊推出了这款美版 ACT 的介绍。

### 丰富多彩的攻击方式

相信还没玩过这款游戏的玩家，当他首次接触这款游戏时，一定会被其中那扑面而来的新鲜感所吸引——怎么还有这么与众不同的游戏？！ 其实不然，说它与众不同，并不是游戏与众不同，而是说它那种表现形式，你想，在 ACT 游戏中，以 ACT 这一个类型的游戏形式来说，这种形式确实让人眼前一亮，它完全以一种传统的美式漫画格调来表现游戏，游戏场景就是漫画格——是漫画面板式游戏——而不是一般的游戏场景，就这一点来说，这款游戏就已经确实地给玩家们留下了一种深刻的印象。

另外从攻击力的角度来说，本作的攻击手法之丰富，也实在让人惊叹——无论是普通的直冲拳，还是对空踢技，甚至是纸飞机，在游戏中都是有效的攻击手段，当然还有更多更多精彩的格斗技术，总结来说攻击手段极其丰富，以下就简要介绍一些。

### 必杀技

| 技名 | 指令 |
|---|---|
| 升龙拳 | ← A |
| 对空腿技 | ↑ A |
| 左冲右突 | → A |
| 重击腿技 | A（连打）← A 组合 |
| 折纸飞机 | 按 A 键片刻 |

### 熟悉每一种道具的性能

在游戏关卡的进行当中，你会遇到多种多样的道具，每一种道具在关卡中都有其独特而有用的功能，你必须熟练地掌握所有道具的功能，才能充分利用每一个道具，在关键时刻发挥出其最大的效用。

### 宠物鼠的三大功用

在游戏角色 TURNER 的身边，还有一只可爱的小宠物鼠，千万不要小看了这个小家伙，它在游戏中也有着举足轻重的作用，具体来说有以下三个功能：

①协助主角攻击敌人；
②发现前方隐藏道具；
③恢复主角活力。

## 游戏的难点非常多，不过有了本文……

### 第一关 A 部分

```
     A────B
     │    │
     C    D
     │    │
     E────F──G
               │
               H──I
```

A. 闯门即击
注意千万不要轻易开枪，因为子弹是有限的，你的主要任务是格斗。

B. 进入漫画中时，要留意前后方向，因为子弹数量有限，不要随便往里走，尽量让敌人靠近你再予以格斗打击，一步一步打过去——一步一步按固定的路线移动，要注意往里面探的次序。

C. 先是不要急着往前走——
D. 发现了子弹留下来继续利用，在前面的来的那几步之中让敌人出来，再以固定路线一步一步往里，一组一格地走。
E. 绝招 3 连击一次就可过这一格。
F. 此格也有子弹出没。

▲第一关 A 部分第一格——注意整个一格中的生物，就是在那里的各种敌人以各种形式出现，不是那么轻易就能解决的，一定要注意路线。

注意：漫画不同于普通游戏，在游戏中有各种不同的方向行进——其中子弹数量的问题要记住——向前走一格以后，就让所有敌人出来，再以固定路线消灭它们，然后再一步步走后面的，这是攻略《漫画地带》最佳的方法，而后几关都是如此。

▲进入一个漫画格以后，敌人就从四周涌出来了，这时要沉着应对。

▲第一关用这种格斗一步一步走过去最为稳妥。

C. 在这里遇到特殊障碍，要注意踢腿的技巧不要错失。
D. ↓B 格完成——

### 第一关 B 部分

```
     A────B
     │    │
  D──C    E
  │        │
  F────G──H
```

A. 因用绳索慢慢悠悠把敌人拉过去的，
B. 这里时关卡外围一格，注意路线。
C. 敌人非常多，这里是第一关格局较复杂的一格，使用招数的时机很关键，敌人以较多的数量同时攻来时，可以不时转向；要注意后面格子的情况，稳扎稳打地推进，注意路线与招数的运用，一个格子里面的敌人太多时就要先转入下一格，再退回来打——
F. 此格也不容轻视。

▲这里能够发现子弹，一定要好好利用。

### 第三关 A 部分

```
     A────B
     │    
  C──D────E
  │
  F──G
     │
     H
```

A. 前几关的经验很可以活用在这里，只不过这里的敌人更加厉害，以多种不同方式同时向你攻击，一定要把握格斗时机，不被多方向的敌人逼到墙角是第三关攻攻关关键。

B. 前面以招式为主，进入了以后就以多方向 5 次的连续的格斗进行——以 5 次为一组的进行，每完成一组就向下一格进发，这是这一段的技巧。（注意确保有 HP，否则中间的格子很难）

C. 发现子弹，使用子弹把这一格的敌人打倒。

D. 这里的情形与第一关有些相似但更难，注意利用招数组合。

E. 以格斗为主而不是以子弹，这里的格斗需要迅速、果断地发动，不要给对方任何喘息的机会，同时注意地图路线，不要误入死格——

F. 第三关 A 部分最难的一格，以大量的 3 连击向单一的敌人连续攻击，不让对方喘息，配合折纸飞机等特殊招数往往更为有效，注意：连续格斗的时机也非常重要，以 3 组的格斗为主，以 B 组的招数完成——

▲第三关地形已经是这里了，最为困难的一关。

### 第二关 A 部分

```
  A────B
  │    │
  C    D─E
  │       │
  F─G──H──I
```

A. 在开始时把 3 条路从下往上逐一走过。

B. 从下往中路一步步进去，那里有 3 个敌人，一次 3 连击以后转身再来一次就可以。

C. 利用道具中的手雷把后方多个敌人一次解决，当然也可以用子弹。

D. 进入格子以后就是密集的格斗，持续以快速的组合攻击压制对手。

E. 走上来以后，先格斗清场然后再推进，以确保尽量留住 HP 为最重要。

F. 在第二关里，这一格是难度最高的，既要注意正面进攻，又要注意侧面的突袭，以 3 连击为主，注意招数的循环。

G. 子弹在这里，不要错过。

H. 使用道具控制手把，可以达到改变路线的效果，这里的路线非常复杂。

I. 在最后一格，此关 BOSS 就在这里出现了，使用子弹以后，3 连击加上迅速的升龙拳可以极大地减少 BOSS 的 HP，一步一步地压迫——最后以格斗决定 BOSS 的命运，结局如何就看玩家的反应了。

▲到了第二关这一格，敌人以各种形式从四方攻来。

▲在第二关 B 处进入第三关后遇到敌人，这时要特别注意路线。

### 第二关 B 部分

```
  A──B
     │
  C──D──E
         │
     F──G──H
```

A. 进来以后让敌人往你这里靠近，以你自身位置为中心向外格斗，不要主动出击，等待合适时机。

B. 此格要特别注意，向前方格斗时需注意来自侧面的攻击，以快速的格斗组合攻击为主，不给敌人任何喘息的机会。

C. 在这格里要善用折纸飞机——折纸飞机这一招对于应付多个敌人同时进攻非常有效，配合普通的拳击招式，可以有效清场。

D. 难度很高的一格，需要以顺序一格一格地推进，不要跳步。

E. 进入这一格后，格斗技巧要比第一关时更为成熟，要更加准确地把握格斗时机，同时注意回收本格内的道具。

F. 稳扎稳打，注意从侧面出现的敌人。

G. 子弹可以在此格发现，要留意格子内的隐藏物——以宠物鼠的能力来协助探测隐藏道具。

H. 最后一格——以格斗收尾，注意：在游戏中，BAMM！等漫画音效字会出现在格斗画面中，增加漫画感，非常独特，可以好好享受这种游戏体验。

### 第三关 B 部分

```
  A──B
  │  │
  C  D──E
  │      │
  F──G   H
         │
         I
```

A. 进来以后把 3 种敌人——以前几关的经验来推进——稳步向前，一个个格子打过去——

B. 格斗以主，以格斗解决单个敌人——不要一次进多个格子。

C. 若碰到道具，可以为你恢复一定的 HP，极为珍贵，注意不要浪费——来找一找你身边的小宠物鼠，它也许能给你带来意外的惊喜。

D. 有一个门出现，进入门以后就到了下一格。

E. 这是关键一格，进入后会遇到大量敌人，快速连续格斗加折纸飞机是清场最有效的方法，一定要尽快消灭——

F. 子弹出现，以后的格子也有子弹，不要随便用，留到遭遇 BOSS 时用最为适合。

G. 这里格斗要特别小心，来自上下的敌人交替进攻，不要乱了方寸，保持冷静，以稳健的格斗推进。

H. 几乎就是到达了。

I. 第三关 BOSS——这位 BOSS 非常厉害，要使用连续的 3 连击和升龙拳以及子弹来结合攻击，是本作难度最高的 BOSS 之一。以格斗不断逼压，并充分利用折纸飞机和宠物鼠的帮助，相信玩家能够以此顺利地通关！
