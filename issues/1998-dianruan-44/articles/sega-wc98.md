# 世嘉98世界杯·近期世嘉土星上的体育类游戏大介绍

> PDF p.75-76 / 次世代天空 / SS SPG SEGA / 责编：小天天

## 世嘉98世界杯足球

小天打过这里最流行的很多款 GUN 类STG，由SEGA'S V-STRIKER2 [?]，让我将注意力转向年大比拼实战，对于足球游戏，SEGA土星已经有了为玩家们准备的好版本，所以这里不需要谈太多关于足球模拟技术的内容，只要说说这款主打“欢笑”精神的体育类游戏，就已经十分令人期待了！

**厂商**：SEGA SPORTS　**类型**：SPG　**负责编辑**：小天天

## 继高人气度的V GOAL'96(97) 以来，玩家对这款新游戏能否满意？

家例如以往，同意对KONAMI而来，SEGA一款一款“不平等”的特色不平等游戏，比较两款SEGA下面带来的一款是相当成功的，SEGA T的SEGA'S V STRIKER2有别于上一代的V GOAL,大型足球游戏设置也相当成功，以KONAMI 为基准，如以下这一套……

在不同以往的风格出发，SEGA改变了比赛中的镜头控制方法，也在原来的基础上增加了很多新的元素，游戏的整体节奏变得更加明快，非常适合不同水平的玩家上手，特别是对于休闲玩家来说，《世嘉98世界杯》最大的特色就在于其出色的娱乐性——足球比赛中的动作充满了夸张但不失真实的运动表现，让玩家即使不了解足球，也能在游戏中找到乐趣。

游戏配有完整的世界杯赛制，涵盖了小组赛、淘汰赛直至决赛的完整赛程，支持多支参赛队伍，包括各国国家队，玩家可以选择自己喜爱的队伍参加世界杯的争夺。

◆ 在不同难度下，游戏的AI对手会展现出截然不同的战术特点，建议初学者从较低难度开始，逐步适应游戏节奏！

## 近期世嘉土星上的体育类游戏大介绍！！

> 责任编辑/小天天

### '98VR冬季奥运会

几年打过这里最受欢迎的“十大”平台上体育类游戏之后，时至最近，体育游戏已越发精彩，立于上方，世界上多个厂商都在争相发布体育游戏，土星上的体育游戏也越来越精彩，上了这道十大玩法的那道风景之后，这款名为《'98VR冬季奥运会》的游戏，可谓是其中的翘楚。

个人觉得这款游戏真的非常棒，以4人为单位实现了一款体育综合游戏，在技术感与娱乐感之间取得了完美的平衡，让玩家在冬奥会项目中体验独特的运动快感，如滑雪跳台、花样滑冰、冰球等项目都以高精度进行还原，加入了VR影像处理技术，视觉效果十分出色。

即使是对冬季运动不甚了解的玩家，也能从中感受到那股充沛的运动热情，游戏节奏轻快流畅，多人模式更是让玩家可以与家人和朋友一同分享运动乐趣。

◆ 什么都可以做，上方让人想到了一个——健康！

### 实况J联盟足球

KCE加持遊戏的，把KONAMI的HOC进行到底，一来是一般“实况足球”的效果，从那时就是一个属于平台，让很多人都心动，在中间之后变成了一款平台的游戏中玩，两人再来体验这款经典的……

同时，另外的Kick off设定，也为更多年龄层的玩家提供了更多样的游戏体验，从KCE立足点上，也比原来更多了一个特殊体验者的玩法，并在两个人的人物体验方面有所不同，再来说到了404——以三个独立方式展开的过程，两个人玩的时候，特别有自由表现……

这款游戏的人物参数设定如下：

| 参数 | 说明 |
|---|---|
| SPE | 速度 |
| STA | 体力 |
| OFF | 进攻 |
| KIC | 射门力 |
| DEF | 防守 |
| GRV | 成长力 |
| POW | 力量 |
| JMP | 弹跳 |
| AVE | 平均 |

### ●设计一个动作虽然想干嘛？

这里是一个用来展现游戏设计理念的板块，游戏设计者为每一个动作设计了独特的手感——无论是射门时的脚感、过人时的节奏、还是防守时的把握，都经过了精心调整，力求为玩家带来真实而流畅的足球体验。

体育类游戏轻松欢笑，别有一番情趣！选择喜爱的！
