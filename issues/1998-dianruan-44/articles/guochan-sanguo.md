# 国产中文游戏万花筒——《三国志》之“雄霸天下”

> 天堂任鸟飞 / 刊页 23 / 机种：FC / 游戏类型：SLG / 游戏厂商：外星科技

操作过“三国”系列的玩家们，一定已经痴迷于三国鏖战的那段历史。恰好出版社也想出这样一本书，外星科技在 98 年春节刚刚推出这款“三国志雄霸天下”，可以说从主题来说是“三国”系列的一个全新发挥，国内游戏爱好者怎能错过？

当然这款游戏的大主将打打打，首先说到底，也要靠一些策略才行，以下就“攻城”、“对战孔明”等招式作简单介绍，以及使用注意事项。到达关键时候，一定用于带领主力突击一前才算完美。

要进行攻战，不能以所有个人物力，更不能以所有兵力外出，保持国内供应的前提，才能打胜仗，才能稳住地盘。你方防线固守好之后，才能积极进攻。当然进攻也要有策略，一天一天、逐渐挺进也是比较实在。

若对方只剩一个城了，也不必急着去打，不能以所有力量去打那城——因为旁边还有外力会来救的。等到时机成熟，再乘胜追击，才能收得到最大的效果。

前期结盟——一两个朋友是有用的，互相不侵犯，各自发展，在合适时机反目，令对方措手不及。但此手段不能使用过早，否则容易引起三国其他势力的反弹。

大将的分配——许多有特殊技能的武将，如关羽、张飞、许褚、吕布等，其军力及内政能力均有不同，不要贪心把所有大将都用于一处，分散使用才能发挥最大效用。内政上，注重各自的长处来分配职务，将有军事能力的人放在对应职位……（以下略）

不多说了，这是笔者进行“三国志雄霸天下”时的一些粗浅体会，希望对有兴趣的玩家有所帮助。一气呵成，争取大一统！

三国志——这款游戏的美中不足在于画面略为简陋，但操控简便，相信会有众多玩家喜爱。而且相信外星科技会推出更加出色的后续产品，为国内游戏界注入活力。
