## 系统介绍及招式说明　SYSTEM

其实这没什么好介绍的，所有的格斗游戏天师都以为玩家们一秒钟便能适应过来，因为它们全都一样嘛！只是”傲剑狂刀”吸取了许多同类游戏的长处，我们倒要”仔细地研究一下”。

### 基本操作

使用八方向摇杆与四个按键，拳（斩）与脚各两个，有轻重之分；基本从上到右（由 A 到 D）为轻斩、轻脚、重斩、重踢。特殊技、必杀技、大绝招、摔投等，必须输入摇杆特定方向并与按键组合后方可使出（读者：这不废话嘛！）。像一般游戏的冲进、疾退、二段跳等，玩家一试便知。另 A＋B 为闪避，而 B＋C 为小跳跃（脚胯位不能按）。

### 攻击及防御集气

有两种气表，一种为攻击集气，对方攻打，本身挡空招都可集气，气表红色；另一种为防御集气，攻击时对方防御成功，自己或对方都会集气，气表为蓝色；防御集气亦可自行按键集气，按键为 A＋C。

### 大绝招

有超必杀、密传套路（十连斩）、逆袭灭三种。

①超必杀必须在攻击气表满时使用。
②密传套路必须在防御气表满时，以 C＋D 启动后开始连斩。
③逆袭灭：必须先进行潜能爆发（A＋B＋C），这时时间暂停，时间段内按 B＋C＋D 袭击。逆袭灭的伤力依照血量剩余多少而计算（原则上：自己伤愈重，使用逆袭灭的威力就愈大——天话：怎么有似曾相识之感？——使用此技后攻击气表消失，不能再换手反击及使用超杀）。

### 换手及反击

选中两人来玩，当你觉得想换人出场时，若攻击气表已满 1/2 并闪动，此时可在防御的同时按 A＋D，就可产生队友跳进场内并作反击动作，所以此技又称换手反击技，消耗攻击气表 1/2 的气量。

**人物的密传套路分为四种类型**

| 类型 | 按键顺序 |
|---|---|
| 一型 | A A C C D D A C C C |
| 二型 | A C D A C D A C D C |
| 三型 | B B C C D D B C C C |
| 四型 | A C D B A C D B A C |

编者注：大绝招中除了各人的超杀完全不同外，密传套路就始终局限在上面四种类型之中（后面在角色图片中标明）；至于那“逆袭灭”，则是共通的一种模式。
