# 任天堂64DD大剖析

> 作者：黄昌星 / 刊页 43 / 栏目：科普园地

---

**HERE COMES A NINTENDO 64 DISK DRIVE!**

首先声明，N64 史上最高档的机器，没有磁盘驱动器是“64 DISK DRIVE！”的小文合称了。为这份给予了“只有N64 才能改变明天”的情方而忘忧的我们，到底在这里介绍一下所谓的——N64的宝宝们。

## 具有双重的功能

不只入把 64DD 磁盘机一般磁盘机工了等等，而且还有 “64 DD 的容量当然！”磁盘比 CD-ROM 卡带更快，64DD 将一般时间64的媒体设备，面对著具有如此多种多样已改变身体。

**64DD 基本资料**

| | |
|---|---|
| 总盘数 | 64MB DISK |
| DISK容量 | 64MB |
| DISK直径 | 64mm |
| 最高传输速度 | 约 1MB/s |
| 连接接口 | 专用连接线（N64底部） |
| 外形尺寸 | 约 204 × 150 × 58mm |
| 重量 | 约 840g |

▶适应系统：NINTENDO 64主机

通过上述的比较可以看出，64DD 的基本参数，其实 64DISK 采用的是一种特殊的 2.5 英寸磁盘。读写速度（约500转/分）相当于三倍速的 CD-ROM。此外其读写精度十分精准，且读写功率消耗及其他功能设施。

由于磁盘比 ROM 卡带，且拥有 64DISK 基本上是直接使用磁盘来读取游戏数据。ROM 区域基本数量 32MB，数据上存的最大量的 32MB，而 RAM 区域最大容量为 16000（约 16KB），不超 ROM 存储情况，因而原本非常简单，用于游戏存储。这样，相比 64 的 64DD 卡带，让玩家享受到真实，已经是大卡带 64 磁盘而 ROM-卡带一样，总容量高达 64 + 64 = 128 的同时带来功能

ROM 卡带的扩充方面：64DD 磁盘 ROM 数据方面做到了 32MB，所谓内存卡，64DD 同时作为存储器——**所以64DD本体、N64DISK、扩充内存卡，3个加起来才是64DD的全部了！**

如果游戏内容是 ROM-卡带中完整装载了 128MB，同时通过 64DISK 磁盘的方式，保存了 2 人以上的系统数据了，让玩家分享到了 N64 的强大存储器。
