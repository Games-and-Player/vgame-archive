# 007黄金眼

> 次世代天空 / PDF p.27 / 刊页 25 / N64 / 3D-STG / 任天堂 / 96M / 责编天师

**间谍之眼 粉碎卫星**

宝贝代代喷射缝之手 F1 一段，N64 最畅销代表游戏，据说在游戏史上的表现是最强壮的，这一作游戏因为很出名，令人满意时 N64 的转型中令人惊叹，N64 给入的特效极大，最大的功劳首要归功于 RARE，多亏了制作者们的卓越设计，才让此"007"游戏一发行在手中，的确给玩家带来了超级游戏享受，用以把以 007 电影体验改编成游戏方式，给 N64 再次带来了久违的爽快感。

厂商：任天堂　类型：3D-STG（3D 射击）　容量：96M

## 詹姆斯·邦德在 N64 上再展银幕中身手

这就邦德游戏的肉容，游戏全名"GOLDEN EYE 007"，在题目里面有了 N64 最经典的一部，这款游戏"007"从成为游戏的一刻之起，借助了 N64 大幅优化的帧数处理，不计游戏中所有 N64 的精准格局的高速，向玩家提供了充满魅力的时代大作，确实也是让 N64 风靡一时，游戏内的各类场景再次以 3D 移植的形式出现，让玩家以完全互动的游戏模式来享受电影中的一切。

以"007"诞生的头号人气角色 JAMES，这时已成为了游戏的头号主角，一切以经典的电影英雄标准来呈现，使游戏中的每一关卡都具备电影级的场景设计，由此以 N64 标配的完美画质来加以呈现，令人叹为观止。

▲一关之后，到达里的每一处，连接到一步步完成各指定任务的感觉十分好，让游戏的冒险氛围中时时令玩家忘返。

▲到处留下了完整的 007 游戏效果，在每一段射击里，已经制作出了最好的 007 效果来。

### 谍报片中的道具助你脱离险境

我游戏在赤裸裸地向观众说明白，里面常有的特务情报行动，在完成具有高风险的秘密任务时，需要利用情报里获取的以下高科技产品，比如手表、特工眼镜，外加隐藏武器等多样化游戏道具，以尽可能在各关卡的险境中顺利脱身。

在游戏过程中，不仅会遭受敌方的正面攻击，还会遭遇各种迷惑人心的障碍，从枪击对抗到以智取胜的战术策略，这样的游戏情节充分发挥了以"GOLDEN EYE"命名之作的间谍精髓。游戏关卡与关卡之间，也经常会穿插各种情节，引发 N64 游戏独特的互动体验，以进一步烘托玩家的游戏热情。

### 间谍战也能以对战形式进行游戏？

游戏当中特别的四分屏多人模式，更将众多玩家聚在一起形成同屏对战场景，来自电影风格的场景设计，以特定任务与各关卡的多人模式并行展开，而本次游戏的局面，通过多人同机方式对战时，游戏中令人心跳加速的紧张气氛一一展现，由多名玩家在 N64 平台上同台角力，不断互相比较操作技艺，增加了这款游戏极高的玩家互动趣味性，是 N64 上专属的顶级游戏享受，将这款游戏的娱乐深度进一步推向了极致。

▲GOLDEN EYE，以大概概率下以整个 N64 平台的游戏标准，它在 3D 射击领域已完全超越其他同类产品。

▲从赛车大型，以 N64 专属的高帧率来处理呈现，让各类运动场景的流畅感达到最高标准。
