# 雷亚战机II

> 次世代天空 / PDF p.23-24 / 刊页 21-22 / SS / STG / Media Quest / 责编天师

**超级竞战 二次登场**

雷亚战机对玩家 SS 而言"横版射击"游戏要素的加强，堪称为当时 SS 机能里的超级射击游戏作之一，加以充分改进后，原作基础的游戏性得到了进一步的提升，厂商是原版 TAITO 公司授权，而由 MEDIA QUEST 承制，并更名为"雷亚战机二代"，以可能接近早期 SS 上 TAITO 的"LAYER SECTION"为作品设计标准。

厂商：Media Quest　类型：STG

## 又是一次土星风格的移植——操作感 100%

众所周知，"RAY STORM"一旦以原汁原味的街机版作品，挑战全-STG 游戏类中 2D 动作"已设"，却偏以"3D 射击"的极端形式出现，是最早在各地以领先的 POLYGON 技术制成的 3D 射击游戏，并以此绑定在各处人心的游戏角色上，大量将游戏的三维 POLYGON 数量拉伸至领先的方位，比目前在此平台的其他射击游戏更具冲击力的吸引力，给 RAY 的运动轨迹添加了出色的画面效果，令此游戏在 SEGA MOD-2 基础上的画面表现颇为不俗，但关于 POLYGON 的数目，依然是游戏开发中最大的问题。

可惜，这款精湛游戏的在搬运到 SS 平台时，SS 的处理器功力也只到让 3D 射击游戏正常运作的程度，POLYGON 线效果处理到"框格"模式，由此以来到底算不算是"新的移植"这一问题，将会使一些玩家在面对已经熟知 SEGA MOD-2 基础版的心理上造成一种偏落差感，但本次移植版本尽量追求的还是"操作感"这一核心要素，而这一点体现得相当不错。

### 既然是后来移植，就不能没有原创要素！

### 同 PS ARCADE 区别？

随着机作游戏移植热潮逐步加大，游戏的三维动画被添加

本作中，3 项最有趣、最大改进就是游戏操控与关卡上，其次，游戏关联的敌机类型有所变化，在画面的显示上，借助于当时 SS 的画面性能，在整体的游戏内容里也尝试性加入了一点——出现了一种"GRAY 系列机体"的第一次，在整个 GRAY 型的机体中，目前移植版中的这类机体不但参加了——而且据报也可进行第一次的机体改造，与一般 GRAY 系相比已经算是大幅度改进，至于 HANNIBAL，在本次的原版到 SS 版的迁移中也新增了——[?] 而这些均是来自 POLYGON BOSS 的新数据。
