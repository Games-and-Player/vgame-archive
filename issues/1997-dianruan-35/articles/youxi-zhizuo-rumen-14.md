# 游戏制作入门（十四）

> 作者：魔法师 / 刊页 61-62 / 栏目：科普园地

---

> 内涵 START　之 魔法师

---

大家好！又见面了。

现在社会上对电子游戏的批判越来越多了，电子游戏的游戏性的问题，已经不只是玩家界关注的问题，连整个社会及大众传媒都纷纷发表意见了。说到电子游戏的游戏性的问题，已经到了一个很普遍的状态，不少家长甚至将孩子的学业成绩下滑归咎于电子游戏，这种情形令不少游戏厂商也开始大力鼓吹「益智」类游戏的开发。

但无论如何，游戏始终是游戏，尤其是那些享誉全球的一流大作，其制作水准、创意构思都是首屈一指的。今天，我们继续「游戏制作入门」的课程，探讨一下游戏中 AVG（冒险游戏）、RPG（角色扮演游戏）等类型游戏的人物对话内容制作，以及如何利用 START 脚本系统来实现游戏中丰富的剧情互动。

---

## 电影与游戏的关系

以往的 AVG（冒险游戏）画面，人们说那是「互动电影」，而现在的 3D、VR 和彩色电影的结合，更加令游戏与电影之间的界限趋于模糊。在某种程度上，大型的 RPG 游戏，真的是有着电影一般的故事叙述。

目前市面上有很多 RPG 游戏，包括 FINAL FANTASY 系列，记得在 TV 版的（MACROSS）杂志三中有以下台词描述道：「我们的生命，真的是太短暂了……」——只有短短的几句话，但却道出了多少人内心的 Super，我们又想到了多少？RPG 中的对白，不只是故事发展的工具，更是承载着人物情感的媒介，好的对白能令玩家产生共鸣，被游戏的世界所吸引。

最重要的是设定好人物性格之后，在各段对白的编写中时时刻刻要保持这个性格特征，不要前后矛盾，这样塑造出来的角色才是一个有血有肉、真实可信的人物。

---

## 什么是「内涵 START」？

游戏制作工具「START」是本系列文章主要介绍的游戏制作系统之一，让大家可以利用这套工具制作出有对白、有互动的游戏。「内涵 START」的意思，就是着重介绍如何利用 START 脚本语言来制作游戏中有丰富「内容」的对话及剧情系统。

---

## 如何利用 START 制作游戏对白？

「内涵」即是「有内容」的简称。在上期的文章中，我们学习了如何建立游戏的基本框架，那么这期就来学习如何在框架中加入游戏内容——人物对白及剧情。

### 人物（CHARACTER）

最重要、最基本的东西，当然是游戏中的人物了。所谓人物，就是玩家在游戏中会遇到的各种角色，包括主角、友方人物、中立 NPC（Non-Player Character，非玩家角色），以及敌方人物。

建立好一个人物之后，最重要就是给他们定出性格，因为这个性格，将决定游戏中这个角色说的每一句话、做的每一个动作，以及各种情况下的反应。如果不能前后保持一致，整个人物便会在玩家眼中显得不真实，甚至令人哗然。

RPG 的人物各有不同性格，有的勇猛，有的软弱，有的睿智，有的鲁莽，有的开朗，有的内向……不一而足。如果在游戏中加入了一个多种性格集于一身的人物，是相当困难去掌握的，但是 RPG 当中这样的设定却比比皆是，因为人是多面的。如何令这个人物的多种性格都能在游戏中得以体现，在对白及互动中表达出来，这是一个非常大的课题。

不过，简单来说，一个人物的性格，可以在游戏中透过以下几种方式表达：

1. **对白**：人物说的话，是最直接表达性格的方式
2. **行动**：人物所作的决定和行为，包括战斗时的战术选择
3. **反应**：面对不同事件，人物的情绪反应

---

## C&C 框架介绍

游戏制作工具 START 的 C&C（Character & Command）框架，是用来管理游戏人物及其可执行命令的系统。在这个框架下：

- **C**：CHARACTER（人物），指游戏中的各类角色
- **C**：COMMAND（指令），指玩家或 NPC 可以执行的各种指令

通过 C&C 系统，游戏开发者可以为每个人物定义其可执行的指令集合，例如：「移动」、「说话」、「攻击」、「使用道具」等，并透过脚本语言控制这些指令在何时、何种条件下被触发执行。

举例来说，一个简单的对白触发脚本可能如下：

```
当 主角 接触 NPC_老人 时：
    触发对白：「年轻人，你要去的地方很危险！」
    设置标志：老人已警告 = TRUE
```

这类脚本语言的设计，使得即使不懂编程的游戏设计者，也能轻松制作出有丰富剧情的游戏。
