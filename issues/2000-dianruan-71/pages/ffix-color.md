# 最终幻想IX专访

## 插6（PDF p.8）

【图】《最终幻想IX》专访彩页第一页。大幅展示角色CG画面，主角吉坦（Zidane）和女主角加奈特（Garnet/Dagger）的精美CG图。

SQUARE最新大作《最终幻想IX》专访。本作回归系列原点，采用中世纪幻想风格。坂口博信监督表示，本作是 + r1 + 的作品。

## 插7（PDF p.9）

【图】展示更多角色CG和游戏画面。包括比比（Vivi）、斯坦纳（Steiner）、弗赖亚（Freya）等角色的设定画。战斗系统回归ATB（Active Time Battle）传统模式。

- 世界观回归中世纪奇幻风格
- 角色设计采用SD（Q版）比例
- 新系统 + r1 + （能力学习）通过装备习得技能

## 插8（PDF p.10）

【图】续上页，展示游戏世界地图、城镇画面、召唤兽画面。介绍 + r1 + （恍惚）系统和ATE（Active Time Event）系统。

- Trance系统：角色在战斗中受到一定伤害后进入强化状态
- ATE系统：在城镇中可切换视角观看不同队友的行动
- 迷你游戏 + r1 + 卡牌对战
