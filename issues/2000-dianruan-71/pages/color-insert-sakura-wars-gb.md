## 大神一郎登场！

叙述主角——大神一郎和樱等花组成员之间故事的 + r1 + 系列作品。在本次的GB版中，大神一郎将不再是主角，只是其中的人物之一而已！而这回的主角据说是一位到帝国·花组体验一个月生活的青年，他将以见习队长的身份活动于游戏中哦！

## LIPS Live Interactive Picture System 健在

提到 + r1 + 系列的最大特征那就不能不提一下 + r1 + 。这次的GB版 + r1 + 也当然存在着哦！而所谓的 + r1 + 就是指出现在会话中的选项，以及在限制时间内选择选项。故事会随玩家的选择而产生不同的结果。由于游戏没设定有时间限制，所以玩家可享受到比现实生活的交还还要快、充满紧张感的会话。记住千万别着急，保持冷静，沉着思考，选择其中最适合的选项哦！

## 又有动画作品了！

不仅日本的内玩家也很支持我们的「樱大战·连国」GB版游戏新情报，这回又另外听到从8月份开始即将播出动画。日本的TBS电视台，想看……嗯，我们也……

### 藤枝 玛丽亚也登场了！

这次的 + r1 + 也和之前的系列作品一样，会出现由事件交叉而成的迷你游戏。而这回应该也是在通过迷你游戏后，就可以随时享受这些迷你游戏哦！不知道能玩到怎样的内容，真令人好奇。

## Game Boy上登场！

### 帝国华击团终于在

以下就要为大家介绍，自99年春天发表以来，就不再有任何新情报传出的Game Boy版 + r1 + 。受到许多玩友关心的新要素、新情报都将呈现在大家面前！

这回的游戏不仅加深了和各个性格独特人物间交流的冒险部分，以及为守护帝都与敌人作战的模拟战略部分……其中更是满载了Game Boy版独创的各种要素。现在就让我们从附图面来推测游戏的各个项目吧！除了欣赏本作在Game Boy上精细、美丽的表现外，各位玩家也别忘了将本篇彻底熟读，以便未来一买到游戏就能立即上手哦！

### 角色介绍

**大神一郎**
帝国华击团花组的队长。在 + r1 + 时已从少尉升级到中尉，而这回的官衔仍是少尉……难道GB版的时间是在 + r1 + 的终盘？

**桐岛丸奈**
帝国华击团的开心果。出生于冲绳的武道家，使用琉球空手道。

**李红兰**
搞笑人物兼开心果……是个操着关西口音的中国大才科学家。

**玛莉亚·塔奇芭娜**
帝击的领导人。出生于俄国。父亲是日本人，母亲是俄国人。

**神崎堇**
以樱的对手身份存在，大财阀——神崎重工的独生女。

**爱莉丝**
帝击的守护神。出身于法国。家族上是大富豪的独生女儿。

**真宫寺樱**
帝国华击团的新进队员。无论是战斗能力或是舞台上的技巧都不断地在进步。不过有时会显露出蕴藏在内部的优良素质。

### 全新系统

在以往的 + r1 + 系列作品中，游戏进行到各章节结束时都可以将资料储存起来。不过这回本作的设计则是可以一天为单位来储存资料。这应该与DC版的 + r1 + 不同，是改用了新系统吧？

这次的 + r1 + 也和之前的系列作品一样，会出现由事件交叉而成的迷你游戏。而这回应该也是在通过迷你游戏后，就可以随时享受这些迷你游戏哦！不知道能玩到怎样的内容，真令人好奇。
