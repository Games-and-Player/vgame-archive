## 冒险舞台从日本延伸到魔界……

超人气RPG + r1 + 系列的最新作品即将在Game Boy登场哦！这次的故事发生在2000年7月经济不景气的日本。主角——刹那与未来在和突然袭来的魔物作战时，不小心误入了魔界……魔界中，有许许多多的魔物正虎视眈眈地等着两人……到底，刹那与未来会遭受到何种命运呢？

↑原本过着非凡生活的主角，这天他竞然来到魔界……

←↑人类的社会因为经济不景气及天气的异常变化而天下大乱。这是不是魔物们的坏勾当呢？

→魔界中有6个大国家。和人类所居住的世界一样，魔界的魔物们也是以同样的方式生活着。

###  + r1 +  + r1 + 

 + r1 + 的主角是个名为甲斐刹那的男孩； + r1 + 的主角则是个女孩叫要未来。此外， + r1 + 的库尔及 + r1 + 中的贝尔都各自从魔物变成主角冒险时的好伙伴。

**主人公就是这2位**

## 可与魔物激战也可与魔物变成同伴

相信各位玩家都知道在 + r1 + 系列的游戏里，不只可以和魔物战斗，也可以把魔物变成好朋友（仲魔），而且还能让 + r1 + 代替主角作战！但是，要如何将魔物变成 + r1 + 呢？那当然一定得在战争中和魔物交涉成功才行。而本作更搭载了可以将变成 + r1 + 的魔物登录其中的 + r1 + ，共有200种以上的魔物可让您收集、登录哦！

### 魔物出现！

←↑魔物有时会以集团发动攻击。如果不严格锻炼能力的话是无法得胜的。

↑虽然是魔物，却也在镇上过着和平的生活。也可以和它交谈哦！

### 单靠自己无法得胜

↑光靠人类的力量无法得胜，所以就请变成仲魔的魔物帮帮忙。

【图】战斗画面截图，显示セツナ和ミライ的状态界面，以及ヴィネコン、アイテム、システム、ステータス、マホウ、デビホン等菜单选项。魔物ケルベロス和グリフォン的能力数值。

## 魔物合体的威力提升了！

将变成 + r1 + 的魔物同伴们合体，便能制作出新的魔物……这就叫做 + r1 + 。而这个在系列中最大家熟悉的系统这次也变得更加厉害啦！只要使用通信缆线，玩家便可以将自己和朋友所拥有的魔物合体。听说有些珍贵的魔只能靠合体才可以得到，所以只要好好利用通信缆线，想收集所有的魔物不是梦想！让我们朝向完成 + r1 + 的目标迈进吧！

↑以合体来制造魔物。依玩家组合的不同，就会产生珍贵稀有的魔物。

### 要不要一起作战啊？

←为了让魔物成为仲魔可不是件简单的事情！在战斗中与魔物进行交涉，要将所有的魔物都变成仲魔的话可不是件简单的事！

包在我们身上！
