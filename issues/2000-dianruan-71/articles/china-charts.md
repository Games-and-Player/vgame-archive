###  + r1 + 顺调进行！

由于许多玩友是第一次参加 + r1 + 投票，对于一些细节不太清楚。主要表现在书写游戏名称的时候，不但没有按照本刊给出的名称，而且不注明机种。这样的选票不但无效，而且还给统计工作造成大量不便，请玩友一定要注意。举例来说，有些玩友投票时只写 + r1 + ，令我们统计时无从下手，或者写着 + r1 + ，也属于无法统计的，因为GB上的 + r1 + 有三款游戏，所以投票时一定要写清游戏的副标题。具体注意事项可以看本刊去年排行榜栏目中所刊登的那些，只有规范的投票才能起到支持自己喜欢游戏的作用，才会有更多的中奖机会。

在PS榜上，本期又有三款新游戏上榜，分别是SNK的 + r1 + 、SQUARE的 + r1 + 和ENIX的 + r1 + 。一直以来，SNK在PS上推出的对战游戏都在移植度和读盘速度上令玩友不满意，不过这次 + r1 + 的表现，也为众多PS玩友扳平了关于2D对战游戏的 + r1 + 。电影化气氛非常浓的 + r1 + 是SQUARE长期在此方面实验后心血的结晶，游戏的各个方面都可以说是非常优秀的，难怪日刊 + r1 + 给它评了满分40。ENIX的 + r1 + 本月终于上榜，以它的实力其实早就该上榜了。

本期街机榜方面，居然会有新游戏上榜，这可是很长时间不见的新气象。新登榜的游戏是目前红透半边天的跳舞游戏 + r1 + 。自从KONAMI推出这款游戏以来，可以说为盗版界开拓出无限的商机，电脑上、影碟机上（甚至还有用8位机开发的），无数的盗版DDR，简直令人目不暇接。无数男女老幼乐此不疲，本次它登上街机榜也证明了这一点。

日本方面，本月PS2无论在主机销量方面还是在软件方面都取得了压倒性的优势，销量榜上有五个PS2游戏。但令人期待的ENIX大作 + r1 + 却依然没有确定发售日，着实让人猜不透ENIX到底做何打算。如果某天突然传出 + r1 + 将在PS2上推出，我一点儿也不会吃惊的。
