## 信长的野望·烈风传

机种 PS　厂商 KOEI　类型 SLG　媒体 CD-ROM

说来惭愧，自认是SLG狂人的笔者此前竟然从未玩过堪称所有SLG标牌的 + r1 + 系列中的任何一款，不过这也为笔者提供了一个从完全客观的角度去审视这款 + r1 + 的绝好机会。

系统很陌涩。玩惯了 + r1 + 的人想必会很不习惯，但越是复杂的系统，将其吃熟于心的成就感就越强烈。这是适用于一切真正玩家的通则，但对于喜欢轻轻松松享受游戏乐趣或者性急的人来说， + r1 + 在节奏上的拖拉就显得太致命了。

极端化的例子就是攻城战。一座城门一座城门的敲，一圈一圈的国者城池绕，再加上笔者选用的是本身就不擅攻城战的武田家，以至每到攻城时就有被抛扔自己头发的冲动，甚至突然奇想，为什么日本当时的铁炮（洋枪）技术那么发达，但却没人想到使用炸药呢？窝的一声齐白哟，把那些 + r1 +  + r1 + 什么的炸上天去……

久闻大名，始闻其形的 + r1 + 大概就是为了解决这个问题吧。把想做的事留给自己做，比如野战；把讨厌做的事留给军团们，比如攻城战。这也算是个体贴的设计吧。

说到体贴，虽然光荣在系统制作上一直对玩家很 + r1 + ，但其帮助系统却是所有日本游戏公司里做得最好的（虽然有促销之嫌）。 + r1 + 更是要承袭这一传统，并冠以 + r1 + 的有趣标题，其中穿插无数搞笑成分，让历史上（也是游戏王！）心比天高、命比纸薄的今川义元和他内斗一样的儿子好好的在这场双簧戏中当了一回主角。相信也只有采取这种形式——只有真正有耐心把那元长的系统磨完一气看完。

一款真正的SLG。

## 龙骑士传说（LEGEND OF DRAGOON）

机种 PS　厂商 SCEI　类型 RPG　媒体 CD-ROM×4

虽然SCE很想借助高度的宣传攻势和足以成为话题的4枚组大容量来为自己在一个新的领域中博得一席之地，但结果却很不乐观。

很早就曾见过 + r1 + 的官传用游戏画面，这是一段精心剪辑过的长达一分钟的CG，其精美程度和气势是绝不逊于 + r1 + 的——SCE在CG制作方面一直很出色，从 + r1 + 到 + r1 + ，到 + r1 + ，均是CG制作中的杰作。但 + r1 + 和这里举出的三款游戏不同，当一个游戏在出色的CG背后没有同样出色的游戏内涵来支撑时，我想带给玩家的就只剩失望。

不知道早已叫惯的各位是否已经熟悉甚至掌握了其唯一有新意的战斗键连击系统，虽然在理论上这会使RPG的战斗变得不再枯燥！但不论这种新鲜感能否保持到第4张CD！，但就这一系统带来的负面效应也非常明显——游戏从头到尾的每一场BOSS战都异常漫长、漫长（平均在半小时以上），想靠普通的操作和战略战术来轻松战胜大规模了一句空话。游戏始终维维蒸级冤旺的操枯，等级只角色最大增量到纯斗机的 + r1 + 无法相提并论。

空有4CD的容量，但笔者感觉游戏中的世界却只有在后殊玩 + r1 + 一般的空间。

笔者认为，RPG如果没有一个能让玩家自由遨游的世界的话，那简直让人无法忍受。而把世界简缩成若干个点就更让人觉得概似了。好不容易坐船出海，可前线还是限定死的，让人觉得好像直接面对干巴巴的游戏制则。虽然从本质上来说RPG都是这样，但至少应该像 + r1 + 、 + r1 + 那样伪得聪明一点儿。

人物的模式化问题也很严重，从 + r1 + 到 + r1 + 、 + r1 + ，游戏制作者大概太注重人物的外壳变了，对人物内在的刻画却显得有气无力！除了那位亦正亦邪的紫色女龙骑士！

 + r1 + 其实是一款很中规中矩的RPG游戏，但玩家对于RPG和SCE的期待度越来越高，这就注定了 + r1 + 的不成功。

## 合金装备·幽灵通天塔（METAL GEAR·Ghost Babel）

机种 GBC　厂商 KONAMI　类型 ACT　媒体 卡带

凭借《合金装备》在PS上的人气，登陆GB也是理所当然的事情了。唯一让人期待的是KONAMI到底能否让这款作品像前作一样让玩家在玩的过程中充满紧张感，毕竟这只是8位的GB。

对于那些熟悉PS版的玩家来说，这是一款很容易上手的游戏，虽然在GB上没有多变的视角和各种华丽的特效，但令人感到惊讶的事原作的气氛完全保留了下来，玩起来的时候那种如临和紧张感、流畅的通过敌区的爽快感真的让我有触敌无不到可能的感觉，看来KONAMI还真的有一手。

在游戏中虽然要尽可能少和敌人发生战斗，但有时在未被敌人发现的情况下空手消灭对手，就有可能从敌人身上获得一些道具，并且可以通过此手段多次从同一敌人身上取得道具，这对于对关非常有帮助的。在VR训练模式中玩家不仅可以和前作一样练习在各种地形下躲避敌人的技巧，而且还可以在此模式中熟悉各种武器（如遥控飞弹）的使用方法，这在游戏后期是非常有帮助的。

唯一让人感到遗憾的是本作敌人的AI和PC上类似的潜入式作品的《盟军敢死队》相比真的有太大的差距，而这望这只是GB机能所限而造成的。总的来说这款游戏绝对值得让各位FANS买来一玩。

## 狼之标记（饿狼 MARK OF THE WOLVES）

机种 ARC　厂商 SNK　类型 FTG　媒体 MVS

原以为自从3D饿狼诞生之后就再也不能在NG的街机上见到2D饿狼的新作了，但没想到SNK仍在3D横行的今天忘记众多的2D饿狼FANS们。也许是因为SNK的几款3D作品并不是很成功的原因，也有可能SNK已明白自己的优势是什么，因而没有和其他厂商一起盲目追风。

这款作品首先值得一提的是新加入的几个系统，让人觉得耳目一新，而且让人觉得更具有挑战性，特别是 + r1 + 系统中在防住对手的攻击之后能得到体力的少许回复，这使得一味追攻的玩家在攻击对手的破绽和弱点，而不至让对手在进攻被拦的同时得不到太多好处的死守派。甚至出现了让攻击者地置的几乎完全命中的超杀连接杀技连技！如ROCK！一个小小的失误就有可能排来对手的两次闪光（超杀）。而且其真攻击和道的指令输入也大大的简化了，几乎所有人物都可以→→↓↘＋A×4来轻松发出所有超杀和潜在能力。

曾经一向让我扑头鸣疵的 + r1 + 终于在本作消声匿迹了，不过增加了20方分为上下段的闪避攻击。虽然看来很简单，但这确是一项非常实用的后发制人的防身技防。不过，要感觉到灵活运用的话还真的需花点时间多多练习。

又一款 + r1 + 游戏！单从名字上来理解： + r1 + 。好象自生化走红后，以僵尸鬼物为 + r1 + 的游戏是开始泛滥了。

不过细细体味后，感觉此GAME还是比较有特色（另类）的。故事的背景设定在魔界，游戏一开始就是一黝壁（魔导士？!）将你引入魔界，游戏气氛很 + r1 + 的，还真有那么一点 + r1 + 。主角（颇似刘×华）在充满了鬼物的世界寻找魔界中的最强兵器，你必须寻找不同的枪支和魔法道具来装备自己。其中最重要的系统是武器的升级强化，每当你斩杀一个新的魔物时就会吸入其灵魂，到一定数量的时候就可把其注入武器中（即升级）。而且，本作简单解谜要素，时段有趣的之分，有点圈头！通常以格斗游戏见长的SNK能做到这个水平实属不易。

## 天诛·忍百选

机种 PS　厂商 SME　类型 ACT　媒体 CD-ROM

PC游戏界十分流行的 + r1 + 之风也吹到TV游戏界了，继 + r1 + 之后，索尼也推出了自家品牌 + r1 + 的资料片。

游戏内容是从日本全国玩家中征集的一百个关卡，十分具有挑战性。在前作没有兴趣的人可以在本作居中过足忍者瘾了。现实生活中没人能成为忍者，可是在游戏中却能将素身体验到功夫片中飞身上房的感觉，在敌人眼底下小心翼翼的行动也是魅力之一。而忍者的绝技—— + r1 + 那种暗杀的感觉也绝非一个架字所能形容的。

比起前作 + r1 + 来说难度要高一些：没有地图，也不能遥望接近具，而且有些关卡的设计确实有些变态。忍者这个词儿语说这个不太合适。

天诛的系统类型叫 + r1 + ＋ + r1 + 的混合体，但与 + r1 + 的VR训练般的轻松多变相比， + r1 + 颇显色泽苦涩。由于忍者的特殊历史设定，使关卡的目标场景就来云就是那么几个：游戏也始终是黑夜城镇为主，玩的时间长使人心情抑压和倍感沉重！只在SFC + r1 + 的100层死者之宫殿般体会还有什么！再有，刚关开始初的读盘速度令人感叹，赶上美版游戏了；标来画面变血腥、虐杀和无聊，虽然少儿不宜一类。

值得一提的是，本作品收录了若干动画可供欣赏，将近十头就酷酷看着中其中不乏忍术心法，看后保你大开眼界。连有不少忍术公司访谈录等珍贵资料，日均是真人报道，但对于只奇五十音的0.9个日文的笔者而言
