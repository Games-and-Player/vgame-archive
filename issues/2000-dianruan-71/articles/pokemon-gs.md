## 精灵特殊进化条件一览

在《口袋妖怪·金银》中，不少精灵的进化方式与初代有所不同，出现了许多新的进化条件。这里将特殊进化条件进行整理。

### 通信交换进化

| 进化前 | 进化后 | 条件 |
|------|------|------|
| 勇基拉 | 怪力 | 通信交换 |
| 豪力 | 怪力 | 通信交换 |
| 鬼斯通 | 耿鬼 | 通信交换 |
| 大岩蛇 | 大钢蛇 | 携带金属外衣通信交换 |
| 飞天螳螂 | 巨钳螳螂 | 携带金属外衣通信交换 |
| 呆壳兽 | 河马王 | 携带王者之证通信交换 |
| 尼多兰 | 尼多后 | 月之石 |
| 波利沃格 | 牛蛙君 | 携带王者之证通信交换 |

### 亲密度进化

以下精灵需要亲密度达到一定值后升级才能进化：

| 进化前 | 进化后 | 条件 |
|------|------|------|
| 皮丘 | 皮卡丘 | 亲密度高时升级 |
| 皮宝宝 | 皮皮 | 亲密度高时升级 |
| 宝宝丁 | 胖丁 | 亲密度高时升级 |
| 波克比 | 波克基古 | 亲密度高时升级 |
| 伊布 | 太阳精灵 | 亲密度高+白天升级 |
| 伊布 | 月亮精灵 | 亲密度高+夜晚升级 |
| 小果然 | 果然翁 | LV15以上升级 |

### 提升亲密度的方法

1. 将精灵放在队伍首位，多行走
2. 给精灵喂食营养药剂（HP增强剂等）
3. 给精灵理发（在金银城的理发店，周二限定）
4. 不要让精灵濒死
5. 不要给精灵喂苦味药

### 时间段进化

金银版引入了昼夜系统，部分精灵的进化与时间段有关：

- 白天（10:00-18:00）：太阳精灵
- 夜晚（20:00-4:00）：月亮精灵
- 黑暗球、虫网球等新型精灵球也与时间相关
