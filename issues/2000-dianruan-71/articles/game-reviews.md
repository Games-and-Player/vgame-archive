## 街霸III 3RD（PDF p.23 / 刊页13）

**角色增加　系统完善**

嘉富康的大部分街机游戏已移植到DC上。由于该公司利用DC的网络机能轰轰烈烈地搞了一场 + r1 + ，本作的注目度较之以往又提高了不少（前两部网络对战是 + r1 + 和 + r1 + ）。图片是DC版，虽是黑白，但仍可看出原街机版的用色是多么丰富。

本作总计19名街头霸王，右图即是5名新角色：●REMY是一名从头型到着装皆酷似八神庵的家伙，内心只为复仇而跳动，乃蓄力系高手。●以华丽腿技见长的春丽大姐，是伴随嘉宣康格斗成长的 + r1 + ，本作她的新超杀 + r1 + 不但对空有奇效，而且无敌时间特长。●身着30年代美国芝加哥黑社会专用服装的谜样人物Q，以突进系必杀技见长，不知是机器还是人。●非常可爱的 + r1 + まこと真真（MAKOTO），战斗的目的是为了重振名门道场 + r1 + 的雄风，是个空手道专家。●白色杀戮机器人，代号12，乃前作柔软超人 + r1 + 的改良型产品。

## FIFA足球（PDF p.24 / 刊页14）

| 项目 | 信息 |
|------|------|
| 对应机种 | PS2 |
| 厂商 | EA SPORTS |
| 类型 | SPG |
| 发售日 | 3月30日 |

PS2首发软件中唯一的一款足球游戏。EA一贯以来的FIFA系列以其拥有正式授权为卖点，但在日本市场上始终无法与KONAMI的 + r1 + 系列相抗衡。这次借PS2首发的东风，EA把FIFA系列第一次搬上了次世代主机，画面精度自然是令人耳目一新。

球员的动作捕捉延续了系列一贯的高水准，过人、铲球、射门等动作都非常流畅。球场的草皮质感、灯光效果等也充分体现了PS2的图形处理能力。

然而，不得不说的是，即便画面进化了，FIFA在操控手感上与 + r1 + 系列相比仍有一定差距。AI方面也显得不够智能，队友的跑位有时令人抓狂。但作为PS2首发的足球游戏，它还是值得一玩的。

## DDR 3RD MIX（PDF p.25 / 刊页15）

| 项目 | 信息 |
|------|------|
| 对应机种 | PS |
| 厂商 | KONAMI |
| 类型 | ACT |
| 发售日 | 2000.4.27 |

KONAMI的跳舞机系列第三作来了！本作在前两作的基础上进行了大幅进化。

新增曲目方面，收录了大量来自街机版的人气曲目，总共约有100首歌曲可供选择。难度方面也有所提升，追加了 + r1 + 模式，挑战性十足。

EDIT模式是本作的一大亮点，玩家可以自己编排舞步，创造出独一无二的跳舞谱面。编辑好的谱面还可以通过记忆卡与朋友分享。

画面效果虽然在PS的机能限制下没有质的飞跃，但背景动画更加丰富多彩，节拍标记也更加清晰。

总体而言，DDR 3RD MIX是系列目前最完整、内容最丰富的一作，无论是新手还是老玩家都能从中获得乐趣。配合跳舞毯使用效果更佳！

## 瓦里奥世界3·不思议的八音盒（PDF p.26 / 刊页16）

| 项目 | 信息 |
|------|------|
| 对应机种 | GBC |
| 厂商 | 任天堂 |
| 类型 | ACT |
| 发售日 | 2000.3.21 |

任天堂的怪叔叔瓦里奥又回来了！这次他被吸入了一个神秘的八音盒世界，必须收集各种宝物才能逃出。

与前作最大的不同是，本作中瓦里奥 + r1 + 。无论是被敌人攻击还是掉入深渊，瓦里奥都不会GAME OVER，只是会被弹回或变成特殊状态。这种设计让游戏的重心完全转移到了解谜和探索上。

瓦里奥可以通过被敌人攻击而变成各种特殊形态：被火烧会变成火焰瓦里奥，被压扁会变成扁平瓦里奥等。这些特殊形态都有各自的用途，是解谜的关键。

游戏共有多个世界，每个世界都有独特的主题和谜题。音乐盒的元素贯穿始终，不同世界的BGM也各具特色。

GBC的机能被发挥得淋漓尽致，画面色彩丰富，角色动画流畅。操作简单直观，但谜题设计巧妙，需要动脑筋才能找到所有宝物。这是一款适合所有年龄段玩家的佳作。

## 塞尔达传说·姆吉拉的假面（PDF p.27 / 刊页17）

| 项目 | 信息 |
|------|------|
| 对应机种 | N64 |
| 厂商 | 任天堂 |
| 类型 | A·RPG |
| 发售日 | 2000.4.27 |

继《时之笛》之后的又一传世之作！本作以 + r1 + 为循环的独特时间系统成为系列最具实验性的一作。

游戏的故事发生在 + r1 + 之后，林克来到了一个名为 + r1 + 的异世界。邪恶的骷髅小丑偷走了面具商人的 + r1 + ，并用其力量将月亮拉向地面，3天后世界将毁灭。林克必须在这72小时（游戏内时间）内拯救这个世界。

最独特的设计是 + r1 + 系统——当72小时耗尽时，林克可以用时之笛将时间倒回第一天。但大部分道具和解谜进度会被重置，只有关键道具和已学会的歌曲会保留。这种设计既给玩家制造了紧迫感，又提供了反复尝试的机会。

面具系统是另一大亮点。林克可以佩戴各种面具获得不同能力，其中德库面具、哥隆面具、佐拉面具可以将林克变身为对应种族，获得完全不同的行动能力和攻击方式。

全游戏共有4个大迷宫，虽然数量不及 + r1 + ，但每个迷宫的设计都极为精巧。加上大量的支线任务和面具收集要素，总游戏量不输前作。

画面在N64上达到了巅峰水准，需要搭配4MB扩展包使用。音乐依旧是任天堂一流水准，尤其是月亮即将坠落时的BGM令人毛骨悚然。

## 科学小飞侠（PDF p.28 / 刊页18）

| 项目 | 信息 |
|------|------|
| 对应机种 | PS |
| 厂商 | ASCII |
| 类型 | ACT |
| 发售日 | 2000.4.27 |

《科学忍者队ガッチャマン》——这部70年代的经典动画在千禧年被搬上了PS平台！

游戏忠实再现了动画中的经典场景和角色。五位科学忍者队成员——大�的�的健、乔、甚平、纯和龙——都可操作，每人有各自的武器和特殊技能。

游戏分为若干关卡，每关都以动画中的经典战斗为蓝本。Boss战设计出彩，特别是与银河之狐（ベルクカッツェ）的对决令人热血沸腾。火鸟号的登场演出也非常帅气。

画面中规中矩，3D建模的角色在PS上略显粗糙，但动画式的演出效果弥补了这一不足。BGM方面大量使用了原作动画的经典曲目，对老粉丝来说是满满的回忆杀。

操作简单易上手，动作爽快感不错。但游戏流程偏短，熟练的话5-6小时即可通关。对于科学小飞侠的粉丝来说，这是一款不可错过的情怀之作；但对于不了解原作的玩家来说，吸引力就要打个折扣了。

## 恶魔人（PDF p.29 / 刊页19）

| 项目 | 信息 |
|------|------|
| 对应机种 | PS |
| 厂商 | BANDAI |
| 类型 | ACT |
| 发售日 | 2000.3.23 |

永井豪的经典漫画《恶魔人》终于登上了PS平台！

游戏以原作漫画的剧情为基础，讲述了不动明成为恶魔人后与恶魔族战斗的故事。主角可以在人类形态和恶魔人形态之间切换，恶魔人形态下拥有飞行和强力攻击能力，但使用过度会导致暴走。

3D战斗场景在PS上表现得中规中矩，恶魔人的变身演出有一定的视觉冲击力。敌人的设计忠实还原了永井豪笔下那些奇形怪状的恶魔们，Boss战的压迫感不错。

剧情方面，游戏挑选了原作中的经典桥段进行改编，通过过场动画来推进故事。动画质量尚可，配音阵容也颇为豪华。

但游戏的缺点也很明显：操作手感偏硬，视角问题时有发生，关卡设计缺乏变化。对于原作粉丝来说可以一玩，但作为一款ACT游戏来评价，只能说是中等偏下的水准。
