## PS2海外输出与社会问题

●最新一期的日本权威游戏情报杂志《FAMI通》及其他游戏媒体都在显著位置刊登有关PS2海外输出问题。日本通产省（相当于我国的经贸委）认定PS2使用与东芝共同开发的专用CPU，拥有与超级电脑并驾其驱的影像处理性能（超过法律所限定的数值）。而将来也预定会有游戏、音乐软件的传输服务，因此发行（传输服务）费用时所必须的密码机能也设定在记忆卡中内。通产省认为这两项性能过高而指摘其与 + r1 + ，因此将PS2列为涉及日本外汇法的 + r1 + 。据日本外汇法规定，武器和炸弹以及高性能电脑等情报相关类机器被列为 + r1 + ，要取得许可证才能输出。换言之，输出申报价格超过50000日元的话，在带至他国或输出时需有通产大臣（经贸部长）的许可才行。以一台PS2售价39800日元而言，带两台PS2就必须要得到这个许可（违者处以拘役或罚款）。游戏机被指定为 + r1 + ，这在游戏机历史上还是小姑娘上轿——头一回（笑）。

通产省的此项规定受到日本业界国有的广泛批评，许多游戏厂商提出： + r1 + ，反映了业界在激烈竞争中的无奈心情。在现在的游戏机市场上，PS2的技术处于领先地位。但预定于明年秋天发售的微软X BOX，性能将远远超过PS2。游戏业的高科技竞争势在必然，无人能够回避。为此索尼方面对通产省的规定反应强烈，表示 + r1 + ，要 + r1 + 。据日本方面报道，索尼在4月18日宣布已取得通产省准许外销PS2的输出许可证！PS2获准销往除伊朗、伊拉克、利比亚及北朝鲜以外的任何国家。因此计划中的今年秋季PS2销往欧美地区应没有问题，而亚洲地区的发售日期还未确定。

## PS不读碟维修小知识

我曾在游戏店里干过，积累了不少知识，下面讲一讲如何简单修理PS游戏机不读碟问题。

A. 由于光盘问题不能读盘的，有些是D盘不平整，形成了一个曲面，使光头不能顺利读碟，本人建议找一张安全平整的碟，把两张碟同时放入光驱内，让上面的碟把下面的碟压平，便能顺利读碟。B. 由于OPEN盖子上方的按钮松了。（按钮上方有一个小柱子，经常使用按钮便会很松）只要将按钮加厚即可。本人建议在按钮上方的柱子上加一个垫子，加长柱子的长度便可。C. 由于光头移位，PS在读盘时，有时会发生光头移位，当受到震动、等待后，机子也不读碟，建议将机器放在安全、平稳的地方，用手轻触光头旁边黑色的部分，不可手触光头，用棉布把周围擦一擦，吹一吹，如果不读碟，请将机器送往 + r1 + 。D. 假机，一些返修机大量的存在于市场当中，假机光头为淡蓝色，或蓝中有白，即为返修机。在返修机中里面有很多灰尘，物镜也不怎么干净，不过是换了一个美丽的外壳。这种机子一是光头读盘不太稳定，二是不到一年就会成废品，也许会被电击穿基板，我一朋友上星光头剩一半还能读 + r1 + ，还是原装的好，以上为本人的见解。

兰州　雪慧军

这样言简意赅，有实质内容的稿件是杂志最欢迎的，我们想也是读者最受益的。欢迎雪慧军和 + r1 + ！
