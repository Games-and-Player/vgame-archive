# 老杂志数字化约定

面向《电子游戏软件》《Game 集中营》等 90 年代中文游戏杂志的本地归档 + 建站项目。

## 目录结构

```
ocr-workspace/
├── CONVENTIONS.md            # 本文件
├── issues/
│   └── <YYYY-vol/期号>/       # 如 1994-vol1
│       ├── README.md         # 本期元信息 + 目录索引
│       ├── pages/            # 按物理页码归档的 md（扉页、目录、短栏目）
│       │   └── p003.md
│       ├── articles/         # 长文章按文章归档（跨多页的攻略/评测）
│       │   └── dq5-walkthrough.md
│       └── assets/           # 页面切图、截图等（如需要）
└── index/
    ├── games.md              # 游戏标题反查：游戏 → 出现在哪几期哪几页
    └── terms.md              # 民间译名对照表
```

**原则**：短栏目按页归档，长文章按文章归档，二者在 `README.md` 交叉引用。

## 文件命名

- `pages/pNNN.md`：`NNN` 为 3 位数物理页码（含彩页，记为 `c01`/`c02`）
- `articles/<slug>.md`：用游戏 ASCII slug，如 `dq5-walkthrough.md`、`sf-pc-edition.md`

## 转录策略（按内容类型区分）

| 类型 | 处理方式 |
|---|---|
| 卷首语、编辑手记、读者来信 | 可逐字转录（短、独立价值高）|
| 目录 | 结构化 Markdown 表格 |
| 游戏攻略正文 | **摘要 + 结构化提取**（剧情梗概、关键节点、必杀技表、地图要点），**不做整段复刻** |
| 操作表 / 必杀技指令 / 数据表 | Markdown 表格 |
| 图注、截图说明 | 摘要 |
| 广告、花絮 | 提及存在即可，不转录 |

说明：这是版权材料，即便个人归档也优先"可检索的结构化数据"而非"完整复刻"，这样对建站检索也更有用。

## 游戏标题处理

- **首次出现**记录全部形式：民间译名、原名、平台、年份
- **后续**统一用民间译名（保留时代感），原名放脚注
- 译名对照集中在 `index/terms.md`

示例：
```markdown
《勇者斗恶龙 V》[^dq5]

[^dq5]: 原名《ドラゴンクエストV 天空の花嫁》/ Dragon Quest V，SFC，1992 年 Enix 发行
```

## 栏目标签（本刊）

- `天堂任鸟飞`：任天堂系（SFC/FC/GB）
- `世嘉世家`：世嘉系（MD/GG）
- `PC 游戏室`：PC 游戏
- `卷首语` / `编辑手记` / `目录`

## 校对工作流

1. 我（Claude）产出 md 初稿 → 存到对应文件
